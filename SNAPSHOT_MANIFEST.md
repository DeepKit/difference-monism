## 1. 快照角色

本快照用于公开展示当前理论母库、导航入口与必要接口说明。

它不是：

- 所有文件同等权威的材料堆
- 执行层与经营层全文开放包
- 已经完成最终同行评审与长期实证的终版封口稿

当前最稳理解方式是：

- 公开法源链：`DM -> ASTO -> ECET -> TAT -> ODD`
- 扩展执行链：`ASTO -> LMM -> COP -> TAT -> ODD -> Harness`

补充说明：

- `ASTO` 当前不仅是桥接解释层，也负责向下游各层提供统一结构编码、前置字段与节律判断接口。
- `WSM` 当前收纳在 `LMM` 之下，属于其内部二级理论，只负责前线识局与弱测量，不单列为新的顶层法源。
