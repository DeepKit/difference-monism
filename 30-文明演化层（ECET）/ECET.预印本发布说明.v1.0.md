## 3. 推荐摘要

中文摘要建议：

> 演化约束存在论（Evolutionary Constraint Existential Theory, ECET）是建立在差异一元论（DM）与属集变迁存在论（ASTO）之上的文明演化与治理判断框架。它以能量约束、适应选择约束与不完备约束为最小工作公理，讨论影子主权、责任对称、反终极化、AI 归属与元治理授权等问题。当前公开版本以 A/B/C 主文档组的形式发布，适合预印本占位、公开征评与公开比较，但不应被表述为已经完成大规模经验验证的闭合终稿。其仿真附录仅提供概念验证级支持，不构成经验完成声明。

英文摘要建议：

> Evolutionary Constraint Existential Theory (ECET) is a civilizational evolution and governance framework built on Difference Monism (DM) and Attribute-Set Transition Ontology (ASTO). Using the energy constraint, adaptive selection constraint, and incompleteness constraint as its minimal working axioms, ECET addresses shadow sovereignty, responsibility symmetry, anti-finalization, AI attribution, and meta-governance authorization. The current public release is presented as an A/B/C document set suitable for preprint placeholder use, public review, and comparative discussion, but it should not be described as a fully closed theory with large-scale empirical validation. Its simulation appendix provides only proof-of-concept level support rather than completed empirical confirmation.

### 3.1 统一发布模板

- `适用范围`：文明演化、治理约束、元治理、影子主权与跨域问责问题
- `不是什么`：第二终极本体论、成熟文明定律、现成制度文本
- `边界`：哲学地基回引 DM；应用桥接回引 ASTO；责任主体闭合回引 TAT；工程执行回引 ODD
- `与上游关系`：ECET 是 `DM -> ASTO -> ECET -> TAT -> ODD` 的文明演化与治理判断层
