# ECET.P99.5 比较历史社会学骨架与跨文明对照模板

> **公开口径**：`工具层文档 / 公开征评版`

---

## 一、用途与边界

本文不是新的文明总论，

而是给 `ECET` 提供一个最小比较骨架，用来回答三类问题：

1. 不同文明或治理单元，能否用同一套变量被描述？
2. 某个历史个案进入 `ECET` 时，最低要交代哪些结构信息？
3. 什么样的对照，才算可比较；什么样的对照，只能算类比？

本文不负责：

- 代替主链正文
- 直接给出价值裁决
- 用一个模板消灭所有历史特殊性

---

## 二、最小案例卡片

每个准备进入 `ECET` 的历史或文明个案，最低应交代以下字段。

| 字段 | 说明 | 最低写法 |
|------|------|---------|
| `case_id` | 个案唯一标识 | 如 `ROME_LATE_EMPIRE_01` |
| `时间窗口` | 分析所指向的时段 | 必须说明起止范围 |
| `空间单位` | 文明 / 帝国 / 平台 / 组织 / 制度区段 | 不可混写 |
| `问题类型` | 资源约束 / 合法性失稳 / 责任逃逸 / 治理断链等 | 先写问题，不先下结论 |
| `比较单位` | 本次比较到底比什么 | 制度、责任链、治理接口、扩张压力等 |
| `能量约束画像` | 资源、财政、组织承载、维护成本 | 写清主要瓶颈 |
| `适应选择约束画像` | 外部竞争、战争、市场、技术、人口压力 | 写清筛选压力 |
| `不完备约束画像` | 认知盲区、信息断裂、制度视野缺口 | 写清看不见什么 |
| `责任拓扑` | 决策权、执行权、后果承担如何分布 | 说明是否发生断链 |
| `影子主权形态` | 谁在决策，谁不承担后果 | 若不存在，也应明确写无 |
| `修复机制` | 该系统如何试图重建闭环 | 税制、法制、仪式、组织改革等 |
| `失稳信号` | 何种现象表明结构失稳 | 漂移、逃责、成本爆炸、合法性断裂等 |
| `证据级别` | `E0-E4` 中哪一级 | 参照 `P99.2` |
| `替代解释` | 至少一条 | 防止自我封闭 |
| `可反驳点` | 至少一条 | 说明何种材料会逼迫改写 |

## 四、最小比较包纪律

一份可公开讨论的 `ECET` 比较稿，最低建议包含四件东西：

1. `同构比较`
   - 至少一个与你的主张方向一致的可比个案
2. `异构比较`
   - 至少一个结构不同但可形成对照的个案
3. `负例或边界例`
   - 至少一个不容易被 `ECET` 顺手解释掉的个案
4. `替代解释`
   - 至少列出一条非 `ECET` 的强竞争解释

如果四者缺一，

则当前稿更像“理论举例”，还不是“可比较分析”。

---

## 五、输出顺序建议

任何文明比较稿，建议按以下顺序写：

1. `先说明比较单位`
2. `再写三约束画像`
3. `再写责任拓扑与影子主权`
4. `再写修复与失稳机制`
5. `最后才写规范判断`

禁止顺序：

1. 先下价值裁决
2. 再倒找结构词汇去证明它

## 七、与现有主链的关系

- `A00 / A01` 负责说明 `ECET` 的层级边界与阅读路径。
- `B03` 提供三约束的最小工作公理。
- `B07` 提供闭环、漂移与失效模式的动力学框架。
- `C13` 提供影子主权、责任拓扑与治理原则。
- `B09` 提供假设接口与证据级别。
- 本文只负责把这些内容压成历史比较时可重复使用的工作壳体。
