# ECET数学符号规范

> 本文档定义ECET理论体系的标准数学符号系统，确保形式化表达的一致性。

---

## 一、符号设计原则

1. **一致性原则**：同一概念在所有文档中使用相同符号
2. **直观性原则**：符号应直观反映概念
3. **可扩展性原则**：使用下标/上标区分变体

---

## 二、状态空间符号

| 符号 | 含义 | 定义 |
|------|------|------|
| S | 真实状态空间 | S ⊂ ℝⁿ |
| Ŝ | 认知状态空间 | Ŝ ⊂ ℝᵐ, m < n |
| Sₜ | t时刻真实状态 | Sₜ ∈ S |
| Ŝₜ | t时刻认知状态 | Ŝₜ ∈ Ŝ |

---

## 三、偏差符号系统

**基本定义**：B = Ŝ - S（偏差 = 认知 - 真实）

| 符号 | 含义 | 公式 |
|------|------|------|
| B | 偏差向量 | B = Ŝ - S |
| Bₜ | t时刻偏差 | Bₜ = Ŝₜ - Sₜ |
| ‖B‖ | 偏差范数 | ‖B‖ = √(∑Bᵢ²) |
| B_cog | 认知偏差 | B_cog = Ŝ_mental - S_real |
| B_act | 行动偏差 | B_act = A_exec - A_intend |
| B_sys | 系统偏差 | B_sys = S_emergent - ∑S_parts |

---

## 四、约束符号系统

**标准表示**：C = {C_E, C_A, C_I}

| 符号 | 含义 | 数学表达 |
|------|------|---------|
| C | 约束集合 | C = {C_E, C_A, C_I} |
| C_E | 有限能量约束 | ‖Ŝ‖ ≤ E_max |
| C_A | 适应选择约束 | A(Ŝ, S) ≥ A_min |
| C_I | 不完备约束 | dim(Ŝ) < dim(S) |

---

## 五、演化与动态符号

| 符号 | 含义 | 说明 |
|------|------|------|
| t | 时间变量 | 连续或离散时间 |
| Δt | 时间步长 | Δt = t_{i+1} - t_i |
| T | 时间范围 | t ∈ [0, T] |
| d/dt | 时间导数 | 连续时间变化率 |
| Δ | 差分 | ΔSₜ = S_{t+1} - Sₜ |

## 八、标准公式模板

### 8.1 偏差生成模型

**标准形式**：
Ŝ_{t+1} = f(Ŝₜ, Sₜ, Cₜ) + εₜ

**随机微分形式**：
dŜₜ = μ(Ŝₜ, Sₜ, C)dt + σ(Ŝₜ, C)dWₜ

### 8.2 闭环适应模型

**状态更新**：
S_{t+1} = Sₜ + α · F(Sₜ, Oₜ)

**策略更新**（梯度上升）：
π_{t+1} = πₜ + η · ∇_π J(πₜ)

### 8.3 多目标优化

**目标函数**：
max_S U(S) = ∑ᵢ wᵢ Uᵢ(S)

**约束条件**：
s.t. S ∈ 𝒮_valid = {S | gⱼ(S) ≤ 0, j=1...m}

## 九、排版规范

### 9.1 公式排版

- 行内公式：使用$包围，如$B = Ŝ - S$
- 独立公式：使用$$包围并编号
- 多行公式：使用aligned环境对齐

### 9.2 符号大小写

- 大写：集合/空间（S, Ŝ, C）、矩阵（A, B, M）
- 小写：标量（x, y, α）、向量元素（sᵢ, bⱼ）

### 9.3 字体规范

- 变量：斜体（x）
- 函数：正体（exp）
- 集合：黑板粗体（ℝ）
- 分布：花体（𝒩）
