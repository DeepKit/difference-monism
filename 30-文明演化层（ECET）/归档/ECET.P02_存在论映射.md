# ECET.P02 存在论映射

## 1. 存在论核心概念

### 1.1 实体 vs 属性集合

传统存在论将"存在"理解为**实体（Entity）**的集合。ECET 提出扩展视角：

> **存在 = 属性集合及其关系网络**

这一观点强调：
- 存在不仅是独立实体，更是实体间的关系
- 属性是动态变化的，而非静态本质
- 关系网络塑造存在的状态

### 1.2 本体论层级

ECET 采用层级化的存在论框架：

```
系统网络层级
    ↑
系统层级
    ↑
集合层级
    ↑
个体层级
```

**各层级特征**：

| 层级 | 存在特征 | ECET 关注点 |
|-----|---------|-----------|
| 个体 | 独立属性、边界明确 | 认知偏差来源 |
| 集合 | 群体属性、统计规律 | 偏差分布与演化 |
| 系统 | 涌现属性、自组织 | 适应性选择机制 |
| 系统网络 | 多系统交互、复杂适应 | 全局约束与局部自由的平衡 |

### 1.3 介入性三元关系

ECET 引入**介入性三元关系**作为实践框架：

```
    需求（主体）
         \
          → 契约（属性集合）
         /
    产出物（功能）
```

**三元关系解释**：
- **需求**：主体的目标或问题
- **契约**：连接需求与产出物的属性集合（规则、资源、约束）
- **产出物**：实现的功能或结果

这一结构说明：系统运作和认知被约束，但通过介入和优化可以部分改变状态。

## 2. 演化约束的存在论映射

### 2.1 能量约束映射

| 存在论层面 | 映射内容 | 实例 |
|-----------|---------|------|
| 物理层面 | 资源有限性 | 计算资源、时间、人力 |
| 认知层面 | 处理能力上限 | 工作记忆容量、注意力限制 |
| 社会层面 | 制度成本 | 决策程序、组织成本 |

**存在论意义**：能量约束不是外部限制，而是存在本身的结构性特征。没有无限资源的存在形式，只能在有限性中寻求最优配置。

### 2.2 适应选择约束映射

| 存在论层面 | 映射内容 | 实例 |
|-----------|---------|------|
| 生物层面 | 自然选择 | 物种适应环境压力 |
| 认知层面 | 信念选择 | 保留有效预测模型 |
| 社会层面 | 制度演化 | 有效制度被保留传播 |

**存在论意义**：适应选择不是外部力量施加于存在，而是存在维持自身的内在机制。不适应的存在形式自我消解。

### 2.3 不完备约束映射

| 存在论层面 | 映射内容 | 实例 |
|-----------|---------|------|
| 认识论层面 | 认知局限 | 模型不完备、信息不完全 |
| 本体论层面 | 存在开放性 | 属性未完全实现、关系未完全确定 |
| 实践层面 | 行动不确定性 | 结果不可完全预测 |

**存在论意义**：不完备不是缺陷，而是存在的动态性和开放性特征。完备的存在将是静止的、死亡的。

## 3. 系统与存在的映射

### 3.1 系统作为存在的具体化

在 ECET 框架中，"系统"是"存在"的具体表现形式：

**系统 = 边界 + 内部结构 + 环境交互**

这与存在论的映射关系：

| 存在论概念 | 系统对应 |
|---------|---------|
| 实体 | 系统组件 |
| 属性 | 组件特征与状态 |
| 关系 | 组件间交互规则 |
| 环境 | 系统边界外的存在 |

### 3.2 认知系统的存在论分析

认知系统作为 ECET 的核心研究对象：

**认知系统的三层结构**：

```
元认知层（监控与调节）
    ↑
认知层（信息处理与偏差生成）
    ↑
感知层（环境信息采集）
```

**各层的存在论地位**：

1. **感知层**：存在与世界的直接接触点，受物理约束限制
2. **认知层**：存在的表征层面，受不完备约束限制
3. **元认知层**：存在的自我反思层面，试图突破但仍在约束内

### 3.3 行动系统的存在论分析

行动系统是存在的实践维度：

**行动系统的三层结构**：

```
策略层（目标设定与路径规划）
    ↑
决策层（选择与执行）
    ↑
反馈层（结果评估与调整）
```

**行动的存在论意义**：
- 行动是存在改变自身与世界的途径
- 行动受能量和适应选择约束
- 行动结果反馈重塑存在状态

## 4. 存在论映射的实践意义

### 4.1 设计原则

基于存在论映射，ECET 提出系统设计原则：

1. **尊重约束原则**：设计必须承认并接受演化约束，而非试图消除
2. **利用偏差原则**：将认知偏差视为探索资源，而非必须纠正的错误
3. **动态平衡原则**：在局部适应与全局稳定之间寻求动态平衡
4. **开放性原则**：保持系统的开放性，避免过早封闭

### 4.2 映射关系的动态性

存在论映射不是静态的，而是随系统演化而变化：

```
环境变化 → 约束条件变化 → 存在状态变化 → 映射关系重构
```

这意味着：
- 没有一劳永逸的映射
- 映射本身需要持续更新
- 映射更新是适应性的一部分
