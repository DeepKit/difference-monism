# ECET 演化约束存在论
## Evolutionary Constraint Existential Theory

> **当前公开口径**：`v1.0 公开征评包 / 预印本支撑文档组`
>
> **定位**：ECET 是建立在 `DM -> ASTO` 之上的文明演化与治理判断层。
> 它不是与 DM、ASTO 平行竞争的第二终极本体论，而是把结构语法继续推进到约束、公理、问责、影子主权与元治理问题的跨域框架。
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）

---

## 当前最稳定位

ECET 当前最适合被公开描述为：

> **一个面向文明演化、治理结构与问责问题的跨域元框架，其主文档组已经适合公开征评与预印本式发布，但仍不是完成大规模经验验证的闭合终稿。**

这意味着：

- 可以公开发布、公开比较、公开征评
- 可以作为预印本平台的占位与引用对象
- 不应被包装为已经完成同行评审或经验验证的终局体系

## 统一发布模板

### 适用范围

- 用于处理文明演化、治理约束、元治理、影子主权与跨域问责问题
- 用于把 `DM -> ASTO` 的结构语法继续推进到文明层的约束、公理、假设和治理判断
- 用于公开征评、预印本支撑文档组和跨领域比较

### 不是什么

- 不是与 DM、ASTO 平行竞争的第二终极本体论
- 不是已经完成大规模经验验证的文明定律
- 不是可直接替代现实政治、法律和治理程序的现成制度文本

### 边界

- 哲学地基问题优先回引 DM，应用桥接语法优先回引 ASTO
- ECET 负责文明层约束与治理判断，不直接承担责任主体闭合和工程执行闭环
- `C17` 等仿真材料只应按概念验证级支撑件理解，不应抬升为充分实证

### 与上游关系

- ECET 建立在 `DM -> ASTO` 之上，是五层公开主链中的第三层
- TAT 继续把 ECET 的文明治理判断压成责任门槛与责任架构
- ODD 若涉及治理与责任边界，应先回引 ECET 与 TAT

## 公开纪律

- 不把 ECET 说成已经完成同行评审的成熟终稿。
