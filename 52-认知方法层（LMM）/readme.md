## 发布纪律

- **public/** 中的内容可按 A/B 类规则公开和推广
- **private/** 中的内容属于 C 类（收入保留），默认不对外全文公开
- 对外展示 private/ 内容时，只展示原则、结构图、脱敏样例
- 不把完整模板、参数、阈值、报价逻辑、客户案例放入 public/
