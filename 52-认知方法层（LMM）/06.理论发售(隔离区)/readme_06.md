# 06.理论发售(隔离区) - 守门人警语

> **状态**：依据“买我们理论的要移出去”的最高指示设立
> **定位**：这不是 LMM 的核心作战库，这是为内部保留的“元知识/防化隔离区”。

### 为什么设这个区？
我们的现金流主轴是 **卖铁器（为企业提供诊断、止损干预、重组架构等实质服务）**。
然而，在理论发展早期，难免会产生类似“怎么教别人这套营销课”、“怎么发售发刊词”、“让多少位门徒买单”等带有浓烈“教人打铁”情绪的宣发废稿。

这种带有**培训式发售、大师式收税**的话语体系，若掺和进前面的 `01-05库`，会造成严重的认知污染，大幅拉低整个干预行动组的冷酷感与威慑力。

### 此库的物理动作
* **隔离**：任何关于“如何把 LMM 当作一种私董会或培训课卖出”的经验与操作手则，全部打入冷宫在此隔离。
* **戒严**：前线作战连队只能调用 `03`、`04`、`05`进行物理实战。本库内容不得直接当做外部获客或港口承接话术。
* **存档**：本次行动中，在销毁旧档时，诸如“零基础怎样靠卖 LMM 核心变现”、“一个老头顶一万路人”等极度功利性的残骸思想已被直接焚烧，未予保留，以此确保总库纯洁。

### 操作规则（圆桌补充，2026-03-26）

**R1 — 主动归筐规则**：
在 01–05 库工作过程中，一旦发现任何"以教别人掌握 LMM 理论为目的的发售内容"（如：LMM 课程提纲、大师式收费手册、让门徒买单的话术包），**必须在发现当日移入本库**，并在文件头注明：来源库、发现时间、归筐原因。

**R2 — 出库解禁规则**：
本库内容并非永久封存。若某份文件经过以下改造，可提请主理人审查并批准迁回对应功能库：
- ① 去除"教人打铁赚钱"的课程化/收税式语气
- ② 改造为"实战干预服务的入口工具或承接文件"
- ③ 通过 01库责任闭合七项边界检验

**R3 — 永久封存红线**：
以下类型内容永久不得出库，不参与 R2 流程：
- 任何夸大成功率的培训式承诺文案
- 任何以"复制 LMM 体系"为卖点的授权加盟设计
- 任何绕过 R=1 责任约束的"轻松躺赚"表述
