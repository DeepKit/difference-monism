# LMM.变现.009 — 产品定价逻辑与报价触发条件

> **版本**：v1.1 | 日期：2026-03-27
> **属性**：05.理论变现 — 报价决策框架，内部使用，不对外发布全文

---

## 一、定价原则（两条铁律）

**铁律 A：结果费，不是时间费。**
我们不按"小时数 × 日费率"报价。我们按"对什么问题负责，责任范围有多大"定价。时间是结果的容器，不是价格的依据。

**铁律 B：R=1 对应风险溢价。**
我们的价格里包含了退款保证。因此我们的定价应比"只出报告不负责"的传统咨询**高 30%-50%**，这个溢价不是傲慢，是责任成本的显性化。

## 二、三条产品线定价框架

### LMM.产品.005 — CEO品牌责任诊断探针包（通用高客单）

| 变量 | 内容 |
|:---|:---|
| **定价区间** | ¥8,000 – ¥30,000（按干预范围和企业规模浮动） |
| **报价触发条件** | BOS ≥ 45 + 决策人直接参与 + 可交付成果可定义 |
| **一次性/持续** | 以项目制计，首期不超过 3 个月 |
| **退款触发条件** | 经 R-Eval 验证，约定样本中的关键判断误差超过约定阈值，或约定交付物无法完成 |

### LMM.产品.007 — 责任架构首诊包（TAT老板线主产品）

| 变量 | 内容 |
|:---|:---|
| **定价区间** | ¥2,000 – ¥6,000（视组织规模和准备材料的完整度） |
| **报价触发条件** | 决策人参与确认 + 对方已描述至少一个明确损耗信号 |
| **一次性/持续** | 单次首诊（60-90分钟），首诊后可选进入持续干预 |
| **退款触发条件** | 经 R-Eval 验证，约定样本中的关键判断误差超过约定阈值，则退还本次费用 |
| **升降价原则** | 超过3人参与的首诊，+20%；远程 vs 现场，等价不差价 |

### LMM.产品.009 — 结构错位自检包（思维越狱知乎线入口）

| 变量 | 内容 |
|:---|:---|
| **定价区间** | 免费（前台自测版）/ ¥0-99（轻量数字版） |
| **报价触发条件** | 自动交付，不进入港口报价对话 |
| **升级路径** | 完成自测并出现物理接近信号后，可升级至产品.007 港口承接，差价结算 |

## 三、阶梯报价触发条件

**报高价的四个条件（全部满足时可报上限）**：
1. 决策人（CEO/董事长级别）亲自参与
2. 问题场 BOS ≥ 45
3. 对方已明确表达"想解决"的时间窗口
4. 我方当前有足够的承接能力

**应报低价或试点价的条件（满足任意一条）**：
1. 当前 P1 阶段，需要积累真实案例
2. 对方是早期关系户，价值在于背书而非现金
3. 问题场复杂程度低于预期，范围需收窄

---

## 四、禁止定价行为

- ❌ 按"给你打个折"方式降价（破坏责任定价逻辑）
- ❌ 先报价再谈内容（应先确定范围再报价）
- ❌ 把前台自测层写成港口报价层（位阶错乱）
- ❌ 未签 R-Min Spec 条款就收款（无责任边界的收款）
- ❌ 超出当前承接能力接单（有钱的陷阱）

---

## 五、报价单格式规范

每次报价必须以书面形式包含：

```
服务名称：[产品名称]
诊断范围：[具体描述]
交付内容：[具体列举，不写"综合建议"]
时间安排：[日期区间]
报价金额：[数字]
退款条件：[具体触发条款]
有效期：本报价有效期 [X] 天
```

不符合上述格式的口头报价，视为非正式沟通，不构成成交。

## 六、责任深度定价调节 (R-Depth Adjustment)

报价应根据我方承担责任的“深度”进行动态调节，而非固定死值。

| 责任级别 | 定义 | 价格调节 | 备注 |
|:---|:---|:---|:---|
| **标准 R=1** | 经 R-Eval 验证未达约定阈值则退款 | **基准价** | 默认选项，不作为卖点，是纪律。 |
| **深度 R=1** | 诊断 + 实施方案止损承诺 | **基准价 +30%-100%** | 当我们需要对后续实施环节作具体担保时。**必须强制挂载【索赔封顶机制 (Cap on Liability)】：灯塔的赔付上限永远以本次干预费用的 100% 为绝对物理上限，绝不承担客户内部后续连锁崩塌的无限连带责任。** |
| **有限责任 (R<1)** | 仅提供信息参考，无退款项 | **基准价 -30%** | **严控使用**。仅限场域数据极度不透明、我方无法验证逻辑的边缘场景。 |
| **排除责任 (R=0)** | 纯知识分享/讲座 | **固定小时费** | 这种形态不涉及 LMM 核心诊断，应归入“市场教育”。 |

### 报价策略提示：
- **拒绝 R=1 的风险补偿**：如果客户因为“不想谈责任”而拒绝签署附件，不要简单降价，而应将其视为“高噪音客体”，报价应维持原价，但合同内必须转为“咨询服务”且注明无退款保障。
- **高 BOS 值调节**：BOS 分数每增加 5 分（45以上），可建议报价上浮 10%。理由不是“对方越痛越该多收”，而是此时场域的边界锁定成本、更高的误判风险、更多的协同与复核资源投入，都会同步抬升我方的承接复杂度与责任成本。
