## 私有承接路径图

### 主战线（TAT老板线）路径
```
[03库灯塔内容/触点]
  → 问题显影与弱测量信号确认
    → LMM.产品.007 首诊包（港口接引与正式承接）
      → LMM.产品.011 责任滑逸CT首诊模板（单病种首诊）
        → LMM.产品.013 责任接驳桩（有界处方前门）
          → 确认单 / R=1 条款 / 必要时 R-Eval
            → 交付 → 进入 05库私有回流层
```

### 思维越狱前台线路径
```
[微信/知乎触点] → LMM.产品.009 结构错位自检包
  → 自检完成 → 按结果分流：
    ├─→ 继续阅读 → 导读包/全书
    ├─→ 需要深度 → LMM.产品.007 升级
    └─→ AI场景 → Agentic子场入口
```

### Agentic-AI 垂直子场路径
```
[03库/Agentic触点] → AI.003五分钟认知相变自测 → AI.007自测归档入口
  → AI.009/011首诊前确认 → AI.013/015材料与模板
    → AI.017/019提交与页面 → AI.021/025/027阶段A受理与放行
      → AI.031港口承接与R=1合约 → AI.005正式首诊包 → 交付闭环
```

## 快速判定：用哪个承接件？

| 对象类型 | 第一承接件 | 后续路径 |
|:---|:---|:---|
| CEO/董事长（治理断链） | `LMM.产品.005` 或 `LMM.产品.007` | → `LMM.产品.011` → `LMM.产品.013` |
| 知乎职场读者 | `LMM.产品.009` 自检包 | → `LMM.产品.007` 首诊 |
| AI工作流需求方 | `LMM.产品-AI.007` 自测入口 | → `LMM.产品-AI.031` 港口承接 |
| 不确定类型 | 请先用 `LMM.定位.007` BOS评分 | → 再定产品 |
