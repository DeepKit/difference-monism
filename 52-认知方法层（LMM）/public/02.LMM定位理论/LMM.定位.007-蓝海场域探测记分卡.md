## 一、蓝海的操作定义

灯塔营销的蓝海不是物理空间，而是：

> **"认知结构处于高压损耗、且责任供应缺失的真空场域。"**

我们不找"有钱人"，我们找的是"**正在由于结构性错误而持续支付隐性损耗**"的那个坐标点。

### 一点五、趋势报告的位阶说明

从本轮起，

`刘润 / 吴晓波 / 香帅团队` 一类年度趋势材料，

在 `BOS` 中只承担一个作用：

> **帮助缩小“先去哪一片海域找流血场”的范围。**

它们不能直接替代：

- `R` 认知刚性
- `L` 损耗强度
- `V` 责任真空
- `T` 相变触发性

也就是说：

1. 趋势热，不等于 `BOS` 高分。
2. 某行业利润高，不等于新品牌就适合先打。
3. 年度报告只能告诉你“风往哪吹”，不能替你证明“这个老板现在就在流血”。

### 一点六、当前优先产业带补注

基于 `2025-2026` 这一轮趋势材料与官方行业数据，

当前最值得优先探测的，不是某一个明星企业，

而是：

> **产业智能化升级中的连接层产业带**

优先顺序建议：

1. `机器人 / 工业自动化系统集成`

3. `工业软件 / 企业AI集成 / 系统实施 / 信息技术服务`
4. `高端医疗器械制造`（高价值后备）

当前不建议优先首发：

- `航空航天 / 军工链`
- `纯半导体大厂 / 超大型IC龙头`

而是：

- 进入慢

- 新品牌信任门槛高

## 二、BOS 记分卡公式

$$Blue\ Ocean\ Score = (R + L + V) \times T$$

### 使用红线

在进入评分前，

必须先承认 BOS 的位阶边界：

- BOS 是**入场筛查工具**，不是市场规模预测器。
- BOS 是**方向性判断框架**，不是精确科学仪器。
- BOS 高分只代表“值得进一步建塔”，不代表“必然进入正式承接”或“必然可做大”。

### 【算法底层声明：为何 T 是全局绝对乘数？】

在 BOS 算法设计中，$T$（相变触发性/扰动精度）不是一个权重加数，而是**一票否决式的乘数**。这是一个刻意为之的伦理保险丝：
当 $T=1$ 或 $T=0$ 时（意味着：我方极度缺乏能刺穿对方防御的物理算据，或只要一开口就会引发对方极大的狂躁反扑与认知闭锁），无论这个场域的内部流血（L）有多恐怖、责任真空（V）有多大，**其探测结果最高只等于 15 分，被物理算法直接锁死在“禁止入场区”**。
这是一种克制且必要的原则：**如果你的手电光不能让别人感觉安全，只会制造更强的防御反弹，那就不允许开灯。** 缺乏触发精度的强行干预，不叫“灯塔救援”，而是失当的侵入式干预。$T$ 作为绝对乘数，是保护 LMM 不向粗暴干预滑落的最强物理城墙。

| 变量 | 名称 | 含义 | 评分区间 |
|:---:|:---|:---|:---:|
| **R** | 认知刚性 (Rigidity) | 受众的路径依赖有多强？改变的阻力有多大？ | 1–5 |
| **L** | 损耗强度 (Loss) | 受众每天在"结构黑洞"里流多少血？ | 1–5 |
| **V** | 责任真空 (Vacuum) | 现有供应商是否只管卖、不管交付结果？ | 1–5 |
| **T** | 相变触发性 (Trigger) | 扰动精度：能否在短时间内建立对“认知别扭感”的物理可验证性？（即用物理算据迫使对方自己推断出损耗，而非利用销售恐吓制造单向的外部压迫） | 1–5 |

### 最低证据要求（建议写入评分备注）

| 维度 | 最低证据例 | 若证据缺失的处理建议 |
|:---:|:---|:---|
| **R** | 行业内反复出现“只能这样”“一直如此”的真实表述；第三方材料记录其强路径依赖 | 暂不打高分；无外部支撑时建议自动降一级 |
| **L** | 可量化的时间、返工、停机、资金、协同损耗数据 | 无法量化时，不宜给 4-5 分 |
| **V** | 现有服务商只卖工具/建议、不写结果性补偿条款的证据 | 若已有强责任承接者，应下调至低分 |
| **T** | 至少两次真实对象测试中，能在短时间内引发自我识别或物理让渡动作 | 仅靠主观想象时，不宜给高分 |

### 评分程序（建议标准动作）

1. 先逐项写证据，再打分，不允许先给结论后补理由。
2. 对任一维度，若只有内部推断、没有外部或真实场景支撑，建议该维度**自动降一级**。
3. 有条件时，采用双评估者独立打分；若同一维度分差 `>= 2`，必须复核原因。
4. 当 `T <= 2` 时，即使总分勉强好看，也原则上不进入建塔阶段，应先补触发材料与首切样本。

**判定标准**：

| 得分 | 判定 | 行动 |
|:---:|:---:|:---|
| ≥ 45 | 🔴 核心蓝海 | 可立项建塔，但仍须通过 `R-Min` 与承接上限检查 |
| 30–44 | 🟡 候补场域 | 继续探测，优先补证据与提升 `T` |
| < 30 | ⚫ 禁止入场 | 场域噪音过高，或损耗不可见，跳过 |

## 三、禁入场域的典型特征

以下场域即使客单价高，也严禁强行入场：

1. **受众损耗不自知** — 他们以为自己运营得很好，没有"别扭感"流变信号
2. **无责任真空** — 现有服务商已经敢对结果负责（极少见，但一旦出现即跳过）
3. **T 分 < 1** — 在出具物理算据（显隐损耗财务评估）面前，对方完全不产生认知共鸣或拒绝面对现实，结构无法被撬动。
4. **承接能力不足** — 我们目前无法配置足够资源承接该场域的规模

## 四、与 LMM.定位.003（场-路-人）的接口

BOS 记分卡是入场前的**"寻域"**工具，回答 $F_{ield}$ 的问题。
一旦 BOS ≥ 45 确认入场，则启动 `LMM.定位.003` 的场-路-人动力学，进入**"确路"**和**"定人"**阶段。

```
BOS 记分（寻域） → 场路人三体运动（确路+定人） → LMM.实践.005（立塔+发光+接引）
```

补充纪律：

- `BOS >= 45` 不是免责令牌。
- 只有当 `BOS`、`R-Min`、`最小承接能力` 三者同时成立，建塔才合法。
