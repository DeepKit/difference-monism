## 一、先分清四类陈述

DM 不是所有句子都声称同一种“真”。

| 陈述类型 | 当前角色 | 对外要求 |
|------|------|------|
| 本体承诺 | 给出最小世界图式 | 说清主张边界，不伪装成已验证科学定律 |
| 规范承诺 | 给出价值锚点与政治底线 | 说清它是价值选择，不偷装成纯经验描述 |
| 机制命题 | 解释意识、责任、纠错等如何成立或为何重要 | 必须允许竞争解释与反例进入 |
| 工具协议 | 案例档案、失败线、可比性清理、诊断协议 | 必须可复核、可回写、可暂停自动判断 |

最短判断：

1. `DM 可以有不可证伪的研究纲领硬核，但不能把所有内容都藏进硬核。`
2. `凡声称解释现实机制的部分，都必须留下被击败或被缩限的通道。`

## 二、最小本体承诺清单

| 承诺 | 最小说法 | 不在这里额外声称什么 | 主回读 |
|------|------|------|------|
| 差异 | 世界不是先给定一堆现成实体，而是先有可区分、不可完全同一的成立条件 | 不声称“差异”是神秘实体 | P00-P01 |
| 非任意约束性 | 并非一切组合都同等可行，存在稳定限制关系 | 不声称已把所有物理定律还原为一句话 | P00 |
| 结构与边界 | 稳定对象、系统与角色来自约束组织与成员性条件 | 不声称边界永远清晰或不可变 | P00-P03 |
| 继承序 | 时间性首先表现为痕迹进入后续成立条件的序位 | 不声称钟表时间可被一句偏序完全替代 | P00 |
| 层级涌现 | 上层结构可以形成新的约束位置与新的失效方式 | 不声称出现超自然断裂 | P02-P03 |
| 内部性 | 某些结构的内部状态差异会不对称地进入其自身后续成立条件 | 不声称已经完成意识第三人称闭合 | P04-P05 |

## 三、术语对接表

| DM 术语 | 外部近邻概念 | DM 当前差异点 |
|------|------|------|
| 差异 | distinction / non-identity condition | DM 把它当成本体起点，而不只是语言区分动作 |
| 结构 | constraint organization | DM 强调结构来自非任意约束，而不只是描述便利 |
| 边界 | membership condition / system boundary | DM 把边界当成“成其为一”的条件，不只当观察者切分 |
| 继承序 | path dependence / order of realization | DM 先谈成立条件继承，再谈钟表时间与过程词 |
| 反向约束 | downward constraint / top-down constraint | DM 要求它在可行状态空间上留下约束痕迹，而不是只会换说法 |
| 内部性 | internal asymmetry / self-conditioning | DM 用它解释第一人称轮廓的最小不可取消位置 |
| 责任 | attribution / accountability relation | DM 与 TAT 只把它先压成可归属性与可承接结构，不自动等于道德责备 |
| 危机压缩必须可复归 | reversible emergency governance | DM 关注的是例外状态是否留有恢复、复审与退出接口 |
| 纠错能力 | corrective capacity / self-revision capacity | DM 把它当作制度和文明韧性的核心变量之一，而不是装饰性美德 |

## 五、DM 当前明确不宣称什么

1. `不宣称已经完成同行评审与长期外部验证。`
2. `不宣称已经完成意识问题的第三人称终局闭合。`
3. `不宣称已经给出制度或文明的终局裁决。`
4. `不宣称任何下游协议都可以脱离责任门槛、工程审计与人工复核独立运行。`

## 六、DM 立足六条件

若外部读者要问“`DM` 为什么不只是另一套宏大叙事”，当前最稳的回答是：

1. `有最小硬核`
地基先压到 `DM-min`，不把所有强结论都神圣化

2. `有陈述分层`
区分 `本体承诺 / 规范承诺 / 机制命题 / 工具协议`

3. `有真实失败线`
关键命题接受缩限、降格与重写

4. `有公共接口`
外部可以按同一规则读、比、驳

5. `有比较力`
能说明自己比对手多解释了什么、最怕什么反例

6. `有实践裁决力`
能把上游判断压到责任、危机、制度与文明分析

正式说明见：

- [DM.P99.34.内部.DM立足六条件](./DM.P99.34.内部.DM立足六条件.md)
