## 一、统一可比性清理（必须先做）

制度案例最容易“比错对象”。每条档案必须先交代：

- 时间窗：危机期/战时/平时/转型期
- 外部威胁强度：高/中/低
- 技术条件：监控能力、传播结构、组织动员能力
- 退出成本：政治、经济、社会信任的不可逆损伤风险

若这四项只做到粗对齐，就只允许输出：

> `缩限结论`

而不允许输出：

> `强结论`

若连粗对齐都做不到，则按 DM.P99.27.内部.跨案例可比性清理模板.md 记为 `C 档`：

> `材料保留，不下总体判断`

### 2.1 档案使用门槛

本文的引用等级遵循 DM.P99.26.内部.发表级引用核验工作流.md，可比性档位遵循 DM.P99.27.内部.跨案例可比性清理模板.md。

制度案例若未同时完成引用核验与可比性清理，不得直接进入公开论证。

制度域的公开使用门槛：

- `A 级引用 + 任一可比档`：只用于内部压测与公开背景提示；进入送审包前必须先升到 `B 级`
- `B 级引用 + C 档可比性`：只保留材料，不下总体判断
- `B 级引用 + B 档可比性`：可进入对外送审文本，但只允许 `缩限结论`、条件性支持或局部反例压力
- `B 级引用 + A 档可比性`：可进入对外送审文本的主论证段；若要承担旗舰案例或跨法域关键外推，宜继续升到 `C 级`
- `C 级引用 + B 档可比性`：可支持公开文章中的条件性比较与限定性外推
- `C 级引用 + A 档可比性`：方可支撑公开文章/答辩中的旗舰案例、较强外推与反例裁决
- `任一引用等级 + C 档可比性`：只保留材料，不做总体胜负判断

默认状态：

- 除非个案条目另行注明，本文现有案例默认仍按 `A 级引用` 管理
- 各案例现写的“简版可比性清理”默认只支持 `B 档或待定` 使用，不视为 `A 档` 完整可比性清理

纪律补充：

- 本文各案例开头的“简版可比性清理”只用于快速判断
- 凡要进入送审包的制度比较，默认再补一轮 `P99.27` 的七维清理记录

### 2.2 四项硬字段与最小测量规则

从本版起，
制度域第一层案例默认还要显式填写四项硬字段：

| 字段 | 最低要问什么 | 最低记录方式 |
|------|--------------|--------------|
| `纠错能力` | 错误能否被看见、送回复审并真实改回 | `强 / 中 / 弱` + 一句理由 |
| `纠错载体` | 谁在记录错误、触发复审、形成反制衡、推动回退 | 至少列出 `2-5` 个具体载体 |
| `退出兑现度` | 临时措施是否按公开条件退出，谁能逼迫回退 | `高 / 中 / 低` + 触发器说明 |
| `深度伤害` | 是否存在去人格化、长期权利剥夺、承诺结构撕毁与难恢复损伤 | `高 / 中 / 低` + 范围/持续时间说明 |

使用纪律：

1. `第一层案例不得只写“有纠错 / 有退出 / 有伤害”`
必须至少按上述四项填一次最小记录。

2. `纠错载体必须是具体名词`
如 `法院 / 审计 / 档案 / 独立统计 / 地方试验 / 市场退出`，不能只写“社会会纠错”。

3. `退出兑现度` 不能只看法条有没有 sunset clause
还要看是否真实回退，以及继续限制时谁承担举证责任。

4. `深度伤害` 不能只看短期秩序
还要看是否存在去人格化、长期污名、永久剥夺申诉与承诺结构破坏。

### 2.3 禁止输出清单

以下输出在制度档案中默认禁止：

1. `未回指 P99.27 正式记录，就直接宣称某制度整体优于某制度`
2. `只凭单一短期效率或单一法条，直接裁决制度健康度`
3. `把高控制自动写成低纠错，或把开放外形自动写成高纠错`
4. `把 C 档可比材料或 A 级引用草料直接拿来做公开强结论`

## 三、档案 1：魏玛紧急状态（Article 48）到例外常态化

### 3.1 档案锚点

- 魏玛宪法的紧急权力条款（Article 48）
- 危机中权力压缩频繁使用
- 退出机制失灵，例外状态逐步常态化

### 3.2 核心争点

`危机治理` 如何退化为 `永久例外`？

### 3.3 DM 被压测的主张

- P09：危机压缩必须可复归；退出判定权不能只由压缩权力自身决定
- P09：法律外形不等于制度正当，纠错通道消失会导致文明退化

### 3.4 六维粗评分（示例）

- A 尊严：1
- B 权利：0-1
- C 纠错：0-1
- D 路径质量：0-1
- E 韧性：2（短期动员可能提升，但长期退化风险高）
- F 退出：0

结论倾向：`红`

### 3.5 四项硬字段（初版）

| 字段 | 初判 | 最小理由 |
| --- | --- | --- |
| 纠错能力 | 弱 | 宪法文本、议会与法院在形式上存在，但紧急扩权与授权法把这些纠错接口快速架空，导致危机中缺少有效复归能力。 |
| 纠错载体 | 宪法文本、议会、法院、州级结构 | 本案并非完全没有制度载体，而是这些载体在例外常态化过程中被吸空，因此恰好能检验“有载体而失兑现”的失败型。 |
| 退出兑现度 | 低 | 紧急状态缺少被独立强制执行的复归阈值，退出判定逐步被紧急权拥有者吞并，常规宪制回路失效。 |
| 深度伤害 | 高 | 基本权利暂停、反对派清除与长期承诺结构破坏叠加出现，不只是短期秩序压缩，而是制度人格化边界被整体侵蚀。 |

### 3.6 支持/缩限/反例纪律

`支持 DM`：

- 例外常态化与纠错通道消失、信任崩塌之间存在稳定结构耦合

`缩限 DM`：

- 若能证明例外条款本身并非关键，而是其他变量（战争失败、经济崩溃）完全解释制度坍塌，则 DM 需降低“退出设计”的优先度

`反例压力（触发重写）`：

- 若大量可比案例显示“例外常态化”反而长期更稳、深度伤害更低、纠错能力并未衰减，则 P09 危机纪律需重写

## 四、档案 2：日裔美国人拘禁（战时族群性拘押）

### 4.1 档案锚点

- 二战时期的族群性怀疑与集中拘禁
- 以国家安全为理由的大规模权利压缩

### 4.2 核心争点

`集体惩罚/族群性拘押` 是否可能被“续存收益”正当化？

### 4.3 DM 被压测的主张

- P08：责任与可归责性边界
- P09：权利作为承诺结构，不能被单方任意撤销
- P09：尊严底线与去人格化风险

### 4.4 六维粗评分（示例）

- A 尊严：0-1
- B 权利：0
- C 纠错：1（当时存在司法，但并不足以阻止）
- D 路径质量：0-1（异议成本与污名化高）
- E 韧性：2（短期“秩序感”可能提高，但长期信任受损）
- F 退出：1（退出并非由清楚触发器保障）

结论倾向：`红`

### 4.5 四项硬字段（初版）

| 字段 | 初判 | 最小理由 |
| --- | --- | --- |
| 纠错能力 | 弱到中 | 当时司法并未阻止族群性拘押，但后续仍出现 `Ex parte Endo`、官方调查与赔偿立法，说明纠错并非全无，只是迟到且代价极高。 |
| 纠错载体 | 司法、官方调查、后续立法赔偿 | 真正起作用的纠错载体主要不在初始行政命令内部，而在事后司法纠偏、委员会调查与国会修复法案。 |
| 退出兑现度 | 低 | 被拘禁者并未因清楚触发器而获得及时退出，退出更像战时政策松动后的迟缓回退，而非制度化兑现。 |
| 深度伤害 | 高 | 族群性怀疑、去人格化、财产与生活轨道破坏、长期污名与修复成本都显示其伤害已超出一般紧急措施。 |

### 4.6 支持/缩限/反例纪律

`支持 DM`：

- 该类机制制造长期不可逆信任损伤，并与后续制度修复成本高度相关

`缩限 DM`：

- 若能证明此类拘禁在可比威胁条件下确属不可替代且无更低损伤方案，则 DM 需更细化“最小替代性”判准

`反例压力（触发重写）`：

- 若大量可比案例显示族群性拘押长期显著更优、深度伤害更低且不破坏承诺结构，则 P08-P09 的责任/权利框架将被击穿（DM 的保守判断：不应出现）

## 五、档案 3：USA PATRIOT Act 与长期安全监控扩张

### 5.1 档案锚点

- 重大恐袭后的紧急立法与长期安全逻辑
- 监控能力与制度常态化风险

### 5.2 核心争点

`高安全收益` 是否能长期覆盖 `承诺结构与纠错通道` 的侵蚀？

### 5.3 DM 被压测的主张

- P09：反恐长期监控作为“常态化风险路径”
- P09：权利是合作可预期性的基础设施

### 5.4 六维粗评分（示例）

本案高度依赖具体条款与监督制度，先给“区间”：

- A 尊严：1-3
- B 权利：1-2
- C 纠错：1-3
- D 路径质量：1-2
- E 韧性：2-3
- F 退出：0-2（若无 sunset/复审硬化则偏低）

结论倾向：`黄`

### 5.5 四项硬字段（初版）

| 字段 | 初判 | 最小理由 |
| --- | --- | --- |
| 纠错能力 | 中 | 本案既有长期监控扩张，也出现法院、PCLOB、白宫复审与国会修法回缩，说明制度存在可工作的纠偏，但不是即时强纠偏。 |
| 纠错载体 | 法院、PCLOB、白宫复审、国会修法 | 关键不在单一 sunset 条款，而在多处载体叠加后，才把常态化监控从扩张链中拉回。 |
| 退出兑现度 | 中 | 相比魏玛或族群拘押，本案确有制度回缩，但回缩依赖多年争议、复审与修法，兑现速度和完整度都有限。 |
| 深度伤害 | 中 | 对隐私、自我审查与公民自由的侵蚀显著，但目前公开链更稳地支撑“结构性风险与冷却效应”，尚不足以直接升到最高伤害档。 |

### 5.6 三分结论纪律

`支持 DM`：

- 长期常态化监控与自我审查、异议成本上升、纠错通道收缩出现稳定耦合

`缩限 DM`：

- 若某些可核查制度设计确能同时保留独立审查、低深度伤害与强纠错，则 DM 需细化“监控何时退化成暴政”的阈值

`反例压力（触发重写）`：

- 若跨案例长期稳定显示：高监控体制更少深度伤害、更强纠错、更高韧性，则需重写 P09 对常态化监控的风险排序
