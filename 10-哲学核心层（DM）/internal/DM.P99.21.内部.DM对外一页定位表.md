## 1) DM 是什么（一句话）

> **DM（差异一元论）是一套用“差异-结构-边界-继承序-层级涌现”同一语法，贯通本体、心灵、认识与规范的研究纲领。**

### 1.1 可宣称 / 不可宣称

`可宣称`：

- 已形成统一主链、失败条件接口与首批案例档案
- 可作为 `送审 / 征评 / 预印本` 入口包公开说明
- 制度域与文明域的第一层案例已补入 `四项硬字段`，公开比较只在回指 `P99.27` 正式记录时下判断；当前已形成 `八组` 正式记录

`不可宣称`：

- 已完成同行评审、长期外部验证与整包终局定稿
- 已完成意识问题的第三人称终局闭合
- `AI 回评` 不构成外部背书；它只应被视为 `内部预审工具 / 外部征评辅助`

## 2) DM 不是什么（防误读）

- `不是强还原论`
DM 不承诺“低层原则上完整决定高层”

- `不是心物二元论或神秘主义`
DM 不另设灵魂实体起点，也不把意识/规范性当超自然命令

- `不是万能解释器`
DM 明示失败条件与缩限纪律（见 P99.6 / P05.12.5.8 / P99.17）

## 3) 三条最小新增性（可对外复述版）

1. `时间性压缩`
把“时间箭头”从过程词退回到“痕迹继承进入后续成立条件”的偏序结构（P00）

2. `意识定位`
把意识写成“内部性结构前提 + 生物实现的多层收窄”，不伪装第三人称闭合（P04-P05）

3. `规范与政治的生成链`
把责任/权利/尊严写成主体间结构中的角色位置与高损伤路径防御，并明确区分“发生学解释”与“规范力量地位”（P08-P09）

## 4) DM 的最小可受损性（你怎么可能错）

DM 允许三种命运：

- `保留`
- `缩限`
- `重写/放弃局部主张`

最危险的四条失败线（压缩版）：

- 若上层因果地位在所有合理重描述下都退化为一一映射下层摘要，`反向约束`降格为描述便利（P02）
- 若纯外部功能模型能长期无剩余解释稳定第一人称区分轮廓，`最小内部性`需重写（P04）
- 若四层半模型在各层同时长期失配，`感质桥接`需缩限或总体失败（P05）
- 若历史比较稳定显示低纠错高控制文明长期更优，`文明韧性押注`需缩限/改写（P09）

完整表：见 [DM.P99.6.内部.失败条件总表](./DM.P99.6.内部.失败条件总表.md)。

### 4.1 DM 为什么现在更容易站住（压缩版）

- 有 `DM-min`，不再把全部强结论都塞进地基
- 有 `本体 / 规范 / 机制 / 工具` 的陈述分层
- 有最危险命题的公开失败线
- 有一页定位、问答包与公共接口包
- 有对主要对手的比较窗口
- 有 `P08-P09` 向责任、危机、制度、文明判断下压的导向

## 6) 对外输出时的五件硬件（避免只讲原则）

1. `案例档案`
意识/制度/文明三域的首批可引用档案（P99.18-20）

2. `制度程序`
危机与合法性的八段式程序 + 六维度表（P99.15-16）

3. `证据链与反例处置`
意识域吃证据与怕反例的纪律（P99.17）

4. `当前公开引用优先序`
意识/制度/文明三域都已写明第一层/第二层/第三层使用纪律；对外默认优先动用第一层案例，且只做限定性主论证。制度域与文明域的第一层案例现已随案写入 `纠错能力 / 纠错载体 / 退出兑现度 / 深度伤害`，公开比较句默认必须回指 `P99.27` 正式记录；当前可直接回指 `八组` 正式记录（P99.18-20、P99.27）

5. `公共接口包`
把 `本体承诺 / 规范承诺 / 机制命题 / 工具协议` 分层，附最小术语对接与可受损条件，作为拆除“私语言”门槛的第一入口（P99.32）
