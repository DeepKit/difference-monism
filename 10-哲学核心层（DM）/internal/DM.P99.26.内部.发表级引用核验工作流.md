## 一、为什么需要单独的核验工作流

DM 当前已经有：

- 可检索作者/年份/题名
- 推荐版本
- 少量对外可复用档案

但这还不等于“发表级引用”。

发表级引用至少还要求：

1. `版本锁定`
2. `页码或章节定位`
3. `原句/原义核对`
4. `主张-证据的匹配记录`
5. `争议性二手文献与一手文献的区分`

压缩判断：

> **可检索锚点解决“能不能找到”，发表级核验解决“找到之后有没有引准”。**

## 二、三种引用等级

### 2.1 A 级：可检索锚点

最低要求：

- 作者
- 年份
- 题名
- 期刊/出版社或稳定机构入口

当前 `P99.18-20` 大多已达到此级。

### 2.2 B 级：发表准备引用

在 A 级基础上，至少再补：

- 版本信息
- 章节或页码范围
- 引用用途说明
- 一手/二手来源标记

### 2.3 C 级：发表级可核验引用

在 B 级基础上，至少再补：

- 逐条核页
- 关键引文原文复核
- 若为转引，必须标记“转引”并优先回到一手来源
- 若有 DOI、稳定链接、馆藏版本或归档链接，应留档

纪律：

> **只有 B/C 级引用，才允许进入对外送审包与公开文章主引文区。**

## 三、六步核验流程

### 3.1 第一步：先锁定“这条引用支撑什么”

每条引用必须先写明其功能：

- `事实锚点`
- `经典表述`
- `经验结果`
- `比较框架`
- `反例压力`

若功能写不清，则该引用不应进入正文。

### 3.2 第二步：区分一手与二手

至少标清：

- 是否为原作者原文
- 是否为综述或二手解释
- 是否存在更适合的原始出处

纪律：

> **涉及 DM 与外部理论的关键比较时，优先使用一手来源。**

### 3.3 第三步：锁定版本

至少记录：

- 版本年份
- 出版社/期刊
- 译本信息（若使用译本）
- 电子版或纸本来源

若同一作品有多个常用版本，应在记录中说明当前采用哪一版。

### 3.4 第四步：核页与核义

至少做到：

- 对关键表述核页
- 对争议性表述核上下文
- 防止只引最有利句子而漏掉限制句

若无法确认页码或上下文，则该条只能保留为 A 级锚点，不得冒充发表级。

### 3.5 第五步：记录“为何引用它”

每条引用需写明：

- 它支持哪一句 DM 表述
- 若拿掉这条引用，会损失什么
- 是否有更强替代引文

### 3.6 第六步：回写到引用核验记录

核验完成后，至少同步到：

- 当前档案文件
- `P99.28` 若该材料将进入送审包
- 必要时回写 `P99.7`，若核验结果导致旧写法需修正

---

## 四、最小核验记录模板

建议每条记录至少有下列字段：

| 字段 | 内容 |
|------|------|
| 引用编号 | 如 `CIV-V01-01` |
| 所属文档 | `P99.18/19/20/21/22/28` 等 |
| 引用功能 | 事实锚点 / 经典表述 / 经验结果 / 反例 |
| 一手/二手 | 一手 / 二手 / 转引 |
| 书目信息 | 作者、年份、题名、出版信息 |
| 版本锁定 | 版本、译本、链接或馆藏信息 |
| 页码/章节 | 精确到页或最小章节 |
| 核验状态 | A / B / C |
| 核验者 | 谁核的 |
| 日期 | 核验日期 |
| 备注 | 是否仍有争议、是否需回查一手文献 |

## 六、失败线与禁止事项

### 6.1 不允许的做法

- 只写“某某说过”而不锁定出处
- 用二手综述替代关键一手来源却不标记
- 用最有利摘句代替上下文
- 明知未核页却在对外稿中冒充精确引用

### 6.2 触发回退的情形

若出现以下情况，相关条目必须降级：

- 找不到稳定版本
- 页码核不出来
- 发现原义与当前转述不符
- 发现更强一手来源与现用二手来源发生冲突

结果：

- 从 `C` 降到 `B`
- 或从 `B` 降到 `A`
- 必要时回写正文，改成更弱表述

## 七、当前内部结论

最短压缩句：

1. `P99.18-20 解决了“有没有档案”，P99.26 解决的是“档案能不能拿出去”。`
2. `引用不核页，送审时最容易先断；因此发表级核验不是修辞装饰，而是外部对话的底线工序。`
