## 一、三种状态

### 1.1 软定稿

- 主链不再继续大改
- 本版可以拿去给外部读者批评
- 后续只允许做表述收紧、案例缩限、引用核验与局部修订

### 1.2 硬定稿

- 经过至少一轮外部压力测试
- 主要异议已回写
- 没有新的核心级重写被触发

### 1.3 宣传期

- 可以稳定对外说“这是当前定稿版”
- 宣传内容不再频繁被回头改写

### 1.4 按 DM 现行实践停线的补充判据

若把理论的意义压成一句：

> **理论的意义 = 解释世界，并指导实践。**

那么 DM 的停稿线还应补一条更直接的务实标准：

1. `解释够用`
它已能比主要竞争框架更稳定地解释自己最看重的那组现象

2. `指导够用`
它已能在制度、危机、责任与文明判断上给出可复查、可操作、可争论的实践导向

3. `继续大改的收益明显下降`
新增修改主要只是在换词，而不再显著提升解释力或实践裁决力

## 二、DM 现在处于哪一档

> **DM 现处于“v1.1 统一送审版”：已进入软定稿与首轮送审后的收束状态，并已达到现行内部所说的“第 3 状态”。**

原因：

- 主干与工具包已基本收束
- 统一送审包已生成，且已根据外部征评与 AI 内部预审做过连续本地收紧
- 已收束出单一默认外发文件 `DM.统一送审包.v1.1`
- 三域案例档案已形成“当前公开引用优先序”，对外口径已可避免超出证据强度的外推
- 制度域与文明域的高频比较对已开始写入 `P99.27` 的正式七维清理记录
- `AI 回评` 已被降格定位为：`内部预审工具 / 外部征评辅助`
- 最新一轮正文级修订已把剩余核心卡点收敛处理到 `B→C 发生桥`、`阈值本体论` 与 `跨文明纠错载体识别`
- 因此按现行内部标准，`解释够用 + 指导够用` 已达到停稿线；后续主要转入维护、核引与外部长期验证

## 三、进入硬定稿的七个门槛

### 3.1 至少一轮外部阅读

- 至少 `3` 位外部读者
- 至少来自 `2` 个不同方向

补充纪律：

- `AI 回评` 可用于内部预审、补齐问答与提前暴露薄弱点
- 但 `AI 回评` 不单独计入 `3` 位外部读者门槛
- 它最多只能作为正式外部阅读之前的辅助筛查

### 3.2 外部批评没有触发核心级重写

允许：

- 表述过满被压低
- 个别案例被缩限
- 若干引文需要重核

不允许仍未处理：

- 主链断裂
- 失败线失真
- 核心案例无法支撑对应主张

### 3.3 三域第一层案例至少各有可公开使用起点

最低要求：

- `P99.18/19/20` 已明确第一层/第二层/第三层公开引用优先序
- 意识/制度/文明三域至少各有 `1` 个第一层案例达到 `B` 级起点
- 若某案例承担旗舰答辩点或高争议外部比较，宜达到 `C` 级或附精确段落/判决页位
- 主张-证据匹配说明
- 一页 `审稿人摘要` 已可稳定配合统一送审包外发

### 3.4 制度域与文明域完成可比性清理

- 送审中承担 `跨案例比较` 或 `公开对照` 功能的制度/文明高频案例对，已写入 `P99.27` 的正式记录
- 若某条只承担 `单案例制度描述` 或 `单案例文明压力`，可不强行写成总体胜负比较
- 不再存在明显“不可比但硬比”的结论
- 公开比较句已形成 `无 P99.27 记录则不下判断` 的统一纪律

### 3.5 一个修订窗口内无核心改动

建议窗口：

- `4-8` 周

在这个窗口内：

- 没有新核心概念
- 没有新的一级张力
- 没有新增导致主链改写的重大异议

### 3.6 达到“解释与实践够用”

这是按 DM 现行实践停线新增的停稿门槛。

最低要求：

1. `解释层`
对外高频批评已开始重复，
且不再出现新的一级问题

2. `实践层`
P08-P09 已能给出可复查的责任、制度、危机与文明判断模板，
而不是只会讲原则

3. `比较层`
DM 已能稳定回答三句：

```text
它比主要对手多解释了什么
它最怕什么反例
它如何指导制度与危机实践
```

### 3.7 达到“DM 立足六条件”

这条门槛用于防止 `DM` 重新滑回“大而全但不够硬”的状态。

最低要求：

1. `最小硬核`
公开口径稳定压回 `DM-min`，不把全部强结论都偷塞进地基

2. `陈述分层`
对外文件已明确区分 `本体承诺 / 规范承诺 / 机制命题 / 工具协议`

3. `失败线`
最危险命题仍保留真实缩限或重写触发，不被“总能再解释”偷抹平

4. `公共接口`
外部读者可经由 `README / P99.21 / P99.32 / P99.34` 按同一规则进入

5. `比较力`
能稳定回答“比主要对手多解释了什么、最怕什么反例”

6. `实践裁决力`
`P08-P09` 已继续承担责任、危机、制度、文明判断入口，而非只保留抽象原则句

## 四、什么时候可以开始宣传

### 4.1 现在就可以做的宣传

- `低承诺定位宣传`
- `招募读者式宣传`
- `工作坊/预印本/征评式传播`

可用口径：

> `DM 已形成统一送审版（v1.1），现进入外部征评、批评吸收与长期验证阶段。`

补充纪律：

- 对外默认优先动用三域第一层案例
- 第二层案例只作边界说明与机制提示
- 第三层案例不进入公开主引文区
- `AI 回评` 不出现在对外口径里充当外部背书

### 4.2 还不建议做的宣传

- “理论已最终完成”的宣传
- “已被证明/已成熟封版”的宣传
- 过度高调的结论式品牌传播
- 拿第二层/第三层案例冒充已完成核验的旗舰证据

### 4.3 何时可以转为正式宣传

建议在第三部分七条门槛同时满足后，再进入：

- `正式定稿宣传`
- `版本号固定宣传`
- `系统性对外比较与公开答辩`

## 六、内部结论

1. `DM 现在已经可以停止大规模内部重写；按现行内部标准，它已达到第 3 状态。`
2. `“解释够用 + 指导实践够用” 现已达成；后续工作的重心不再是补主链，而是核引、定点维护、扩展可比性记录与长期外部比较。`
3. `宣传现在可以做，但应以“统一送审版 / 征评版 / 已达到现行实践停线的研究纲领”口径展开，并按三域公开引用优先序使用案例，而不应冒充终局真理。`
4. `后续若再判断 DM 是否站得住，默认不再靠抽象印象，而要回到 P99.34 的六条件总表。`
