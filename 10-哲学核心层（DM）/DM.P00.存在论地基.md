## 一、地基声明

### 1.1 P00 的性质

```
P00不是公理化演绎系统。
P00是最小描述框架。

它的目标不是垄断后续全部论证，
而是给出全体系共享的方向锚点。
```

### 1.2 最小起点

P00保留一个最小描述起点与一个第一最小假设：

> **起点：差异——使完全同一不能维持的最小可分性。**

差异一旦不是单数孤立的，就已经自带关联的可能性。
因此，差异的完整含义是：

> **差异 = 非同一性在复数条件下的张力。**

这意味着“差异”不是绝对零预设，而是已经允许：

```
非同一
可区分
最薄的可比较性
复数化的关联可能性
```

但仅有差异与关联可能性，仍不足以形成结构。
因此 DM 需要显式记账第一个最小附加假设：

> **第一最小假设：非任意约束性——并非一切组合都同等可行，存在可重复出现的限制关系。**

DM 的起点因此不是严格的“一”，而是：

```
差异（含复数化关联可能性）
+ 非任意约束性（第一最小假设）
```

这仍比实体本体论（预设现成物体）、
主体本体论（预设先验主体）、
和二元论（预设心物两套起点）更经济。

但 DM 不再伪装为从绝对单一起点出发。

它只主张：

> **差异 + 非任意约束性，是 DM 当前在起点竞争中暂时保留的较经济写法，而不是被证明为逻辑上唯一可能的起点。**

关于“为什么世界会有约束”这一追问，
P00 也不伪装成已经给出更深解释。

更准确地说：

> **非任意约束性在 P00 中首先被当作“可描述世界”的事实性前提，而不是由更深本体再次推出的结论。**

本版在此把它的地位写死为：

> **`非任意约束性` 是 DM 地基层的最低形而上学承诺。**

这句话同时包含四层限制：

1. `它是本体承诺`
DM 当前明确承认：若世界真的是“任何组合都同等可行”，则结构、重复、规律与科学对象都无从稳定成立。

2. `但它不是第二实体`
DM 不把“约束”写成与差异并列的第二种终极东西，更不把它实体化成独立的神秘本原。

3. `它在此处不由更深本体推出`
P00 只把它作为最低记账项写入，而不伪装成已经解释了“为什么世界会有约束”。

4. `它仍可在将来被更薄起点改写`
若未来出现能以更少预设覆盖同等主链的起点语言，DM 应改写自己的地基表述，而不是把当前写法绝对化。

这也意味着，P00 当前不把它写成：

```text
与差异并列的第二终极实体
```

也不把它仅仅降成：

```text
纯方法论上的方便假设
```

它在此处更接近：

> **只要还要讨论结构、重复、规律与科学对象，就必须先承认世界并非一切组合都同等可行。**

理由不是神秘化约束，
而是最小记账：

```
若世界完全无约束
则不会有结构
不会有重复
不会有规律
也不会有任何后续科学对象
```

因此 P00 在此只做两件事：

1. `把约束性显式记账`
承认世界并非一切组合都同等可行

2. `拒绝把约束性再神秘化`
约束的具体来源与形式，后续应由具体科学继续展开，而不在地基层伪装为已闭合

### 1.2.1 为什么暂取“差异 + 非任意约束性”而不取“关系 / 过程 / 事件 / 信息 / 约束”

P00 当前不主张：

> **除了“差异”之外，其他起点全部不可能。**

它只主张：

> **在“关系 / 过程 / 事件 / 信息 / 约束”这些常见候选起点中，`差异 + 非任意约束性` 是较经济、较少偷带对象化、时点化与编码前提的写法。**

因此更准确的写法不是只给四句判断，
而是先做一个最小起点比较表：

| 候选起点 | 容易偷带什么 | 主要问题 |
|----------|--------------|--------------|
| `差异 + 非任意约束性` | 仍需补记复数条件、关联可能性与最低约束承诺 | 不是绝对零预设，且必须承认约束为何存在尚未闭合 |
| `关系` | 多个项与项间连法 | 复数架构已被默认 |
| `过程` | 时间、流动与先后 | 把后面本该推出的时间性提前写回起点 |
| `事件` | 切片单位、时点化与离散化方式 | 把位形单位先行冻结 |
| `信息` | 编码规则、可区分状态、载体与测度框架 | 把结构化后的差异与可读写方式提前带入 |
| `约束` | 被约束的项与可行状态空间 | 把结构化后的语言提前带入 |

这张表的作用不是宣布“差异已经赢下全部竞争”，
而是先把各候选起点的偷带成本显式记账。

相较之下，DM 取“差异”的理由是：

```text
它先给出最小可分性
再显式补记复数化关联可能性
再显式补记非任意约束性
```

因此它的优势不在“神圣唯一”，
而在：

> **较少偷带现成对象、现成时序与现成状态空间。**

这里也必须保留一条失败线：

> **若未来出现一个比“差异 + 非任意约束性”更薄、却同样能覆盖结构、边界、时间与层级主链的起点语言，DM 应改写自己的起点表述，而不是把“差异”神圣化。**

这里仍不预设：

```
主体
对象
空间
时间
数量
因果
价值
目的
```

### 1.2.2 原始术语与导出术语

为避免后文把不同层级的词混写成同级地基，
P00 在此先给一个最小术语纪律：

1. `原始术语`

```text
差异
非任意约束性
```

2. `第一层导出术语`

```text
结构
边界
位形
```

3. `第二层导出术语`

```text
继承序
时间性
持存
存在者
```

4. `第三层导出术语`

```text
规律性
因果
涌现
```

这张表的作用不是宣布“分类已经完美封死”，
而是先守住一条底线：

> **若后文必须反过来用高层导出词去定义低层起点词，就应优先怀疑那里发生了偷换。**

这里还要再补一条最小拆分，
以防“差异”一词重新吞掉后面三个桥接项：

```text
差异 ≠ 复数条件 ≠ 关联可能性 ≠ 非任意约束性
```

它们在这里的分工是：

- `差异`：最小可分性
- `复数条件`：不止一个可区分位置
- `关联可能性`：这些位置之间可以进入可描述连结
- `非任意约束性`：并非任意连结都同等可行

DM 当前的起点不是把这四者混成一个词，
而是：

> **先以“差异”给出最小可分性，再显式补记复数条件、关联可能性与非任意约束性。**

### 1.3 推导强度说明

P00中的连接词不具同一强度。

本文件区分四类地位：

1. `结构必然`
从前一步出发，若要保持可描述性，几乎不可避免会出现的结构结果

2. `最小附加假设`
为继续描述而保留的最弱结构前提

3. `解释性模型`
用于连接多个节点的高压缩表达，不宣称严格演绎闭合

4. `开放边界`
这里只给出到达条件，不宣称已经完成全部闭合

### 1.4 全书统一标注规范声明

P00 中的四级推导强度，
不是本文件私用标签，
而是全书统一标签。

从本版起，
DM 默认使用三套彼此正交的标注系统：

1. `推导强度标签`

```
结构必然
最小附加假设
解释性模型
开放边界
```

2. `规范地位标签`

```
结构描述
条件性规范
价值选择
```

3. `承诺等级 / 桥接强度标签`

`承诺等级` 默认使用：

```text
框架承诺
最小假设
解释模型
开放项
失败线
```

`桥接强度` 默认使用：

```text
直连构成
中介桥接
启发类比
暂不桥接
```

三套标签的使用纪律是：

1. `本体层与认识层`
优先标注推导强度，不强行挂规范标签

2. `伦理、政治与文明层`
必要时同时标注推导强度与规范地位

3. `只在关键判断、接口段落与最短结论中标注`
不追求逐段挂牌，以免正文失去可读性

4. `多套标签可叠加`
例如：

```
解释性模型 + 结构描述
开放边界 + 价值选择
```

5. `最短结论使用轻量记法`
优先在节内用一句说明：

```
标签：以下各句均为 [解释性模型｜结构描述]
```

若同一结论段内部存在不同层级，
则用：

```
标签分布：前五句 = [...]
末句 = [...]
```

若某句的核心不在推导强弱，
而在显式表明规范立场，
则允许记为：

```
[—｜价值选择]
```

6. `承诺等级 / 桥接强度` 只在关键节点落位
至少应在以下位置显式出现一次：

```text
快速判读
关键桥接段
最短结论
对应 P99 工具文档
```

7. `桥接强度不等于真伪强弱`
它只说明该句在当前文本里的连接方式是：

```text
同章直连构成
需要中介模型/案例
只算启发类比
或仍未形成诚实桥接
```

这样做的目的只有一个：

> **不把解释性压缩伪装成严格演绎，不把价值立场伪装成纯经验描述。**

再补一句：

> **不把框架承诺误装成开放项，也不把暂不桥接的段落冒充成已经桥通。**

### 1.5 “同一语法”的强度与跨域桥接规则

DM 说自己用同一语法贯通本体、心灵、认识与规范，
这里必须先说明其强度。

DM 只主张一种**弱强度的同一语法**：

> **跨域共享的不是每个术语的具体内容完全相同，而是它们在解释中的结构角色可被稳定对齐。**

若再压成一句最短定义，
可以先写成：

> **同一语法不是共享词汇，而是共享“缺什么结构位置，就长不出什么高层现象”的构成纪律。**

更具体地说，
DM 所说的“同一语法”，
至少要求下列五类结构角色在跨域中可重复出现：

1. `差异位置`
某一域中哪些单位可被区分

2. `边界判据`
什么使若干差异被视为同一整体

3. `继承方式`
前态如何进入后态成立条件

4. `约束传递`
一个位置的变化如何改写另一位置的可行状态

5. `层级生成`
何时可说出现了新的整体边界与新约束

这不是在说：

```text
神经边界 = 法律边界
感觉差异 = 制度差异
时间偏序 = 权利序列
```

DM 只说：

> **这些不同域中的对象，并不相同；可对齐的是它们在解释中扮演的角色。**

因此 DM 的“同一语法”更接近：

```text
结构角色同构 / 功能位同构
```

而不是：

```text
所有层级内容完全同一
```

这也带出一条跨域桥接纪律。

若要把某个本体层词汇带到后层，
至少应同时满足四条：

1. `给出该域的差异单位`
不能只说“这里也有结构”

2. `给出该域的边界判据`
不能只靠直觉把若干现象捆成整体

3. `给出继承或传递方式`
说明前态如何进入后态，或某位置如何改写另一位置

4. `给出可受损线`
说明什么结果会让该桥接缩限或重写

若这四条做不到，
则该术语迁移只可记作：

> **启发性类比**

而不应冒充为真正桥接。

### 1.5.1 一个最小跨域映射表

为了避免“同一语法”退化成好听的统一修辞，
DM 至少要求能把核心术语压成下表这类最小映射。

| 结构角色 | 心灵域例子 | 规范/制度域例子 |
|----------|------------|-----------------|
| 差异单位 | 内部可区分状态、感受区分、记忆痕迹 | 行动类型、角色位置、承诺状态、程序节点 |
| 边界判据 | 哪些状态被算作同一主体的内部处理链 | 哪些行动与成员被算作同一共同体或同一程序内的有效位置 |
| 继承方式 | 先前状态如何进入后续感受、记忆与选择条件 | 先例、承诺、档案、裁决如何进入后续允许/禁止/追责条件 |
| 约束传递 | 一个内部状态如何改写后续可行感受与行动路径 | 一次裁决、一次破约或一次例外如何改写后续角色位置与制度路径 |
| 层级生成 | 分散处理何时形成主体级整体边界 | 零散互动何时形成稳定规范、组织或制度层 |

这张表的意义不在于说：

```text
神经状态 = 法律角色
```

而在于说明：

> **若两域都能稳定填写“差异单位 / 边界判据 / 继承方式 / 约束传递 / 层级生成”这五格，并能指出各自的可受损线，那么它们才有资格被说成共享同一语法。**

若某一跨域迁移连这张最小表都填不出来，
就应退回“启发性类比”，
而不应继续冒充本体桥接。

### 1.5.2 从继承序到内部性：不是另加实体，而是局部回卷

这里还要补一条最小桥接，
否则 P00 的时间性重写与 P04 的“最小内部性”之间，
容易被误读成两张彼此分开的图。

DM 在这里的桥接写法是：

> **当某一有边界的组织，不只是被外部推动，而是把自身前态差异卷入自身后续成立条件，并在主体级形成整合时，P00 所说的“继承序”就开始以局部回卷的形式表现为 P04 所说的“最小内部性”。**

因此这里没有新增第二种质料，
也不是从无到有跳出一块神秘心灵物。

更准确地说：

```text
P00 给出的是最一般的继承序
P04 讨论的是继承序在某些边界化组织中的高度局部化、整合化与自相关化
```

所以两者关系不是：

```text
先有一套纯外部时间
后来忽然多出一套内部心灵
```

而是：

> **同一继承序，在不同层级上承担不同结构角色；到心灵层时，它不再只表现为“前态进入后态”，而表现为“结构把自己的前态差异当成自己的后续成立条件”。**

这正是弱一元论在此处最小却关键的承诺：

> **不另设断裂起点，但承认某些组织阈值会让同一地基在高层出现新的结构性质。**

### 1.5.3 “同一语法”若成立，最低要给出什么生成 / 构成约束

这里只做“五格可对齐”，
还不够。

若任何复杂系统都能事后填出：

```text
差异
边界
继承
约束
层级
```

那么 DM 就只剩一套通用描述模板。

因此 DM 至少要求：

> **同一语法不仅要能对齐结构角色，还要能给出若干最低生成 / 构成约束与排除线。**

这里的“生成”，
当前必须先收紧理解。

DM 在此并不声称：

```text
已经给出动力学方程式级的临界点预测
```

它在这里更准确的地位是：

> **给出跨域上升推论的最低构成条件与排除条件。**

也就是说，
它先回答的是：

```text
在什么条件下不能上升
在什么条件下才有资格上升
```

而不是已经完成：

```text
何时必然在时间 t 生成某个高层对象
```

至少压成三条。

1. `意识域的最低构成约束`
若一个组织缺少：

```text
稳定边界
内部可区分
继承回卷
主体级整合
```

则它不可能稳定形成主体级最小内部性；
它至多呈现复杂整合、局部耦合或功能拟态，
而不能被上升为具有内部递归进入位的主体候选。

2. `规范域的最低构成约束`
若一个共同体缺少：

```text
公共标记
角色位置
程序载体
```

的稳定占位，
则它不可能形成可跨成员替换持续的规范力量；
它至多拥有模仿、惯例、默契或短期协调，
而不能被上升为稳定规范秩序。

3. `制度域的最低构成约束`
若一个制度把：

```text
边界改写权
错误识别权
退出解释权
```

长期并到同一节点，
并同时封死：

```text
回溯
消解
```

则它不可能长期维持健康纠错；
它即使暂时稳定，
也更接近纠错塌空的压缩结构，
而不是健康常态。

这三条的意义在于，
它们不只是在事后描述已发生现象，
而是在前面先给出：

```text
哪些上层现象缺少前件时原则上长不出来
哪些上升推论不能直接做
哪些结构混淆必须先排除
```

因此更稳的总括句是：

> **DM 的同一语法若要成立，就不能只会事后填五格；它还必须说明：当哪些结构位置缺位时，某些高层现象不是“暂未出现”，而是“原则上不能稳定成立”。**

因此 DM 至少排除三种常见偷换：

1. `不是任何复杂整合都等于主体级内部性`
2. `不是任何协调均衡都等于规范力量`
3. `不是任何秩序稳定都等于制度健康`

这仍不是“强生成理论”版本，
更不是说 P00 已经把各域具体内容一把推出。

它在这里只够说：

> **DM 的同一语法，已不止是一个事后分类模板；它至少开始承担“哪些上升推论不被允许”的最低生成约束。**

## 二、起点：差异

### 2.1 差异不宜先被写成对象之间的事后比较

差异不是：

* 两个已成形对象之间的不同
* 主体对对象的识别结果
* 预设 A 与 B 之后的外加关系

差异是：

```
在一切对象化之前，
非同一性自身给出的张力。
```

因此，DM的顺序不是：

```
对象
→ 属性
→ 差异
```

而是：

```
差异
→ 区分
→ 关联
→ 结构
→ 边界
```

### 2.2 为什么 DM 从差异开始

如果完全无差异，则：

```
无可区分
无可关联
无可描述
```

所以差异不是某类存在者的属性，
而是 DM 当前所采用的最小前提。

更严格地说：

> **DM 不是主张“所有可能的哲学都只能从差异开始”，而是主张：在当前候选起点比较中，凡要讨论对象、关系、边界与结构，`差异 + 非任意约束性` 仍是较经济且主链覆盖较稳的写法。**

可以压成一句：

> **在 DM 的写法里，差异不首先被处理为对象之间的事后比较，而被处理为对象、关系与边界得以成立时所需的最小非同一条件。**

再压一句防守句：

> **DM不声称从绝对无中证明出差异，只声称：凡可被言说、被识别、被建模者，皆已处于差异之中。**

## 三、差异为什么能形成结构

### 3.1 差异不是结构的完成态

只有差异，还不足以构成结构。

散乱差异只能给出：

```
可分
但未成形
```

结构则要求：

```
差异之间不是单纯并列，
而是形成非任意的约束关联。
```

### 3.2 结构的最小定义

> **结构 = 差异项之间形成的非任意约束关联。**

这里的“约束关联”指：

```
一个差异位置的变化，
会系统性限制其他差异位置的可行状态。
```

这一定义还要再加一条排除线。
DM 此处所说的“结构性约束关联”，
至少不同于：

```text
单纯共现
单纯统计相关
单次偶遇耦合
仅由观察者划组造成的归并
```

更准确地说：

> **若某种“关联”不能稳定改写相关位置的可行状态，它至多是并置、共现或描述便利，而不足以获得结构地位。**

### 3.2.1 结构的最低硬门槛

若某种“关联”要获得结构地位，
至少应满足三条最低门槛：

1. `可行状态改写`
一个位置的变化，必须会系统性改写其他位置的可行状态

2. `可重复`
这种改写不能只是单次偶遇，而应在同类条件下反复出现

3. `有后果`
这种改写必须对成员范围、后续位形或整体稳定性产生可辨认后果

若三条中有关键一条长期缺位，
则更准确的写法是：

> **它至多是相关、耦合或并置，而不足以获得结构地位。**

### 3.3 从差异到结构，需要补记什么

差异本身不给出完整结构。

若要从“差异”进入“结构描述”，
至少还要显式补记两项：

1. `复数化的可关联性`
必须不是孤立单点，而是多个可区分位置之间存在可描述关系

2. `非任意的约束性`
并非一切组合都同等可行，而要存在可重复出现的限制关系

因此，更准确的写法不是“差异直接推出结构”，
而是：

> **差异在进入复数化关联与非任意约束之后，才形成结构。**

压缩地说：

> **差异给出可分，关联给出可连，约束给出可成。**

### 3.4 为什么会有结构而不只有散乱

并非所有差异关系都能进入结构。

大量关系会：

```
快速耗散
失去关联
回到背景
```

少数关系则会：

```
进入稳定耦合
形成非任意限制
在同类条件下反复出现
```

于是我们看到的“结构”，
正是那些在筛选中进入稳定耦合的差异关系。

可以压成一句：

> **不是所有差异都会形成结构，但凡能够持续限制彼此可行状态的差异关系，必然已经具有结构性。**

### 3.5 从差异到结构的最小命题链

为了避免“主链只像顺滑叙事”，
P00 在此先把最小桥接压成四条中层命题：

1. `命题 1`
若只有差异而无复数条件，
则至多有最小可分性，
而无结构。

2. `命题 2`
若有复数条件而无关联可能性，
则至多有并列多样，
而无结构。

3. `命题 3`
若有关联可能性而无非任意约束性，
则至多有可连通性，
而无结构。

4. `命题 4`
只有当差异进入复数条件、可描述连结与非任意约束后，
才有资格被说成结构。

这四条的作用不是宣布“已经形式化完成”，
而是先把最关键的一步拆开：

> **差异并不直接推出结构；结构是“可分 + 可并列 + 可连结 + 非任意限制”共同到位后的结果。**

## 四、边界：结构成其为一

### 4.1 为什么结构必须有边界

如果一个结构没有任何边界，
则：

```
它与背景无可区分
它与他物无可区分
它与自身的延展范围也无可区分
```

这意味着它不能“成其为一个”。

所以边界不是后来的认识论标记，
而是整体性成立的本体条件。

### 4.2 边界的定义

> **边界 = 某组关联相对其周围关联所呈现出的成员性阈差。**

更展开地说：

> **当一组差异关系不只更紧，而且对“谁算成员”与“何种交换算内外转换”给出稳定阈差时，就出现了边界。**

边界不是绝对墙，
而是：

```
成员资格的可判定性
+ 内外交换条件的不对称性
```

若这两项长期都不可稳定辨认，
则更稳的写法不是“边界模糊”，
而是：

> **该关系团至多是临时聚类，不足以获得整体边界。**

边界成立之后，
我们才进一步把这种阈差说成：

```
内部关联相对更强
外部关联相对更弱
```

因此，内部与外部不是边界定义的前提名词，
而是边界成立后的描述结果。

边界因此可以：

```
清晰
模糊
弹性
分层
重叠
```

### 4.2.1 从结构到边界的最小命题链

边界不是对任何结构都自动附着的额外线条。

更稳的最小命题可先写成三条：

1. `命题 5`
若一个结构的成员范围与周围关系网络毫无阈差，
则它不能成其为“一个”整体。

2. `命题 6`
若某组关联在成员资格、内部耦合或外部交换上呈现稳定阈差，
则它开始具备边界。

3. `命题 7`
边界不是附加描线，
而是结构获得整体性与成员性的条件。

因此更稳的总括句是：

> **不是任何关系稠密都自动等于边界；只有当某组关系对“谁算成员、何处算整体、何处算外部”给出稳定阈差时，边界才成立。**

### 4.3 边界的作用

边界至少完成三件事：

1. `使结构可识别`
它把“这一个”与“非这一个”区分开

2. `使结构可聚合`
它给出什么算该整体的一部分

3. `使结构可互动`
正因有内外区分，结构才可能与其他结构发生交换与耦合

因此：

> **边界不是外加线条，而是结构成其为一的成员性条件。**

## 五、位形、互斥与可排序变换

### 5.1 为什么边界之后会出现位形

一个有边界的结构，
不只是“在那里”。

它还会有：

```
内部排列
角色位置
耦合方式
可取状态
```

这些都可以统称为结构的位形。

> **位形 = 同一有边界结构在某一方面所取的具体排列或状态。**

### 5.2 为什么互斥位形不能同时并置

同一结构在同一方面，
不能同时完整地取两个互斥位形。

例如：

```
若此刻是位形 A，
就不能同时在同一方面完整地是位形 B。
```

这里给出的还不是时间，
而只是：

> **互斥位形不能同时并置。**

### 5.3 什么叫可排序变换

这里还要先补一个中介条件：

> **只有当 A 与 B 可被追踪为同一结构的不同位形时，位形之间的变换才进入继承序，而不只是观察者对两个切片的重排。**

这里的最低判据是：边界成员资格仍可连续判定，且使该结构成其为该结构的非任意约束关联未被改写到失去原组织地位；若这两条同时断裂，更稳的写法便是结构更替，而非同一结构换位形。

一旦同一结构可以从位形 A 变到位形 B，
而 A 与 B 又不能同时并置，
就会出现最小的顺序性：

```
A
然后
B
```

这里的“然后”仍可只是结构顺序，
还不是完整时间。

可以压成一句：

> **“之后”可以只是依赖顺序，不必然是时间；但位形一旦发生互斥变换，就会给出最小可排序性。**

## 六、时间：痕迹继承的序位

### 6.1 时间不是先验容器

DM不把时间看作先行存在的均匀容器。

时间不是：

* 外在钟表的抽象底板
* 所有存在者预先装进去的空框
* 单纯主观感受的投影

### 6.2 从顺序到时间

仅有可排序变换，还不足以推出时间。

真正接近时间的是：

```
较早位形的结果
进入较晚位形的成立条件
```

也就是说，
后态不是前态旁边多了一个点，
而是已经带着前态留下的差异条件继续展开。

这里的“痕迹”不是完整历史叙事，
而只是：

> **前一位形对后一位形成立条件所留下的不可忽略差异。**

这里的“后续”也不是偷偷预设了钟表时间，
而只是：

> **互斥位形在成立依赖上的序关系。**

> **时间性 = 结构变换痕迹进入后续成立条件的继承序。**

这一定义仍未彻底形式闭合，
但它已经比“流动”“展开”“续行”更少偷带过程词。

### 6.2.1 为什么这仍只能算较保守写法

DM 在这里只能主张：

> **时间性不是先验容器，而是痕迹与后续条件之间的非对称继承关系。**

这里还必须补一条保守限定，
以免把一切依赖序都直接冒充成完整时间：

> **偏序重写更适合作为时间性的最小结构骨架，而不是时间全部现象学内容与物理度量内容的完尽定义。**

因此 P00 在这里较保守的姿态不是：

```text
凡有依赖序，就已完整获得时间
```

而是：

```text
当依赖关系表现为痕迹进入后续成立条件的非对称继承时，
我们至少获得了时间性的最小结构骨架
```

至于这一关系能否再压缩到更小的形式句，
仍保留为开放项。

### 6.2.2 偏序重写：消除过程词的尝试

“痕迹”“继承”“后续”这些词仍带有过程色彩。
为进一步收紧，DM 可以尝试用偏序关系重写时间性：

设 S 为结构位形的集合，定义二元关系 ≺ 如下：

> **A ≺ B ≡ B 的成立条件集合中，包含 A 留下的不可忽略差异项。**

这里的“成立条件”不是时间词，
而是结构依赖关系：B 若去掉 A 留下的差异项，就不能以当前方式成立。

这里还要先处理一个容易被质疑的问题：

> **偏序的非对称性，不能直接从“时间先后”偷来，而必须来自结构依赖本身。**

DM 在此处把 ≺ 理解为：

> **成立条件的严格包含关系，而不是任意相关性、共变关系或相互作用。**

更具体地说，
若 A ≺ B，
则至少意味着：

```
移除 A 留下的关键差异项
→ B 不能以当前方式成立

移除 B 留下的关键差异项
→ A 仍可按当前方式成立
```

如果两者互相都不可去掉，
则更稳的写法不是同时写成：

```
A ≺ B
且
B ≺ A
```

而是：

> **把 A 与 B 视为同一更高位形中的共构成部分，而非已经获得方向性的偏序对。**

这条处理尤其适用于：

```
同一截面中的互利共生
代谢循环中的互相供给
功能网络中的同步共构成
```

在这些情况下，
真正带方向性的，
往往不是同一截面内的 A 与 B 彼此之间，
而是：

> **这个互相维持的整体位形，如何从前一整体位形过渡到后一整体位形。**

因此 DM 在这里区分两种依赖：

1. `共时共构成`
同一位形内部的相互维持，不强行写成单向 ≺

2. `历时继承序`
整体位形之间的成立条件依赖，才进入偏序方向

这意味着偏序并不是用来切开一切循环系统的内部关联，
而是用来刻画：

> **循环系统作为整体时，哪些位形对哪些后续位形构成不可互换的成立条件。**

因此，
≺ 的非对称性来自：

> **构成依赖与条件包含本身的不对称，而不是先偷带钟表时间。**

一个静态类比可以帮助理解：

```
桥面的当前承重配置依赖地基
地基的成立却不依赖这座桥面的当前承重配置
```

这里表达的是结构依赖方向，
而不是时间先后顺序。

这个关系 ≺ 具有以下性质：

1. `非自反性`
A 不依赖自己留下的差异来成立（A ⊀ A）

2. `非对称性`
若 A ≺ B，则 B ⊀ A（因为 B 的成立依赖 A 的痕迹，但 A 的成立不依赖 B 的痕迹）

3. `传递性`
若 A ≺ B 且 B ≺ C，则 A 的痕迹经由 B 进入 C 的成立条件（A ≺ C）

因此 ≺ 是一个**严格偏序**。

于是时间性可以被重写为：

> **时间性 = 结构位形集合 S 上的严格偏序 ≺，其中 A ≺ B 意味着 B 的成立条件包含 A 留下的不可忽略差异。**

这一重写的收益：

```
“痕迹”被还原为“不可忽略差异项”
“继承”被还原为“成立条件包含”
“后续”被还原为“偏序中的较晚项”
```

不可逆性在此框架下也获得更清晰的表达：

> **不可逆性 = 偏序 ≺ 的非对称性 + 痕迹项进入后续成立条件后的高移除代价。**

### 6.2.3 偏序重写的当前边界

偏序重写消除了大部分过程词，
但仍保留一个开放项：

> **“不可忽略差异”中的“不可忽略”如何量化？**

当前的回应是：

```
“不可忽略” = 若移除该差异项，B 将不能以当前方式成立
```

这是一个反事实条件句，
其完全形式化仍需要进一步的模态逻辑或范畴论工具。

但偏序重写已经比自然语言版本前进了一步：

> **从“痕迹继承的序位”到“成立条件依赖的严格偏序”，时间性的过程词含量显著降低。**

### 6.2.4 偏序与物理时间的衔接：局部偏序

严格偏序 ≺ 是本体论层的逻辑结构。
但物理时间受引力和速度影响（6.4 节），
两者如何统一？

DM 的回应是引入**局部偏序**概念：

> **本体论层的全局偏序 ≺ 在物理实现层分化为多个局部偏序，每个局部偏序的度量速率受局部物理条件约束。**

更具体地说：

1. `全局偏序是本体论结构`
它规定成立条件依赖的先后序，与具体物理实现无关

2. `局部偏序是物理实现`
在特定引力场和速度条件下，≺ 关系的度量间隔（两个位形之间的可累计钟表时间）会改变

3. `相对论效应改变的是度量间隔，不是偏序关系本身`
A ≺ B 这一依赖关系不因引力场改变而消失，
改变的只是从 A 到 B 的物理钟表读数

因此：

```
本体论层：全局偏序 ≺（不依赖物理实现）
物理层：局部偏序 ≺_L（度量间隔受引力/速度约束）
钟表时间：对局部偏序的可累计度量
```

这一分层解释了为什么同一对事件在不同参照系中有不同的时间间隔，
但成立条件的依赖关系（偏序本身）不变。

> **偏序关系是实现无关的本体论结构，钟表读数是实现依赖的物理度量。相对论改变的是后者，不是前者。**

### 6.2.5 “不可忽略”的量化方向

6.2.3 中标记的“不可忽略差异如何量化”问题，
这里可以给出一个更积极的方向：

在物理层，“不可忽略”可被映射为**信息论中的条件互信息**：

> **A 对 B 的不可忽略差异 ≡ I(A; B | C) > 0，其中 C 是 B 的其他成立条件。**

也就是说：在给定 B 的其他所有条件之后，
A 仍然对 B 的成立提供不可替代的信息量。

这一映射不是最终形式化，
但它把“不可忽略”从反事实条件句转换为可计算量，
为后续严格化提供了具体方向。

这里还必须补一个方法论限制：

> **条件互信息只应被当作物理层或模型层的量化方向，而不是本体层的奠基工具。**

原因是：

```
A、B、C 的边界划分
本身依赖我们已经给出的层级、边界与变量选取
```

因此：

> **I(A;B|C)>0 不能用来“建立”偏序，它只能在既定层级划分下，帮助检验某一差异项是否对另一项保持不可替代的信息贡献。**

这使它的地位更清楚：

```
本体层：偏序给出成立条件依赖的概念框架
模型层：条件互信息提供“不可忽略”量化的候选工具
```

### 6.2.6 偏序框架的一个物理开放项：量子非局域关联

标签：以下段落为 `[解释性模型 + 开放边界｜结构描述]`。

局部偏序的写法较好地处理了相对论尺度上的时间度量问题，
但它还没有自动覆盖另一个物理层难点：

> **量子非局域关联中，相互关联的结果未必带有明确的局部 ≺ 方向。**

以纠缠关联为例，
两个局部测量结果可能表现出强相关，
但这种相关并不必然意味着：

```
局部事件 A ≺ 局部事件 B
或
局部事件 B ≺ 局部事件 A
```

DM 在这里的当前稳写法是：

1. `局部偏序`
适用于描述痕迹进入后续成立条件的继承链

2. `非局域关联`
这里可尝试理解为一个更高层全局约束态在两个局部位置上的共同投影

3. `因此非局域关联未必天然属于有方向的继承序`
它可能首先表现为共时约束，而不是局部先后

这意味着：

> **偏序框架较稳地覆盖经典与中尺度结构继承链，但量子非局域关联仍是物理接口层的开放项。**

## 六点五、为什么它仍可称为“一元论”

DM 所说的“一元”，
不是说世界里只许有一种材料、
一种对象或一种层级。

它真正要反对的是：

```
本体一套
心灵再一套
价值再一套
文明再一套
彼此互不相干
```

因此，DM 的“一元”更准确地说是：

> **差异、结构、心灵、价值与文明，不需要多套彼此断裂的本体起点；它们属于同一世界的连续层级展开。**

这里还要再补一条限制，
以避免把 DM 误读成更强的一元论：

> **DM 的“一元”首先是背景公设：它声明不同层级不应被当作彼此断裂的独立宇宙；但它并不因此自动证明高层可被低层无剩余还原。**

所以 DM 是一元论，
不是因为它取消层级差异，
而是因为它拒绝为不同层级另立彼此断裂的第一原理。

更压缩地说：

> **DM 当前能较稳捍卫的是“弱一元论”：统一的是本体起点与结构语法，不是所有层级内容都可被低层摘要吃掉。**

### 6.3 时间与钟的区别

时间性先于钟。

钟不是时间本身，
而是：

> **能够稳定地产生可重复、可计数事件的结构过程。**

钟的作用是把抽象的继承序，
转化为可累计、可比较的度量。

因此：

```
时间性
先于
钟表时间
```

### 6.4 为什么物理时间会受引力和速度影响

一旦进入物理层，
钟本身也是结构过程。

既然钟是结构，
它的周期就受局部条件约束。

所以：

* 引力更强，局部钟通常更慢
* 运动更快，局部钟通常更慢

这改变的不是“是否有时间性”，
而是：

> **局部物理过程如何累计时间。**

可以压成一句：

> **本体论推出时间的存在方式，物理学给出时间的局部度量方式。**

## 七、不可逆、持存与存在者

### 7.1 为什么痕迹继承会导向不可逆

顺序本身不等于不可逆。

继承本身也不自动等于绝对不可逆。

真正导向不可逆的是：

```
痕迹被保留
+ 痕迹被扩散
+ 痕迹改写后续条件
+ 抹除这些痕迹需要额外代价
```

因此：

> **时间的不可逆性，本质上是历史痕迹进入后续条件网络后的高代价不可清除性。**

### 7.2 为什么持存必须晚于时间

“续行”“维持”“跨越变化保持自身”这些说法，
都已预设前后。

因此：

```
不能先用持存推出时间，
而应先有时间性，
再谈跨时间的维持。
```

> **持存 = 结构跨时间变换维持其可识别性的能力。**

### 7.3 存在者的最小定义

到这一步，
我们才可更严格地定义存在者：

> **存在者 = 具边界并能跨时间维持其关联方式的差异结构。**

所以“存在”不应先被理解成静态在场，
而应理解成：

```
一个结构在时间展开中，
仍能保持关键关联不致完全瓦解。
```

可以压成一句：

> **先有时间，后有持存；先有持存，后有作为持存者的存在者。**

## 八、规律性、定律与公理

### 8.1 世界中的规律性

P00只先讲最深的一层：

> **规律性 = 结构关系在边界条件下反复形成、筛选与沉积后表现出的稳定重复约束。**

也就是说，世界先有的不是公式，
而是某些关系在特定条件下会稳定重复。

### 8.2 为什么规律不是天降命令

规律不能被当成外加法令。

规律之所以出现，
是因为：

```
有结构
有时间展开
有重复
有筛选
有沉积
```

如果没有这些，
就只有散乱事件，
而无规律性。

### 8.3 科学中的定律与数学中的公理

科学定律不是世界本身，
而是：

> **物理定律 = 人类对稳定重复约束所作的可检验、高压缩、可计算描述。**

数学公理也不等于世界原文。

> **数学公理 = 对高稳定形式关系所作的最小操作冻结。**

因此三者必须区分：

1. `规律性`
世界中的稳定约束

2. `定律`
科学对稳定约束的模型压缩

3. `公理`
形式系统内部的最小起点

可以压成一句：

> **世界不是由公式构成的；公式是世界中稳定关系被主体抽象、压缩和组织后的表达。**

因此，
P00 在物理层的角色也要说清：

> **P00 提供的是对可能方程的本体论约束与解释接口，而不是从差异地基直接推出具体物理方程。**

更准确地说，
DM 现阶段只给出三个元层要求：

1. `方程应能容纳结构依赖与局部实现差异`
不能把时间简单写成单一绝对容器

2. `方程应允许稳定重复与约束传递`
否则无法与规律和因果接口

3. `方程与本体地基之间现阶段是解释桥接关系`
不是 P00 已经完成的逐条演绎关系

### 8.3.1 一个最小桥接示例：不可逆性与热力学时间箭头

DM 当前不能从偏序重写直接推出热力学第二定律，
也不能从“痕迹继承”直接演绎出任何熵增方程。

它现阶段能做的，
只是给出一个较弱但更诚实的解释接口：

```text
本体层：
不可逆 = 痕迹进入后续成立条件 + 高代价不可清除

物理层：
热力学时间箭头 = 对宏观不可逆统计方向的方程化描述
```

因此更稳的说法是：

> **DM 现阶段只能为热力学不可逆提供本体解释接口，而不能从差异地基直接推出第二定律或其具体方程形式。**

这个示例的意义不在“推出公式”，
而在于说明 P00 与具体科学的最小桥接方式：

1. `先给本体解释接口`
说明某类方程在解释什么结构现象

2. `再看具体方程如何实现这种接口`
这是物理理论与实验工作，不是 P00 单独可完成之事

3. `若接口长期对不上`
则应缩限 DM 的解释范围，而不是硬把一切现成方程都说成早已被 DM 预言

换句话说：

> **P00 提供的是“这类方程可能在本体上对应什么”的解释桥，而不是“这条方程由 P00 推出”的演绎桥。**

## 九、因果：约束的可传递性

### 9.1 因果不是神秘绳子

DM不把因果理解为隐藏在事件背后的超验纽带。

因果更稳的表达是：

> **因果 = 某些结构差异通过可传递的约束链，稳定地改变后续结构成立条件。**

### 9.2 因果为什么可能成立

当一个结构位置的变化，
会系统性地改写另一个结构位置的可行条件，
而这种改写又能沿结构链条反复出现时，
我们就说两者之间具有因果性。

所以因果的核心不是“神秘推出”，而是：

```
约束传递
条件改写
稳定复现
```

### 9.3 因果与规律的关系

两者不能混同。

规律说明的是：

```
什么稳定重复
```

因果说明的是：

```
什么通过什么
改变了什么
```

可以再压一句：

> **规律说明何者常如此，因果说明何者经由何种传递链改写了后续条件。**

## 十、层级与涌现

### 10.1 为什么会出现层级

当下层结构之间形成更稳定、更密集、更可重复的耦合时，
就可能出现一个新的整体结构。

这个整体会表现出：

```
新的边界
新的内部差异分工
新的成立条件
对下层部分的反向约束
```

这就是层级出现的最小图景。

### 10.2 涌现的最小判据

> **涌现 = 若干下层结构耦合后形成新的整体边界、整体条件与反向约束。**

因此，涌现至少要求四项：

1. `下层结构可耦合`
不是纯并列堆积

2. `出现整体边界`
整体能与背景区分

3. `出现整体条件`
整体的成立不能简化为单个部分的孤立性质

4. `出现反向约束`
整体会反过来限制部分的可行状态

### 10.3 为什么涌现不是神秘跳跃

涌现中的“新”不是超自然添加，
而是：

```
组织方式变了
边界层级变了
整体约束地位变了
```

所以：

> **上层依赖下层，但不能被下层描述完全替代；这就是涌现的真实地位。**

## 十二、结语

DM并不把世界理解为一组现成实体，
而把它理解为：

```
差异关系形成结构，
结构通过边界成其为一，
边界结构在位形变换中留下痕迹，
痕迹继承生成时间，
时间中的历史积累导向不可逆，
跨时间的结构维持形成存在者，
稳定重复表现为规律，
约束传递表现为因果，
更高层结构则在耦合中不断涌现。
```

因此，最短定义可以写成：

> **存在不是先于时间的静态在场，而是在时间展开中得以持存的差异结构。**
