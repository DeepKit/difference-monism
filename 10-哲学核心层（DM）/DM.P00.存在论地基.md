## 二、起点：差异

### 2.1 差异不是对象之间的比较

差异不是：

* 两个已成形对象之间的不同
* 主体对对象的识别结果
* 预设 A 与 B 之后的外加关系

差异是：

```
在一切对象化之前，
非同一性自身给出的张力。
```

因此，DM的顺序不是：

```
对象
→ 属性
→ 差异
```

而是：

```
差异
→ 区分
→ 关联
→ 结构
→ 边界
```

### 2.2 为什么必须从差异开始

如果完全无差异，则：

```
无可区分
无可关联
无可描述
```

所以差异不是某类存在者的属性，
而是任何存在论讨论得以开始的最小前提。

可以压成一句：

> **差异不是对象之间的事后比较，而是对象、关系与边界得以成立的最小非同一条件。**

再压一句防守句：

> **DM不声称从绝对无中证明出差异，只声称：凡可被言说、被识别、被建模者，皆已处于差异之中。**

---

## 三、差异为什么能形成结构

### 3.1 差异不是结构的完成态

只有差异，还不足以构成结构。

散乱差异只能给出：

```
可分
但未成形
```

结构则要求：

```
差异之间不是单纯并列，
而是形成非任意的约束关联。
```

### 3.2 结构的最小定义

> **结构 = 差异项之间形成的非任意约束关联。**

这里的“约束关联”指：

```
一个差异位置的变化，
会系统性限制其他差异位置的可行状态。
```

### 3.3 结构成立的最小条件

我建议用最简三项表达：

1. `可区分`
必须有 A 不是 B

2. `可关联`
A 与 B 之间不能毫无关系，而要有耦合、依赖、限制或排列

3. `可约束`
并非一切组合都同等可行，而要存在可重复出现的限制关系

因此：

> **差异 + 关联 + 约束 = 结构的最小成立条件。**

### 3.4 为什么会有结构而不只有散乱

并非所有差异关系都能进入结构。

大量关系会：

```
快速耗散
失去关联
回到背景
```

少数关系则会：

```
进入稳定耦合
形成非任意限制
在同类条件下反复出现
```

于是我们看到的“结构”，
正是那些在筛选中进入稳定耦合的差异关系。

可以压成一句：

> **不是所有差异都会形成结构，但凡能够持续限制彼此可行状态的差异关系，必然已经具有结构性。**

## 四、边界：结构成其为一

### 4.1 为什么结构必须有边界

如果一个结构没有任何边界，
则：

```
它与背景无可区分
它与他物无可区分
它与自身的延展范围也无可区分
```

这意味着它不能“成其为一个”。

所以边界不是后来的认识论标记，
而是整体性成立的本体条件。

### 4.2 边界的定义

> **边界 = 结构内部关联强度与外部关联强度之间可区分的梯度。**

边界不是绝对墙，
而是：

```
内外关系权重的差异分布。
```

边界因此可以：

```
清晰
模糊
弹性
分层
重叠
```

### 4.3 边界的作用

边界至少完成三件事：

1. `使结构可识别`
它把“这一个”与“非这一个”区分开

2. `使结构可聚合`
它给出什么算该整体的一部分

3. `使结构可互动`
正因有内外区分，结构才可能与其他结构发生交换与耦合

因此：

> **边界不是外加线条，而是结构成其为一的成员性条件。**

## 五、位形、互斥与可排序变换

### 5.1 为什么边界之后会出现位形

一个有边界的结构，
不只是“在那里”。

它还会有：

```
内部排列
角色位置
耦合方式
可取状态
```

这些都可以统称为结构的位形。

> **位形 = 同一有边界结构在某一方面所取的具体排列或状态。**

### 5.2 为什么互斥位形不能同时并置

同一结构在同一方面，
不能同时完整地取两个互斥位形。

例如：

```
若此刻是位形 A，
就不能同时在同一方面完整地是位形 B。
```

这里给出的还不是时间，
而只是：

> **互斥位形不能同时并置。**

### 5.3 什么叫可排序变换

一旦同一结构可以从位形 A 变到位形 B，
而 A 与 B 又不能同时并置，
就会出现最小的顺序性：

```
A
然后
B
```

这里的“然后”仍可只是结构顺序，
还不是完整时间。

可以压成一句：

> **“之后”可以只是依赖顺序，不必然是时间；但位形一旦发生互斥变换，就会给出最小可排序性。**

## 六、时间：痕迹继承的序位

### 6.1 时间不是先验容器

DM不把时间看作先行存在的均匀容器。

时间不是：

* 外在钟表的抽象底板
* 所有存在者预先装进去的空框
* 单纯主观感受的投影

### 6.2 从顺序到时间

仅有可排序变换，还不足以推出时间。

真正接近时间的是：

```
较早位形的结果
进入较晚位形的成立条件
```

也就是说，
后态不是前态旁边多了一个点，
而是已经带着前态留下的痕迹继续展开。

> **时间性 = 结构变换痕迹进入后续成立条件的继承序。**

### 6.3 时间与钟的区别

时间性先于钟。

钟不是时间本身，
而是：

> **能够稳定地产生可重复、可计数事件的结构过程。**

钟的作用是把抽象的继承序，
转化为可累计、可比较的度量。

因此：

```
时间性
先于
钟表时间
```

### 6.4 为什么物理时间会受引力和速度影响

一旦进入物理层，
钟本身也是结构过程。

既然钟是结构，
它的周期就受局部条件约束。

所以：

* 引力更强，局部钟通常更慢
* 运动更快，局部钟通常更慢

这改变的不是“是否有时间性”，
而是：

> **局部物理过程如何累计时间。**

可以压成一句：

> **本体论推出时间的存在方式，物理学给出时间的局部度量方式。**

## 七、不可逆、持存与存在者

### 7.1 为什么痕迹继承会导向不可逆

顺序本身不等于不可逆。

继承本身也不自动等于绝对不可逆。

真正导向不可逆的是：

```
痕迹被保留
+ 痕迹被扩散
+ 痕迹改写后续条件
+ 抹除这些痕迹需要额外代价
```

因此：

> **时间的不可逆性，本质上是历史痕迹进入后续条件网络后的高代价不可清除性。**

### 7.2 为什么持存必须晚于时间

“续行”“维持”“跨越变化保持自身”这些说法，
都已预设前后。

因此：

```
不能先用持存推出时间，
而应先有时间性，
再谈跨时间的维持。
```

> **持存 = 结构跨时间变换维持其可识别性的能力。**

### 7.3 存在者的最小定义

到这一步，
我们才可更严格地定义存在者：

> **存在者 = 具边界并能跨时间维持其关联方式的差异结构。**

所以“存在”不应先被理解成静态在场，
而应理解成：

```
一个结构在时间展开中，
仍能保持关键关联不致完全瓦解。
```

可以压成一句：

> **先有时间，后有持存；先有持存，后有作为持存者的存在者。**

## 八、规律性、定律与公理

### 8.1 世界中的规律性

P00只先讲最深的一层：

> **规律性 = 结构关系在边界条件下反复形成、筛选与沉积后表现出的稳定重复约束。**

也就是说，世界先有的不是公式，
而是某些关系在特定条件下会稳定重复。

### 8.2 为什么规律不是天降命令

规律不能被当成外加法令。

规律之所以出现，
是因为：

```
有结构
有时间展开
有重复
有筛选
有沉积
```

如果没有这些，
就只有散乱事件，
而无规律性。

### 8.3 科学中的定律与数学中的公理

科学定律不是世界本身，
而是：

> **物理定律 = 人类对稳定重复约束所作的可检验、高压缩、可计算描述。**

数学公理也不等于世界原文。

> **数学公理 = 对高稳定形式关系所作的最小操作冻结。**

因此三者必须区分：

1. `规律性`
世界中的稳定约束

2. `定律`
科学对稳定约束的模型压缩

3. `公理`
形式系统内部的最小起点

可以压成一句：

> **世界不是由公式构成的；公式是世界中稳定关系被主体抽象、压缩和组织后的表达。**

## 九、因果：约束的可传递性

### 9.1 因果不是神秘绳子

DM不把因果理解为隐藏在事件背后的超验纽带。

因果更稳的表达是：

> **因果 = 某些结构差异通过可传递的约束链，稳定地改变后续结构成立条件。**

### 9.2 因果为什么可能成立

当一个结构位置的变化，
会系统性地改写另一个结构位置的可行条件，
而这种改写又能沿结构链条反复出现时，
我们就说两者之间具有因果性。

所以因果的核心不是“神秘推出”，而是：

```
约束传递
条件改写
稳定复现
```

### 9.3 因果与规律的关系

两者不能混同。

规律说明的是：

```
什么稳定重复
```

因果说明的是：

```
什么通过什么
改变了什么
```

可以再压一句：

> **规律说明何者常如此，因果说明何者经由何种传递链改写了后续条件。**

## 十、层级与涌现

### 10.1 为什么会出现层级

当下层结构之间形成更稳定、更密集、更可重复的耦合时，
就可能出现一个新的整体结构。

这个整体会表现出：

```
新的边界
新的内部差异分工
新的成立条件
对下层部分的反向约束
```

这就是层级出现的最小图景。

### 10.2 涌现的最小判据

> **涌现 = 若干下层结构耦合后形成新的整体边界、整体条件与反向约束。**

因此，涌现至少要求四项：

1. `下层结构可耦合`
不是纯并列堆积

2. `出现整体边界`
整体能与背景区分

3. `出现整体条件`
整体的成立不能简化为单个部分的孤立性质

4. `出现反向约束`
整体会反过来限制部分的可行状态

### 10.3 为什么涌现不是神秘跳跃

涌现中的“新”不是超自然添加，
而是：

```
组织方式变了
边界层级变了
整体约束地位变了
```

所以：

> **上层依赖下层，但不能被下层描述完全替代；这就是涌现的真实地位。**

## 十二、结语

DM并不把世界理解为一组现成实体，
而把它理解为：

```
差异关系形成结构，
结构通过边界成其为一，
边界结构在位形变换中留下痕迹，
痕迹继承生成时间，
时间中的历史积累导向不可逆，
跨时间的结构维持形成存在者，
稳定重复表现为规律，
约束传递表现为因果，
更高层结构则在耦合中不断涌现。
```

因此，最短定义可以写成：

> **存在不是先于时间的静态在场，而是在时间展开中得以持存的差异结构。**
