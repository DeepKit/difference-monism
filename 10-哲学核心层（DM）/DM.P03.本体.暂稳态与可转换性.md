# 差异一元论 · 本体
## Difference Monism: Metastability and Convertibility (Sealed v1.0)
### DM.P03 · 暂稳态、刚性稳定与可转换性

> **稳定使结构得以成形，可转换性使结构更能跨时间持存。**

---

## 一、P03 的位置

P00先给出时间、持存与存在者的基本顺序。
P01说明世界应理解为稳定差异结构网络。
P02说明层级与涌现如何可能。

P03只继续追问一个问题：

> **什么样的稳定更适合长期持存：刚性稳定，还是可转换的暂稳态。**

因此，P03仍处于本体层。
这里不直接进入价值、伦理、制度与文明评判。

---

## 二、为什么“稳定”还不够

如果把存在简单理解为“越稳越好”，
就会出现一个问题：

```
环境条件会变
结构耦合会变
扰动方式会变
```

在这种情况下，
一种完全刚性的稳定，
反而可能最先断裂。

所以必须区分两种稳定：

1. `刚性稳定`
2. `暂稳态`

---

## 三、刚性稳定与暂稳态

### 3.1 刚性稳定

刚性稳定指：

```
内部排列高度固定
变化空间极小
面对新条件时几乎无改道能力
```

它的优点：

* 在既定条件下表现整齐
* 局部波动较小
* 短期可预测性强

它的缺点：

* 一旦条件改变，容易整体脆断
* 缺少缓冲区与重组空间
* 只能守住旧条件，难以进入新条件

### 3.2 暂稳态

暂稳态指：

> **在当前条件下能够持存，并在条件变化时仍保留转入新稳态可能性的结构。**

它的特点是：

```
不是彻底松散
也不是彻底僵硬

而是在持存与改道之间保持动态平衡。
```

因此，暂稳态不是“不稳定”，
而是更高阶的稳定。

---

## 四、什么叫可转换性

### 4.1 最小定义

> **可转换性 = 一个结构在条件变化时，以较低断裂代价进入新稳态的能力。**

这里强调两点：

1. `不是随意变形`
如果完全失去自身，不能算转换，只能算瓦解

2. `不是绝对保形`
如果完全不能变化，也不能算高质量持存

所以转换讲的是：

```
在保持延续性的前提下完成改道。
```

### 4.2 可转换性的两个维度

可以先用概念方式表达：

1. `路径成本`
从当前稳态转入新稳态需要付出的结构重组代价

2. `可达范围`
结构能够转入的新稳态数量与种类

因此，高可转换性结构通常表现为：

```
改道成本较低
可达路径较多
```

---

## 五、为什么暂稳态比刚性稳定更适合长期持存

### 5.1 条件网络总在变化

任何结构都不处于绝对静止背景中。

只要存在：

```
扰动
耦合变化
资源波动
关系重排
```

就不存在对所有时刻都同样适用的单一稳态。

### 5.2 长期持存看的是续行，不是局部整齐

刚性稳定可能在局部时段非常整齐，
但一旦环境越出其适配区间，
就可能整体断裂。

暂稳态则可能在局部看起来不那么“完美”，
却因保留改道空间而有更高长期持存概率。

因此：

> **长期持存衡量的不是局部最整齐，而是跨条件变化时的续行能力。**

### 5.3 最短结论

> **刚性稳定保证局部整齐，暂稳态保证跨条件续行。**

---

## 六、可转换性与结构优化

P03在本体层可以给出一个非常关键的判断：

> **可转换性是结构优化的最高属性之一。**

原因不是价值偏好，
而是本体条件：

```
若结构完全不能转换，
则一旦条件越界，它就归零。

若结构能在一定范围内改道，
它就能把单次冲击变成路径重组，
而不是存在终止。
```

所以：

> **稳定保证存在，可转换性保证续存。**

这句话在本体层就成立，
不必等到政治或文明层。

---

## 七、路径依赖与固化

### 7.1 成功结构为什么会固化

任何已经成功持存的结构，
都容易出现路径依赖。

原因很简单：

```
旧结构曾经有效
于是结构会优先重复旧解
```

但这会带来固化倾向：

* 旧解被误当成永恒解
* 新扰动被强行套入旧框架
* 转换能力被成功经验逐步削弱

### 7.2 固化为什么危险

固化本身不是立刻消亡，
但它会让结构：

```
越来越难改道
越来越依赖旧条件
越来越脆而不知脆
```

所以在本体层就可说：

> **结构一旦成功，就会天然带有稳定化与固化倾向。**

因此，更高的稳定不是更硬，
而是更能在连续性中自我调整。

---

## 八、P03 的边界

P03在此处只完成：

1. 区分刚性稳定与暂稳态
2. 定义可转换性
3. 说明为何长期持存偏向高可转换结构
4. 指出成功结构会自然滑向路径依赖与固化

P03不直接推出：

* 价值
* 道德
* 自由
* 制度设计
* 文明韧性

这些都需要更高层前提，
尤其需要生命、主体和社会结构的出现。

---

## 九、P03 的最短结论

P03可压成五句：

`稳定还不够，因为环境与条件网络会变。`
`刚性稳定在局部整齐上占优，暂稳态在长期持存上占优。`
`可转换性是结构在变化中进入新稳态而不整体断裂的能力。`
`长期持存看的是跨条件续行，而不是单一时段的完美固定。`
`结构一旦成功就会趋于路径依赖，因此真正高级的稳定必须保留改道能力。`

---

## 十、结语

P03的核心判断只有一句：

> **在持续扰动的世界中，最可持存的不是最僵硬的结构，而是最能在保持连续性的同时完成自我转换的结构。**
