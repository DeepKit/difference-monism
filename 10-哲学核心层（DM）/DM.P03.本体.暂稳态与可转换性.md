## 二、为什么“稳定”还不够

如果把存在简单理解为“越稳越好”，
就会出现一个问题：

```
环境条件会变
结构耦合会变
扰动方式会变
```

在这种情况下，
一种完全刚性的稳定，
反而可能最先断裂。

所以必须区分两种稳定：

1. `刚性稳定`
2. `暂稳态`

---

## 三、刚性稳定与暂稳态

### 3.1 刚性稳定

刚性稳定指：

```
内部排列高度固定
变化空间极小
面对新条件时几乎无改道能力
```

它的优点：

* 在既定条件下表现整齐
* 局部波动较小
* 短期可预测性强

它的缺点：

* 一旦条件改变，容易整体脆断
* 缺少缓冲区与重组空间
* 只能守住旧条件，难以进入新条件

### 3.2 暂稳态

暂稳态指：

> **在当前条件下能够持存，并在条件变化时仍保留转入新稳态可能性的结构。**

它的特点是：

```
不是彻底松散
也不是彻底僵硬

而是在持存与改道之间保持动态平衡。
```

因此，暂稳态不是“不稳定”，
而是更高阶的稳定。

---

## 四、什么叫可转换性

### 4.1 最小定义

> **可转换性 = 一个结构在条件变化时，以较低断裂代价进入新稳态的能力。**

这里强调两点：

1. `不是随意变形`
如果完全失去自身，不能算转换，只能算瓦解

2. `不是绝对保形`
如果完全不能变化，也不能算高质量持存

所以转换讲的是：

```
在保持延续性的前提下完成改道。
```

### 4.2 可转换性的两个维度

可以先用概念方式表达：

1. `路径成本`
从当前稳态转入新稳态需要付出的结构重组代价

2. `可达范围`
结构能够转入的新稳态数量与种类

因此，高可转换性结构通常表现为：

```
改道成本较低
可达路径较多
```

## 五、为什么暂稳态比刚性稳定更适合长期持存

### 5.1 条件网络总在变化

任何结构都不处于绝对静止背景中。

只要存在：

```
扰动
耦合变化
资源波动
关系重排
```

就不存在对所有时刻都同样适用的单一稳态。

### 5.2 长期持存看的是续行，不是局部整齐

刚性稳定可能在局部时段非常整齐，
但一旦环境越出其适配区间，
就可能整体断裂。

暂稳态则可能在局部看起来不那么“完美”，
却因保留改道空间而有更高长期持存概率。

因此：

> **长期持存衡量的不是局部最整齐，而是跨条件变化时的续行能力。**

但这里还要补一条分层纪律，
否则本体描述会不知不觉滑入价值判断：

1. `纯本体描述`
在持续变化的条件网络中，
高可转换结构在统计上更倾向于继续存在

2. `条件性评价`
若我们选择以长期持存作为评估基准，
则暂稳态优于刚性稳定

因此 P03 的立场是：

> **P03 可以描述哪类结构更可能长期续行；至于“长期续行是否值得优先追求”，则不是 P03 单独能在本体层完成的判断。**

### 5.3 最短结论

标签：该句为 `[解释性模型｜结构描述]`。

> **刚性稳定保证局部整齐，暂稳态保证跨条件续行。**

## 六、可转换性与结构续存能力

P03在本体层可以给出一个非常关键的判断：

> **可转换性是结构长期续存能力的关键属性之一。**

这句话若保持在本体层，
就必须读成中性结构判断，
而不是价值排序。

原因是本体条件：

```
若结构完全不能转换，
则一旦条件越界，它就归零。

若结构能在一定范围内改道，
它就能把单次冲击变成路径重组，
而不是存在终止。
```

所以：

> **稳定保证存在，可转换性保证续存。**

这句话在本体层就成立，
不必等到政治或文明层。

---

## 七、路径依赖与固化

### 7.1 成功结构为什么会固化

任何已经成功持存的结构，
都容易出现路径依赖。

原因很简单：

```
旧结构曾经有效
于是结构会优先重复旧解
```

但这会带来固化倾向：

* 旧解被误当成永恒解
* 新扰动被强行套入旧框架
* 转换能力被成功经验逐步削弱

### 7.2 固化为什么危险

固化本身不是立刻消亡，
但它会让结构：

```
越来越难改道
越来越依赖旧条件
越来越脆而不知脆
```

所以在本体层就可说：

> **结构一旦成功，就会天然带有稳定化与固化倾向。**

因此，更高的稳定不是更硬，
而是更能在连续性中自我调整。

## 八、暂稳态的分层参数表

暂稳态是 DM 的核心跨层概念。
为防止它退化为弹性过大的万能隐喻，
必须对每个层级分别列出其实例化参数。

每次暂稳态跨层使用时，
都必须通过 P02 的涌现四判据（可耦合、可分化、可持存、可反约束），
否则只是类比而非真正的跨层结构概念。

### 8.1 本体层

| 参数 | 内容 |
|------|------|
| 成立条件 | 差异关系进入非任意约束，形成可重复维持的关联网络 |
| 典型时间尺度 | 从亚原子结构（飞秒级）到地质结构（亿年级） |
| 失败模式 | 约束条件越界，结构整体瓦解或不可逆衰变 |
| 转换路径 | 相变、对称性破缺、结构重组进入新稳态 |

### 8.2 生命层

| 参数 | 内容 |
|------|------|
| 成立条件 | 边界维持 + 代谢循环 + 复制继承 + 谱系约束 |
| 典型时间尺度 | 单细胞（分钟至小时级分裂）到个体（年至百年级寿命） |
| 失败模式 | 代谢崩溃、边界破裂、复制失败——即死亡 |
| 转换路径 | 表型可塑性、表观遗传调整、免疫适应、行为改道 |

### 8.3 意识层

| 参数 | 内容 |
|------|------|
| 成立条件 | 具边界的自指结构 + 内部可区分状态空间 + 时间继承 + 主体级短路径 |
| 典型时间尺度 | 毫秒级（惊吓跃迁）到秒至分钟级（情绪流变态） |
| 失败模式 | 感觉适应（暂稳态被吸收）、注意脱落、意识中断（昏迷、麻醉） |
| 转换路径 | 注意转移、情绪切换、新刺激引发的整体状态跃迁 |

### 8.4 文明层

| 参数 | 内容 |
|------|------|
| 成立条件 | 多主体共在 + 制度沉积 + 代际传承 + 认同整合 |
| 典型时间尺度 | 十年至百年级（制度周期）到千年级（文明周期） |
| 失败模式 | 纠错能力衰减、代际断裂、认同撕裂、制度僵死——即文明退化或崩溃 |
| 转换路径 | 制度改革、权力更替、技术范式转换、文化重组 |

因此：

> **暂稳态不是一个泛用隐喻，而是一个受涌现判据约束的跨层结构模板。它在每个层级上必须各自具有不同的成立条件、时间尺度、失败模式与转换路径。**

### 8.5 定量化指引：从定性描述到可测量指标

分层参数表给出了定性描述，
但为防止暂稳态退化为万能解释器，
还需要为每层指出可能的定量化方向。

**本体层**

```
稳定度指标 ← 结构在扰动下恢复原位形的速率
可转换性指标 ← 从当前稳态到可达稳态的路径数 / 平均转换代价
脆性指标 ← 使结构不可逆瓦解的最小扰动幅度
```

与物理学的接口：势能景观中的局部极小深度、鞍点高度、盆地宽度。

**生命层**

```
稳定度指标 ← 代谢恢复速率、损伤修复速率
可转换性指标 ← 表型可塑性范围、免疫适应谱宽
脆性指标 ← 致死最小扰动（LD50 类指标）
```

与生物学的接口：适应度景观、表型空间、生态位宽度。

**意识层**

```
稳定度指标 ← 注意维持时长、情绪稳定性量表
可转换性指标 ← 注意切换速率、情绪灵活性
脆性指标 ← 导致意识中断的最小扰动（麻醉剂量、创伤阈值）
```

与神经科学的接口：EEG 稳定性、神经元发放模式的吸引子分析。

**文明层**

```
稳定度指标 ← 制度存续年限、政权更替后制度保留率
可转换性指标 ← 制度改革频率 × 改革成功率、危机恢复速度
脆性指标 ← 导致文明崩溃的最小冲击规模（人口损失比例、资源断裂阈值）
```

与社会科学的接口：制度经济学的制度韧性指标、文明史的崩溃模型。

这些定量化方向仍是指引而非完成：

> **暂稳态的每层实例化，最终应当能够与该层的科学度量建立可比较的数值接口。在此之前，分层参数表至少保证了概念不会被无约束地跨层滥用。**

这里还要再补一句，
以避免“稳定度”“脆性”“可转换性”被误读成单一跨层标尺：

> **这些指标在跨层上共享的是结构角色，而不是共同量纲。物理层的势阱深度、生命层的修复速率、文明层的制度存续年限，可以同名，但不能直接按同一数轴通约。**

更稳的理解是：

```
可跨层比较的是角色位置
不可直接跨层比较的是数值单位
```

### 8.6 跨层统一性的研究承诺：动力系统吸引子模板

分层参数表展示了暂稳态在各层的差异，
但 DM 作为一元论，也必须给出一个可研究的**跨层统一模板**。

统一性不等于同一性。
DM 主张的是**结构同构**——
各层暂稳态共享同一种数学结构，但在不同层级有不同的实例化：

> **暂稳态在每个层级上都可以被描述为动力系统中的局部吸引子：一个在状态空间中占据有限区域、在扰动下倾向回归但在大扰动下可以跃迁到新吸引子的区域。**

具体映射：

| 暂稳态概念 | 动力系统对应 |
|-----------|------------|
| 暂稳态本身 | 状态空间中的局部吸引子（吸引盆地） |
| 稳定度 | 吸引盆地的深度（恢复力） |
| 可转换性 | 从当前吸引子到邻近吸引子的鞍点高度（转换阈值） |
| 脆性 | 使系统离开当前盆地的最小扰动幅度 |
| 路径依赖 | 吸引盆地对初始条件的敏感性 |
| 相变/跃迁 | 从一个吸引子跳到另一个吸引子 |

更准确地说：

> **这一同构在物理层、生命层与部分意识层，已较接近数学上可操作的对应；在文明层，现阶段仍主要是一种研究纲领式的形式承诺。**

因此下列映射的地位并不完全相同：

- 物理层：势能景观中的局部极小
- 生命层：适应度景观中的适应度峰
- 意识层：神经状态空间中的吸引子态
- 文明层：制度-社会状态空间中的稳定均衡

因此暂稳态的跨层统一性可以更稳地表述为：

> **暂稳态不是泛用隐喻，也不仅是家族相似。它是一个以动力系统局部吸引子为核心的跨层模板；对物理、生命与神经层，可望进入较严格的数学实例化；对文明层，当前仍是等待状态空间与动力学方程被更清楚定义的形式研究方向。**

但这一同构也有边界：

各层的状态空间维度、动力学方程和噪声结构完全不同。
同构保证的是**结构形式的一致性**，
不是**具体内容的可互换性**。

因此物理层的暂稳态不能直接“套用”到文明层，
每次跨层使用仍必须通过 P02 的涌现四判据。

## 九、P03 的边界

P03在此处只完成：

1. 区分刚性稳定与暂稳态
2. 定义可转换性
3. 说明为何长期持存偏向高可转换结构
4. 指出成功结构会自然滑向路径依赖与固化
5. 给出暂稳态的分层参数表与跨层使用纪律

P03不直接推出：

* 价值
* 道德
* 自由
* 制度设计
* 文明韧性

这些都需要更高层前提，
尤其需要生命、主体和社会结构的出现。

---

## 九、P03 的最短结论

P03可压成五句：

标签：以下五句均为 `[解释性模型｜结构描述]`。

`稳定还不够，因为环境与条件网络会变。`
`刚性稳定在局部整齐上占优，暂稳态在长期持存上占优。`
`可转换性是结构在变化中进入新稳态而不整体断裂的能力。`
`长期持存看的是跨条件续行，而不是单一时段的完美固定。`
`结构一旦成功就会趋于路径依赖，因此真正高级的稳定必须保留改道能力。`

## 十、结语

P03的核心判断只有一句：

> **在持续扰动的世界中，最可持存的不是最僵硬的结构，而是最能在保持连续性的同时完成自我转换的结构。**
