## 文件命名规则

所有内容文件统一采用以下格式：

```
[RM].P01.[分类名].文件名.md
 └─┘  └─┘   └──┬───┘     └──┬───┘
   │    │       │            └── 具体文件名（中文）
   │    │       └── 分类名（本体/心灵/认识/伦理/政治/对话）
   │    └── 编号（P01-P99 / ECET / TAT / ODD / PAPERS）
   └── 体系前缀（固定为RM）
```

**编号规则：**
- `RM.P00-P19`：哲学核心层（10-哲学核心层/）
- `RM.ECET`：文明演化层（30-文明演化层/）
- `RM.TAT`：责任架构层（40-责任架构层/）
- `RM.ODD`：工程方法层（50-工程方法层/）
- `RM.PAPERS`：学术投稿（80-学术投稿/）

**唯一真相源：** 00-99 编号目录体系，根目录仅保留 README.md 作为入口。
