# 差异一元论 / Difference Monism

> 一个以`差异`为最小起点的哲学本体论框架。
> 它讨论的核心问题不是“世界由哪些现成实体构成”，而是“存在如何可能”。

---

## 当前定位

本仓库当前以`哲学版本`为主，不再把自己包装成：

- 可直接证伪的自然科学理论
- 商业方法论入口
- 产品化/变现型项目主页

当前主轴是：

1. `本体论地基`
差异、结构、边界、持存、时间、规律、因果、涌现

2. `层级展开`
从结构到生命、主体、认识、责任与文明

3. `与主要传统对话`
同结构实在论、过程哲学、现象学、佛学缘起论等展开比较

---

## 最短主链

```
差异
→ 结构
→ 边界
→ 持存
→ 时间
→ 规律
→ 因果
→ 涌现
```

一句话压缩：

> **存在不是静止之物，而是差异结构在约束中的暂时续行。**
