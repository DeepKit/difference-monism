# DM 代表作对外入口

## 冠军命题

DM 当前最强的公共主张不是“万物统一”式总宣言，而是两条更窄、也更可裁决的判断：

1. 在意识问题上，硬问题首先不是两种实体如何桥接，而是内部性如何形成。
2. 在危机治理问题上，危机压缩若不可复归，就会反向侵蚀它声称要保卫的共同体承诺结构。

这两条命题分别由两篇代表作承担，不要求读者先接受整套体系。

## 两篇代表作

### 1. 意识问题代表作

- 文件：`DM-P04.硬问题不是两种实体的桥接而是内部性的形成.投稿体草案`
- 当前状态：`送审清稿`
- 核心问题：硬问题应否先被改写为内部性的形成问题
- 最小新增性：不发明新意识实体，而把分歧压到组织阈值、边界案例与失败线
- 主要对手：`GNW / IIT / HOT / 强版 illusionism`
- 失败条件：若纯第三人称路线能跨 `报告 / 记忆 / 后续选择` 与多组边界案例稳定吃掉主体级差额，则本文必须弱化或撤回

### 2. 法政问题代表作

- 文件：`DM-P09.危机压缩必须可复归.投稿体草案`
- 当前状态：`送审清稿`
- 核心问题：什么样的危机压缩仍可被称为可辩护的长期治理
- 最小新增性：不做制度终局排名，而只建立 `纠错能力 / 纠错载体 / 退出兑现度 / 深度伤害` 四项硬字段
- 主要样本：`魏玛 / 日裔拘禁 / USA PATRIOT Act / 纽伦堡`
- 失败条件：若四项硬字段无法稳定区分失控压缩与可复归程序，且跨案可比性持续被击穿，则本文结论必须下调

## 三条边界

1. 这套理论当前不靠“体系更大更全”取胜，而靠少数代表作的外部裁决力。
2. 这套理论当前不提供终局排序，只提供更窄的判断协议、失败线与禁止结论。
3. 这套理论当前仍需要真人批评、逐条核引与正式送审反馈来决定是否还能继续提级。

## 当前最稳的读法

- 若你关心意识问题，请先读 `P04`。
- 若你关心危机治理与制度复归，请先读 `P09`。
- 若你只想知道这套理论什么时候会输，请同时看两篇代表作中的 `失败线 / 反对路径 / 缩限` 小节。
