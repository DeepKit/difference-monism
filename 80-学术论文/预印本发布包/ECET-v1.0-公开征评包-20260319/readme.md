# ECET v1.0 公开征评包

> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）
>
> **公开边界**：`A00 + A01 + B03 + B05 + B07 + B09 + C11 + C13 + C15 + C17`

本目录为 ECET 的首轮外发包，适用于预印本平台占位、公开征评与公开比较。

本包的**主公开边界**仍是 `A/B/C 主文档组`。此外可附带两份**可选单篇论文稿**，用于平台更偏好单文稿上传的场景：

1. `ECET_论文_中文版_v1.md`
2. `ECET_Paper_v1.md`

这两份稿件属于对 `A/B/C` 主文档组的论文化重写，不应反客为主替代整个公开主包的边界说明。

若平台只允许一个主文，建议上传：

1. `ECET.A01_入口.md`

若平台支持主文 + 附件，建议顺序：

1. `ECET.预印本发布说明.v1.0.md`
2. `ECET.A01_入口.md`
3. `ECET.A00_理论基础.md`
4. `ECET.B03_三大公理.md`
5. `ECET.B09_可检验假设.md`
6. `ECET.C13_影子主权与治理原则.md`
7. 其余 `B05 / B07 / C11 / C15 / C17`
8. `ECET_论文_中文版_v1.md` 或 `ECET_Paper_v1.md`（可选）
9. `sim_results/` 与仿真脚本

对外口径固定为：

> **ECET 是建立在 DM 与 ASTO 之上的文明演化与治理判断层。当前版本适合作为公开征评预印本发布，不应表述为已经完成大规模经验验证的闭合终稿。**

补充纪律：

- `C17` 只按“概念验证级仿真附录”外发
- 单篇论文稿也必须维持“公开征评预印本”口径
- 不把历史归档、草稿或未纳入主包的材料一并上传
