# 预印本发布包

> 定位：本目录用于手工上传预印本平台或做公开征评占位。
> 口径：这里只处理公开法源线，不处理单篇投稿线。

---

## 一、当前职责

本目录只负责：

- 理论全本预印本包
- 公开征评包
- 法源链展示包

若目标是单篇论文外投、匿名送审或投稿材料组织，请改看：

- `../投稿申请包/README.md`

---

## 二、当前包含的发布包

当前稳定的预印本发布线共 `5` 条：

1. `DM-v1.1-统一送审预印本包`
2. `ASTO-首轮公开预印本包-20260319`
3. `ECET-v1.0-公开征评包-20260319`
4. `TAT-v1.0-公开征评包-20260319`
5. `ODD-v1.0-公开征评包-20260319`

当前公开主链按：

`DM -> ASTO -> ECET -> TAT -> ODD`

理解。

---

## 三、最小上传建议

- `DM`：优先上传 `DM.统一送审包.v1.1.md`
- `ASTO`：若平台只允许一个主文，优先上传 `ASTO.U00.属集变迁存在论简明核心框架.md`
- `ECET`：若平台只允许一个主文，优先上传 `ECET.A01_入口.md`
- `TAT`：若平台只允许一个主文，优先上传 `责任架构理论宣言与白皮书.md`
- `ODD`：若平台只允许一个主文，优先上传 `论文/论文-学术版.md`

---

## 四、当前边界

1. 不要把 `better.md`、缓存稿、归档稿、历史 PDF 当成首轮公开主包。
2. 不要把 `RM / Resonance Monism / 共振一元论 / 共扰一元论` 阶段材料误当成当前默认上传对象。
3. 不要把 `COP / Harness / WSM / BetterCiv` 当前文档混进独立预印本主链；它们现在都不是独立预印本线。
4. `ECET / TAT / ODD` 的当前公开包，都只应按“公开征评 / 限定边界”口径外发，不应包装成终局验证完成。

---

## 五、最短结论

`预印本发布包/` 现在只承担一件事：

**为 `DM -> ASTO -> ECET -> TAT -> ODD` 这条公开法源链提供稳定的对外上传入口。**
