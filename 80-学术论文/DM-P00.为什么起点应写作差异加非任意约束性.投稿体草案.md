## 零点五、本文贡献

本文当前最稳的三项贡献可以压成三句：

1. 它把起点竞争从“谁更本源”的修辞竞争，改写为“谁偷带更少、谁覆盖更广”的前提记账竞争。
2. 它把 `差异 + 非任意约束性` 从容易被神圣化的首词，改写为一个可比较、可受损、可被替代的较经济起点组合。
3. 它把这场竞争的最低裁决标准与失败线显式写进正文，避免起点问题继续停留在口号层。

## 一、引言

形而上学写作最容易犯的错误之一，是把自己的起点当成“天然最薄”，却不肯认真检查它究竟已经偷带了什么。差异一元论过去也有过这种风险：若把“差异”直接写成绝对零预设，再从中一路推出结构、边界、时间、规律与涌现，就很容易把本该显式记账的复数条件、关联可能性与约束条件偷渡进同一个词里。反过来，若为了避免这一风险而直接改用“关系”“过程”“事件”“信息”或“约束”作为起点，又常常会把对象化、时点化、编码规则或状态空间提前写回地基。

因此，本文处理的不是“哪个词最漂亮”，而是一个更窄、也更难回避的问题：在起点竞争中，究竟哪一种写法偷带更少，却仍能覆盖更长的主链。本文主张，`差异 + 非任意约束性` 当前仍值得被保留，不是因为它已被证明为逻辑上唯一的终极起点，而是因为它在现阶段较少偷带现成对象、现成时间与现成编码框架，同时又能较稳定地进入 `结构 / 边界 / 时间 / 层级` 这条生成链。

这意味着本文不再追求“无条件赢下起点之争”。更准确地说，本文只捍卫一个较弱但更诚实的命题：`差异 + 非任意约束性` 是当前较经济的起点组合，而不是不可竞争的神圣首词。只要出现一个更薄、却能完成同等主链覆盖的替代写法，DM 就应主动改写自己的地基声明。

## 二、文献位置

本文并不把自己写成一篇覆盖全部形而上学传统的总论，而只与四类最相关路线发生对话。第一，它与结构实在论共享“关系优先于独立实体”的底线，但不同于只停留在“结构是什么”的讨论，本文更关心结构如何进入边界、时间与层级生成。<sup>1</sup><sup>2</sup> 第二，它与过程哲学共享反静态实体论的倾向，但拒绝把“过程”直接当成无需再解释的首词，因为这会把时间流动与先后方向提前写回起点。<sup>3</sup> 第三，它与信息本体论或信息优先路线共享对可区分状态与编码问题的敏感性，但不接受把编码规则、载体与测度框架直接当成地基。<sup>4</sup> 第四，它与一般的非还原自然化计划共享“拒绝额外神秘实体”的底线，却不愿把本体地基直接写成“物理”或其他较晚层的对象化语言。<sup>5</sup>

因此，本文的切口并不是证明其他起点一律不可能，而是把起点竞争压到最低可裁决形式：哪些写法更薄，哪些写法虽然强大却已经提前加载了它后面本应推出的结构。

## 三、核心命题：起点之争首先是前提记账之争

本文的核心命题可以压成两句。第一，起点竞争不应再被写成“谁的词最本源”，而应被写成“谁偷带更少、却仍覆盖更广”。第二，只要讨论对象、关系、边界、时间与层级如何成立，`差异 + 非任意约束性` 目前仍是较经济的起点组合。

这两个命题都在主动降格。本文不再宣称“纯差异”已经足以独立长出一切，也不再把“非任意约束性”伪装成完全由更深本体推出的结论。更准确地说，本文承认 DM 的地基层至少包含两个显式记账项：一是最小可分性，二是并非一切组合都同等可行的最低约束承诺。

这样一来，起点之争就不再是修辞上的抢占制高点，而变成了一个较清楚的比较问题：若某一路线声称自己比 DM 更薄，它就必须说明自己是否已经偷带了对象、时序、事件单位、编码规则、状态空间或其他更厚结构；若它声称自己覆盖更广，它也必须说明自己能否不借更重前提而完成相同主链。

## 四、表 1：起点竞争的最小比较表

| 候选起点 | 容易偷带什么 | 当前优势 | 当前弱点 |
| --- | --- | --- | --- |
| `差异 + 非任意约束性` | 仍需补记复数条件、关联可能性与最低约束承诺 | 较少偷带现成对象、时点化切片与编码框架 | 不是绝对零预设，且必须承认约束来源尚未闭合 |
| `关系` | 多个项、项间连法、关系位次 | 直接进入结构语言 | 复数架构与连法已被默认 |
| `过程` | 时间、流动、先后方向 | 便于表达生成与连续性 | 把时间性提前写回起点 |
| `事件` | 切片单位、离散化规则、时点化方式 | 便于进入历史与发生叙述 | 把位形单位先行冻结 |
| `信息` | 编码规则、可区分状态、载体、测度框架 | 便于进入认知与系统论语言 | 把结构化差异与可读写方式提前带入 |
| `约束` | 被约束的项、状态空间、可行域 | 便于进入规律、筛选与稳定性讨论 | 把较后层的结构语言提前写成地基 |

这张表的功能不是宣布 DM 已经赢下起点竞争，而是把各候选起点的前提成本摊开。只有在这一步做完之后，“谁更薄”才不是修辞上的自我褒奖。

更具体地说，`关系` 之所以比 DM 更厚，不在于它错误，而在于它太快默认了“已有多个项，并且这些项之间已经存在可描述连法”。`过程` 的问题也不在于它不能进入世界，而在于它把流动、先后与展开方向提前写回地基。`事件` 的难点，则在于它常常要先给出切片单位与离散化方式，才能开始说“发生了什么”。`信息` 之所以对后续理论很有吸引力，是因为它便于进入编码、传播与认知语言；但作为地基，它又往往默认了可区分状态、载体与测度框架。至于 `约束`，它最接近 DM 的第二项，却也最容易反向预设“被约束的项”和“可行状态空间”已经在场。

因此，本文保留 `差异 + 非任意约束性` 的理由，不是它比这些候选更有气势，而是它允许把对象、时间、事件单位与编码框架再往后放一步。

## 五、为什么 DM 暂时保留“差异 + 非任意约束性”

DM 当前保留这一起点组合，不是因为它比其他候选更宏大，而是因为它的记账顺序较稳。第一步，`差异` 给出最小可分性，使“完全同一不能维持”成为起点层的最低断裂。第二步，差异一旦不是孤立单数，就必须允许复数化关联可能性进入。第三步，若要让这些关联进一步沉积为结构、重复与规律，就必须承认并非任意连结都同等可行，也即最低限度的 `非任意约束性`。

这个顺序的优势，在于它尽量把更厚的词后移。本文无需一开始就承认现成对象、现成时序、现成事件切片或现成信息编码，只需要先承认：世界至少有最小可分性，而且并非一切组合都同样成立。正是因为这种后移策略，`差异 + 非任意约束性` 当前比“关系”“过程”“事件”“信息”更少偷带较晚层的构造。

但这种保留是有代价的。本文必须同时承认三点：其一，`差异` 单独不足以直接成结构；其二，`非任意约束性` 目前仍是最低形而上学承诺，而非已闭合的更深解释；其三，DM 尚不能回答“为什么世界会有约束”这一更深追问。若不把这些代价一起写出，`差异 + 非任意约束性` 就会重新滑回被神圣化的首词。

## 六、主链覆盖的最低要求

起点竞争不能只比谁“听起来更薄”，还要比谁能覆盖更长主链。对本文而言，最低主链不是全部 DM，而只是四个承重点：`结构 / 边界 / 时间 / 层级`。

若把这条主链再压成最小中层命题，可先写成四句：

1. `命题 1`
只有当最小可分性进入复数条件、可描述连结与非任意限制时，差异才有资格被写成结构。

2. `命题 2`
只有当一组结构关系对成员资格与内外转换给出稳定阈差时，它才有资格被写成边界。

3. `命题 3`
只有当结构位形的变换痕迹进入后续成立条件时，时间性才有资格被写成继承序，而不是先验容器。

4. `命题 4`
只有当若干边界化结构的耦合产生新的整体条件、整体边界与反向约束时，层级才有资格被写成真实生成位置。

这四句的作用不是宣称形式化已经完成，而是防止起点竞争继续停留在口号层。谁若声称自己拥有更薄起点，就必须说明自己如何以不更厚的方式穿过这四步。

### （一）结构

任何起点若想真正取代 DM，首先必须说明：它如何从最小前提进入可重复的结构，而不是只把结构当作已给定背景。对本文而言，`差异 + 非任意约束性` 的优势在于，它先给出可分，再通过约束解释为何某些差异关系会反复沉积为稳定结构。若一个候选起点必须直接预设“结构网络”“编码规则”或“状态空间”，它就已经比 DM 更厚。

### （二）边界

起点若不能进入边界问题，就很难说明“何以成其为一”。DM 在这里的最低主张是：边界不是附加描线，而是某组关系对“谁算成员、何处算整体、何处算外部”给出稳定阈差的结果。<sup>6</sup> 这要求候选起点不仅能谈关联，还能解释成员性条件如何出现。关系语言若默认了关系项，事件语言若默认了切片单位，都会在边界问题上显得更厚。

### （三）时间

本文拒绝把“过程”直接升格为首词，正因为这会把时间流动与先后性提前写回地基。DM 当前更保守的写法是：时间性不是先验容器，而是结构变换痕迹进入后续成立条件的继承序。<sup>7</sup> 这一定义当然仍是解释性模型，但它至少把时间后移到结构与痕迹之后，而不是直接把“流”当作不需再解释的本原。

### （四）层级

最后，任何候选起点若要真正胜出，还必须说明层级如何出现。DM 的最低要求不是一口气推出生命、意识与规范，而是说明：当差异关系在约束中沉积为新的整体边界、整体条件与反向约束时，层级才有资格被写成真实结构位置。<sup>8</sup> 若某候选起点必须先默认对象层、功能层或编码层的稳定单位，才能进入层级讨论，那么它并没有比 DM 更薄。

因此，本文给出的最低裁决标准不是“谁最抽象”，而是：谁能以更少偷带完成 `结构 / 边界 / 时间 / 层级` 的主链覆盖。

## 七、与强对手的最低分界

若上述比较仍太抽象，就必须进一步说明本文与几类常见路线的最低分界。

第一，针对“关系优先”路线，本文的分界不是反对关系，而是反对把关系项与关系位次直接预设进地基。第二，针对过程哲学，本文的分界不是反对生成，而是反对把时间流动与先后性直接写成本原。第三，针对事件本体论，本文的分界不是反对发生，而是反对把切片单位与离散化方式先行冻结。第四，针对信息优先路线，本文的分界不是反对编码与传输，而是反对把编码规则、载体与测度框架当成无需再解释的基础。第五，针对“约束优先”路线，本文的分界不是反对稳定性与筛选，而是反对把状态空间与可行域直接写在起点层。

这些分界的共同功能，是把本文的主张压缩为一个可受损命题：`差异 + 非任意约束性` 当前之所以值得保留，只是因为它比以上候选写法更少偷带其后本应推出的结构，而不是因为它已经永久获胜。

## 八、表 2：起点竞争的更新规则

本文不把起点问题写成一劳永逸的宣判，而把它写成可更新的比较协议。

| 压测项 | 若支持本文，可写什么 | 若出现稳定反向结果，本文须怎样缩限 |
| --- | --- | --- |
| `更薄替代起点` 尚未出现 | 可继续把 `差异 + 非任意约束性` 写成当前较经济起点 | 若出现更薄且覆盖同等主链的替代语言，须改写地基表述 |
| 各候选起点仍明显偷带对象、时间或编码前提 | 可继续强调 DM 的较低偷带成本 | 若比较显示 DM 自身同样持续偷带这些前提，须下调“较经济”声明 |
| `结构 / 边界 / 时间 / 层级` 主链仍可由 DM 较顺畅覆盖 | 可把 DM 写成当前覆盖面更大的起点组合 | 若主链覆盖必须反复借用更厚术语，须缩限覆盖声明 |
| `非任意约束性` 仍可作为最低记账项保留 | 可继续把它写成最低形而上学承诺 | 若它被证明只是更厚前提的模糊别名，须重写起点组合 |

表 2 的作用不是把哲学争论伪装成实验协议，而是迫使本文承担清楚的更新纪律。

更准确地说，本文对“替代起点胜出”的认定，至少要求同时满足四步：它必须更薄；它必须不靠更重前提进入 `结构 / 边界 / 时间 / 层级`；它必须在母本正文中稳定替代 DM 的起点句而不造成新的偷换；它还必须把 DM 当前依赖 `差异 + 非任意约束性` 才能完成的承重点一并接住。若只满足其中一两步，更稳的写法仍应是“形成压力”，而不是“已经击败”。

## 九、失败线：哪些情况会击败本文

为了避免把“差异 + 非任意约束性”写成不可触碰的护城河，本文把失败线压成三条。

第一，`替代起点击败`。若未来出现一个比 `差异 + 非任意约束性` 更薄、却仍能覆盖 `结构 / 边界 / 时间 / 层级` 主链的写法，DM 就应主动改写自己的地基表述。

第二，`约束记账失败`。若 `非任意约束性` 最终被证明只是更厚前提的模糊别名，或必须反复借更晚层术语才能维持其可理解性，那么把它保留为最低形而上学承诺的做法就会失分。

第三，`主链覆盖失败`。若 DM 在核心推导中持续偷偷调用对象、时点、事件单位或编码框架，而这些偷带程度并不低于其对手，那么“较经济起点”这句话就必须显著收缩。

因此，本文的核心句不应再写成“差异已经赢下起点之争”，而应更准确地写成：`差异 + 非任意约束性` 目前仍有保留理由，但它必须继续接受竞争、比较与改写。

## 十、本文的缩限

本文只处理 `P00` 的起点竞争问题，不负责直接推出完整物理学、生命、意识、规范或文明理论；只比较候选起点对主链前四格的覆盖能力，不讨论后续全部章节；只建立最小比较表与失败线，不提供起点问题的终局裁决。

因此，本文的成功标准并不是“证明所有哲学都只能从差异开始”，而是至少做到三件事：第一，把起点竞争从修辞争夺改写成前提记账；第二，把 `差异 + 非任意约束性` 的优势与代价一起写清；第三，把击败条件显式留在文内，而不是把当前写法神圣化。

## 十一、结论

本文要捍卫的不是“差异是唯一合法起点”，而是一个更窄但更稳的判断：与 `关系 / 过程 / 事件 / 信息 / 约束` 相比，`差异 + 非任意约束性` 目前仍是较经济、较少偷带前提的起点组合。它的优势不是抽象度更高，而是更晚才引入对象、时间、事件单位与编码框架；它的代价则是必须承认自己并非绝对零预设，并把 `非任意约束性` 诚实写成最低形而上学承诺。

因此，本文真正推进的不是某种终局宣判，而是一条更清楚的比较纪律：任何候选起点若要击败 DM，都必须同时满足两点，一是比 `差异 + 非任意约束性` 更薄，二是在不额外偷带前提的情况下完成同等主链覆盖。只要这两点尚未被满足，DM 就仍有理由保留自己的起点写法；一旦它们被满足，DM 也应主动改写，而不是继续把“差异”写成不可竞争的神圣首词。

## 注释

1. 本文与结构实在论共享关系优先的底线，但不同于只停留在“结构是什么”的层面；本文更关注结构如何进入边界、时间与层级生成；参见 Ladyman and Ross, *Every Thing Must Go*.
2. 对 French 一类结构实在论写法而言，结构更容易被理解为物理表征中的数学关系；本文的分歧主要在于，DM 要继续处理结构如何通过边界与持存获得动态地位；参见 French, *The Structure of the World*.
3. 本文与 Whitehead 共享反静态实体论的直觉，但拒绝把过程、流动与先后直接升格为起点层首词；本文更愿意把时间性压回“痕迹继承”的较晚位置；参见 Whitehead, *Process and Reality*.
4. 本文不反对信息语言在模型层的高效性，只反对把编码规则、载体与测度框架直接写回地基；对于“信息”在哲学中作为基础词的典型写法，可参见 Floridi, *The Philosophy of Information*.
5. 本文的自然化取向并不意味着直接把本体地基写成“物理”或其他现成对象层；与此最接近的外部对照，可参考 Ladyman and Ross 对自然化形而上学的推进。
6. 本文在边界问题上的最低主张是：只有当某组关系对成员资格与内外转换给出稳定阈差时，边界才成立；这一点是 DM 相对一般关系优先写法的新增压力。
7. 本文当前把时间性写成“结构变换痕迹进入后续成立条件的继承序”；它是解释性模型，而不是时间全部内容的完尽定义。
8. 本文在层级问题上的最低要求，只是说明何时可说出现了新的整体边界、整体条件与反向约束，而不是直接推出生命、意识或规范的全部闭合。
