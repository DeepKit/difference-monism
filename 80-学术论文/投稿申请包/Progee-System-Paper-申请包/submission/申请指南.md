# Progee System Paper 申请指南

## 当前定位

这是系统 / 工具论文，应按“可复现系统研究”提交，不应伪装成纯理论方法宣言。

## 包内使用顺序

1. 在未最终拍板前，先按 `英文系统期刊默认执行模板` 组织包内材料
2. 先看 `匿名终核清单`，确认匿名稿未残留直接识别项
3. 再决定是否转回国内期刊或会议路线
4. 再看 `materials/` 下的 artifact 交付目录、结果映射与审稿人复核路径
5. 最后补投稿信、作者元数据与真实平台字段

## 当前剩余外部动作

1. 目标期刊 / 会议定案
2. 匿名版最终识别项核查
3. artifact 与复现说明按目标 venue 体例统一并冻结交付版本
4. 作者信息与平台字段填实
5. 平台账号、预印本确权与正式提交

## 默认模板入口

- 英文系统期刊默认执行模板.md
- 匿名终核清单.md
