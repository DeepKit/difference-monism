# Progee System Paper 投稿信与摘要资料

本文提出一个以状态机治理与证据优先持久化为核心的、可复现且可审计的 AI 辅助软件工程系统。

## 中文短摘要

本文提出 Progee v2，通过 Validation Gate 状态机与 Evidence-First 持久化，形成可审计的证据链，并给出基于 E2E 测试的最小可复现评估路径。

## 英文短摘要

This paper presents Progee v2, an AI-assisted software engineering system centered on validation-gate state-machine governance and evidence-first persistence. It provides a minimal reproducible evaluation path built on E2E tests and auditable evidence objects.
