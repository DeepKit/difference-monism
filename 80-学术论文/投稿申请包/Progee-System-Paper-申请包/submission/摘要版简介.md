# Progee System Paper 摘要版简介

## 中文短简介

本文提出 Progee v2，一套以 Validation Gate 状态机治理与 Evidence-First 持久化为核心的 AI 辅助软件工程系统。文章的重点不在于夸大自动化收益，而在于把执行主张、证据对象与最小可复现评估路径绑定起来，从而提高系统研究的可审计性与可复核性。

## English Short Summary

This paper presents Progee v2, an AI-assisted software engineering system centered on validation-gate state-machine governance and evidence-first persistence. Its contribution is not an inflated claim about automation gains, but a tighter binding between execution claims, evidence objects, and a minimal reproducible evaluation path, thereby improving auditability and reviewability in system research.
