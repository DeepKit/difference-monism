# Progee System Paper 退修最小回应模板

## 使用原则

- 先区分 `系统定位误读 / 评估不足 / artifact 不清 / 匿名化问题 / 表达问题`
- 能通过补 artifact 说明解决的，不顺手扩大性能宣称
- 能通过补方法边界解决的，不顺手新增营销式贡献
- 只有在核心系统定位被击中时，才调整标题、贡献表述与结论强度
