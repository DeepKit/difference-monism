# DM-P04 申请指南

## 当前定位

这是一篇单焦点意识哲学 / 认知方法论投稿稿，不是整套 DM 理论总包。

## 包内使用顺序

1. 先看 `匿名终核清单`，确认匿名底稿未残留直接识别项
2. 再按 `目标期刊适配表` 收口摘要、关键词、标题级次与参考文献
3. 再按 `逐条核引审计表` 做第一次逐项核引
4. 最后按本目录中的 `投稿信模板 / 作者信息与投稿元数据 / 摘要版简介` 填平台信息

## 当前剩余外部动作

1. 依据 `目标期刊适配表` 把正文压到 `《自然辩证法研究》` 单刊要求
2. 依据 `投稿检查清单`、`逐条核引审计表` 与 `匿名终核清单` 完成核引、匿名终核与平台提交
3. 具体人工项可直接按 `平台提交前最后动作单` 执行
