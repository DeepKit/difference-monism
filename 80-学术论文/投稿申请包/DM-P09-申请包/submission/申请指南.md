# DM-P09 申请指南

## 当前定位

这是一篇单焦点政治理论 / 制度程序论文，不是文明总论，也不是 DM 全本总包。

## 包内使用顺序

1. 先看 `匿名终核清单`，确认匿名底稿与作者信息页已分离
2. 再按 `目标期刊适配表` 统一摘要、关键词、脚注与参考文献
3. 再按 `逐条核引审计表` 做第一次逐项核引
4. 最后按本目录中的 `投稿信模板 / 作者信息与投稿元数据 / 摘要版简介` 填平台信息

## 当前剩余外部动作

1. 依据 `目标期刊适配表` 把正文压到 `《环球法律评论》` 单刊要求
2. 依据 `投稿检查清单`、`逐条核引审计表` 与 `匿名终核清单` 完成核引、匿名终核与平台提交
3. 具体人工项可直接按 `平台提交前最后动作单` 执行
