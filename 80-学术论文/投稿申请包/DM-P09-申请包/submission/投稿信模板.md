# DM-P09 投稿信模板

尊敬的《环球法律评论》编辑部：

现提交拙稿《危机压缩必须可复归：纠错能力、退出兑现与深度伤害的最小程序理论》，申请贵刊审阅。

本文处理的问题是：什么样的危机压缩仍可被称为可辩护的长期治理。文章的核心推进有三点：第一，把危机治理中的纠错问题压缩成 `外部复审入口 / 退出触发器 / 继续限制的举证责任`；第二，把 `纠错能力 / 纠错载体 / 退出兑现度 / 深度伤害` 写成最低记录格式；第三，以 `魏玛 / 日裔拘禁 / USA PATRIOT Act / 纽伦堡` 四组材料说明，危机压缩若不可复归，将反向侵蚀共同体的承诺结构。

本文不是文明总论，也不做制度终局排名；它只提出一组更窄的程序门槛、案例快照、禁止结论与失败路径。若贵刊认为危机治理、法政理论与制度纠错之间的程序性裁决问题仍有发表价值，此稿愿提交匿名审查。

本文未一稿多投。当前稿件已按作者信息页与匿名主文分开准备；若贵刊需要匿名版、作者信息、摘要版简介或补充说明，可按要求即时提供。
