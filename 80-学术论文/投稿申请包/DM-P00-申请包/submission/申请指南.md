# DM-P00 申请指南

## 当前定位

这是一篇起点竞争短稿，应按单焦点形而上学 / 方法论短文投稿处理，不应把它重新写回 DM 总包。

## 包内使用顺序

1. 先看 `目标期刊适配表`，决定唯一目标期刊或短文路线。
2. 再看 `匿名终核清单` 与 `平台提交前最后动作单`，明确匿名与平台动作边界。
3. 匿名提交文件应从当前主稿导出，不在 `paper/blind` 入口页长期维护第二份正文。
4. 最后统一 `投稿信 / 摘要版简介 / 作者信息与投稿元数据`。

## 当前剩余外部动作

1. 目标期刊最终定案
2. 从现行主稿导出完整匿名提交文件，并做人眼终核
3. 英文版术语与引注体例终修
4. 平台账号、预印本确权与正式提交
