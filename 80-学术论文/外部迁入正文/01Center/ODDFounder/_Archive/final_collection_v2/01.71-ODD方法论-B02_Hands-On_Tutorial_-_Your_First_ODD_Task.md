# 01.71 B02: Hands-On Tutorial - Your First ODD Task
## 实战教程：10分钟完成你的第一个 ODD 任务

> **难度**: 入门
> **时间**: 10 分钟
> **目标**: 使用 Progee 理念，从零实现一个用户注册 API。

---

## 准备工作

虽然 Progee 是自动化工具，但本教程将带你**手动**体验 ODD 的思维过程。你需要：
1.  一个文本编辑器（VS Code）。
2.  一个 LLM 聊天窗口（ChatGPT/Claude）。

---

## 步骤 1: Define (定义)

不要直接写代码。先写**契约 (Contract)**。
创建一个文件 `contract.yaml`：

```yaml
# 产出物定义
artifact:
  type: python_function
  name: register_user
  
# 输入规格
input:
  username: string (min 3 chars)
  password: string (min 8 chars)
  email: string (email format)

# 输出规格
output:
  success: boolean
  user_id: string (uuid)
  error: string (optional)

# 验收标准 (最重要的部分！)
acceptance:
  - Case 1: 输入合法信息 -> 返回 success=True, user_id非空
  - Case 2: 密码太短 -> 返回 success=False, error="Password too short"
  - Case 3: 邮箱格式错误 -> 返回 success=False, error="Invalid email"
```

> **ODD 思维**: 在这一步，你已经"完成"了开发。剩下的只是实现细节。

---

## 步骤 2: Execute (执行)

将上面的 YAML 发给 AI，附带指令：
> "作为 Worker，请根据此契约实现 Python 代码。不依赖数据库，使用内存字典模拟存储。"

AI 会生成类似代码：

```python
import uuid
import re

users_db = {}

def register_user(username, password, email):
    if len(username) < 3:
        return {"success": False, "error": "Username too short"}
    if len(password) < 8:
        return {"success": False, "error": "Password too short"}
    if not re.match(r"[^@]+@[^@]+\.[^@]+", email):
        return {"success": False, "error": "Invalid email"}
        
    user_id = str(uuid.uuid4())
    users_db[user_id] = {"username": username, "email": email}
    return {"success": True, "user_id": user_id}
```

---

## 步骤 3: Verify (验证)

不要肉眼检查代码。让 AI 根据 `acceptance` 生成测试代码。
指令：
> "作为 Tester，请根据契约中的 acceptance 部分，为上述代码编写 pytest 测试用例。"

运行测试：
```bash
pytest test_register.py
```

*   **如果通过**: 恭喜，任务完成。
*   **如果失败**: 将报错信息发给 AI，让它修正代码（Execute 循环），直到通过。

---

## 步骤 4: Seal (封印)

测试通过后，这段代码就是**可信**的。
将其保存为 `register.py`，并打上 `v1.0` 标签。在未来的开发中，你可以放心地调用它，因为它符合契约。

---

## 总结

你刚刚完成了一次 ODD 循环：
1.  **Define**: 明确要什么。
2.  **Execute**: AI 填空。
3.  **Verify**: 机器验证。
4.  **Seal**: 交付资产。

这就是 Progee 每天在后台自动做成千上万次的事情。当你掌握了这个思维，你就是一个单人软件工厂。
