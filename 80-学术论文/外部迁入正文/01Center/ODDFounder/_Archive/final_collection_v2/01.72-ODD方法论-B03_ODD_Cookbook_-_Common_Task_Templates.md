# 01.72 B03: ODD Cookbook - Common Task Templates
## ODD 食谱：常用任务模板库

> **简介**: 不要从零开始。复制这些模板，填入你的需求，立刻启动开发。

---

## 1. 数据库表 (Database Table)

**Artifact Type**: `pg_table`

```yaml
contract:
  name: Create{{TableName}}
  type: pg_table
  spec:
    name: {{table_name}}
    columns:
      - name: id
        type: uuid
        default: gen_random_uuid()
        primary_key: true
      - name: created_at
        type: timestamptz
        default: now()
      # Add your columns here
  constraints:
    - no_triggers: true
    - naming_convention: snake_case
  acceptance:
    - verify_table_exists
    - verify_crud_operations
```

## 2. REST API 端点 (REST Endpoint)

**Artifact Type**: `fastapi_route`

```yaml
contract:
  name: Api{{ResourceName}}
  type: python_fastapi_route
  spec:
    method: POST
    path: /api/v1/{{resource}}
    request_model: {{Resource}}CreateRequest
    response_model: {{Resource}}Response
  context:
    - auth_middleware: required
    - db_session: required
  acceptance:
    - test_success_200
    - test_validation_error_422
    - test_unauthorized_401
```

## 3. React 组件 (UI Component)

**Artifact Type**: `react_component`

```yaml
contract:
  name: {{ComponentName}}
  type: ts_react_component
  spec:
    props:
      - label: string
      - onClick: function
      - variant: 'primary' | 'secondary'
    style: tailwind
  constraints:
    - functional_component: true
    - no_class_component: true
  acceptance:
    - render_snapshot_match
    - test_click_event
```

## 4. 数据迁移 (Data Migration)

**Artifact Type**: `migration_script`

```yaml
contract:
  name: Migrate{{Operation}}
  type: migration_script
  spec:
    up: {{SQL_to_apply_change}}
    down: {{SQL_to_revert_change}}
  safety:
    - transaction_wrapped: true
    - no_data_loss_check: true
  acceptance:
    - verify_up_success
    - verify_down_success
    - verify_idempotency
```

## 5. 自动化脚本 (Automation Script)

**Artifact Type**: `python_script`

```yaml
contract:
  name: Auto{{TaskName}}
  type: python_script
  spec:
    entry_point: main()
    dependencies: [pandas, requests]
  logic:
    - step1: Read input from {{source}}
    - step2: Process data
    - step3: Save to {{dest}}
  acceptance:
    - run_successfully_with_sample_data
    - check_output_format
```

---

**使用方法**:
1.  复制 YAML。
2.  替换 `{{...}}` 中的变量。
3.  输入给 Progee 或 AI。
4.  享受代码。
