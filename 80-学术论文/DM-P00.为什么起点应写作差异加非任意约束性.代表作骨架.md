# 为什么起点应写作“差异 + 非任意约束性”
*一篇面向起点竞争的最小比较稿*

**版本**：代表作骨架 v0.1

**英文标题**：Why the Starting Point Should Be Written as Difference Plus Non-Arbitrary Constraint: A Minimal Comparative Defense

**摘要**：本文不再把 `差异` 写成逻辑上唯一可能的终极起点，而只捍卫一个更窄的判断：在 `关系 / 过程 / 事件 / 信息 / 约束` 这些常见候选写法中，`差异 + 非任意约束性` 目前是更经济、较少偷带现成对象、时点化切片与编码框架的起点组合。文章的目标不是从绝对零预设推出整个世界，而是把不同候选起点各自偷带的前提显式记账，并说明为什么 DM 当前宁愿把 `非任意约束性` 诚实写成最低形而上学承诺，也不愿把对象、时间、事件单位或信息编码偷偷提前写回地基。本文同时保留明确失败线：若未来出现一个更薄、却能覆盖 `结构 / 边界 / 时间 / 层级` 主链的起点语言，DM 就应改写自己的起点表述，而不是把“差异”神圣化。

**关键词**：起点竞争；差异；非任意约束性；存在论地基；结构；时间性

## 一、本文主张什么

1. 本文不证明 `差异 + 非任意约束性` 是逻辑上唯一可能的起点，只主张它是 DM 当前较经济的起点组合。
2. 起点竞争不应靠口号裁决，而应比较每种候选写法偷带了什么前提。
3. `差异` 的优势在于先给出最小可分性，再显式补记复数关联与约束，而不是把对象、时间或编码规则预设进地基。
4. `非任意约束性` 必须被诚实写成最低形而上学承诺，而不是伪装成已从更深本体推出的结论。
5. 只要出现更薄且覆盖同等主链的替代起点，DM 就必须缩限或改写自己的地基声明。

## 二、本文不主张什么

- 不把 `差异` 写成神圣、不可竞争的唯一首词。
- 不把 `非任意约束性` 写成第二实体或神秘本原。
- 不在本文中直接推出完整物理学、生命、意识或规范理论。
- 不把“较经济起点”偷换成“已被证明为真”。

## 三、为什么问题必须改写为“起点竞争”

过去更容易出错的写法有两种：

1. `绝对零预设写法`
试图把 DM 写成从一个没有任何附带条件的“纯差异”直推出整个世界。

2. `偷带前提写法`
表面上说自己起点很薄，
实际上却已偷偷带入对象、过程、事件单位、信息编码或状态空间。

本文主张，
更稳的做法不是在两种错误之间摆动，
而是直接承认：

> **起点之争首先是一场前提记账之争。**

因此需要比较的不是：

- 哪个词听起来最宏大

而是：

- 哪个起点组合偷带更少
- 哪个起点组合能覆盖更长主链
- 哪个起点组合在失败时更容易被改写

## 四、最小起点比较表

| 候选起点 | 容易偷带什么 | 当前优势 | 当前弱点 |
| --- | --- | --- | --- |
| `差异 + 非任意约束性` | 仍需补记复数化关联可能性与最低约束承诺 | 较少偷带现成对象、时点化切片与编码框架 | 不能再宣称绝对零预设，且必须承认约束来源未闭合 |
| `关系` | 多个项、项间连法、关系位次 | 直接进入结构语言 | 复数架构与连法已被默认 |
| `过程` | 时间、流动、先后方向 | 便于表达生成 | 把时间性提前写回起点 |
| `事件` | 切片单位、离散化规则、时点化方式 | 便于进入历史叙述 | 把位形单位先行冻结 |
| `信息` | 编码规则、可区分状态、载体、测度 | 便于进入认知与系统论表述 | 把结构化差异与可读写方式提前带入 |
| `约束` | 被约束的项、状态空间、可行域 | 便于进入规律与稳定性讨论 | 把较后层的结构语言提前写成地基 |

这张表的用处不是宣布 DM 已经赢下起点竞争，
而是先把竞争对象的成本摊开。

## 五、为什么 DM 暂时保留“差异 + 非任意约束性”

DM 当前保留这一起点组合，
原因不在于它最响亮，
而在于它的记账顺序较稳：

1. `先给最小可分性`
差异先说明完全同一不能维持，
从而保留最薄的可区分性。

2. `再承认复数化关联可能性`
差异一旦不是孤立单数，
就必须允许关联可能性进入。

3. `再显式补记非任意约束性`
世界若是一切组合都同等可行，
则结构、重复、规律与对象都难以稳定成立。

这个顺序的好处是：

- 不必先承认现成对象
- 不必先承认现成时间
- 不必先承认现成事件切片
- 不必先承认现成信息编码

它的代价也必须一并写出：

- `差异` 单独不足以成结构
- `非任意约束性` 目前仍是最低形而上学承诺
- DM 还不能解释“为什么世界会有约束”这一更深追问

## 六、DM 在这里真正承认的代价

本文必须把三点写死，
否则起点竞争会重新滑回自我神化。

1. `不是唯一`
DM 当前只主张较经济，
不主张逻辑唯一。

2. `不是零预设`
只要保留 `非任意约束性`，
DM 就必须承认自己已有最低本体承诺。

3. `不是终局封口`
起点竞争在本文中不是被关闭，
而是被重新写成可持续比较的问题。

## 七、失败线与下调条件

本文至少接受三条失败线：

1. `替代起点击败`
若未来出现一个比 `差异 + 非任意约束性` 更薄、却仍能覆盖 `结构 / 边界 / 时间 / 层级` 主链的写法，
DM 就应改写自己的地基表述。

2. `约束记账失败`
若 `非任意约束性` 最终被证明只是更厚前提的模糊别名，
而不能作为最低记账项稳定存在，
DM 也应重写起点组合。

3. `主链覆盖失败`
若 DM 在核心推导中仍持续偷偷调用对象、时点、事件单位或编码框架，
那“起点较经济”这句话就会显著失分。

因此本文的结论不是：

> **差异已经获胜。**

而是：

> **在当前可比的候选起点里，DM 暂时有理由保留“差异 + 非任意约束性”，但它必须继续接受竞争、比较与改写。**

## 八、扩写时优先补什么

1. 把 `关系 / 过程 / 事件 / 信息 / 约束` 五类候选起点各自补成一段更正式的对照论证。
2. 把 `结构 / 边界 / 时间 / 层级` 四段主链压成更清楚的中层命题，避免只剩口号式比较。
3. 把 `起点竞争失败` 与 `偏序/物理接口失败` 区分开来，避免不同层级的败局互相遮蔽。
4. 把这篇稿子的最强句继续压短，方便在 `README / 审稿人摘要 / P11` 中反复复用。

## 九、当前最短结论

本文要捍卫的不是“差异是唯一合法起点”，而是一个更窄的判断：与 `关系 / 过程 / 事件 / 信息 / 约束` 相比，`差异 + 非任意约束性` 目前是较经济、较少偷带前提的起点组合；但它仍必须接受持续竞争，并在更薄替代方案出现时主动改写。
