# 学术投稿区说明

> 定位：本目录是 `一元论` 的论文外发入口，不是新的理论层。
> 口径：当前同时维护两条外发线，分别是 `预印本发布包` 与 `投稿申请包`。

---

## 一、当前分工

### 1. 预印本发布包

用于：

- 理论全本公开
- 征评占位
- 法源链展示

当前稳定的预印本发布线共 `5` 条：

- `DM`
- `ASTO`
- `ECET`
- `TAT`
- `ODD`

### 2. 投稿申请包

用于：

- 单篇论文外投
- 匿名送审
- 投稿材料归拢
- supplement / artifact / 复现说明归档

当前稳定建立的单篇申请包共 `6` 条：

- `DM-P00`
- `DM-P04`
- `DM-P09`
- `Progee-System-Paper`
- `LMM-理论总述`
- `ExecNorm`

---

## 二、当前最短使用规则

1. 若要公开法源、做预印本占位或给外部读者总览，优先使用 `预印本发布包/`。
2. 若要推进真实投稿、匿名送审与单篇外投，优先使用 `投稿申请包/`。
3. 若某条论文线同时存在外部迁入正文与申请包，以申请包作为对外执行入口，以外部迁入正文作为来源归档。
4. `RM / Resonance Monism / 共振一元论 / 共扰一元论` 一类旧命名材料，只按历史探索稿理解，不再作为当前默认公开主包。
