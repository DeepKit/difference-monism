## V5：时间路径预测

### 核心变化
前面所有版本回答："你现在卡在哪里？"
V5 回答：**"如果你不改变，接下来会怎么死。"**

### 新增 Q10（行为倾向题）
> 当问题出现时，你更倾向于：
> * A. 继续加大投入
> * B. 再沟通试试
> * C. 拖一拖再说
> * D. 直接中断

### 路径树（每种类型的默认崩溃路径）

**Type B（死锁型）崩溃路径**：
```
t0：流程卡住
→ 用户继续沟通（错误行为）
t1：沟通成本上升
→ 双方更谨慎（结构强化）
t2：责任更不清晰
→ 推进彻底停滞
t3：项目失败
```

**Type D（沉没成本型）崩溃路径**：
```
t0：已投入较多
→ 用户继续追加
t1：亏损扩大
→ 心理更难退出
t2：加码加速
→ 现金流断裂
t3：系统崩溃
```

### 输出格式升级（预测报告）
```
🔮 未来路径预测

如果你维持当前策略：

第1阶段（1-2周）：投入继续增加，情绪压力上升
第2阶段（2-4周）：收益未改善，退出难度指数上升
第3阶段（1-2个月）：资金/精力枯竭，被动崩溃

⚠️ 系统结论：当前路径 = 确定性亏损轨道

但如果你现在强制归零：
t0 → 中断投入 → t1 短期损失确认 → t2 资源回收 → t3 系统恢复
```

## V6：行为干预引擎

### 三级干预分级

| 等级 | 适用场景 | 输出形式 |
|:---|:---|:---|
| 🟢 Level 1（认知提醒） | 低风险，初次偏移 | 告知："你正在进入XX路径" |
| 🟡 Level 2（结构指令） | 已偏离，行为重复 | 给出具体动作清单 |
| 🔴 Level 3（强制干预） | 临界点逼近 | 行为锁定+后果警告 |

### 三个触发条件
* **Trigger 1（路径锁定）**：连续 2 次行为 = 同一错误路径
* **Trigger 2（风险跃迁）**：风险值从 <0.4 急升至 >0.6
* **Trigger 3（临界点逼近）**：已连续追加投入 / 已连续推迟决策

### 行为承诺锁（V6 灵魂）
用户在接到干预后，必须二选一：
```
选项1：我承诺执行干预动作
选项2：我选择继续当前路径

如果选2，系统输出：
⚠️ 你正在主动选择进入：沉没成本崩溃路径（预计损失：高）
```
> 本质：让用户为错误负责，解除自欺。

### 干预 API
```json
POST /intervene
输入：{ "user_id": "xxx", "current_type": "D", "behavior": "A", "risk": 0.72 }
输出：{ "intervention_level": "HIGH", "message": "禁止继续追加投入",
        "required_action": ["停止资金投入","计算未来收益"], "lock": true }
```
