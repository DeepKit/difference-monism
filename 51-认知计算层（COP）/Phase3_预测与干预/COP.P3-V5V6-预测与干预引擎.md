# COP · Phase 3 · V5+V6 — 路径预警与有界干预建议

> **启动条件**：只有在 `Phase 2` 的复核一致性、版本回滚与误判成本控制都达标后，才允许进入本阶段。
> **核心边界**：本阶段只负责 `路径预警 + 干预建议 + 升级接口`，不负责强制干预授权，不直接绕过 `TAT / ODD / HUMAN_REVIEW`。

---

## V5：时间路径预警

### 核心变化

前面所有版本主要回答：

> “你现在更像哪一种结构状态？”

`V5` 进一步回答：

> “如果当前结构和行为倾向维持不变，接下来更可能朝哪类失稳路径漂移？”

这仍然是：

- 条件性预测
- 候选路径预警
- 风险分流接口

而不是确定性命运宣判。

### 新增 Q10（行为倾向题，当前为可选扩展）

> 当问题出现时，你更倾向于：
> * A. 继续加大投入
> * B. 再沟通试试
> * C. 拖一拖再说
> * D. 直接中断

### 当前路径树（工作性假设）

**Type B（死锁型）可能路径**

```text
t0：依赖冲突存在
-> 持续单点协调
t1：沟通成本上升
-> 责任边界进一步模糊
t2：推进停滞风险升高
```

**Type D（沉没成本型）可能路径**

```text
t0：既有投入已显著影响判断
-> 持续追加投入
t1：退出门槛升高
-> 损失吸收能力继续下降
t2：资源与判断弹性同步恶化
```

### 预测报告最低输出

```text
🔮 路径预警

当前候选路径：Type D drift
路径置信度：MEDIUM
时间窗口：短期 / 中期
当前最强风险：退出门槛持续升高
当前更稳处理：补录阈值条件 + 升级人工复核
required_review：HUMAN_REVIEW / TAT_REVIEW
```

### 当前固定纪律

1. 预测输出必须写成 `候选路径`，不能写成确定性命运。
2. 预测输出必须带 `时间窗口` 与 `路径置信度`。
3. 高风险路径默认附带 `required_review`，不得自动闭环。

## V6：有界干预建议

### 干预分级

| 等级 | 适用场景 | 当前允许输出 |
|:---|:---|:---|
| `Level 1` | 低风险，初次偏移 | 认知提醒与补采样建议 |
| `Level 2` | 已偏离，结构问题重复出现 | 结构性建议清单与升级提示 |
| `Level 3` | 高风险、不可逆后果或越过责任门槛 | 只允许升级建议，不允许系统单独执行强制动作 |

### 三个触发条件

* **Trigger 1（路径重复）**：连续多次进入同一路径候选
* **Trigger 2（风险跃迁）**：`structural_risk` 在短窗口内明显升高
* **Trigger 3（临界信号）**：连续追加投入、连续推迟决策或连续绕过阈值

### 用户确认回执

`V6` 可以记录用户如何回应建议，

但不能把回执写成强制锁定。

当前允许的回执状态：

- `accept`：接受建议
- `defer`：暂缓执行
- `dispute`：提出异议并请求人工复核

若用户选择 `defer` 或 `dispute`，

系统更稳的动作是：

- 记录分歧
- 保留路径预警
- 升级人工复核

而不是输出“你已被系统锁定”。

### 干预 API（建议模式）

```json
POST /intervene
{
  "user_id": "xxx",
  "current_type": "D",
  "behavior": "A",
  "structural_risk": "HIGH",
  "path_confidence": "MEDIUM"
}
```

```json
{
  "intervention_level": "LEVEL_3",
  "mode": "suggest_only",
  "message": "当前更稳的处理是暂停追加投入并升级人工复核。",
  "recommended_actions": [
    "暂停新增承诺",
    "补录退出条件",
    "请求人工复核未来收益与责任承接"
  ],
  "required_review": ["HUMAN_REVIEW", "TAT_REVIEW"],
  "forbidden_actions": [
    "把预测结果当成自动执行许可",
    "绕过责任门槛继续扩张"
  ]
}
```

### 当前最硬边界

1. `COP` 可以建议暂停，不可以单独强制暂停。
2. `COP` 可以提示高风险，不可以单独授权高风险动作。
3. 一旦触及责任、补偿、冻结、召回或工程留痕，
   必须回接 `TAT / ODD / HUMAN_REVIEW`。

---

## 最短压缩句

1. `V5/V6` 的价值，不是“替你做决定”，而是更早暴露可能的失稳路径。
2. `高风险阶段最稳的动作，不是系统自己更强势，而是更早升级到有责任的复核接口。`
