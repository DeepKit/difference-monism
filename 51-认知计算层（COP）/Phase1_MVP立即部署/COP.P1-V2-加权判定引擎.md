## 二、判定逻辑

### Step 1：计算四类总分
```
Score_A = 所有 A 列加总
Score_B = 所有 B 列加总
Score_C = 所有 C 列加总
Score_D = 所有 D 列加总
```

### Step 2：选主类型
```
主类型 = 分数最高的类型
```

### Step 3：副类型判定
```
副类型 = 第二高分（必须 ≥ 主类型的 60%）
否则：不显示副类型（避免噪音）
```

---

## 三、冲突消解规则（优先级机制）

> 当多个类型分数接近时，以下规则强制覆盖：

**规则1（死锁优先）**：
若 Q4 = D（多方循环依赖） → **强制主类型 = 死锁型（B）**
（物理结构问题，优先级最高）

**规则2（沉没成本锁定）**：
若 Q5 = D 且 Q2 = D → **强制主类型 = 沉没成本型（D）**

**规则3（共识幻觉覆盖）**：
若 Q2 = A 且 Q3 = C 或 D → **主类型优先 = 共识幻觉型（A）**

---

## 四、风险等级计算

```
Total Score = A + B + C + D
Max Score = max(A, B, C, D)

风险值（Confidence）= Max Score / Total Score

< 0.4  → 🟢 低风险
0.4-0.6 → 🟡 中风险
> 0.6  → 🔴 高风险
```
