## 发布纪律

- **public/** 内容：五协议框架、诊断逻辑、理论推导——可公开确权
- **private/** 内容：判定阈值、权重配置、误判样本库、引擎参数——C 类独享
- 对外展示 COP 时只展示 public/ 的框架和走通示例
