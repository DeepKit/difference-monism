# DM.O04 · 跨理论比较速查表

> 定位：为快速回答"差异一元论 vs X"提供标准化比较框架。不是替代 DM.P10/P11 的完整学术对话，而是传播与问答场景的速查工具。
> 作者：Yi Fu（付毅，ODDFounder）

---

## 比较总表

| 理论传统 | 共同点 | 关键差异 | DM 的增量 |
|----------|--------|---------|----------|
| 实体本体论（亚里士多德） | 都关心"存在者如何成立" | DM 不以现成实体为起点，从差异出发 | 不需要先偷带"物体"概念 |
| 过程哲学（怀特海） | 都拒绝静态实体优先 | 过程哲学以"事件/过程"为起点，DM 以"差异及其约束"为起点 | DM 的"结构持存"解释了为什么不是一切都在流变 |
| 佛学缘起论 | 都拒绝自性/实体 | 佛学以"空"为终点，DM 以"结构层级涌现"为展开方向 | DM 贯通了从本体到制度的正向建构 |
| 现象学（胡塞尔/海德格尔） | 都重视"显现"和"在世存在" | 现象学以主体经验为方法，DM 以结构分析为方法 | DM 提供了"内部性"的非现象学解释 |
| 结构实在论（Worrall/Ladyman） | 都认为结构优先于实体 | 结构实在论以科学理论为锚，DM 以本体论为锚 | DM 提供了结构的生成条件（差异→结构→边界） |
| 非还原物理主义 | 都拒绝独立的"心灵实体" | 物理主义仍以"物理"为基本层，DM 以"差异"为基本层 | DM 避免了"物理"概念本身的形而上学预设 |
| 批判理论（哈贝马斯） | 都关注规范与交往条件 | 哈贝马斯以理想言说情境为规范来源，DM 以结构可归属性为规范来源 | DM 的规范不需要预设理性共识 |

## 与主要传统的详细对比

### DM vs 实体本体论

| | 实体本体论 | DM |
|------|-----------|-----|
| 起点 | 存在者（物体、事物） | 差异 |
| 构成 | 实体+属性 | 差异→结构→边界 |
| 变化 | 实体属性变化 | 结构位形变换 |
| 关系 | 实体的外在关系 | 关系内在于结构形成 |
| 优势 | 直观、与日常语言兼容 | 不需要预设"物体"概念 |
| 劣势 | 难以解释关系本身 | 不如实体本体论直观 |

### DM vs 过程哲学

| | 过程哲学 | DM |
|------|---------|-----|
| 起点 | 实际缘现（actual occasions） | 差异 |
| 持存 | 社会的/历史的稳定模式 | 结构在边界内的变换不变性 |
| 时间 | 过程本身 | 结构变换痕迹的继承序 |
| 主体 | 每一个 actual occasion 都有主体极 | 主体在特定结构复杂度的内部性中形成 |
| 共识 | 都反对"实体是终极的" | 都反对"实体是终极的" |
| 分歧点 | 过程哲学把持存解释为"社会"的稳定模式 | DM 把持存解释为结构差异在边界内的继承 |

### DM vs 佛学缘起论

| | 佛学缘起论 | DM |
|------|-----------|-----|
| 对"实体"的立场 | 自性空——无独立实体 | 差异优先——不需预设实体 |
| 对"苦/困局"的诊断 | 执取导致苦 | 结构失衡导致责任流的错误汇聚 |
| 出路 | 解脱——脱离轮回 | 规范——通过结构约束防止失衡 |
| 关键分歧 | 解脱是终点；世界是问题 | 结构优化是目标；世界是起点 |
| 共识 | "空"与"差异"在否定实体自性上是盟友 | |
