# 《哲学研究》投稿 - 建议回避专家及其他意见

## 📌 填写原则

### **什么时候需要填写回避专家？**

✅ **建议填写的情况：**
1. 与你有**直接利益冲突**的专家（导师、合作者、同单位）
2. 与你有**学术争议**的专家（批评过你的观点）
3. 你的研究**直接批评**了某位知名学者的核心观点

⚠️ **不建议填写的情况：**
1. 仅仅是"担心某专家可能不认可"（没有实际冲突）
2. 列出一长串名单（会让编辑部反感）
3. 回避本领域所有知名专家（显得不自信）

## ✍️ 填写建议（三个方案）

### **方案A：不填写**（推荐）⭐⭐⭐⭐⭐

**理由：**
- 你没有明显的利益冲突
- 不填写显得更自信和坦诚
- 编辑部会根据匿名原则选择合适的审稿人

**填写内容：**
```
无
```

或者留空不填。

---

### **方案B：礼貌性说明**（次优选择）⭐⭐⭐⭐

如果你担心某些特定领域的专家可能不够了解你的跨学科视角，可以这样写：

**填写内容：**
```
本文采用跨学科研究方法，融合认识论、科技哲学、中国哲学等视角。
建议选择对STS研究、4E认知科学或中国哲学有一定了解的审稿专家，
以便更全面地评价本文的学术贡献。感谢编辑部！
```

**优点：**
- 不是回避专家，而是提出建设性建议
- 显得专业和体贴

**缺点：**
- 可能被理解为"挑审稿人"
- 不如方案A直接

---

### **方案C：明确回避（仅在有具体原因时使用）**⭐⭐

**仅当以下情况时使用：**
1. 你确实在某篇论文中直接批评了某位知名专家
2. 该专家可能认出你的身份（尽管是盲审）
3. 有明确的利益冲突

**填写模板：**
```
建议回避专家：XXX教授（XX大学）
理由：本文部分观点与该专家的核心理论存在学术争议，
为保证评审公正性，建议回避。
```

**⚠️ 警告：不要轻易使用方案C！**
- 可能显得不够自信
- 编辑部可能反感"指定审稿人"
- 除非有充分理由，否则不建议

## ⚠️ 填写禁忌

**千万不要这样写：**

❌ "本文批评了XXX教授的观点，建议回避"
- → 显得攻击性强，编辑部反感

❌ "建议回避以下10位专家：XXX、XXX……"
- → 名单太长，显得不自信

❌ "希望找到认可我观点的审稿人"
- → 显得学术不端

❌ "本文非常创新，希望编辑部重视"
- → 显得傲慢
