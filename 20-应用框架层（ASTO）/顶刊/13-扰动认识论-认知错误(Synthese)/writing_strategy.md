# 写作策略：扰动认识论 (Perturbation Epistemology)

**目标期刊**: *Synthese*
**定位**: 哲学主线（最高理论位）
**核心任务**: 为“错误”建立本体论地位，去 ASTO 化。

---

## 1. 核心论点 (The One Thing)
**"Cognitive error is not a failure of representation, but a structural necessity of perturbation management."**
（认知错误不是表征的失败，而是扰动管理的结构性必然。）

## 2. 避坑指南 (Anti-Patterns)
*   **严禁** 提及 "ASTO"、"System"、"Framework"。
*   **严禁** 使用工程隐喻（如 "Software"、"Code"），必须使用认知科学/生物学隐喻（如 "Organism"、"Environment"）。
*   **避免** 宏大叙事，专注于微观的认识论机制。

## 3. 结构大纲 (Structure)
1.  **Introduction**: 挑战传统的 "Truth-tracking" 认识论。提出问题：如果错误是常态，那么认知的本质是什么？
2.  **The Problem of Error**: 批判表征主义（Representationalism）无法解释错误的结构性功能。
3.  **Perturbation & Boundary**: 引入 "Perturbation" 概念。认知即是在边界上管理扰动。
4.  **Structural Error**: 论证错误是维持边界的代价（The cost of individuation）。
5.  **Conclusion**: 认识论应从 "Avoiding Error" 转向 "Managing Perturbation"。

## 4. 写作战术 (Tactics)
*   **对话对象**: 引用 Andy Clark (Predictive Processing), Francisco Varela (Enactivism)。
*   **语气**: 冷静、分析性、高密度。
*   **术语替换**:
    *   Input -> Perturbation
    *   System -> Agent / Organism
    *   Bug -> Structural Mismatch
