This glossary has undergone rigorous philosophical disambiguation to avoid misreading by Western philosophy scholars:

**✅ Resolved Risks:**
1. **Peircean Phenomenology Confusion** → Changed Firstness/Secondness/Thirdness to Monistic Layer/Observational Duality/Mediated Triadic Structure
2. **Field Ambiguity** → Added precise definition to distinguish from physics/Bourdieu/Lewin fields
3. **Perturbation Determinism** → Clarified as non-intentional influence, not direct causal control
4. **Sartrean Existentialism** → Marked In-itself as descriptive usage only
5. **Engineering Context** → Provided Fundamental Unit/Taboo Zone alternatives for technical docs

**✅ Confirmed Correct:**
- Motility: Unique term with no English equivalent; correctly annotated
- Re-origin: Original and appropriate; traditional alternative provided
- Untouchable Dimension: Precise emphasis on non-instrumentalization

## 译名变更与争议记录 (Change Log & Debates)

### **一级修正（哲学歧义风险）**

**1. 一元/二元/三元 (CRITICAL CHANGE)**
- **旧译**: Firstness/Secondness/Thirdness
- **新译**: Monistic Layer / Observational Duality / Mediated Triadic Structure
- **原因**: 避免与皮尔士 (Peirce) 现象学混淆。ASTO的三层结构起点、因果逻辑与本体论立场与皮尔士完全不同。
- **使用原则**: 必须在首次出现时明确声明: "This is not Peircean Firstness–Secondness–Thirdness."
- **Monistic Layer 注意**: "Layer" does not imply ontological stratification, but epistemic accessibility — 不是本体分层，而是认识论的可接近性分层。此注释应放在P04正文首次定义处。

**2. Field (场域)**
- **完整定义**: Field (Mediated condition-space through which perturbations propagate and transform Attribute-Sets)
- **原因**: 防止被误读为物理场/布迪厄社会场/勒温心理场。ASTO的Field是“变化中介场”，不是力场也不是社会空间。
- **关键澄清**: Field不生产变化，只负责传播与转译扰动。

**3. Perturbation (扰动)**
- **补充定义**: Perturbation (Non-intentional influence entering the Field, not direct causal control)
- **原因**: 防止被物理化为“外力偏移”或因果决定论。ASTO的Perturbation是非意志介入，不是直接因果控制。

**4. In-itself (自在态)**
- **补充声明**: Used descriptively for pre-articulated state, not Sartrean être-en-soi
- **原因**: 该词是萨特存在主义专用术语。必须声明仅作描述性使用，不涉及存在主义哲学立场。

### **二级优化（工程语境适配）**

**5. Fundamental/Taboo**
- **工程语境优化**: Fundamental Unit / Taboo Zone
- **原因**: 在工程文档中增加空间感与单元感，清晰区分于哲学的"Fundamental"（形而上基础）。
- **使用原则**: 哲学文档用 Fundamental/Taboo；工程文档用 Fundamental Unit/Taboo Zone。

### **三级保留（风格与原创性）**

**6. Untouchable Dimension vs. Inviolable Dimension**
- 当前采用 Untouchable Dimension，强调“不可被工具化/操作化”。

**7. Motility (动变性)** ✅ 完全正确
- 用于“动变性”，同时表达：可演化 (evolve) + 可回滚 (rollback) + 可替换 (swap) + 可重组 (reconfigure)。
- **为何保留**: 英语中几乎没有第二个词能同时承载这四重语义。这是ASTO的原创核心术语。
- **风险**: 生物学意味强，易被理解为“生命本体论”。
- **处理策略**: 每次出现时强制注释 "(capacity to evolve/rollback/swap)"。
- **结论**: 已经采取了原创术语的标准操作，不需修改。

**8. Re-origin (归元)**
- 对应“归元”，保留作者语义（归返本源），避免“Return”过于平庸。
- **备选**: 如需更传统，可改为 Reversion to Source。
