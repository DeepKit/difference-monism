# Strategic Decision: ASTO Brand Timeline

---

## The Question

要不要：

- 用真实"ASTO"术语出现？
- 还是前3个月不提ASTO，只做结构分析？

---

## Recommendation

> 前三个月不要强调ASTO品牌。
> 先建立"结构清晰度"。
> 等读者认可解释力，再引出体系。

---

## Rationale

### Phase 1: Structural Clarity (Months 1-3)

- Focus: Demonstrating structural analysis capability
- Method: Pure structural language, no brand mention
- Goal: Build credibility through clarity

If readers say "this analysis is clearer" — you have succeeded.

### Phase 2: Method Introduction (Months 4-6)

- Focus: Introducing consistent terminology
- Method: Gradual introduction of ASTO terms
- Goal: Build vocabulary recognition

When readers can use terms like "structural tension" — you have succeeded.

### Phase 3: Framework Reveal (Months 7-12)

- Focus: Presenting ASTO as the underlying framework
- Method: Connect structural analysis to ASTO
- Goal: Framework adoption

When readers ask "what is ASTO?" — you have succeeded.

---

## Execution Rules

### Months 1-3: Pure Structure

- ❌ Do NOT mention "ASTO"
- ❌ Do NOT reference the theory
- ✅ Use neutral terms: "structural view," "tension analysis," "attribute mapping"
- ✅ Let the clarity speak for itself

### Months 4-6: Terminology Building

- ✅ Introduce terms gradually: "attribute set," "structural constraint," "tension overflow"
- ❌ Do NOT explain the full framework yet
- ✅ Use terms as if they are common vocabulary

### Months 7-12: Framework Introduction

- ✅ Mention "ASTO" naturally in conclusion
- ✅ Provide context about the framework
- ✅ Invite readers to explore more
- ✅ Maintain structural analysis as primary focus

---

## Success Metrics

### Phase 1 (Months 1-3)

- Reader feedback: "This is a clearer way to think about X"
- No brand awareness yet — this is correct

### Phase 2 (Months 4-6)

- Reader usage: Readers start using "structural tension" in discussions
- Terminology recognition — this is success

### Phase 3 (Months 7-12)

- Direct questions: "What is ASTO? Tell me more"
- Framework curiosity — this is success

---

## Key Insight

> Brand follows clarity, not the other way around.

If you lead with "this is ASTO," readers will dismiss it as theory.
If you lead with clarity, readers will seek the theory.

The order matters.
