## 选题评估标准

### 优先级判断

| 维度 | 高优先级 | 低优先级 |
| -- | -- | -- |
| 冲突明显性 | 公开争议事件 | 模糊隐含冲突 |
| 结构典型性 | 属集边界清晰 | 属集交织复杂 |
| 可解释性 | 结构优势明显 | 解释力一般 |
| 读者兴趣 | 关注度高 | 冷门话题 |

### 选题来源

1. **新闻事件** - 科技/商业/社会热点
2. **经典案例** - 历史知名事件
3. **读者征集** - 读者提供案例
4. **个人观察** - 身边典型现象

## 执行要点

### 每周节奏

1. 周一：确定本周选题
2. 周二-周三：资料收集
3. 周四-周五：撰写初稿
4. 周六：修改定稿
5. 周日：发布与反馈

### 质量控制

- 每篇至少修改 2 遍
- 关键论点需有实例支撑
- 避免主观判断，强调结构分析
- 术语使用要自然流畅
