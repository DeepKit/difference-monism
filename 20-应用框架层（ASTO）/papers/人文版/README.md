# ASTO 人文版目录说明

> **定位**：本目录收纳 ASTO 的人文转译稿、低门槛理解稿与故事化表达。
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）
>
> **当前状态**：本目录文件属于 `公开人文支线稿`，用于理解、传播与概念转译，不纳入 ASTO 首轮公开主包。

## 边界

- 本目录不是 ASTO 的正式哲学主链
- 本目录中的叙事化表达不替代 `P` 系列与 `U00`
- 若需正式引用与公开发布说明，优先回到 `../readme.md` 与 `../ASTO.预印本发布说明.v1.0.md`

## 推荐用法

- 用作读者导读、传播转译和辅助理解材料
- 若对外附带发布，建议明确标注为“人文支线稿”
