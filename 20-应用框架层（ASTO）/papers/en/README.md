# ASTO English Directory Note

> **Positioning**: This directory contains English drafts, public review texts, translated support materials, and reference indexes for the ASTO document system.
>
> **Author**: Yi Fu (付毅, ODDFounder, fuyi.it@live.cn)
>
> **Current Status**: This directory is an English support layer. It does not mean every file here belongs to the first-round ASTO public release package.

## Boundary

- The canonical first-round public chain remains `U00 + P03 + P05a + P06 + P04 + P99.1-5 + preprint note`
- `P01 / P02` are peripheral primers, not part of the minimum core chain
- `P05b / P07-P11` are extension or scenario texts, included only when topical expansion is needed
- `P12 / P13`, `read_p*.txt`, `read_p*_full.txt`, archived directories, PDF bundles, and release-history files are not part of the first-round public package
- Files explicitly marked as `Historical English Draft`, `Working Extension Draft`, `Working English Support Draft`, or reference sets should be treated as support materials
- For formal release framing, always return to `../readme.md` and `../ASTO.预印本发布说明.v1.0.md`
