# 归档目录说明

> **定位**：本目录收纳历史稿、旧命名阶段材料与保留性参考件。
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）
>
> **当前状态**：本目录属于历史留档区，不属于当前 `DM -> ASTO -> ECET -> TAT -> ODD` 公开主包。

## 边界

- 本目录用于保存旧稿、旧命名阶段材料与溯源参考
- 其中内容可供研究史、版本演化与自我校正使用
- 若对外正式引用，应优先引用当前主链正文、发布说明与预印本包，而不是直接引用归档稿

## 当前保留原则

- 已被当前正式目录明确吸收、且已有稳定现行替代件的旧稿，可从本目录移除
- 当前仍保留的内容，主要包括：旧数学前导、旧论文体系、旧科学强推导稿与其他尚未完全转写的历史材料
- 因此，本目录当前不再保留已被 `10-哲学核心层（DM）` 与 `60-科学对话层` 明确吸收的顶层哲学主干旧稿
- 早期 `RM.papers` 论文写作链，现统一保留在 `80-学术论文` 作为历史探索稿与写作记录；本目录不再重复保留第二份
- 早期 `RM` 导航件、层级跟踪件与若干旧层入口摘要，若已被 `00-索引与导航` 与对应正式层 `README` 接管，也不再在本目录重复保留
- 早期 `RM` 的薄摘要件（如时空、价值、规范、认识等短文），若核心论点已被当前主链正文吸收，则也不再重复保留
- 早期 `RM.101/102/201/401/404` 这类按层压缩的旧摘要主稿，若其主题已分别被当前 `DM / ECET / TAT / ODD / 对话层` 承接，则也不再重复保留
- 早期 `MFO_SCI` 科学稿中，若后续已有更成熟的 `RM_SCI` 或 `RM参考` 保留版本，则本目录只保留后者，不再重复保留更早草稿
- 早期 `MFO.*` 分层长稿若其主题已被当前 `DM / 30 / 40 / 50 / 60 / 70` 各正式目录吸收，则本目录仅保留更原始的底稿与推导起点，不再重复保留整条中间演化链
- 因此 `根目录原始文件` 当前只保留极少数更原始的底稿，不再保留已存在后续保留版本的中间科学化稿
- `RM参考` 中若只是会话记录、重构草案、示例稿或缓存式张力草稿，也可移除，只保留较高价值的推导汇总、审查意见与少数关键参考件
- 进一步地，若 `RM参考` 中的评估稿、宣传式比较稿与旧审查中间件已无独立保留必要，也可移除；当前优先保留的是少数关键推导档案
- `SCI旧版` 若继续保留，优先保留“旧科学化主干”而不是行业手册、修正中间件等支撑稿；支撑件可先行移除
- 当前 `SCI旧版` 已进一步收缩为最小旧科学主干，只保留较核心的公理/形式化稿
- 若继续极限压缩，则每条历史链只保留一份最具代表性的文件：最早底稿链保留原始稿，推导链保留汇总稿，旧科学链保留单份代表性接口稿
