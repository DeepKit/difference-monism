在一切命题之前，RM必须先声明自己的认识论地位：

> **RM本身是意识这个幸存结构产生的解释模式。它不是终极真理，而是当前条件网络下最少预设的稳定解释结构。它的有效性来自与其他幸存结构之间形成的暂时稳定的共振关系，不来自对宇宙终极真相的把握。**

这不是谦辞，是理论的内在要求。一个把自己也纳入所描述框架的理论，比任何声称掌握终极真理的理论都更诚实。

## 一、宣言：从物体到流变结构

传统本体论将世界视为**物体**的集合——原子、粒子、实体。这种假设预设了：

- 存在是静态的
- 边界是固定的
- 本质是内在的

**共扰一元论拒绝这种假设。**

世界不是由独立的静态物体构成，而是由**差异活动中维持的稳定流变结构**构成。

- **结构**是关系性的，不是实体性的
- **稳定**是动态的，不是静止的
- **存在**是流变中的暂稳态，不是永恒的实体
- **边界**是幅值梯度的物理事实，不是固定的墙

这不是语义游戏，是根本的概念转换。

## 三、核心命题

### 命题一：差异活动是存在的起点

> **差异不需要解释，无差异才需要解释。**

RM-A0确立了理论的边界条件：

> 混沌是未结构化的差异场。它不是无，也不是无差异，而是充满扰动却没有任何稳定闭合结构的纯粹差异活动。

完全无差异在物理上不可实现（不确定性原理）。差异活动是默认态。莱布尼茨之问在物理层被关闭：无比有需要更多条件，而无在物理上无法实现。

### 命题二：存在是差异活动中稳定闭合有边界的流变

> **存在 = 差异活动中稳定闭合有边界的流变**

四个关键词，缺一不可：

```
稳定   不随机消散，能在扰动中维持
闭合   形成自维持的正反馈回路
有边界 内外可以区分，幅值梯度存在
流变   持续动态更新，不是固定实体
```

**三层概念精确区分：**

```
存在（Being）
= 闭合形成的那一刻
= 差异活动第一次产生稳定边界

持存（Persistence）
= 在持续扰动中维持闭合的能力
= tr < τ（回归时间短于扰动间隔）

幸存（Survival）
= 持存在时间尺度上的统计结果
= 我们能够观察和讨论的存在
```

瞬间稳定闭合：完成了存在，未完成持存。
持续稳定闭合：完成了存在，也完成了持存。

**因此旧有的"存在=幸存下来的稳定结构"不是错的，而是从存在推论到幸存的自然结果——在持续扰动的差异场中，我们能够观察和讨论的存在，都是幸存下来的稳定流变结构。**

### 命题三：闭合是意外，筛选是统计

差异场中的闭合不是必然的，是偶然的：

> 绝大多数差异活动随机消散。极少数差异路径偶然折回自身，形成第一个稳定闭合。这个偶然是一切存在的起点。

宇宙不是被设计的，不是被预定的，是**无数次偶然中极少数幸存的结构集合**。

闭合之后，筛选机制涌现：

**最小回归原理（MRP）不是公设，是统计筛选的必然结果。**

回归时间 $t_r < $ 扰动间隔 $\tau$ 的结构系统性留下来，$t_r > \tau$ 的结构系统性消失。不需要预设这个原理，持续扰动加上偶然闭合，MRP自然涌现。

### 命题四：结构之间存在三种关系

稳定结构并非孤立存在，它们之间持续发生关系：

**相容**：两个结构的回路互相强化，形成更大的稳定结构。复杂性从这里增长。

**排斥**：两个结构的回路互相干涉破坏，其中一个或两个消失。

**吞并**：强结构吸收弱结构，扩大自身回路。

层级组合只在相容条件下发生。宇宙的复杂化不是目的论的进步，是相容关系在统计筛选下的自然积累。

### 命题五：复杂结构有统计优势，但没有目的论特权

复杂结构具有三个稳定优势：

**冗余**：多个回路互相备份，局部破坏不导致整体崩溃。

**自修复**：局部破坏可以被其他回路补偿，等效回归时间缩短。

**扰动吸收**：复杂结构可以将外部扰动纳入自身回路，而不是被摧毁。

因此：

$$P(\text{持存} \mid \text{复杂结构}) > P(\text{持存} \mid \text{简单结构})$$

**这不是目的论。** 宇宙没有偏好复杂结构。这只是统计偏向——在持续扰动的差异场中，复杂结构碰巧有更高的持存概率，因此我们观察到的世界里复杂结构更多。这是幸存者偏差，不是宇宙意志。

## 四、涌现层级

从混沌到意识，结构在多个层级涌现：

```
RM-A0
混沌 = 未结构化的差异场
差异 → 扰动 → 闭合 → 结构
        ↓
物理层
稳定结构的统计行为 = 粒子、场、物理定律
（幸存结构的稳定窗口，非客观必然）
        ↓
化学层
相容结构的层级组合 = 原子、分子、化学键
        ↓
生物层
自维持、自复制、自修复的复杂结构 = 生命
        ↓
认知层
预测-误差-更新的闭环结构 = 认知
        ↓
意识层
高耗散自指闭合结构的内部经历 = 意识
        ↓
社会层
跨个体的共扰网络 = 文明
```

每一层都是前一层条件网络的展开，不是独立事件的集合。每一层的结构都是幸存者，没有任何一层有目的论特权。

## 五、时间与空间

### 时间

时间不是宇宙的背景容器，而是结构变化的涌现属性：

**拓扑时间**：差异事件之间的因果偏序。混沌中已存在，不需要稳定结构。

**度量时间**：稳定闭合结构的不可逆跃迁累积。需要第一个天然时钟（稳定循环结构）出现后才有可测量的计次。

$$T = \sum_i \Delta\Phi_i$$

没有稳定结构，就没有度量时间。时间不是流逝的背景，是结构变化的内生序列。

### 空间

空间不是宇宙的容器，而是差异关系网络的几何表达：

$$\kappa(i) = \sum_{j \sim i}\bigl(w(i) - w(j)\bigr)$$

节点曲率由与相邻节点的差异强度差定义。路径分布的不均匀性定义距离，独立变化方向的数量定义维度。

空间的维度不是预设的，是关系网络节点连接分布的统计特征涌现出来的。

## 六、数学与物理定律的地位

### 数学

数学是稳定结构规律的压缩描述语言。数学的有效性来自两个方向的筛选：

只有能被数学描述的物理结构才能稳定持存，只有能描述幸存物理结构的数学才被保留和发展。

**数学和物理是互相筛选的幸存者。** 这不是数学神奇地符合物理，而是两者在同一个差异场中经历了相互的筛选过程。

### 物理定律

物理定律是这个宇宙中幸存结构的稳定描述模式。它们不是宇宙的客观必然，而是在当前条件窗口下稳定的统计规律。

换一套初始条件，不同的东西可能是"稳定的"，不同的规律可能是"定律"。我们的物理定律是这个幸运儿宇宙的地方性事实。

## 七、意识的地位

意识在RM框架里的地位需要在两个层面精确区分：

**存在论层面：**
意识是众多幸存结构之一，没有目的论特权，和粒子、生命平等。它是高耗散自指流变结构在演化筛选中偶然幸存的一种形态。

**认识论层面：**
意识不是RM的逻辑前提（无意识不代表RM为假），但意识是RM的历史条件（无意识则RM无人建构）。RM的真理地位不依赖意识，但RM的被陈述依赖意识。

**这两个层面合在一起给出了意识唯一真正特殊的性质：**

> 意识是唯一一个能够宣布自己没有特权的结构。这个宣布的动作本身，无法被纳入幸存者队列消解。

## 八、评估标准

作为哲学研究纲领，RM的评估标准是：

**内部一致性**：推导链无逻辑矛盾，每一步都从前一步推出。

**最小预设**：用最少的假设解释最多的现象。RM的停止点是差异场的存在，这比上帝、量子真空、多宇宙等起点要求更少。

**诚实的边界标注**：理论明确声明自己无法解释的地方，不用流畅的论证掩盖断裂。

**可对话性**：与物理学、生物学、神经科学、哲学传统保持明确的关系和对话。

**不是评估标准**：波普尔证伪性（哲学本体论不直接产生可证伪预测）、定量精度（当前形式化阶段的限制）。
