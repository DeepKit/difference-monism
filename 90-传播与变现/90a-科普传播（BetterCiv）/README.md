# BetterCiv

Static website for [betterciv.org](https://betterciv.org) — For a Better Civilization.

## Structure

```
index.html      — Home
theories.html   — ODD / ECET / ASTO
manifesto.html  — Core manifesto
join.html       — Recruitment
style.css       — Single stylesheet
```

## Deploy

Push to `main` branch → GitHub Pages auto-deploys.

```bash
git add .
git commit -m "update"
git push
```
