# 素材编号：ECO-001
# 素材名称：汉武帝盐铁专卖·垄断定价
# 算法类型：垄断定价 / 市场操控 / 约束优化

## 核心算法逻辑

```
目标：最大化财政收入
约束：不能过度征税导致民变
算法：控制供给（垄断）→ 控制价格 → 提取剩余价值
```

## 历史背景

- **时间**：公元前119年（桑弘羊推行）
- **政策**：盐铁官营，均输平准
- **核心创新**：国家作为唯一供给者

## 算法拆解

### 垄断定价模型

**完全竞争市场**：
- 价格 = 边际成本
- 利润 ≈ 0（长期均衡）

**垄断市场**（盐铁专卖）：
- 价格 >> 边际成本
- 利润 = (价格 - 成本) × 数量
- 最优价格：需求弹性 = -1

### 均输平准算法

**输入**：各地物价差异、运输成本
**算法**：
1. 低价区收购 → 高价区出售
2. 消除套利空间
3. 国家赚取差价

**效果**：
- 稳定物价（减少波动）
- 增加财政收入
- 打击商人投机
