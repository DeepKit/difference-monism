# 素材编号：ECO-004
# 素材名称：范蠡计然之策·经济周期
# 算法类型：周期预测 / 逆向投资 / 动态规划

## 核心算法逻辑

```
观测：农业丰歉的12年周期（岁星周期）
策略：旱则资舟，水则资车（逆向布局）
算法：在周期低谷买入，高峰卖出
```

## 历史背景

- **时间**：春秋末期（范蠡辅助勾践）
- **来源**：《史记·货殖列传》
- **核心思想**："论其有余不足，则知贵贱"
- **成就**：三致千金，三聚三散

## 算法拆解

### 周期识别

**农业周期**（12年）：
- 岁星（木星）12年绕天一周
- 气候、农业、经济随之波动
- **特征提取**：寻找周期性模式

**逆向投资策略**：
- 大众追涨（高价买入）→ 贪婪
- 范蠡追跌（低价囤积）→ 理性
- ** contrarian algorithm**

### 动态规划

**状态空间**：
- S(t) = 时间t的市场状态（价格、供需）
- A(t) = 行动（买入/卖出/持有）

**贝尔曼方程**（现代表述）：
```
V(S,t) = max_A [R(S,A) + γ × V(S',t+1)]
```

**范蠡直觉**：
- 在低谷时买入（低R，高未来V）
- 在高峰时卖出（高R，低未来V）
