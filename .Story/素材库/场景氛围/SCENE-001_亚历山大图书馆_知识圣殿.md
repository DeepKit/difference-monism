# 素材编号：SCENE-001
# 素材名称：亚历山大图书馆·知识圣殿
# 场景类型：物理空间 / 符号意象 / 时间容器

## 空间描述

**建筑特征**：
- 位置：埃及亚历山大港
- 规模：50万卷手稿（古代最大）
- 结构：大厅、研究室、抄写室、仓库
- 装饰：缪斯女神像、几何图案地板

**感官细节**：
- **视觉**：莎草纸的黄色、橄榄油灯的光晕
- **嗅觉**：墨水、旧羊皮纸、樟脑（防虫）
- **听觉**：翻页声、低声讨论、地中海的风
- **触觉**：莎草纸的粗糙、羊皮纸的柔软

## 符号意义

**知识即权力**：
- 托勒密王朝通过收集书籍控制知识
- "拥有书籍 = 拥有解释世界的权力"

**脆弱性**：
- 多次火灾、战乱损毁
- 知识的物理脆弱性
- **隐喻**：算法需要载体，载体易逝

**时间性**：
- 图书馆是时间的对抗者
- 保存过去，连接未来
- 主角在此理解"传承"的意义

## 可植入场景

**Episode 300-400**（第1纪）：
- 萨洛贡到达亚历山大
- 第一次见识系统性知识保存
- "原来算法可以被写下来，被传递"

**Episode 4800**（第5纪）：
- 数字时代的"亚历山大图书馆"（互联网）
- 对比：物理易毁 vs 数字易逝（格式过时）
- "载体变了，脆弱性没变"
