# 素材编号：COST-010
# 素材名称：图灵迫害·算法英雄的悲剧
# 维度：算法代价（COST-010）

## 核心事件

```
时间：1952-1954年（第4纪末）
地点：英国曼彻斯特
人物：阿兰·图灵（Alan Turing）
事件：因同性恋被迫害，自杀身亡
```

## 代价类型

**系统性的忘恩负义**：
- 拯救国家的英雄被抛弃
- 算法的创造者在算法社会中没有容身之处
- "不同"成为被惩罚的理由

## 历史背景

**图灵的贡献**：
- 1936：图灵机，计算机科学理论基础
- 1940-1944：破解Enigma，缩短二战2年，拯救1400万生命
- 1945-1950：早期计算机研究，AI先驱

**迫害**：
- 1952：因同性恋被起诉（当时英国法律禁止）
- 选择：监狱 or 化学阉割（雌激素注射）
- 选择化学阉割
- 1954：咬下毒苹果，氰化物中毒身亡

## 算法拆解

### 社会的算法

**1950年代英国的法律算法**：
- 同性恋 = 犯罪
- 惩罚 = 监禁 or 化学阉割
- 逻辑严密，执行无情

**图灵的"错误"**：
- 不是计算错误，是"存在错误"
- 不符合社会算法的输入要求
- 被当作"异常"处理

### 讽刺

**图灵研究停机问题**：
- 算法无法判定某些程序是否会停止
- 社会无法容忍某些"不同"的存在
- 两者都是"不可判定"

**第73柱的呼应**：
- 萨尔贡选择"成为错误"（第99集）
- 图灵被迫成为"错误"（第4000集）
- 两代信使，同一种命运
