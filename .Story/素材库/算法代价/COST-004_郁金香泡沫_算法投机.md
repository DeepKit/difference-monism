
## 算法拆解

### 金融算法的早期形式

**期货合约**：
- 承诺未来以某价格购买郁金香球茎
- 不需要实际拥有，只需合约
- 算法：价格预期 × 时间价值

**杠杆与投机**：
- 小额定金控制大额合约
- 价格上涨时，收益放大
- 价格下跌时，亏损放大

**正反馈循环**：
```
价格上涨 → 更多人买入 → 价格继续上涨
                    ↓
              泡沫形成
                    ↓
恐慌性抛售 → 价格崩溃 → 更多人抛售
```
