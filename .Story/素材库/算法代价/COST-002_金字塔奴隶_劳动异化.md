# 素材编号：COST-002
# 素材名称：金字塔建造·劳动异化
# 维度：算法代价（COST-002）

## 核心事件

```
时间：公元前2580-2510年（第1纪早期）
地点：埃及吉萨
规模：230万块巨石，10-20万劳工，20年
代价：数千人死亡，终身奴役
```

## 代价类型

**劳动异化**：
- 几何算法用于精确压榨劳动
- 人的价值被降为"人力单位"
- 算法效率 vs 人的尊严

## 算法拆解

### 几何学的双重用途

**正面**：
- 精确的建筑设计
- 天文对齐（指向北极星）
- 工程奇迹

**负面**：
- 计算每一块石头的搬运路径（最优效率）
- 规划劳动队列（最小化空闲）
- 奴隶的"最优"使用（最大化产出）

### 算法的冷酷

**效率计算**：
- 每块石头需要多少"人日"
- 如何排列劳动班次（24小时不停工）
- 死亡率的"可接受"水平

**人的物化**：
- 劳工 = 可消耗的资源
- 死亡 = 成本的一部分
- 几何学的精确应用于人的压榨

## 可植入场景

**第1纪·第150集（第2时代：圆与圆周率）**

**场景**：
- 萨尔贡被征召参与建造（梦境/前世）
- 体验算法如何精确计算人的劳动
- 意识到：几何学可以用于压迫

**对话**：
> 工头："这块石头需要40人搬运，最优路径是那条斜坡。"
> 萨尔贡："如果有人倒下？"
> 工头："替换。算法已经计算了损耗率。"

**第5000集回收**：
- "每一个伟大的建筑，下面都埋着无名者的骨。"
- "算法不知道痛，但执行算法的人应该知道。"

## 与主线5（算法代价）的关联

**代价类型**：劳动异化  
**延续到现代**：
- 第3纪：工厂制度（时间异化）
- 第4纪：泰勒制（动作优化）
- 第5纪：算法管理（零工经济）

**模式**：
```
算法优化效率 → 人的价值被降低 → 异化与剥削
```

## 文学化描写

**建造现场**：
> "几何学在纸上是美的。但在烈日下，它是皮鞭，是绳索，是监工的吼叫。每一道完美的直线，都沾着汗与血。"

**萨尔贡的觉醒**：
> 他学会了用几何计算石头的角度。他也看到，那个角度决定了多少人的腰将被压断。算法没有道德，但使用算法的人应该有。
