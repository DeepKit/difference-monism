# 素材编号：COST-005
# 素材名称：工厂制度·时间异化
# 维度：算法代价（COST-005）

## 核心事件

```
时间：18-19世纪（第3-4纪）
地点：英国工业革命
算法：标准化时间 + 流水线 + 效率最大化
代价：人的生物节律被破坏，劳动异化
```

## 算法拆解

### 时间的算法化

**机械钟的普及**：
- 从"日出而作日落而息"到"朝九晚五"
- 时间被切割成标准单位
- 人的存在被时间规训

**泰勒制**：
- 动作研究：每个动作精确到秒
- 最优路径：消除一切"浪费"动作
- 人成为机器的附属

### 异化（Alienation）

**马克思的观察**：
- 工人与产品分离（不知道为谁生产）
- 工人与生产过程分离（重复单一动作）
- 工人与类本质分离（无法实现创造力）

## 可植入场景

**第3纪·第2800-2850集**
- 工人进入工厂，听到机器的轰鸣
- 12小时工作制，没有休息日
- 童工在纺织机下死去

**第5纪对比**：
- 算法管理（零工经济）
- 不是进步，是异化的新形式

**代价延续**：
- 第3纪：时间异化
- 第4纪：泰勒制
- 第5纪：算法管理、零工经济
