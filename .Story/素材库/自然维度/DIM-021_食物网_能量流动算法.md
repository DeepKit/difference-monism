# 素材编号：DIM-021
# 素材名称：食物网·能量流动算法
# 维度：生态网络（DIM-021）

## 核心算法

```
输入：太阳能
传递：生产者 → 初级消费者 → 次级消费者
效率：10%能量传递（90%损失）
输出：金字塔形的生物量分布
```

## 算法特征

**10%法则**：
- 能量每传递一级，损失90%
- 为什么食物链通常不超过4-5级
- 效率限制导致金字塔结构

**网络复杂性**：
- 不是简单链条，而是复杂网络
- 冗余连接增加稳定性
- 关键种（Keystone Species）的 disproportionate 影响

## 可植入场景

**各纪元的生态隐喻**：
- 第1纪：农业的兴起（人类进入食物链顶层）
- 第3纪：城市化（脱离自然食物网）
- 第5纪：人工生态系统（生态算法的人工化）

## 文学化隐喻

> "能量的河流，经过十层堤坝，只剩十分之一。这就是生命的代价。"
