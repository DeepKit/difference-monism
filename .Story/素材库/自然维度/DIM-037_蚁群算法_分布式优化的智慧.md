## 核心算法

```
蚁群优化（ACO）：
1. 蚂蚁随机探索
2. 找到食物后释放信息素
3. 信息素随时间挥发
4. 其他蚂蚁倾向于跟随信息素浓度高的路径
5. 最优路径信息素积累最快

正反馈 + 负反馈 = 优化
```

## 历史背景

- **时间**：蚁群进化于约1.4亿年前（第1纪）
- **算法发现**：1992年，马尔科·多里戈
- **灵感来源**：阿根廷蚁的觅食行为
- **应用**：路由优化、调度问题、网络设计

## 算法拆解

### 分布式智能

**没有中央控制**：
- 没有"蚁后"指挥
- 每只蚂蚁只遵循局部规则
- 集体涌现智能

**正反馈机制**：
- 好的路径吸引更多蚂蚁
- 信息素积累形成"记忆"
- 群体学习

### 算法的鲁棒性

**容错性**：
- 个体蚂蚁可以死亡
- 系统仍然运行
- "去中心化"的优势

**适应性**：
- 环境变化时快速适应
- 新路径的发现
- 动态优化
