# 维度：生态网络（DIM-022）

## 核心算法

```
输入：两个物种的互补需求
算法：互利共生（Mutualism）
输出：双赢的稳定均衡
例子：蜜蜂与花、地衣（藻+菌）
```

## 算法特征

**正和博弈**：
- 不是零和竞争
- 1+1 > 2的协同效应
- 算法的"合作"维度

**稳定均衡**：
- 背叛的代价高于合作
- 长期关系锁定
- 演化稳定的策略（ESS）

## 可植入场景

**第2纪**：
- 丝绸之路的贸易共生
- 文明的互利交流

**第5纪**：
- 人类与AI的共生关系
- 第5000集的选择：共生而非对抗
