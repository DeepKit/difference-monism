# 素材编号：DIM-004
# 素材名称：牛顿万有引力·宇宙算法
# 维度：物理算法（DIM-004）

## 核心算法

```
F = G * (m1 * m2) / r²
输入：两个物体的质量和距离
算法：平方反比律
输出：引力大小
```

## 历史背景

- **时间**：1687年（第4纪）
- **人物**：艾萨克·牛顿
- **来源**：开普勒定律 + 伽利略落体实验
- **意义**：天地统一的算法（苹果与月亮遵循同一法则）

## 算法特征

**普适性**：
- 从苹果到行星，同一公式
- 宇宙的"源代码"被破解
- 确定性的巅峰

**预测能力**：
- 预测哈雷彗星回归（1758年）
- 发现海王星（1846年，通过计算）
- 宇宙成为可计算的机器
