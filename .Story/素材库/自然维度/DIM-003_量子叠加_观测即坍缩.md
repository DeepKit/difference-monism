# 素材编号：DIM-003
# 素材名称：量子叠加·观测即坍缩
# 维度：物理算法（DIM-003）

## 核心算法

```
输入：量子系统（电子、光子）
算法：叠加态（|ψ⟩ = α|0⟩ + β|1⟩）
观测：选择其中一个本征态（概率 |α|² 或 |β|²）
输出：确定态（坍缩后）
```

## 历史背景

- **时间**：1925-1930年（第4纪末-第5纪初）
- **人物**：薛定谔、海森堡、玻尔、爱因斯坦
- **争议**：爱因斯坦-玻尔论战（"上帝掷骰子"）
- **意义**：确定性的终结，概率的本体化

## 算法拆解

### 叠加态的诡异

**薛定谔的猫**：
- 量子系统可以同时处于多个状态
- 观测前：既死又活
- 观测后：确定一个状态

**哥本哈根诠释**：
- 观测导致坍缩
- 观测者是量子系统的一部分
- 没有"客观的"独立实在

### 算法特征

**不确定性**：
- 无法同时知道位置和动量（海森堡不确定性）
- 不是测量误差，是本质属性
- 自由意志的物理学基础？

**与计算的关系**：
- 量子计算利用叠加态并行计算
- 量子纠缠实现超距通信
- 第5纪的量子算法时代

## 与主线的关联

**算法进化**：
- 从确定性（经典）到概率性（量子）
- 算法的本质随机性

**算法代价**：
- 失去确定性，人类失去"安全"感
- 无法预测，无法完全控制
