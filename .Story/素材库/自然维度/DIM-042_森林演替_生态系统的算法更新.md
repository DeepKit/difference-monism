## 核心算法

```
演替阶段：
1. 裸岩/裸地（先锋物种）
2. 草本阶段（一年生 → 多年生）
3. 灌木阶段
4. 乔木阶段（先锋树种）
5. 顶极群落（climax community）

驱动机制：
- 物种改变环境（土壤、光照、湿度）
- 新环境适合新物种
- 正反馈循环

干扰：
- 火灾、风暴、伐木重置演替
- 中等干扰假说（生物多样性最高）
```

## 历史背景

- **时间**：生态演替概念形成于19世纪末（第4纪）
- **先驱**：
  - 梭罗（1860s）：森林演替观察
  - 考尔斯（1899）：密歇根湖沙丘演替研究
  - 克莱门茨（1916）：顶极群落理论
- **修正**：顶极群落并非静态，持续变化

## 算法拆解

### 自组织过程

**无需设计者**：
- 物种竞争、共生、演替
- 生态系统自我组织
- 算法的"自发性"

**可预测性**：
- 给定条件，演替方向可预测
- 但具体时间、物种组成有随机性
- 确定性与随机性的平衡

### 干扰的必要性

**中等干扰假说**：
- 完全没有干扰：优势种垄断
- 过度干扰：无法建立群落
- 适度干扰：生物多样性最高
- "算法的重启机制"
