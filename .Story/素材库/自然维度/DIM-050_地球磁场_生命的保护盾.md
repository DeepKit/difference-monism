## 核心算法

```
地磁发电机理论：
1. 外核液态铁镍对流
2. 科里奥利力使流动螺旋化
3. 自激发电机效应
4. 产生偶极磁场

磁场特征：
- 磁极会移动（现在快速北移）
- 地磁倒转（平均~20万年一次）
- 上一次倒转：~78万年前（布容期）

保护作用：
- 偏转太阳风（带电粒子）
- 保护大气不被剥离
- 屏蔽宇宙射线
```

## 历史背景

- **时间**：地磁场存在至少35亿年（第1纪）
- **人类认知**：
  - 古代中国：司南（指南针前身）
  - 11世纪：指南针用于航海
  - 1600年：吉尔伯特《论磁》
- **现代研究**：
  - 1906年：布容发现地磁倒转
  - 1960s：地磁发电机理论建立
- **倒转周期**：不规律，最短~1万年，最长~5000万年

## 算法拆解

### 行星的护盾

**没有磁场的后果**：
- 火星：磁场消失 → 大气剥离 → 干燥寒冷
- 金星：没有磁场，但质量大，保留大气
- 地球：磁场保护生命

**倒转的影响**：
- 磁场减弱期间，更多宇宙射线
- 可能影响生物演化
- 算法的"重启"

### 导航的算法

**生物磁感应**：
- 候鸟、海龟、蜜蜂利用地磁导航
- 磁小体（magnetite）
- 生物算法的量子效应

**人类利用**：
- 指南针改变世界
- 导航技术的起点
- 从自然算法到人工算法
