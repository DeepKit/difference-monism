## 核心算法

```
板块构造：
- 地幔对流驱动板块移动
- 汇聚边界：碰撞 → 造山
- 离散边界：张裂 → 海洋
- 转换边界：剪切 → 地震

造山过程：
1. 大陆碰撞（印度-欧亚，~5000万年前）
2. 地壳压缩、褶皱、断裂
3. 岩石变质、岩浆侵入
4. 持续数百万年
5. 侵蚀与隆升平衡

喜马拉雅：
- 每年上升~5-10mm
- 珠穆朗玛峰仍在升高
```

## 历史背景

- **时间**：板块构造论建立于1960s（第5纪）
- **先驱**：
  - 魏格纳（1912）：大陆漂移假说
  - 赫斯（1960s）：海底扩张
  - 麦肯齐、帕克：板块构造理论
- **证据**：
  - 海底磁条带
  - 地震分布
  - 大陆轮廓匹配

## 算法拆解

### 地质时间尺度

**人类无法感知**：
- 山脉形成需要数百万年
- 一次地震只是一瞬间
- 算法的"慢变量"

**持续变化**：
- 喜马拉雅仍在上升
- 阿尔卑斯、安第斯山仍在形成
- 算法的"进行时"

### 地球引擎

**地幔对流**：
- 地球内部的热量驱动
- 放射性衰变 + 原始热
- "地球的算法核心"

**碳循环**：
- 造山运动加速岩石风化
- CO₂被吸收，气候变冷
- 地质尺度的气候调节
