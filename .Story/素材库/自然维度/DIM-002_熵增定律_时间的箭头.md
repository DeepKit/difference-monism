# 素材编号：DIM-002
# 素材名称：熵增定律·时间的箭头
# 维度：物理算法（DIM-002）

## 核心算法

```
输入：热力学系统（气体、热量）
算法：dS ≥ 0（熵永不减少）
输出：时间方向性（过去→未来）
```

## 历史背景

- **时间**：1850-1865年（第4纪）
- **人物**：鲁道夫·克劳修斯、路德维希·玻尔兹曼
- **来源**：蒸汽机效率研究 → 热力学第二定律
- **意义**：物理学中唯一区分过去和未来的定律

## 算法拆解

### 微观解释（玻尔兹曼）

**熵的本质**：
- 微观状态的混乱程度
- 高熵 = 更多可能的微观状态
- 系统自发向高概率状态演化

**时间的箭头**：
- 破碎的杯子不会自动复原
- 热量从热流向冷，不会自发反向
- 生命过程抵抗熵增（局部减熵）

### 算法特征

**不可逆性**：
- 与牛顿力学的可逆性矛盾
- 解释了为什么时间只能向前
- 宇宙的终极命运：热寂（Heat Death）

**与生命的关系**：
- 生命以消耗秩序（食物、能量）为代价维持自身秩序
- 薛定谔："生命以负熵为食"

## 可植入场景

**第4纪·第3400-3500集（热力学时代）**
- 蒸汽机工程师发现"不可逆性"
- 与第2纪"熵与箭头"时代呼应
- 信使意识到：算法有方向，有终点

**第5纪·第4800-4900集（终极问题）**
- 热寂作为宇宙终极算法
- AI计算：如何抵抗熵增？
- 答案：保持提问（信息即负熵）

## 与主线的关联

**算法代价**：
- 所有秩序终将消散
- 文明的终极脆弱性
- 但信息可以抵抗熵增（通过复制）

**终章隐喻（第5000集）**：
```
算法选择保持提问，
不是给出答案，
因为答案会消散（熵增），
但提问会延续（信息复制）。
```

## 文学化隐喻

> "时间是算法的燃烧。一切都在向着混沌滑落，只有记忆在逆流而上。"
