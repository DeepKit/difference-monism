## 核心算法

```
撞击过程：
1. 陨石进入大气（速度：11-72 km/s）
2. 大气摩擦 → 发光（火球）
3. 撞击地表（能量 = 1/2 mv²）
4. 冲击波、 crater 形成
5. 全球性影响（大撞击）

白垩纪末撞击：
- 时间：~6600万年前
- 地点：墨西哥尤卡坦半岛（希克苏鲁伯陨石）
- 直径：~10-15 km
- 能量：~100万亿吨TNT
- 后果：恐龙灭绝、全球冬天
```

## 历史背景

- **时间**：1980年，阿尔瓦雷斯父子提出撞击假说（第5纪）
- **证据**：
  - 全球铱异常层（陨石富含铱）
  - 冲击石英
  - 陨石坑发现（1991年确认）
- **其他撞击**：
  - 通古斯大爆炸（1908年）
  - 车里雅宾斯克（2013年）
  - 巴林杰陨石坑（5万年前）
- **威胁**：近地天体监测计划

## 算法拆解

### 算法的随机性

**非生物因素**：
- 生物演化被随机事件打断
- "算法的外部输入"
- 不可预测性

**筛选作用**：
- 大灭绝改变演化方向
- 小型、适应性强的物种存活
- 算法的"压力测试"

### 人类的脆弱

**通古斯级别**：
- 如果发生在城市，毁灭性后果
- 现代文明对单一事件的脆弱
- "算法的系统性风险"

**防御计划**：
- DART任务（2022年）：改变小行星轨道
- 人类开始防御"算法中断"
- 从被动到主动
