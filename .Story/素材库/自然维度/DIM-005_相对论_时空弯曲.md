# 素材编号：DIM-005
# 素材名称：相对论·时空弯曲
# 维度：物理算法（DIM-005）

## 核心算法

```
E = mc²
时空告诉物质如何运动，物质告诉时空如何弯曲。
```

## 历史背景

- **时间**：1905（狭义）-1915（广义）（第4纪末）
- **人物**：阿尔伯特·爱因斯坦
- **突破**：引力不是力，是时空弯曲
- **验证**：1919年爱丁顿日食观测

## 算法特征

**相对性**：
- 没有绝对时间，没有绝对空间
- 观测者的运动状态影响测量
- 同时性是相对的

**弯曲的时空**：
- 质量弯曲时空（像重物压在弹簧床上）
- 光线在引力场中偏折
- 黑洞：时空的极端弯曲

## 可植入场景

**第4纪·第3850-3900集**
- 爱因斯坦的"思想实验"
- 追逐光束的想象
- 信使意识到：算法是相对的

**代价**：
- 确定性的进一步丧失
- 人类在宇宙中的位置更加渺小
- 但打开了新的可能性（核能）

## 文学化隐喻

> "质量弯曲了时空，就像爱情弯曲了时间。靠近你，时间变慢。"
