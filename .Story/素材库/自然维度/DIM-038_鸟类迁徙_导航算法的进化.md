## 核心算法

```
导航机制：
1. 太阳罗盘（白天）
2. 星辰罗盘（夜晚）
3. 地磁感应（磁小体/隐花色素）
4. 地标记忆（视觉地图）
5. 嗅觉线索（梯度追踪）

迁徙路线：
- 遗传编码 + 学习改进
- 多代优化路径
- 群体导航（V形队列减少阻力）
```

## 历史背景

- **时间**：鸟类进化迁徙于约5000万年前（第1纪）
- **研究历史**：
  - 1822年：白鹳带箭证明迁徙
  - 1950s：克雷默的旋转笼实验（太阳罗盘）
  - 1970s：威尔施科的磁感应研究
- **最远距离**：北极燕鸥每年往返南极-北极，~70,000公里

## 算法拆解

### 多模态导航系统

**太阳罗盘**：
- 根据太阳位置和时间确定方向
- 内部时钟的校准
- 阴天时的备用方案

**地磁感应**：
- 隐花色素蛋白的量子效应
- 感知磁场强度和倾角
- "内置GPS"

### 群体智能

**V形编队**：
- 减少空气阻力
- 轮流领头（节省体力）
- 通信协调

**遗传与学习**：
- 第一次迁徙靠本能
- 后续改进路线
- 代际知识传递
