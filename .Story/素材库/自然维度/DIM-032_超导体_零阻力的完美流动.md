## 核心算法

```
超导临界条件：
T < Tc（临界温度）
电阻 R = 0

迈斯纳效应：
磁场被完全排斥（B = 0 inside）

库珀对：
电子配对形成玻色子，无阻碍流动
```

## 历史背景

- **时间**：1911年发现，1957年理论解释（第4纪末-第5纪初）
- **人物**：卡末林·昂尼斯（发现），巴丁-库珀-施里弗（理论，BCS理论）
- **发现**：汞在4.2K时电阻突降为零
- **应用**：MRI、磁悬浮、粒子加速器

## 算法拆解

### 秩序的涌现

**宏观量子现象**：
- 通常量子效应只在微观尺度
- 超导体中，库珀对宏观相干
- "量子的大尺度化"

**无损耗传输**：
- 电流可以永远流动
- 磁悬浮的实现基础
- 未来能源传输的希望

### 高温超导的未解之谜

**1986年突破**：
- 铜氧化物超导体（Tc > 77K，液氮温度）
- BCS理论无法解释
- "算法"尚未被完全理解
