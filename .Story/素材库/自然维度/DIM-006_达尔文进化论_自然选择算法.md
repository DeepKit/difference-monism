# 素材编号：DIM-006
# 素材名称：达尔文进化论·自然选择算法
# 维度：生物进化（DIM-006）

## 核心算法

```
输入：遗传变异 + 环境压力
算法：自然选择（适者生存）
输出：适应性进化
优化目标：生存与繁殖成功率
```

## 历史背景

- **时间**：1859年《物种起源》（第4纪）
- **人物**：查尔斯·达尔文
- **来源**：加拉帕戈斯群岛的雀鸟观察
- **意义**：生命是算法，不需要设计者

## 算法特征

**无目的优化**：
- 没有"进化方向"，只有局部适应
- 突变是随机的，选择是必然的
- 红皇后效应：必须不断奔跑才能留在原地

**效率**：
- 从单细胞到人类，38亿年
- 并行计算（所有生物同时进化）
- 分布式优化（没有中央控制）

## 可植入场景

**第4纪·第3350-3400集**
- 达尔文的小猎犬号航行
- 雀鸟喙形的多样性
- 信使意识到：生命是算法

**代价**：
- 人类不再是"神的设计"
- 社会达尔文主义的滥用
- 伦理价值的危机

## 文学化隐喻

> "没有设计者，只有选择。生命在试错中，写下了自己的代码。"
