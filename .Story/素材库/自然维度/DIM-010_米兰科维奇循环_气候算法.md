# 素材编号：DIM-010
# 素材名称：米兰科维奇循环·气候算法
# 维度：地质时间（DIM-010）

## 核心算法

```
输入：地球轨道参数（偏心率、倾角、岁差）
算法：周期性变化（10万年、4.1万年、2.3万年周期）
输出：冰期-间冰期交替
```

## 历史背景

- **米兰科维奇**（1920s）：塞尔维亚数学家
- **预测**：冰期循环的天文原因
- **验证**：深海沉积物证实（1970s）

## 算法特征

**周期性**：
- 10万年周期：偏心率
- 4.1万年周期：倾角
- 2.3万年周期：岁差

**气候算法**：
- 天文输入 → 气候输出
- 可预测未来冰期
- 与第5纪气候变化呼应
