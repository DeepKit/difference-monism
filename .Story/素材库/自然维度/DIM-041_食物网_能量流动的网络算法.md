# 素材编号：DIM-041
# 素材名称：食物网·能量流动的网络算法
# 维度：生态网络（DIM-041）

## 核心算法

```
能量流动：
太阳 → 生产者（植物）→ 初级消费者 → 次级消费者 → ...

效率：
- 能量传递效率：~10%（林德曼效率）
- 90%能量在每一级散失（呼吸、排泄）
- 食物链长度限制：通常4-5级

网络特征：
- 多路径冗余
- 关键种（keystone species）
- 级联效应（cascading effects）
```

## 历史背景

- **时间**：生态学形成于20世纪初（第4纪末）
- **关键人物**：
  - 林德曼（1942）：能量流动研究
  - 奥德姆兄弟：生态系统理论
  - 皮姆：食物网复杂性研究
- **应用**：生态保护、渔业管理、生物防治

## 算法拆解

### 能量经济学

**10%法则**：
- 为什么老虎数量少？
- 为什么素食可以养活更多人？
- 生态系统的"能量预算"

**网络鲁棒性**：
- 多路径连接提供冗余
- 某些物种的消失可以被补偿
- 但关键种的消失导致崩溃

### 级联效应

**海獭-海胆-海藻**：
- 海獭消失 → 海胆暴增 → 海藻林消失
- 单一物种的连锁影响
- 算法的敏感性

**人类影响**：
- 过度捕捞打破平衡
- 入侵物种扰乱网络
- 食物网的算法重构
