## 核心算法

```
珊瑚建造过程：
1. 珊瑚虫分泌碳酸钙（CaCO₃）
2. 形成钙质骨骼
3. 死亡后骨骼积累
4. 新珊瑚在旧骨骼上生长
5. 数万年形成礁体

共生关系：
- 珊瑚 + 虫黄藻（光合作用提供能量）
- 珊瑚礁生态系统（25%海洋物种依赖）

威胁：
- 海洋酸化（CO₂ → H₂CO₃ → pH下降）
- 温度上升（珊瑚白化）
```

## 历史背景

- **时间**：现代珊瑚出现于2.4亿年前（第1纪）
- **大堡礁**：约5000年历史，世界最大生物结构
- **生态系统服务**：
  - 海岸保护（减弱波浪能量）
  - 渔业资源
  - 旅游业
- **现状**：全球50%珊瑚礁已消失或退化
