# 素材编号：DIM-012
# 素材名称：光合作用·能量捕获算法
# 维度：生态网络（DIM-012）

## 核心算法

```
输入：阳光 + CO₂ + H₂O
算法：叶绿体光反应 + 暗反应
输出：葡萄糖 + O₂
效率：约3-6%（理论极限11%）
```

## 算法特征

**生命的基础**：
- 几乎所有能量来自光合作用
- 食物链的起点
- 氧气的来源

**效率限制**：
- 为什么只有3-6%？
- 光饱和、呼吸损耗
- 算法的物理极限

## 可植入场景

**第3纪**：农业的本质（人工光合作用优化）
**第4纪**：工业革命（化石燃料=古代光合作用储存）
**第5纪**：人工光合作用（突破自然极限？）
