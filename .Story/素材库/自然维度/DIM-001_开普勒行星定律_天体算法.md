# 维度：物理算法（DIM-001）

## 核心算法

```
输入：第谷·布拉赫的观测数据（20年）
算法：椭圆轨道拟合 + 面积速度守恒
输出：三大定律（轨道定律、面积定律、周期定律）
```

## 历史背景

- **时间**：1609-1619年（第4纪）
- **人物**：约翰内斯·开普勒
- **数据**：第谷·布拉赫20年的精密观测（误差<2角分）
- **突破**：从圆形轨道到椭圆轨道的范式转变

## 算法拆解

### 数据驱动发现

**第谷的贡献**：
- 肉眼观测精度达到望远镜时代前最高
- 系统记录火星位置（误差<2角分）
- 数据量：数千个观测点

**开普勒的方法**：
1. 假设圆形轨道 → 拟合失败（8角分误差）
2. 放弃圆形，尝试椭圆 → 拟合成功
3. 归纳三大定律

### 算法特征

**从公理到数据**：
- 古希腊：圆形轨道（美学公理）
- 开普勒：数据决定形状（经验主义）

**预测能力**：
- 可以精确预测行星位置
- 为牛顿万有引力奠定基础
