## 核心算法

```
形成机制：
1. 海陆热力差异（比热容不同）
2. 夏季：海洋→陆地（湿润）
3. 冬季：陆地→海洋（干燥）
4. 青藏高原地形增强

亚洲季风系统：
- 印度季风（6-9月）
- 东亚季风（6-8月）
- 决定数亿人的农业生产

变化：
- 厄尔尼诺影响
- 气候变化导致不稳定
```

## 历史背景

- **时间**：季风存在约4000万年（第1纪）
- **文明影响**：
  - 印度河流域文明（依赖印度季风）
  - 中国古代农业（二十四节气）
  - 印度尼西亚的香料贸易（季风航海）
- **词源**：阿拉伯语"mausim"（季节）

## 算法拆解

### 热力驱动

**海陆差异**：
- 水比热容大，升温慢
- 陆地比热容小，升温快
- 夏季形成低压（陆地）→ 海洋风

**地形放大**：
- 青藏高原的热力作用
- "世界屋脊"影响全球气候
- 地理的算法重要性

### 文明的算法

**农业依赖**：
- 印度：季风决定粮食产量
- 中国：南稻北麦的季风农业
- "靠天吃饭"的算法

**不稳定的风险**：
- 季风失败 → 饥荒
- 季风过强 → 洪灾
- 算法的代价
