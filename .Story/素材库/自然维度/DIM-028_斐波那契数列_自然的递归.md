# 维度：物理算法（DIM-028）

## 核心算法

```
F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2)  [递归定义]

黄金比例：φ = (1+√5)/2 ≈ 1.618
```

## 历史背景

- **时间**：1202年（第2纪）
- **人物**：莱昂纳多·斐波那契（比萨的莱昂纳多）
- **来源**：《计算之书》(Liber Abaci)
- **背景**：从印度-阿拉伯数字系统引入欧洲

## 算法拆解

### 递归之美

**数学特征**：
- 每一项都是前两项之和
- 隐含递归结构
- 与黄金比例的深刻联系

**自然出现**：
- 向日葵花盘的螺旋数（34, 55, 89）
- 鹦鹉螺壳的生长曲线
- 树枝分叉的角度

### 算法的普遍性

**跨文化发现**：
- 印度：冈比塔关于韵律的研究（早于斐波那契）
- 中国：杨辉三角中的对角线
- 自然：不需要人类发明的算法
