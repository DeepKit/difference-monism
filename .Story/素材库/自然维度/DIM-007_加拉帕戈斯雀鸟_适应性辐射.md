# 素材编号：DIM-007
# 素材名称：加拉帕戈斯雀鸟·适应性辐射
# 维度：生物进化（DIM-007）

## 核心算法

```
输入：共同的祖先 + 不同的生态位
算法：适应性辐射（分化适应）
输出：14种不同喙形的雀鸟
优化：最大化各生态位的资源利用
```

## 历史背景

- **时间**：1835年观察，达尔文1859年发表（第4纪）
- **地点**：加拉帕戈斯群岛
- **现象**：同一祖先演化出14种不同雀鸟
- **意义："适应性辐射"的经典案例

## 算法特征

**分化而非进步**：
- 不是从简单到复杂的"进步"
- 而是对特定环境的"适应"
- 没有"更高级"，只有"更合适"

**并行优化**：
- 14种不同的优化解
- 针对不同的食物来源（种子、昆虫、仙人掌）
- 算法的多样性

## 可植入场景

**第4纪·第3360集（与达尔文素材配合）**
- 达尔文观察雀鸟的细节
- 从怀疑到顿悟的过程
- 信使看到：算法会产生多样性

**代价**：
- 特化导致脆弱性（环境变化时易灭绝）
- 没有"完美"的解，只有"暂时合适"的解

## 文学化隐喻

> "同一首曲子，在不同的岛上，演奏出了14个变奏。"
