## 核心算法

```
曼德勃罗集合：
Zₙ₊₁ = Zₙ² + C

如果 |Z| ≤ 2 在迭代中始终成立，则C属于集合

特征：
- 无限复杂的边界
- 任意放大的自相似性
- 分数维度（非整数维度）
```

## 历史背景

- **时间**：1975年（第5纪）
- **人物**：本华·曼德勃罗
- **著作**：《大自然的分形几何学》
- **突破**：维度可以是分数

## 算法拆解

### 无限之美

**数学特征**：
- 有限的面积，无限的周长
- 自相似性（部分与整体相似）
- 分形维度（1 < D < 2）

**自然界中的分形**：
- 海岸线、山脉、云朵
- 血管系统、支气管树
- 闪电、雪花、河流三角洲

### 算法的递归本质

**生成规则**：
- 简单的递归公式
- 产生无限复杂
- "复杂性来自简单规则的重复"
