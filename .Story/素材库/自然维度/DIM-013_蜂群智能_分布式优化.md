## 核心算法

```
个体规则：
1. 侦察蜂随机搜索
2. 发现蜜源跳8字舞（距离+方向编码）
3. 跟随蜂按舞蹈强度选择
4. 动态调整，最优解收敛
```

## 算法特征

**分布式优化**：
- 无中央控制
- 个体简单规则
- 群体智慧涌现

**人工应用**：
- 蜂群算法（优化问题）
- 路由优化、任务分配
- 仿生计算
