# 素材编号：TRANS-001
# 素材名称：第一纪→第二纪·口传算法的文字化
# 维度：纪间传承（TRANS-001）

## 传承事件

```
断裂点：公元前300年前后
传承方向：口耳相传的算法知识 → 莎草纸/竹简的成文系统
关键载体：亚历山大图书馆、秦汉典籍、印度吠陀数学
```

## 传承算法逻辑

```
输入：分散在祭司、工匠、占星师脑中的计算方法
过滤条件：只有被权力认可的知识才能被记录
输出：标准化、可复制、可传授的算法文本
代价：大量口传知识在文字化过程中失真或消亡
```

## 叙事核心

**什么被保留了**：
- 几何测量（土地丈量→欧几里得）
- 天文历法（祭祀需要→精确计算）
- 商业算术（税收需要→标准化）

**什么被丢失了**：
- 女性祭司的草药配方算法
- 游牧民族的迁徙路径算法
- 口传史诗中的记忆宫殿技术

**关键张力**：
文字化是算法的解放，也是算法的第一次囚禁——
只有统治者需要的算法才值得被记录。

## 可用场景

- 亚历山大图书馆的整理者面对浩如烟海的莎草纸
- 秦始皇焚书坑儒：哪些算法被烧掉了？
- 一个祭司决定把秘密写下来，还是带进坟墓
