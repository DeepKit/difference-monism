# 素材编号：TRANS-002
# 素材名称：第二纪→第三纪·算法从神殿走向市场
# 维度：纪间传承（TRANS-002）

## 传承事件

```
断裂点：公元1000年前后（伊斯兰黄金时代→欧洲文艺复兴）
传承方向：宗教/帝国垄断的算法知识 → 商业与印刷术的民主化扩散
关键媒介：阿拉伯数字西传、花拉子密《代数学》拉丁译本、印刷机
```

## 传承算法

- **保留**：欧几里得几何、托勒密天文算法、阿拉伯代数
- **变异**：算法从"神圣计算"变为"商业工具"（复式记账、航海导航）
- **断裂**：大量希腊原典在亚历山大图书馆焚毁后失传，仅存阿拉伯译本

## 叙事碎片

> "花拉子密不知道，他写下的那本书会在三百年后被翻译成拉丁文，然后再过三百年，变成欧洲每一个商人的算盘。"

> "知识的传承从来不是复制，而是翻译。每一次翻译都是一次背叛，也是一次新生。"

## 关键张力

- 谁有权翻译算法？翻译即诠释，诠释即权力
- 伊斯兰学者保存了希腊算法，却在欧洲文艺复兴中被遗忘
- 印刷术让算法第一次脱离了"守门人"

## 可用场景

- 第二纪末：某位阿拉伯学者将希腊手稿译成阿拉伯文的夜晚
- 第三纪初：欧洲商人第一次用阿拉伯数字记账，感到某种陌生的力量
