# 素材编号：MATH-002
# 素材名称：阿基米德穷竭法·极限思想
# 算法类型：逼近算法 / 数值积分 / 无穷小雏形

## 核心算法逻辑

```
目标：计算圆的面积（曲线 → 直线）
方法：内接多边形逼近
算法：n边形 → 2n边形 → ... → 极限
输出：πr²（通过无穷过程定义）
```

## 历史背景

- **时间**：公元前3世纪
- **来源**：《圆的度量》
- **突破**：首次严格处理"无穷"
- **精度**：π ≈ 3.1408 < π < 3.1429

## 算法拆解

### 穷竭法算法

**迭代过程**：
```
Step 0: 内接六边形（6边）
Step 1: 六边形 → 十二边形
Step 2: 十二边形 → 二十四边形
...
Step n: 面积序列收敛于圆面积
```

**收敛性证明**：
- 每次迭代，面积增加
- 但始终 < 圆面积（上界）
- 根据**穷竭原理**：差值可任意小

### 数值积分

**现代对应**：
- 黎曼和 / 梯形法则
- 用离散逼近连续
- **误差分析**：O(1/n²) 收敛速度

### 无穷小的困境

**阿基米德的谨慎**：
- 使用穷竭法，避免直接处理无穷小
- "让差值小于任意给定量"
- **ε-δ语言的雏形**（2000年前！）

**后来者的懒惰**：
- 微积分时代的"无穷小"滥用
- 直到19世纪才严格化（魏尔斯特拉斯）
