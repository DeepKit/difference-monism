# 素材编号：CHAR-020
# 素材名称：算法难民·被替代者
# 所属纪：第五纪（2000年 ~ 未来）
# 角色类型：普通工人 / 时代牺牲品 / 算法受害者

## 核心特质

**社会身份**：
- 曾是熟练工人（会计/司机/客服/工厂工人）
- 被自动化算法彻底替代
- 无法再融入新经济体系

**性格特征**：
- 不是坏人，只是被时代抛弃
- 愤怒但无处发泄
- 对"算法"的恐惧是具体的、日常的

**悲剧元素**：
- 不是英雄，是统计数字
- 代表算法进步的"外部成本"
- 符号：空工厂 / 失业通知书

## 算法拆解

**核心算法逻辑**：自动化替代 = 效率优化 + 成本最小化，但人的价值无法被量化

**历史对应**：
- 工业革命纺织工人（卢德运动）
- 2010年代优步司机 vs 自动驾驶
- 2020年代 ChatGPT vs 内容创作者

## 可植入场景

- 与ASTO的对话：你让我失业了
- 与技术官僚的冲突：效率 vs 尊严
- 内心独白：我的技能是算法，但算法不需要我

## 文学化隐喻

> 他不是被打败的，他只是被跳过了。算法不需要消灭他，只需要忽略他。
