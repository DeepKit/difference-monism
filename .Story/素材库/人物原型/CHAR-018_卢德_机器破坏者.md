# 素材编号：CHAR-018
# 素材名称：卢德·机器破坏者
# 所属纪：第三纪（1811年 ~ 1816年）
# 角色类型：工人领袖 / 时代异化者 / 算法受害者

## 核心特质

**社会身份**：
- 纺织工人，手工技艺传承者
- 卢德运动精神领袖（内德·卢德传说）
- 工业算法的第一批反抗者

**性格特征**：
- 不反对技术本身，反对技术被用于剥削
- 身份认同危机：当双手的价值被机器取代
- 悲剧性：反抗无效，历史车轮碾过

**算法隐喻**：
- 人体算法（手工技艺）vs 机器算法（蒸汽织机）
- 效率的胜利 = 人的失败？
- 第一次"算法失业"浪潮

## 可植入场景

- 维多利亚蒸汽工厂（SCENE-004）对立面
- 与莱布尼茨（CHAR-004）的跨时代对话：机器是解放还是奴役？
- 第三纪→第五纪的历史回响：AI失业潮的先声

## 核心台词方向

> "我不恨机器。我恨的是，机器让我的孩子挨饿，却让工厂主更富。"
