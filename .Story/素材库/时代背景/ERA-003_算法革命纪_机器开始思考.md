# 素材编号：ERA-003
# 素材名称：算法革命纪·机器开始思考
# 所属纪：第三纪（1000年 ~ 1900年）

## 时代概貌

```
时间跨度：公元1000年 ~ 1900年
核心转变：算法从人脑迁移到纸张，再迁移到机器
权力结构：资本 + 教会 + 民族国家 = 三方争夺算法控制权
技术水平：印刷术、微积分、蒸汽机、差分机
```

## 时代算法逻辑

```
输入：工业生产需求 + 科学革命的知识爆炸
约束：算法效率必须服务于资本积累
算法：机械化 + 标准化 + 可重复执行
输出：工业文明 + 算法与人体的第一次分离
副作用：卢德运动 = 人类第一次对算法的集体反抗
```

## 核心矛盾

**算法脱体**：
- 巴贝奇差分机：算法第一次不需要人来执行
- 印刷术：算法知识的第一次大规模民主化
- 郁金香泡沫：金融算法第一次脱离实体经济

**三大断裂**：
1. 手工 vs 机器：人类算法执行者被替代
2. 宗教 vs 科学：神圣算法被世俗算法取代
3. 个人 vs 系统：算法开始大于个人
