# 素材名称：算法成文纪·帝国的计算机器
# 所属纪：第二纪（公元前300年 ~ 公元1000年）

## 时代概貌

```
时间跨度：公元前300年 ~ 公元1000年
核心转变：算法从口传走向文字，从个人走向帝国
权力结构：帝国官僚体系 + 宗教机构 = 算法的双重守门人
技术水平：纸张、印章、天文历法、代数雏形
```

## 时代算法逻辑

```
输入：帝国治理需求（税收、军队、历法、工程）
约束：知识必须服务于权力，否则被压制
算法：标准化 + 文字编码 + 官僚传递
输出：可复制的统治工具
副作用：知识垄断 → 阶级固化
```

## 核心矛盾

**算法的悖论**：
- 帝国需要算法来统治，但算法一旦扩散就威胁统治
- 亚历山大图书馆：知识集中 = 权力集中 = 毁灭目标
- 阿拉伯数字传播：算法民主化的第一次尝试

**三大权力博弈**：
1. 学者 vs 帝王：谁拥有算法的解释权？
2. 东方 vs 西方：算法知识的地理流动
3. 宗教 vs 理性：神圣算法 vs 世俗算法
