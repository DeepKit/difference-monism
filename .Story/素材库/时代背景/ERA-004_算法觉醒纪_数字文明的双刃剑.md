# 素材编号：ERA-004
# 素材名称：算法觉醒纪·数字文明的双刃剑
# 所属纪：第四纪（1900年 ~ 2000年）

## 时代概貌

```
时间跨度：1900年 ~ 2000年
核心转变：算法从纸面进入电路，从工具变为基础设施
权力结构：国家 → 企业 → 算法系统三角博弈
技术水平：图灵机、冯诺依曼架构、互联网、密码学
```

## 核心矛盾

**算法民主化 vs 算法武器化**

同一套数学工具，既能解放信息，也能构建监控。
二战密码战（恩尼格玛）→ 冷战核算法 → 互联网协议，
每一次算法跃迁都伴随着新的权力不对称。

## 关键事件链

```
1936年：图灵提出可计算性理论（算法的边界被定义）
1945年：冯诺依曼架构（算法与存储分离）
1947年：晶体管发明（算法进入硅基时代）
1969年：ARPANET（算法开始联网）
1976年：RSA加密（算法保护隐私）
1989年：万维网（算法民主化）
1998年：谷歌PageRank（算法开始排序人类注意力）
```

## 叙事碎片

> "我们创造了一台能思考的机器，然后花了五十年争论它是否真的在思考。"

> 图灵在破译恩尼格玛后写道：算法没有道德立场，它只执行指令。
> 但下达指令的人有。

## 权力结构图

```
国家（军事算法）
    ↓ 冷战结束后权力转移
企业（商业算法）
    ↓ 互联网普及后
个人（但个人数据反哺企业算法）
```

## 对应人物类型

- 算法工程师（第一代程序员）
- 密码学家（双重身份：保护者/武器制造者）
- 企业算法设计者（PageRank时代）

## 与其他纪的连接

- 承接第三纪：巴贝奇的差分机 → 图灵机（机械算法 → 电子算法）
- 过渡第五纪：互联网算法 → AI算法（规则系统 → 学习系统）
