# 素材名称：孟德尔豌豆实验·遗传定律
# 维度：遗传编码（DIM-106）

## 核心算法

```
输入：豌豆的7对相对性状
算法：分离定律 + 自由组合定律
输出：3:1 和 9:3:3:1 的性状分离比
```

## 历史背景

- **时间**：1856-1863年（第4纪）
- **地点**：奥地利布鲁恩（今捷克布尔诺）修道院
- **人物**：格雷戈尔·孟德尔（修道士）
- **规模**：28,000株豌豆，7年实验
- **被忽视**：1866年发表，1900年才被重新发现

## 算法拆解

### 分离定律

**算法**：
- 性状由成对的因子（基因）控制
- 形成配子时，成对因子分离
- 受精时，随机组合

**输出**：
- F2代显性与隐性比例为3:1
- 完美符合概率计算

### 自由组合定律

**算法**：
- 不同性状的遗传相互独立
- 两对性状：9:3:3:1的比例
- 算法组合爆炸（2^n种可能）
