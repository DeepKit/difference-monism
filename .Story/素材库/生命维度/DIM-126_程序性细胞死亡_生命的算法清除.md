# 素材名称：程序性细胞死亡·生命的算法清除
# 维度：衰老死亡（DIM-126）

## 核心算法

```
凋亡（Apoptosis）：
1. 内源途径：线粒体释放细胞色素c
2. 外源途径：死亡受体激活
3. Caspase级联激活
4. 细胞收缩、DNA断裂
5. 被吞噬细胞清除（无炎症）

坏死（Necrosis）：
- 非程序性，损伤性
- 细胞膜破裂
- 引起炎症

自噬（Autophagy）：
- 细胞自我吞噬
- 回收成分
- 细胞存活或死亡
```

## 历史背景

- **时间**：凋亡研究始于1960s-1970s（第5纪）
- **关键发现**：
  - Kerr（1972）：提出"凋亡"概念
  - Horvitz（1980s-90s）：线虫凋亡基因（诺贝尔奖2002）
  - 半胱天冬酶（Caspase）的发现
- **生理意义**：
  - 胚胎发育（手指形成）
  - 免疫系统（自身反应性细胞清除）
  - 组织稳态
- **疾病**：
  - 凋亡不足：癌症
  - 凋亡过度：神经退行、自身免疫
