# 素材名称：微生物组·第二基因组的算法
# 维度：免疫防御（DIM-127）

## 核心算法

```
人体微生物：
- 数量：~39万亿（与人体细胞相当）
- 基因：~330万基因（人体2万基因的150倍）
- 重量：~1-2kg

功能：
1. 代谢：消化纤维、合成维生素
2. 免疫：训练免疫系统
3. 保护：竞争排斥病原体
4. 脑-肠轴：影响情绪、认知

稳态：
- 菌群多样性
- 关键菌种
- 失调（dysbiosis）→ 疾病
```

## 历史背景

- **时间**：微生物组研究爆发于2000s（第5纪）
- **里程碑**：
  - 2007年：人类微生物组计划启动
  - 2012年：粪菌移植治疗艰难梭菌
  - 2013年：脑-肠轴研究兴起
- **发现**：
  - 个体间菌群差异巨大
  - 与肥胖、糖尿病、自闭症、抑郁相关
- **应用**：
  - 益生菌、益生元
  - 粪菌移植
  - 个性化营养

## 算法拆解

### 共生算法

**互惠关系**：
- 人提供环境、营养
- 微生物提供代谢、免疫服务
- "算法的合作"

**早期定植**：
- 出生方式影响菌群（顺产 vs 剖腹产）
- 母乳喂养 vs 配方奶
- "算法的初始化"

### 菌群与疾病

**肥胖**：
- 肥胖者菌群多样性低
- 移植肥胖者菌群 → 小鼠变胖
- "算法的代谢影响"

**精神健康**：
- 焦虑、抑郁与肠道菌群相关
- 脑-肠轴：迷走神经、免疫、代谢产物
- "算法的扩展"
