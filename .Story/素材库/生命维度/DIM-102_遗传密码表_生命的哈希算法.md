# 维度：遗传编码（DIM-102）

## 核心算法

```
输入：64种密码子（4³）
映射：哈希表 → 20种氨基酸
冗余：多个密码子编码同一氨基酸（容错）
输出：蛋白质序列
```

## 算法特征

**通用性**：
- 从细菌到人类，同一套密码表
- 生命的共同起源
- 算法的普适性

**冗余设计**：
- 64个密码子 → 20种氨基酸
- 多个密码子对应同一氨基酸
- 容错机制（突变不一定改变蛋白质）
