## 核心算法

```
抗体生成：
1. V(D)J重组：基因片段随机组合
2. 体细胞高频突变：进一步多样化
3. 克隆选择：亲和力高的B细胞被选中
4. 类别转换：改变抗体类型（IgM→IgG等）

多样性：
- 理论可达 10¹¹ 种不同抗体
- 远超基因数量
- "算法的组合爆炸"

记忆：
- 再次遇到同一抗原 → 更快更强的反应
- 疫苗的基础
```

## 历史背景

- **时间**：免疫学研究始于19世纪末（第4纪）
- **里程碑**：
  - Behring & Kitasato（1890）：抗毒素
  - Ehrlich（1900）：侧链理论（抗体）
  - Burnet（1957）：克隆选择理论
  - Tonegawa（1987诺贝尔奖）：V(D)J重组机制
- **疫苗**：
  - Jenner（1796）：牛痘疫苗
  - Pasteur（1885）：狂犬病疫苗
  - 现代：mRNA疫苗（COVID-19）

## 算法拆解

### 随机生成 + 选择

**V(D)J重组**：
- V、D、J基因片段随机组合
- 像"洗牌"
- 产生多样性

**克隆选择**：
- 随机生成抗体
- 环境中选择匹配的
- "算法的试错"

### 免疫记忆

**疫苗原理**：
- 模拟感染，不产生疾病
- 建立记忆B/T细胞
- "算法的预训练"

**自身免疫**：
- 免疫系统攻击自身
- "算法的错误"
- 类风湿关节炎、红斑狼疮
