# 素材名称：细胞凋亡·程序性死亡
# 维度：衰老死亡（DIM-111）

## 核心算法

```
触发：DNA损伤/病毒感染/发育需要
信号： caspase级联反应
执行：细胞收缩、DNA片段化、吞噬
功能：清除异常细胞，维持组织健康
```

## 算法特征

**程序性**：
- 不是意外死亡，是预设程序
- 每个细胞都有"自杀代码"
- 多细胞生物的必要机制

**与坏死的区别**：
- 凋亡：有序、不引起炎症
- 坏死：混乱、引起炎症
- 算法的优雅 vs 崩溃

**第4纪**：凋亡机制发现（细胞生物学革命）
**第5纪**：AI是否应该有"自我删除"机制？
**隐喻**：文明的自我净化（代价线的自然基础）
