# 素材编号：WAR-008
# 素材名称：围魏救赵·间接攻击
# 算法类型：间接优化 / 约束松弛 / 攻击转移

## 核心算法逻辑

```
目标：解救赵国（被魏国围攻）
直接法：与魏军正面决战（高成本）
间接法：攻击魏国本土（松弛约束）→ 魏军回援（目标自动达成）
```

## 历史背景

- **时间**：公元前354年
- **地点**：桂陵
- **战术**：孙膑率齐军攻魏都大梁
- **结果**：魏军撤离赵国，回援途中被伏击

## 算法拆解

### 约束优化视角

**原问题**：
- 目标：救赵
- 约束：军力有限，难以正面击败魏军
- **不可行解**

**松弛策略**：
- 不直接求解"救赵"，而是改变问题空间
- 创造新约束（魏都危险）→ 原约束自动松弛
- **转化问题**：从"如何击败魏军"→"如何让魏军离开"

### 间接攻击算法
- 系统的脆弱点 ≠ 最强点
- 找到依赖关系：魏军战斗力依赖本土安全
- 攻击依赖根基，而非直接对抗
