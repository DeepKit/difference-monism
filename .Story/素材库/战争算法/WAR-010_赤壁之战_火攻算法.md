# 素材编号：WAR-010
# 素材名称：赤壁之战·火攻算法
# 算法类型：触发条件 / 环境利用 / 链式反应

## 核心算法逻辑

```
输入：东风 + 火船 + 铁索连舟（敌方的错误设计）
触发条件：风速>阈值 AND 敌船密集 AND 使用铁索
算法：火 → 船 → 蔓延（链式反应）
输出：曹操水军全灭
```

## 历史背景

- **时间**：公元208年
- **地点**：赤壁（长江）
- **关键变量**：东南风（冬季罕见）
- **敌方设计缺陷**：铁索连舟（增加连通性，也增加蔓延速度）

## 算法拆解

### 链式反应（Chain Reaction）

**反应条件**：
- 可燃物（船+油）
- 助燃物（风+氧气）
- 点火源（黄盖的火船）

**敌方助燃设计**：
- 铁索连舟 = 增加热传导路径
- 密集停泊 = 减少蔓延阻力
- 北方士兵不习水战 = 无法自救

### 触发条件优化
- 诸葛亮"借东风" = 等待最优触发窗口
- 过早：风不够，火不蔓延
- 过晚：风停，错过机会

### 复杂度涌现
- 单个火船 = 局部损失
- 铁索网络 + 风 + 密集布局 = 全局灾难
- ** emergent behavior**：设计时未预料的系统性风险
