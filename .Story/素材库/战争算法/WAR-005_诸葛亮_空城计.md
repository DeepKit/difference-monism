# 素材名称：诸葛亮·空城计
# 算法类型：逆向博弈 / 信号传递 / 贝叶斯欺骗

## 核心算法逻辑

```
前提：司马懿知道诸葛亮谨慎（先验概率）
观测：城门大开，诸葛亮弹琴（异常信号）
推断：城中有埋伏（后验概率）
决策：撤退（错误但理性的选择）
```

## 历史背景

- **时间**：公元228年
- **地点**：西城
- **兵力对比**：诸葛亮（0兵力）vs 司马懿（15万）
- **关键资产**：诸葛亮的"谨慎"声誉

## 算法拆解

### 贝叶斯欺骗

**先验知识**：
- P(有埋伏|诸葛亮) = 高（历史数据）
- P(冒险|诸葛亮) = 低

**观测到**：城门大开（异常）

**司马懿的推断**：
- 如果无埋伏，诸葛亮必死（不符合理性）
- 如果有埋伏，开门是诱敌（符合理性）
- 因此：有埋伏（即使看不到）

### 信号成本
- 开城门的高风险 = 高成本信号
- 低成本信号（喊几声）不可信
- 只有极高成本信号才能传递真实信息

### 逆向选择
- 诸葛亮选择"看起来最危险的行动"
- 因为这是唯一能让司马懿相信"有埋伏"的方式
