# 素材编号：WAR-011
# 素材名称：巨鹿之战·破釜沉舟
# 算法类型：承诺机制 / 烧桥策略 / 完全承诺博弈

## 核心算法逻辑

```
问题：士兵可能逃跑（背叛）
策略：烧毁退路（改变收益矩阵）
效果：逃跑成本 = 死亡 > 战斗风险
输出：唯一纳什均衡 = 死战
```

## 历史背景

- **时间**：公元前207年
- **人物**：项羽 vs 秦军
- **兵力**：楚军5万 vs 秦军40万
- **关键行动**：破釜沉舟，烧营舍

## 算法拆解

### 承诺机制（Commitment Device）

**博弈论模型**：
- 原博弈：士兵可以选择战斗或逃跑
- 逃跑收益 > 战斗收益（个体理性）
- 集体悲剧：如果都逃跑，全军覆没

**烧桥策略**：
- 项羽公开烧毁退路（可观察的行动）
- 改变收益：逃跑 = 必死
- **承诺可信**：因为无法撤销

### Thomas Schelling的理论

**冲突策略**（1960年）：
- 限制自己的选择可以增加谈判力
- "疯子策略"看似非理性，实则理性
- **项羽是早期的Schelling式战略家**
