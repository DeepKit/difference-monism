# 素材编号：SOC-002
# 素材名称：都江堰·反馈控制
# 算法类型：负反馈系统 / 自动控制 / PID控制器雏形

## 核心算法逻辑

```
目标：维持水位在安全区间
传感器：水情观测（三石人）
执行器：鱼嘴分水、飞沙堰溢洪、宝瓶口引流
算法：偏差检测 → 自动调节 → 稳态维持
```

## 历史背景

- **时间**：公元前256年（李冰父子修建）
- **地点**：四川成都平原
- **核心设计**："深淘滩低作堰"
- **效果**：2200年无坝引水，至今使用

## 算法拆解

### PID控制器雏形

**P（比例控制）**：
- 水位偏差 → 飞沙堰自动溢洪量
- 偏差越大，溢洪越多

**I（积分控制）**：
- 长期泥沙淤积 → 定期深淘
- 消除累积误差

**D（微分控制）**：
- 水位变化速度 → 预测性调节
- "深淘滩"看淤积趋势，而非当前状态

### 鲁棒性设计

**无计算机控制**：
- 纯物理机制（水流动力学）
- 自适应（洪水期多分流，枯水期少分流）
- 容错（局部损坏不影响整体）

**涌现智能**：
- 李冰设计了规则，而非具体行为
- 系统根据环境自动调整
- 最早的**自组织系统**
