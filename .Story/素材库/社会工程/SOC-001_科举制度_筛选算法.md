# 素材名称：科举制度·筛选算法
# 算法类型：标准化测试 / 分层抽样 / 信号筛选

## 核心算法逻辑

```
输入：百万候选人
约束：只能录取极少数（<1%）
算法：标准化测试 + 多级筛选
输出：高能力信号 + 社会流动
```

## 历史背景

- **时间**：隋唐创立，宋明完善
- **规模**：明清时期每年约50万考生
- **录取率**：进士约1%，举人约5%
- **影响**：打破贵族垄断，但也产生新问题

## 算法拆解

### 筛选机制

**多级漏斗**：
```
童试 → 乡试 → 会试 → 殿试
(50万)   (3万)   (300)   (100)
```

**信息经济学**：
- 考试作为**信号装置**（signaling device）
- 高能力者通过考试的成本 < 低能力者
- 因此：通过考试 = 高能力的可信信号

### 标准化 vs 个性化

**优势**：
- 公平（统一标准）
- 可比较（分数排序）
- 抗腐败（相对客观）

**劣势**：
- 局部最优（应试技巧 > 真实能力）
- 维度丢失（道德、创新无法测量）
- **Goodhart定律**："当指标成为目标，它就不再是好指标"

### 八股文的过度优化

**现象**：
- 考试内容固化（四书五经）
- 形式僵化（八股格式）
- 考生 optimize for 考试，而非真实能力

**现代对应**：
- 标准化考试的"应试教育"
- AI的"奖励黑客"（reward hacking）
