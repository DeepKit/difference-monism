# 素材名称：GDPR·数据主权与算法治理
# 所属纪：第五纪（2018年 ~ 未来）
# 类型：法律哲学 / 数字权利 / 主权重构

## 核心命题

> 数据是新时代的土地，谁拥有数据，谁就拥有算法的原料——也就拥有了权力。

GDPR（通用数据保护条例）是人类第一次系统性地用法律对抗算法权力的尝试，标志着"数字主权"概念的诞生。

## 关键权利框架

- **被遗忘权**：要求算法删除关于自己的数据记录
- **可解释权**：有权要求算法解释其决策逻辑
- **数据可携权**：数据属于个人，而非平台
- **同意原则**：算法使用数据须经明确授权

## 哲学意涵

GDPR本质上是一场关于**算法主权归属**的哲学宣言：

1. **个人主权 vs 平台主权**：数据是个人延伸还是平台资产？
2. **透明度悖论**：要求算法透明，但透明的算法更易被操控
3. **主权的碎片化**：数字时代，主权不再属于国家，而是分散在数据节点之间

## 历史锚点

| 年份 | 事件 |
|------|------|
| 2018 | GDPR正式生效，覆盖欧盟5亿人 |
| 2019 | Google被罚5000万欧元（首个重大GDPR处罚） |
| 2023 | ChatGPT在意大利被临时封禁（数据合规争议） |
| 2024 | 欧盟AI法案通过，算法治理进入新阶段 |

## 叙事张力

- 法律永远滞后于算法：GDPR诞生时，它规范的技术已经迭代三代
- 数据主权的幻觉：个人"拥有"数据，但无力理解或使用它
- 治理的悖论：越严格的监管，越有利于已有数据优势的巨头
