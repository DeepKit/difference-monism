# 所属纪：第二~三纪（公元500年 ~ 1600年）
# 类型：宗教权力 / 知识管控 / 算法异端

## 核心命题

> 教会既是算法知识最大的压制者，也是最大的保存者。
> 修道院抄写员保存了古希腊数学，宗教裁判所却烧毁了哥白尼的继承者。

## 核心张力

**教会的算法悖论**：
- 需要算法（历法、建筑、财务）来维持权力运作
- 又恐惧算法（数学宇宙观）动摇神学权威
- 解决方案：将算法神圣化——"数学是上帝的语言"

**知识管控的算法逻辑**：
```
允许的算法 = 服务神学的算法
禁止的算法 = 挑战神学的算法
```

## 关键人物

- **罗杰·培根**（1214~1292）：修士科学家，因研究光学算法被软禁
- **托马斯·阿奎那**：将亚里士多德逻辑算法整合进神学
- **哥白尼**：死后出版《天体运行论》，算法革命的延迟引爆
