# 所属纪：第一纪（远古 ~ 公元前300年）
# 算法类型：知识垄断 / 神权算法 / 信息不对称

## 权力算法逻辑

```
输入：天文观测数据（日食、月相、洪水周期）
垄断手段：将算法包装为"神谕"，拒绝公开
输出：对农业时间表、战争时机、税收周期的绝对控制
代价：知识持有者一旦失去神圣性，算法即失效
```

## 权力结构

**控制者**：祭司、占星师、神庙数学家
**被控制者**：农民、士兵、商人、国王（名义上）
**算法武器**：历法、占卜、神谕系统
**失控机制**：当算法被世俗化（如希腊哲学家公开几何学），神权崩溃

## 历史原型

- 埃及祭司掌握尼罗河洪水预测算法，控制农业生产节奏
- 巴比伦天文学家垄断日食预测，以此要挟国王
- 玛雅历法系统：精确到极致的算法，但只有祭司能读懂

## 叙事碎片

## 权力转移节点

- 泰勒斯公开预测日食（公元前585年）→ 算法去神圣化开始
- 欧几里得《几何原本》→ 知识从神殿走向学院
