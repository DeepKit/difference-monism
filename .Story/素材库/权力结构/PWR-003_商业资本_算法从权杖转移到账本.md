# 素材编号：PWR-003
# 素材名称：商业资本·算法从权杖转移到账本
# 所属纪：第三纪（1000年 ~ 1900年）
# 算法类型：复式记账 / 市场定价 / 资本积累

## 权力算法逻辑

```
输入：商品价格、贸易路线、信用评级
约束：利润最大化 + 风险对冲
算法：复式记账 × 汇票系统 → 资本流动
输出：财富集中 / 贸易垄断 / 殖民扩张
```

## 历史事件

**美第奇家族复式记账（1397年）**
- 算法核心：借贷双方同时记录，利润可被精确计算
- 权力逻辑：谁掌握账本算法，谁就掌握权力
- 代价：教皇权威被金融算法侵蚀

**东印度公司股份制（1602年）**
- 算法核心：风险分散算法——把一次航行的风险切割成股份
- 权力逻辑：算法让资本可以跨越时间和空间聚合
- 代价：殖民地人民成为算法的外部成本

**郁金香泡沫（1637年）**
- 算法核心：期货定价算法失控
- 权力逻辑：当算法脱离实物，权力结构瞬间崩塌
- 代价：荷兰中产阶级财富清零

## 权力结构图谱

```
金融家族（算法设计者）
    ↓
商业公司（算法运营者）
    ↓
国家政府（算法背书者）
    ↓
殖民地 / 工人（算法成本承担者）
```
