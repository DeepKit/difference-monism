```
地点：美索不达米亚·乌鲁克神庙粮仓
时间：公元前3100年，麦收之后
气候：干燥炎热，尘土飞扬
光线：油灯昏黄，泥板反光
```

## 核心算法事件

人类第一次用符号记录数量——不是为了诗歌，而是为了粮食分配。
楔形文字的诞生不是语言的胜利，是算法的胜利：
把现实压缩成符号，符号可以被操作，操作产生权力。

```
现实 → 符号（泥板刻痕）→ 规则（分配算法）→ 权力（谁掌握泥板）
```

- 气味：晒干的泥土、小麦秸秆、动物油脂

- 视觉：成排的泥板，每块都是一个微型数据库

## 叙事张力

祭司手持铁笔，面对堆积如山的粮食。
他不需要搬运粮食，只需要控制泥板。
**控制符号的人，控制了现实。**

第一个程序员不是在写代码，是在刻泥板。
第一个数据库不在服务器里，在神庙地下室。
