# 素材编号：SCENE-015
# 素材名称：布莱切利园·密码战争
# 所属纪：第四纪（1939~1945年）
# 场景类型：战争算法 / 机器破译 / 算法英雄

## 场景坐标

```
地点：英国白金汉郡·布莱切利庄园
时间：1942年冬，二战最黑暗时期
氛围：压抑、紧迫、天才与机器的共鸣
```

## 核心意象

- 木屋8号：图灵与同事的工作间，纸张堆积如山
- 炸弹机（Bombe）：机械轰鸣，转轮不停旋转
- 恩尼格玛密文：每天午夜重置，算法的时间锁
- 窗外：英国乡村的宁静与战争的荒诞对比

## 叙事功能

1. 算法作为战争武器的具象化场景
2. 图灵原型（CHAR-006）的核心活动空间
3. 展示"破解算法"与"创造算法"的双重张力
4. 引入"算法知道秘密，但人类付出代价"的主题

## 关键细节

- 每天处理约3000条截获电报
- 破译成功后必须伪装成"情报来源不明"，不能直接使用
- 团队成员签署保密协议，胜利后无法公开讲述
- 图灵的算法思维在此首次被国家机器承认，又在战后被同一国家迫害

## 可调用对话钩子

> "机器不会疲倦，但我们会。"
> "我们赢了这场战争，但没有人知道是我们赢的。"
> "算法是中性的。使用它的人，才决定它是武器还是工具。"
