# 素材编号：SCENE-011
# 素材名称：巴格达智慧宫·翻译运动
# 所属纪：第二纪（公元800年前后）
# 场景类型：知识传递 / 跨文明算法融合

## 场景坐标

```
地点：巴格达·智慧宫（Bayt al-Hikma）
时间：公元830年前后，阿拔斯王朝鼎盛期
氛围：烛光、羊皮纸、多语言的低语声
```

## 核心算法事件

花拉子米在此写下《代数学》，将印度数字系统与希腊几何学融合——这是人类历史上最重要的一次算法翻译运动。算法第一次跨越文明边界，成为普世语言。

## 感官细节

- 视觉：数十种语言的手稿堆叠，阿拉伯文、希腊文、梵文并排
- 听觉：翻译者低声朗读，核对数字的发音
- 触觉：羊皮纸的粗糙，墨水未干时的黏腻
- 嗅觉：皮革、墨水、来自东方的香料

## 叙事功能

| 用途 | 说明 |
|------|------|

## 冲突张力

翻译即选择——哪些知识被翻译，哪些被遗忘？翻译者的每一个选择都是一次算法的生死裁决。

## 可用对话钩子

> "我们不是在翻译文字，我们是在翻译思维的形状。"
