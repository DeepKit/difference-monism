# 所属纪：第一纪（公元前17000年前后）
# 场景类型：算法起源 / 符号诞生 / 认知革命

## 场景坐标

```
地点：法国多尔多涅省，拉斯科洞穴深处
时间：旧石器时代晚期，约公元前17000年
光源：动物脂肪灯，火焰摇曳
气味：赭石粉末、动物油脂、潮湿石灰岩
```

## 核心场景

洞壁上，一个人用手指蘸赭石，在石灰岩上画下第一头奔跑的野牛。

这不是艺术。这是算法。

画下的不是眼前的牛，而是"牛"这个类别——一个抽象规则，可以被复用、被传递、被其他人识别。符号从此脱离了具体事物，成为可操作的信息单元。

旁边，另一个人在骨头上刻下29道划痕。月亮的周期。第一张日历。第一个循环算法。

## 叙事锚点

**算法的本质**：在混沌中提取模式，并将模式固化为可复用的符号。

洞穴画家不知道自己在做什么。但他做的事情，和三万年后的程序员做的事情，在本质上没有区别：
- 输入：观察到的现实（野牛的形态）
- 抽象：提取关键特征（四肢、角、奔跑姿态）
- 输出：可复用的符号（洞壁上的图像）
- 传播：其他人可以读取并理解这个符号

## 感官细节

- 火焰在石壁上投下的阴影，让画中的野牛仿佛在奔跑
- 赭石粉末落在手背上，像锈色的尘埃
- 洞穴深处的寂静，只有滴水声和呼吸声
- 画完最后一笔，画家退后一步，第一次看见了"抽象"
