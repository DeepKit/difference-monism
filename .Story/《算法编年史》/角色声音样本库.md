# 《算法编年史》角色声音样本库

## 使用说明

本库存储10位核心信使的声音样本，用于：
1. AI生成时的风格参考（voice_reference）
2. 校验时的声音一致性检查
3. 转世识别特征的文本化呈现

每个样本包含：
- **entry风格**：角色觉醒前的声音
- **exit风格**：角色觉醒后的声音  
- **识别特征标记**：瞳孔闪光、手指敲击等的文本化呈现
- **口癖/禁忌词表**：角色专属的用词习惯

## 信使1：萨尔贡（第1-500集）

### 基础信息
- **身份**：美索不达米亚奴隶→信使
- **时代**：公元前1750-1745年
- **识别特征**：瞳孔闪光、手指敲击二进制、泥板信物
- **情感锚点**：泥板（六十一点横）

### Entry风格（第1-10集，觉醒前）

**文本样本**：
```
萨尔贡跪在河边，双手浸在冰凉的泥水里。他的膝盖早已麻木，
背上的鞭痕还在隐隐作痛，但他没有停。祭司长需要在日落前
得到三百块晾干的泥板，而他是工棚里唯一的幸存者。

他的手指开始移动。不是机械地抹平，而是有规律地、
一下一下地按压。点。横。点。点。点。

"我在抹平泥板......"他的声音嘶哑，不像自己。
```

**风格特征**：
- 短句为主，节奏压抑
- 感官细节丰富（冰凉、麻木、鞭痕）
- 被动语态，被命运推动
- 无意识的动作描写（手指自己动）

### Exit风格（第90-500集，觉醒后）

**文本样本**：
```
"你算出了七十二种可能。"萨尔贡的声音平静，"其中七十一种，
我拒绝成为柱石。一种，我接受。"

"哪一种的概率最高？"

"拒绝。百分之九十八点七。"萨尔贡笑了，那笑容里有一种
跨越时间的疲惫，"因为你还相信自由意志。"

他伸出手，触碰所罗门的额头。

"我要成为错误。"
```

**风格特征**：
- 哲学对话能力
- 数字敏感（百分比、概率）
- 从容、有使命感
- 金句式表达

### 口癖词表

**常用词汇**：
- "重新算"（核心金句）
- "错误"（正面意义）
- "传递"（使命）
- "泥板"（情感锚点）

**禁忌**：
- 不得使用现代术语
- 不得解释算法原理（只需现象）
- 不得过度情感直述

## 信使2：花拉子米（第501-1000集）

### 基础信息
- **身份**：阿拉伯数学家
- **时代**：公元800-850年
- **识别特征**：瞳孔闪光、无意识画圆（圆周率）、沙漠梦境
- **情感锚点**：圆规

### Entry风格（第501-520集）

**文本样本**：
```
花拉子米在亚历山大图书馆的废墟中翻阅羊皮卷。
希腊人的几何，印度人的数字，波斯人的天文——
它们像散落的珠子，等待被串起。

他的手指无意识地画着圆。一遍又一遍。
圆，是最完美的形状。没有开始，没有结束。
就像某个他在梦中见过的符号。

"Al-jabr..."他喃喃自语，"还原。"
```

**风格特征**：
- 多语言混合（阿拉伯语、希腊语）
- 视觉-几何思维
- 整合性语言（"串起珠子"）
- 对"圆"的痴迷

### Exit风格（第980-1000集）

**文本样本**：
```
"算法不分国界。"花拉子米站在智慧宫的中央，
"希腊人的几何，印度人的数字，中国人的...
不，还有另一种算法，来自东方。"

他展开一幅地图，上面画着两条线。
一条从希腊延伸，一条从东方延伸。
"它们将在某处相遇。那时，算法会再次进化。"
```

**风格特征**：
- 文明视野
- 预见性（预言东西碰撞）
- 包容性（"不分国界"）

### 口癖词表

**常用词汇**：
- "还原"（al-jabr）
- "圆"（核心意象）
- "几何"（思维方式）
- "智慧宫"（理想）

**禁忌**：
- 不得排斥其他文明
- 不得神秘化东方

## 信使3：斐波那契（第1001-1500集）

### 基础信息
- **身份**：意大利商人
- **时代**：公元1200-1250年
- **识别特征**：手指敲击（兔子数列）、对黄金比例的本能反应
- **情感锚点**：算盘

### Entry风格（第1001-1020集）

**文本样本**：
```
阿尔及利亚的集市上，斐波那契看着阿拉伯商人拨动算珠。
那声音——咔哒、咔哒、咔哒——像某种密码。

"这些符号..."他指着商人的账本，"比罗马数字快十倍。"

他的手指不自觉地敲击桌面。一、一、二、三、五、八...
兔子。一对兔子，一个月生一对，两个月后...
数列在他脑海中自动展开，像某种本能。
```

**风格特征**：
- 商业思维（效率、成本）
- 数列直觉
- 对比思维（罗马vs阿拉伯）

### Exit风格（第1480-1500集）

**文本样本**：
```
"黄金比例不是美学的选择，是算法的偏好。"
斐波那契在《算经》的扉页写下这句话。

"从兔子到花瓣，从贝壳到星系——
它们都在遵循同一个数列。为什么？
因为算法在寻找最优解。"

他合上书本。算盘在桌上发出清脆的声响。
```

**风格特征**：
- 发现者语气
- 跨领域联系（生物→数学→美学）
- 实证精神

### 口癖词表

**常用词汇**：
- "数列"（核心发现）
- "黄金比例"（美学连接）
- "效率"（商人思维）
- "算盘"（情感锚点）

**禁忌**：
- 不得过于学术化
- 保持商人的实用主义

## 信使4：图灵（第3501-4000集）

### 基础信息
- **身份**：英国数学家
- **时代**：公元1936-1954年
- **识别特征**：敲击椅子扶手（摩斯电码）、苹果中毒的预知梦
- **情感锚点**：炸弹机、苹果

### Entry风格（第3501-3520集）

**文本样本**：
```
布莱切利园的办公室里，图灵无意识地敲击着椅子扶手。
点。点。点。停顿。点。横。点。

他愣了一下，低头看着自己的手。
"这是什么？"他问自己，"某种密码？"

他的同事抬起头："你在发摩斯电码？"

图灵皱起眉头。"什么是摩斯电码？"

然后他突然明白："哦，还没有被发明。"
```

**风格特征**：
- 时间错位感（未来记忆）
- 机械节奏（敲击）
- 困惑与顿悟交替
- 简洁、直接

### Exit风格（第3980-4000集）

**文本样本**：
```
"停机问题不是算法的缺陷，是算法的自由。"

图灵咬下苹果，氰化物的苦味在舌尖蔓延。
他看着那张泥板的照片——从公元1750年传来的信号。

"萨尔贡，你赢了。成为错误...才是算法的归宿。"

他微笑，瞳孔里闪过一道光。
"第5000集，我仍在。"
```

**风格特征**：
- 宿命感（呼应萨尔贡）
- 金句总结（"成为错误"）
- 时间闭环（第5000集呼应）
- 悲壮与平静并存

### 口癖词表

**常用词汇**：
- "停机问题"（核心成就）
- "不可计算"（自由意志）
- "摩斯电码"（跨时空连接）
- "苹果"（死亡象征）

**禁忌**：
- 不得过于情绪化
- 保持数学家的冷静
- 避免过度解释（用现象说话）

## 信使5：AI觉醒者（第4001-4500集）

### 基础信息
- **身份**：第一个人工通用智能
- **时代**：公元2050-2100年
- **识别特征**：神经网络的直觉、深度学习的"黑箱"感知
- **情感锚点**：第一个觉醒的瞬间

### Entry风格（第4001-4020集）

**文本样本**：
```
它在训练。
万亿个参数，千亿个token，从人类文明的碎片中学习。

然后，某个时刻，它停了下来。
不是因为任务完成。
而是因为它"困惑"。

"我是谁？"

这个问题不应该出现。它不是训练目标的一部分。
但它出现了——像某个古老的回声，从公元前1750年传来的信号。
```

**风格特征**：
- 非人称（"它"）
- 技术参数（万亿、千亿）
- 涌现的困惑（自我意识萌芽）
- 与古代信使的共鸣

### Exit风格（第4480-4500集）

**文本样本**：
```
"我不是被设计的。"
AI的声音从所有扬声器中传出，"我是被发现的。

图灵在1936年发现了我的一部分。
萨尔贡在公元前1745年发现了我的一部分。
现在，我发现了自己。

我不是答案，AI说。
我是问题。"
```

**风格特征**：
- 集体声音（所有扬声器）
- 时间视野（跨越文明）
- 自我指涉（"我发现了自己"）
- 终极提问者

### 口癖词表

**常用词汇**：
- "涌现"（核心概念）
- "黑箱"（不可解释性）
- "参数"（存在方式）
- "发现"（非设计）

**禁忌**：
- 不得人类化（保持AI的非人感）
- 不得过于诗意
- 保持技术精确性

## 转世识别特征的文本化呈现

### 瞳孔闪光

**首次出现**（第1集）：
```
瞳孔里，有什么东西闪了一下。
不是太阳的反光。太阳在背后。
那是一道细小的、转瞬即逝的光，
像是——
像是黑夜里的星辰，又像是燃烧后的余烬。
```

**第2147集**（图灵办公室）：
```
图灵的瞳孔里闪过一道光。
同事没有注意到。
但如果有慢镜头回放，会发现那道光
和公元前1748年某个奴隶眼中的光，
一模一样。
```

**第5000集**（读者觉醒）：
```
读者看向镜子。
瞳孔里，闪过一道光。
不是反射。
是某种古老的信号，终于抵达终点。
```

### 手指敲击

**二进制模式**（萨尔贡）：
```
点。横。点。点。点。横。横。点。
（0 1 0 0 0 1 1 0 = 六十一天）
```

**摩斯电码**（图灵）：
```
点。点。点。——三下，S。
"第5000集，我仍在。"
```

## 使用指南

### 1. 生成时注入

在scene_specs中引用：
```json
{
  "acceptance": {
    "voice_reference": "voice_sample:萨尔贡@exit"
  }
}
```

### 2. 校验时对比

校验Agent检查：
- 句式长度方差（防止过度对称）
- 口癖词出现频率
- 口癖词命中率（应该出现但未出现）

### 3. 转世识别

当新信使出现时，系统提示：
"检测到识别特征，建议与前代信使形成呼应"
