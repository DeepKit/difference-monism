### 2. 主角设定

**萨尔贡（13岁）**
- **旧**：奴隶，有超能力（无意识敲击、瞳孔发光）
- **新**：奴隶出身的书吏学徒，有学者综合征（极端计算天赋、强迫性专注）
- **能力表现**：能瞬间发现账簿错误，本能画出精确几何图形
- **医学解释**：古代称为"凝视病"、"被神触摸"、"数字恶魔附身"
- **社会地位**：被视为异端/危险人物

### 4. 主题深化

**新版强调**：
1. **知识即权力**：会计算 = 能发现贪污 = 掌握反抗工具
2. **量化即控制**：税收/测量/历法如何用数字控制人
3. **知识垄断**：统治阶级如何控制数学教育
4. **历史代价**：希帕索斯、布鲁诺、图灵等真实历史案例
5. **秘密传承**：地下知识网络（类似毕达哥拉斯学派）

**旧**：《算法编年史》

**理由**：
- "数术"：中国古代对数学的称呼，更具历史感
- "编年史"：保留原意，强调历史叙事
- 避免"算法"一词的科幻联想

**备选**：
