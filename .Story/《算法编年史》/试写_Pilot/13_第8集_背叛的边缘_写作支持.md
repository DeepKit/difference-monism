## 场景结构（4场景，3500字）

### 场景1：告发（开头，800字）

**时间**：清晨
**地点**：神庙
**人物**：学徒（纳布），祭司长

**场景目标**：
1. 学徒走进神庙（执行告发）
2. 与祭司长的对话
3. 展示学徒的内疚和恐惧
4. 告发完成

**关键段落**：
```
纳布站在神庙的台阶下，手在颤抖。

他已经在这里站了一刻钟。

上，还是不上？

他想起萨尔贡。想起他们一起工作的日子。想起萨尔贡帮他修好的工具。

他想起那十两银子。想起自己的债务。想起未来的生活。

他迈上了第一级台阶。

然后是第二级。第三级。

他走进神庙，向祭司长的房间走去。

"大人，"他说，声音颤抖，"我要举报一个异常者。"

祭司长抬起头，看着他："谁？"

纳布吞咽："萨尔贡。泥板作坊的奴隶。"

祭司长微笑："我知道他。我们一直在观察他。"

纳布惊讶："您...您知道？"

祭司长：（站起来）"当然。我们什么都知道。包括你三天前的犹豫，两天前的试探，昨天的告发决定。"

纳布脸色苍白。

祭司长：（走近）"你做得很好。赏银会给你的。"

他拍拍纳布的肩膀："现在，你可以走了。剩下的事，交给我们。"
```

**学徒的内心**：
- 内疚和恐惧
- 自私的理由
- 知道自己被利用了

**回收的线索**：
- CLUE-APPRENTICE-01：告发决定（第2集埋下，本集回收）

### 场景2：最后的友谊（发展，1000字）

**时间**：当天下午
**地点**：作坊
**人物**：学徒（纳布），萨尔贡

**场景目标**：
1. 学徒与萨尔贡的最后对话
2. 学徒试图掩饰，但失败了
3. 萨尔贡察觉到异常
4. 友谊的终结

**关键对话**：
```
纳布：（走近萨尔贡，试图正常）"今天工作怎么样？"
萨尔贡：（看着他）"你有什么事吗，纳布？"
纳布：（紧张）"没...没什么。只是...想聊聊。"

萨尔贡：（平静）"你告发了我。"

纳布震惊："什么？我..."
萨尔贡：（平静但悲伤）"我知道。我看见了。今天早上，你去了神庙。"
纳布：（后退）"萨尔贡，我...我不是..."
萨尔贡：（打断他）"没关系。我理解。"
纳布：（困惑）"你...理解？"
萨尔贡：（点头）"你想要那十两银子。你想要更好的生活。我不怪你。"
他看着纳布："我只是...以为我们是朋友。"

纳布低下头，无法直视萨尔贡的眼睛。

"对不起，"他低语，"对不起..."

萨尔贡：（转身）"没关系。去吧。拿你的赏银。"

他回到工作台，继续工作，不再看纳布一眼。

纳布站在原地，感到一种空虚。

他得到了他想要的。
但他失去了...他失去了什么？
```

**情感描写**：
- 萨尔贡的平静（比愤怒更伤人）
- 纳布的内疚
- 友谊的终结

**关键转折**：
- 萨尔贡已经知道（他的感知能力增强）
- 不责怪，但失望
- "我以为我们是朋友" - 情感冲击

### 场景3：抓捕准备（转折，1200字）

**时间**：当晚
**地点**：作坊外
**人物**：学徒（纳布，观察），所罗门之门的士兵（暗中），萨尔贡

**场景目标**：
1. 纳布观察抓捕准备
2. 他意识到萨尔贡面临的危险
3. 他的内疚达到顶点
4. 想要警告但犹豫

**关键段落**：
```
纳布躲在作坊外的阴影中，看着所罗门之门的士兵包围作坊。

六个人。武装。

不是普通的抓捕。

纳布心跳加速。这不只是要把萨尔贡带走。这是...

他看到祭司长和士兵队长说话。

"活捉，"祭司长说，"但如果他反抗..."
他停顿："...可以杀死。"

纳布感到一阵寒意。

杀死？萨尔贡只是...只是敲击泥板。他做了什么该死的事？

他想起萨尔贡的话："我以为我们是朋友。"

他想起他们一起工作的日子。

他想起萨尔贡帮他修好的工具。

他握紧拳头。

他应该警告萨尔贡。
他应该...

但他没有动。

他只是站在那里，看着，像所有其他人一样。

自私。懦弱。

这就是他。
```

**内心冲突**：
- 想要救萨尔贡
- 但害怕牵连自己
- 最终的懦弱

**讽刺**：
- 他为了自保告发萨尔贡
- 但现在看到后果，又内疚
- 但依然没有行动

### 场景4：抓捕（结尾，500字）

**时间**：深夜
**地点**：作坊
**人物**：萨尔贡，所罗门之门的士兵

**场景目标**：
1. 士兵闯入抓捕萨尔贡
2. 萨尔贡没有反抗（意料之外）
3. 强烈的钩子
4. 为第9集河边老人的救援铺垫

**关键段落**：
```
深夜，作坊的门被撞开。

六个士兵涌入，手持武器。

萨尔贡坐在工作台前，似乎已经等待多时。

"萨尔贡，"队长说，"你被指控为异常者。跟我们走。"

萨尔贡：（平静地站起来）"我知道。"

他没有反抗。没有逃跑。他只是...接受。

这让士兵们困惑。他们准备好了一场搏斗。

"走吧，"萨尔贡说，"我准备好了。"

他被带出去，穿过街道。

纳布躲在阴影中，看着这一切。

他想要喊："等等！停止！"

但他的声音卡在喉咙里。

萨尔贡被带走，没有回头。

但在经过纳布藏身的阴影时，萨尔贡似乎...停顿了一下。

他们的目光相遇。

萨尔贡的眼神中没有愤怒，没有仇恨。

只有...理解？还是...怜悯？

然后萨尔贡被带走了。

纳布跪倒在地，哭泣。

但他知道，已经太迟了。

太迟了。
```

**钩子**：
> 萨尔贡被带向神庙。
> 但就在神庙的台阶前，一个身影从阴影中走出。
> "等一等，"那人说。
> 是河边老人。

**主题**：
- 背叛的代价
- 自私与懦弱
- 但还有救赎的可能（第9集）
