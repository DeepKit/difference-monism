## 场景结构（4场景，3500字）

### 场景1：学徒的住处（开头，800字）

**时间**：第1集当晚
**地点**：学徒的简陋房间
**人物**：学徒（独自），回忆中的萨尔贡

**场景目标**：
1. 展示学徒的贫穷但自由的身份（对比萨尔贡）
2. 他的内心独白：我该怎么办？
3. 回忆与萨尔贡的友谊
4. 埋下告发的念头

**关键描写**：
- 学徒数着父亲留下的几枚银币（生活的压力）
- 看着窗外的神庙（权力的象征）
- 想起萨尔贡帮他修工具的往事（友谊）

**内心冲突**：
```
如果我告发他：
- 赏银十两（解决生计）
- 祭司长的赏识（前途）
- 但萨尔贡会死...

如果我保持沉默：
- 萨尔贡是好人
- 但万一他真的是...
- 万一我也被牵连...
```

**钩子**：
> 他拿起外衣，准备出门，但又放下了。
> "明天，"他对自己说，"明天再决定。"

---

### 场景2：作坊工作（发展，1000字）

**时间**：第二天上午
**地点**：泥板作坊
**人物**：学徒，萨尔贡（背景）

**场景目标**：
1. 展示学徒观察萨尔贡
2. 萨尔贡表现正常（对比昨天的异常）
3. 学徒的试探（试图确认昨天是否幻觉）
4. 友谊的细节

**关键互动**：
```
学徒：（递水给萨尔贡）"你昨天...还好吗？"
萨尔贡：（困惑）"什么？"
学徒：（试探）"没什么。只是你昨天好像很累。"
萨尔贡："我不记得了。"

学徒观察：萨尔贡的手指，正常地写着楔形文字，没有异常。
```

**内心独白**：
> 也许只是幻觉？也许我看错了？
> 但那个瞳孔里的光...那61个凹点...
> 不，那是真的。

**关键动作**：
- 学徒"不小心"碰倒萨尔贡的泥板（测试他的反应）
- 萨尔贡正常地重新制作（没有无意识敲击）

---

### 场景3：神庙附近（转折，1200字）

**时间**：当天下午
**地点**：通往神庙的街道
**人物**：学徒（独自），祭司长（远处）

**场景目标**：
1. 学徒走向神庙（告发的冲动）
2. 看到祭司长（恐惧）
3. 回忆父亲的警告（异常者=危险）
4. 决定告发

**环境描写**：
- 神庙的高大威严
- 街道上的人群（学徒感到孤独）
- 赏银告示（诱惑）

**关键段落**：
```
学徒站在神庙的台阶下。

十两银子。足够他离开作坊，开一家自己的小店。足够他娶一个妻子，生几个孩子，过上正常人的生活。

他只需要说一句话："我发现了一个异常者。"

他想起萨尔贡。想起他帮他修好的那把刻刀。想起他分享的最后一块面包。

他想起父亲临死前说的话："在这个世界，善良是奢侈品。活下去，才是唯一的目标。"

他迈上了第一级台阶。
```

**情感高潮**：
- 内心的道德挣扎达到顶点
- 自私最终战胜善良
- 为告发找理由："我不是坏人，我只是想活下去"

---

### 场景4：告别（结尾，500字）

**时间**：当天晚上
**地点**：作坊外
**人物**：学徒，萨尔贡

**场景目标**：
1. 学徒与萨尔贡的简短对话（最后的友谊）
2. 学徒的内疚
3. 强烈的钩子：明天告发

**关键对话**：
```
学徒："萨尔贡。"
萨尔贡：（抬头）"嗯？"
学徒：（停顿）"如果...如果有什么不好的事情发生，你会怪我吗？"
萨尔贡：（困惑）"什么？"
学徒：（勉强笑）"没什么。只是...谢谢你。"
萨尔贡："谢什么？"
学徒：（转身）"没什么。明天见。"

学徒走了，没有回头。
他不知道，这是他和萨尔贡最后一次正常的对话。
```

**钩子**：
> 学徒躺在床上，盯着天花板。
> "明天。"
> 他闭上眼睛。
> "明天，一切都会结束。"

## 常见错误避免

### ❌ 错误1：学徒太坏
**避免**：不要把他写成纯粹的恶人
**正确**：展示他的挣扎和无奈，让读者理解他的选择

### ❌ 错误2：内心独白太多
**避免**：不要大段大段的心理描写
**正确**：用动作和对话展示内心，独白要简短有力

### ❌ 错误3：告发决定太快
**避免**：不要让他轻易决定告发
**正确**：展示充分的挣扎，让决定显得艰难
