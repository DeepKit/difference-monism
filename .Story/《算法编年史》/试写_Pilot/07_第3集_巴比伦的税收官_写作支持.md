## 场景结构（4场景，3500字）

### 场景1：税收日（开头，800字）

**时间**：月初（税收日）
**地点**：作坊外的大街
**人物**：萨尔贡（背景），众多市民，税收官

**场景目标**：
1. 建立时代氛围（古代巴比伦的日常生活）
2. 引入税收官的威严和残酷
3. 展示市民的反应（恐惧、顺从）
4. 萨尔贡作为奴隶的特殊视角

**环境描写**：
```
月初的巴比伦，空气中弥漫着紧张和恐惧。

税收官的队伍从神庙出发，沿着主要街道行进。他们穿着统一的制服，手持记录板，身后跟着武装士兵。

每到一家店铺，他们就停下来：
"上个月的营业额？"
"五十舍客勒。"
"税率20%，应交十舍客勒。"

交钱的人脸色苍白。
没钱的人被士兵带走。
```

**萨尔贡的观察**：
> 萨尔贡站在作坊门口，看着这一切。
> 作为奴隶，他不需要交税。但他记得父亲。
> 记得那个因为交不起税而失去一切的夜晚。

**钩子**：
> 税收官的队伍停在了作坊前。
> "萨尔贡在吗？"税收官问。
> 萨尔贡的心跳停止了。
> 他以为奴隶不需要面对这些...

---

### 场景2：父亲的故事（发展，1000字）

**时间**：回忆（闪回）
**地点**：萨尔贡的童年家中
**人物**：小萨尔贡（7岁），父亲

**场景目标**：
1. 闪回展示萨尔贡的身世
2. 父亲是小商人，因税收破产
3. 揭示萨尔贡成为奴隶的原因
4. 建立情感基础

**关键段落**：
```
七岁的萨尔贡躲在门后，看着父亲和税收官对话。

"上个月的营业额，三十舍客勒。"
"税率30%，应交九舍客勒。"

父亲的脸色变了："30%？上个月还是20%！"

税收官面无表情："新规定。国王要修建神庙。"

"但我只有八舍客勒..."

"那就拿货物抵。"

士兵开始搬动家里的东西。父亲的存货，母亲的嫁妆，最后...父亲自己。

"你欠的税，用你自己抵。"税收官说，"从今天起，你是国王的奴隶。"

小萨尔贡看着父亲被带走。

那天晚上，母亲把萨尔贡送到作坊："从今天开始，你在这里工作。至少...至少你还能活下去。"

三天后，母亲投河自尽。
```

**情感描写**：
- 童年的创伤
- 失去双亲的痛苦
- 成为奴隶的屈辱

---

### 场景3：税收算法（转折，1200字）

**时间**：回到现在
**地点**：作坊内
**人物**：萨尔贡，税收官，作坊老板

**场景目标**：
1. 展示税收官的"算法"（ECO-001素材）
2. 揭示税收制度的精确和冷酷
3. 萨尔贡与税收官的直接对话
4. 埋下CLUE-TAX-01

**关键对话**：
```
税收官：（查看记录）"这家作坊，上个月生产了一百块泥板。"
作坊老板："是的，大人。"
税收官："每块泥板价值半舍客勒，总计五十舍客勒。税率20%，应交十舍客勒。"

作坊老板交出钱袋。

税收官转向萨尔贡："你叫萨尔贡？"
萨尔贡：（低头）"是的，大人。"
税收官：（查看记录）"你的父亲，纳布-阿海，欠税三舍客勒，未还清。按照法律，债务 transferable。"
萨尔贡：（震惊）"但我已经是奴隶了..."
税收官：（面无表情）"利息。每年20%。三年，三舍客勒变成五舍客勒。你需要工作十年才能还清。"

萨尔贡感到一阵眩晕。

税收官："这就是算法。精确，公平，无情。"
```

**算法概念植入**：
- 税收制度的"算法"本质（输入-计算-输出）
- 算法的冷酷性（不考虑个人情况）
- 算法的累积效应（利息使债务增长）

**描写要点**：
- 税收官的机械和精确
- 萨尔贡的无助和愤怒
- 制度的庞大和个人的渺小

---

### 场景4：决心（结尾，500字）

**时间**：当晚
**地点**：河边
**人物**：萨尔贡（独自）

**场景目标**：
1. 萨尔贡的愤怒和决心
2. 对"算法"的初步理解
3. 为未来反抗铺垫
4. 钩子：学徒告发的威胁

**关键段落**：
```
萨尔贡跪在河边，手指插入水中。

五舍客勒。十年。

因为他父亲欠了三舍客勒。

这就是算法的冷酷。不考虑你是否已经失去一切，不考虑你是否已经付出全部。它只是计算，执行，索取。

萨尔贡握紧拳头。

"我不接受。"

他低声说，声音被河水的声音淹没。

"我不接受这种算法。"

身后传来脚步声。

萨尔贡转过头，看到学徒站在不远处，表情复杂。

"萨尔贡，"学徒说，"我们需要谈谈。"
```

**钩子**：
> 学徒的表情告诉萨尔贡，这不是普通的谈话。
> 有什么事情要发生了。

**主题**：
- 对算法的初步觉醒
- 反抗的种子
- 个人 vs 制度

## 写作要点

### 闪回的技巧
- 用触发点引入闪回（税收官的出现）
- 闪回要简短但情感强烈
- 回到现在时要有过渡

### 展示制度而非说教
- 通过具体案例展示税收制度的残酷
- 不要大段解释制度，用行动展示
- 让读者自己感受不公平

### 萨尔贡的情感
- 从恐惧（童年记忆）到愤怒（现在）
- 初步的反抗意识
- 为未来的觉醒铺垫
