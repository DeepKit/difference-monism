## 场景结构（4场景，3500字）

### 场景1：摊牌（开头，800字）

**时间**：第3集当晚
**地点**：河边（远离作坊）
**人物**：萨尔贡，学徒

**场景目标**：
1. 学徒向萨尔贡摊牌
2. 透露他知道萨尔贡的秘密
3. 给萨尔贡一个"选择"（配合或...）
4. 展示学徒的复杂心态

**关键对话**：
```
学徒："我知道你是什么。"
萨尔贡：（警惕）"什么？"
学徒："昨天。61个凹点。瞳孔里的光。那不是人类能做到的。"
萨尔贡：（沉默）
学徒："我有两个选择。告发你，获得赏银。或者...帮我一个忙。"
萨尔贡："什么忙？"
学徒：（靠近）"让我也成为异常者。"
萨尔贡：（震惊）"什么？"
学徒：（急切）"教我怎么做到。那种无意识的状态，那种能力。如果我能做到，我就不会告发你。我们可以一起..."
萨尔贡：（后退）"我不知道怎么做到。我不记得..."
学徒：（失望，转愤怒）"你不记得？还是你不想教我？"
```

**转折**：
- 学徒不是要告发，而是要利用萨尔贡
- 揭示了学徒的自私和野心
- 友谊彻底破裂

**钩子**：
> 突然，河边芦苇丛中传来响动。
> 两个人同时转头。
> 一个身影从阴影中走出。
> "有趣，"那人说，"非常有趣。"

### 场景2：祭司长的登场（发展，1000字）

**时间**：紧接着
**地点**：河边
**人物**：萨尔贡，学徒，祭司长（首次登场）

**场景目标**：
1. 祭司长正式登场（所罗门之门的代表）
2. 展示祭司长的威严和神秘
3. 揭示所罗门之门在监视异常者
4. 萨尔贡第一次面对权威

**人物描写**：
```
祭司长穿着深色的长袍，胸前绣着复杂的图案——圆中套着方形，方形中有61个点。

他的眼睛深邃，像是能看透人心。

"我是神庙的祭司长，"他说，"也是...所罗门之门的守门人。"

萨尔贡感到一阵寒意。

"我们一直在观察你，"祭司长说，"从三个月前开始。你的无意识敲击。你的瞳孔异常。你是...特别的人。"
```

**关键对话**：
```
祭司长：（对学徒）"你做得很好。及时发现，及时报告。"
学徒：（震惊）"我没有报告..."
祭司长：（微笑）"不，你报告了。三天前。你在神庙外徘徊，想要告发，但最终没有进门。但我们看到了你。我们总是在看着。"

学徒脸色苍白。

祭司长：（对萨尔贡）"你不用担心他。他只是个工具。我关心的是你。"
萨尔贡："我...我什么都没做。"
祭司长："正是如此。你什么都没做，但你的手指在创造。你的瞳孔在发光。这是...源代码在通过你表达。"
```

**埋下线索**：
- CLUE-CC-001：所罗门之门
- CLUE-SURVEILLANCE-01：监视系统
- CLUE-SOURCE-01：源代码（初次提及）

### 场景3：监视与威胁（转折，1200字）

**时间**：第二天
**地点**：作坊内外
**人物**：萨尔贡，暗中监视者（所罗门之门的探子）

**场景目标**：
1. 展示萨尔贡被监视的状态
2. 他的日常生活被渗透
3. 心理压力的积累
4. 发现监视者的痕迹

**监视的展示**：
```
萨尔贡开始注意到：

- 河边的渔夫，每天都会在同一个位置，但从不钓鱼
- 市场上的卖花女，目光总是追随着他
- 作坊对面的屋顶，偶尔会闪过光点（望远镜？）

所罗门之门的监视无处不在。

他尝试验证：
故意走向神庙的反方向。
那个渔夫立即收起渔具，跟了上来。

他停下来，转头直视渔夫。
渔夫低下头，假装整理渔网。

萨尔贡明白了。
他已经被标记了。
```

**心理压力**：
> 萨尔贡开始不敢睡觉。他害怕在睡梦中再次无意识敲击，害怕再次瞳孔发光。
> 他害怕自己。

**发现**：
```
晚上，萨尔贡假装睡着。

从窗户的缝隙中，他看到对面屋顶上的人影。
那人拿着一个奇怪的东西，对着他的方向。

是记录板？还是某种...仪器？

那人发现萨尔贡在看，立即缩回头。
但萨尔贡已经看到了。

他不仅被监视，还被记录。
被研究。
```

### 场景4：决定（结尾，500字）

**时间**：当晚
**地点**：萨尔贡的住处（奴隶区）
**人物**：萨尔贡（独自），学徒（最后出现）

**场景目标**：
1. 萨尔贡决定不再逃避
2. 学徒的最后忏悔（或威胁）
3. 强烈的钩子：对抗开始
4. 为第5集铺垫

**关键段落**：
```
学徒出现在门口。

"对不起，"学徒说，"我不知道他们会监视所有人。我以为..."
萨尔贡：（平静）"没关系。"
学徒：（惊讶于他的平静）"你不恨我？"
萨尔贡："我恨的不是你。是他们在利用你。就像利用我一样。"

学徒沉默。

萨尔贡：（站起身）"告诉祭司长。告诉他，我不会再躲了。"
学徒：（惊恐）"什么？"
萨尔贡：（看着窗外的监视者）"如果他们想看，就让他们看个够。但我不会再假装正常。我要理解我身上的...这种力量。"

学徒后退一步。

萨尔贡的眼睛，在月光下，闪过一丝光芒。
```

**钩子**：
> "告诉他们，"萨尔贡说，"游戏开始了。"

**主题**：
- 从逃避到面对
- 从受害者到反抗者
- 力量的觉醒
