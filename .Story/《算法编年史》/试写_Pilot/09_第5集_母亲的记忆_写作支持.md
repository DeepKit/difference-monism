## 场景结构（4场景，3500字）

### 场景1：深夜的河边（开头，800字）

**时间**：深夜
**地点**：幼发拉底河畔
**人物**：萨尔贡（独自），监视者在远处

**场景目标**：
1. 萨尔贡无法入睡，来到河边
2. 监视者仍在，但他不再恐惧
3. 触发回忆的契机（看到水中的倒影）
4. 建立孤独和沉思的氛围

**环境描写**：
```
深夜的河边，只有月光和水声。

萨尔贡坐在河岸上，看着水中的倒影。月光把他的脸照得惨白，瞳孔里的光芒若隐若现。

对面芦苇丛中，那个渔夫仍在。但萨尔贡不再在意。

他想起了母亲。

想起她最后几天的眼神。
想起她说过的话。
想起她投河的那个夜晚。
```

**触发回忆**：
> 萨尔贡把手伸入水中，搅碎了倒影。
> 水波荡漾，像是时光的涟漪。
> 他闭上眼睛，让记忆涌来。

---

### 场景2：母亲的过去（发展，1000字）

**时间**：闪回（6年前）
**地点**：萨尔贡的童年家中
**人物**：小萨尔贡（7岁），母亲

**场景目标**：
1. 展示母亲的温柔和智慧
2. 母亲也有"异常"（暗示）
3. 母亲教导萨尔贡生存之道
4. 建立母子深情

**关键段落**：
```
母亲坐在床边，手中拿着一块泥板。

"萨尔贡，"她说，"你知道为什么我们要在晚上工作吗？"
小萨尔贡："因为白天太热？"
母亲：（微笑）"不。因为晚上，我们可以听见自己的心跳。"

她把手放在胸口："在这里，有一个声音。不是语言，是...节奏。"
小萨尔贡：（困惑）"节奏？"
母亲："是的。有些人能听见。有些人不能。"

她看着小萨尔贡的眼睛："你能听见吗？"
小萨尔贡：（侧耳）"我...我不知道。"
母亲：（抚摸他的头）"没关系。有一天，你会听见的。"

小萨尔贡注意到，母亲的手指在轻轻敲击床沿。
嗒、嗒、嗒。
61次。
```

**暗示**：
- 母亲也有无意识敲击的习惯
- 母亲知道这种"能力"
- 这是遗传的

**情感描写**：
- 母爱的温暖
- 神秘的传承
- 童年的安全感

---

### 场景3：母亲的死亡（转折，1200字）

**时间**：闪回继续（母亲去世前）
**地点**：家中，河边
**人物**：小萨尔贡，母亲

**场景目标**：
1. 母亲的异常加剧（无法控制）
2. 她意识到自己的命运
3. 她选择死亡，保护萨尔贡
4. 情感高潮

**关键段落**：
```
母亲的状态越来越糟。

她开始无法控制自己的手指。即使在睡觉时，手指也会无意识地敲击。
她开始出现幻觉，说看到"代码"在空中飘浮。
她开始害怕自己。

"萨尔贡，"她在最后的那个晚上说，"记住，无论发生什么，不要让他们抓到你。"
小萨尔贡：（恐惧）"谁？母亲，谁要抓你？"
母亲：（看着窗外）"所罗门之门。他们一直在看着。我年轻时，他们就想抓我，但我逃了。现在...我逃不了了。"

她拿出一小块泥板，上面刻着61个点。
"这是地图，"她说，"指向真相。但我没能解开它。也许...也许你能。"

小萨尔贡哭泣："不要离开我。"
母亲：（拥抱他）"我不会离开你。我会变成水，变成风，变成你眼中的光。我会一直在你身边。"

那天晚上，她走进了河里。

小萨尔贡追出去，只看到月光下的河水，和水中逐渐消失的光芒。

那光芒，和她瞳孔里的一模一样。
```

**情感高潮**：
- 母亲的选择（牺牲）
- 传承的揭示（泥板上的61点）
- 萨尔贡的创伤

**埋下线索**：
- CLUE-MOTHER-01：母亲的秘密
- CLUE-TABLET-MOTHER：母亲留下的泥板
- CLUE-ESCAPE-01：母亲年轻时逃脱所罗门之门

### 场景4：觉醒（结尾，500字）

**时间**：回到现在
**地点**：河边
**人物**：萨尔贡（独自）

**场景目标**：
1. 萨尔贡理解母亲的牺牲
2. 他决定继承母亲的意志
3. 不再逃避，要解开61点的秘密
4. 强烈的情感钩子

**关键段落**：
```
萨尔贡睁开眼睛。

河水仍在流淌，月光仍在照耀。

他明白了。

母亲不是自杀。她是被所罗门之门逼死的。就像现在他们要逼死他一样。

但母亲留下了希望。那61个点的泥板。指向真相的地图。

萨尔贡站起身。

他不再是一个孤独的奴隶。他是传承的一部分。母亲的传承。那些在他之前的异常者的传承。

"我不会让你白死，母亲。"

他看着水中的倒影，看着瞳孔里的光芒。

"我会解开这个谜。我会找到真相。"

身后的芦苇丛中，监视者仍在。

但这一次，萨尔贡转过身，直视那个方向。

"告诉他们，"他说，声音在夜空中回荡，"告诉他们，萨尔贡不再逃避。"

"告诉他们，第73柱，回来了。"
```

**钩子**：
> 第73柱。
> 母亲曾经逃脱的编号。
> 现在，萨尔贡要继承它。

**主题**：
- 传承与继承
- 从孤独到连接
- 觉醒与决心

## 写作要点

### 闪回的处理
- 用触发点引入（河水的倒影）
- 闪回要情感强烈
- 用细节让回忆真实（母亲的手势，声音）

### 情感的控制
- 不要过度煽情
- 用细节展示情感（母亲的动作，萨尔贡的反应）
- 悲剧要有美感

### 悬念的建立
- 母亲知道什么？（没有完全揭示）
- 61点的泥板在哪里？（萨尔贡需要找到）
- 第73柱是什么意思？（后续揭示）
