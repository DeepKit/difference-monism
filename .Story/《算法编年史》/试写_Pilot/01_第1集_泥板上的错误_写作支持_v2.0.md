## 场景结构（4场景，3500字）

### 场景1：税收泥板（开头，800字）

**时间**：公元前1750年，尼普尔城，洪水退去后
**地点**：神庙泥板作坊
**人物**：萨尔贡（13岁奴隶书吏）

**场景目标**：
1. 建立历史背景（古巴比伦、税收季节）
2. 展示萨尔贡的工作：校对税收泥板
3. 发现哈兰村泥板的错误（伪造/调换）
4. 展示萨尔贡的计算天赋（瞬间发现异常）

**关键细节**：
- 泥板内容：哈兰村，217户，每户3舍客勒，应缴651舍客勒，实缴648舍客勒
- 异常点：笔迹太均匀（左撇子书吏的特征缺失）、泥质不同（下游泥土vs上游沙土）
- 萨尔贡的生理特征：专注时瞳孔极度收缩（"凝视病"）

**历史真实植入**：
- 美索不达米亚的税收制度
- 泥板作为官方记录
- 书吏（scribe）阶层

**钩子**：
> "有人在系统性地偷窃税收。"萨尔贡低语。
> 他不知道，这句话会改变他的一生。

### 场景2：学徒的警告（发展，1000字）

**时间**：当天下午
**地点**：作坊
**人物**：萨尔贡，纳布（17岁学徒）

**场景目标**：
1. 纳布发现萨尔贡发现了错误
2. 警告萨尔贡：这是祭司长/税收官的贪污
3. 揭示"数学作为权力工具"的现实
4. 建立纳布的复杂角色（体制内但清醒）

**关键对话**：
```
纳布："忘掉你看到的。把这块泥板放回去，记录为正常。"
萨尔贡："但有人在偷神庙的税。这是渎神。"
纳布："这是生存。你以为只有这一处吗？是祭司长本人。是他的侄子。是税收总管。"

萨尔贡："那为什么要记录？如果数字可以被伪造..."
纳布："正因为可以被伪造，所以才要计算。你要算得足够精确，才能知道偷了多少。你要控制数字，才能控制人。"
```

**历史知识点**：
- 统治阶级发明数学是为了按人头计税
- 数字可以被用作剥削工具
- 会计算的人掌握权力

**埋下线索**：
- CLUE-TAX-CORRUPTION：税收贪污系统
- CLUE-NABU-001：纳布的双重立场

### 场景3：结绳的秘密（转折，1200字）

**时间**：当晚
**地点**：河边
**人物**：萨尔贡，纳布，河边老人（首次登场）

**场景目标**：
1. 萨尔贡带着泥板来到河边
2. 纳布介绍河边老人
3. 老人展示**结绳计数**（比泥板更原始的技术）
4. 建立秘密知识传承的氛围

**关键内容**：
```
老人手里拿着几根彩色的绳子，上面打着结。

"泥板会被检查，"老人说，"但绳子可以藏在衣服里。结可以被解开再重系。这是...结绳计数。是书吏们垄断泥板之前，普通人用的方法。"

萨尔贡看着那些绳结，突然意识到——这不是简单的记录。这是一种不同的逻辑。泥板是线性的、官方的、固定的。结绳是灵活的、私人的、可变的。

"为什么要教我？"萨尔贡问。

老人看着萨尔贡的眼睛——那双在计算时会缩成针尖的瞳孔："因为，你是三十年来第一个能同时看见绳结和泥板错误的人。你能穿越不同的计数系统。这种能力...很罕见。也很危险。"
```

**历史真实**：
- **结绳记事**（Quipu）：印加帝国、中国古代都有
- 比文字更古老的计数方式
- 非精英的、民间的知识传统

**埋下线索**：
- CLUE-QUIPU-001：结绳计数技术
- CLUE-ELDER-001：老人的身份
- CLUE-SAVANT-001：萨尔贡的学者综合征

### 场景4：觉醒（结尾，500字）

**时间**：深夜
**地点**：河边
**人物**：萨尔贡（独自思考）

**场景目标**：
1. 萨尔贡理解自己的处境
2. 决定不再假装看不见
3. 强烈的钩子（准备好面对危险）
4. 为第2集铺垫

**关键段落**：
```
萨尔贡独自坐在河边，手里拿着那块伪造的泥板和几根结绳。

他想起了纳布的话："上一次尼普尔城有人能这样看数字，是三十年前。那是个老妇人，能瞬间算出任何账目的错误。你知道她后来怎么样了吗？被指控为巫婆。烧死了。"

因为能看穿数字谎言的人，对掌握数字权力的人来说，就是威胁。

萨尔贡看着自己的手指。那双手沾满泥浆，伤痕累累，属于一个奴隶。但这双手能看穿祭司长的贪污。

他握紧拳头。

"我不会再假装看不见了。"

他看着河水。明天，祭司长会发现有人动了那块泥板。明天，危险就会开始。

但今晚，他是自由的。

自由的计数者。
```

**钩子**：
> 明天，太阳升起时，他就会成为"危险人物"。
> 一个能看穿数字谎言的奴隶。
> 一个体制无法控制的变量。

**主题升华**：
- 一旦你能计数，你就再也无法接受被计数而不问为什么
- 数学天赋作为反抗工具
- 个体 vs 体制
