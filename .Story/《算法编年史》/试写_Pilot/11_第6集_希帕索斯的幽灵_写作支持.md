## 场景结构（4场景，3500字）

### 场景1：发现（开头，800字）

**时间**：公元前5世纪，古希腊
**地点**：毕达哥拉斯学派的聚会场所
**人物**：希帕索斯（年轻数学家）

**场景目标**：
1. 建立古希腊的背景
2. 展示希帕索斯的才华
3. 他的发现：√2是无理数
4. 埋下危机的种子

**关键段落**：
```
希帕索斯看着地上的图形。

一个正方形，边长为1。对角线，按照毕达哥拉斯定理，应该是√2。

但他证明了，√2不能表示为两个整数的比。

它无限，不循环，不可捉摸。

"这不可能，"他的老师说过，"万物皆数，数是整数的和谐。"

但希帕索斯知道，他证明了不可能。

他跑向学派的聚会场所，兴奋得忘记了一切。

"我发现了一个不可能的东西！"他喊道。

他不知道自己刚刚签下了死亡 warrant。
```

**数学概念**：
- 毕达哥拉斯定理：a² + b² = c²
- 当a=b=1时，c=√2
- √2 ≈ 1.41421356...（无限不循环）

**钩子**：
> "这不是错误，"希帕索斯说，"这是真理。"
> "真理，"他的老师低声说，"有时候需要用生命换取。"

### 场景2：审判（发展，1000字）

**时间**：几天后
**地点**：毕达哥拉斯学派的秘密法庭
**人物**：希帕索斯，学派长老们

**场景目标**：
1. 展示学派的权力结构
2. 希帕索斯的审判
3. 知识 vs 权力的冲突
4. 揭示算法的代价

**关键对话**：
```
长老："你发现了什么？"
希帕索斯："我证明了√2不能表示为整数比。"
长老："这意味着什么？"
希帕索斯："意味着...世界比我们想象的更复杂。有些数是无限的，不循环的。它们不属于我们已知的秩序。"

长老们交换眼神。

长老："你知道这会带来什么吗？"
希帕索斯：（困惑）"什么？"
长老："混乱。如果我们接受这种'无限'，我们的一切理论都会崩塌。我们的一切权威都会动摇。"

希帕索斯：（坚持）"但这是真理。"
长老：（冷酷）"真理需要服从秩序。你，希帕索斯，破坏了秩序。"
```

**权力逻辑**：
- 新知识威胁旧秩序
- 权力者害怕失去控制
- "真理"被当作武器

**算法代价**：
- COST-001：知识暴力
- 新算法出现 → 威胁旧秩序 → 权力镇压 → 代价（生命）

### 场景3：处决（转折，1200字）

**时间**：当晚
**地点**：海边
**人物**：希帕索斯，学派执行者

**场景目标**：
1. 希帕索斯的最后时刻
2. 他的坚持和勇气
3. 处决的残酷
4. 萨尔贡感应到这个场景（梦境/幻象）

**关键段落**：
```
希帕索斯被带到海边。

月亮高悬，海浪轻拍礁石。

"最后的机会，"执行者说，"收回你的证明。说那是错误的。"

希帕索斯看着大海。

"我不能，"他说，"因为那不是错误。那是真理。即使我死了，真理仍然存在。"

"那么，你就死吧。"

他们把他绑起来，放入一个密封的箱子。

"等等，"希帕索斯说，"让我再看一眼星空。"

他们允许了。

他抬头，看着无限的星空。

"原来如此，"他微笑，"无限...就是宇宙的本质。"

箱子被沉入海底。

气泡上升，然后消失。

一个天才，因为看见真理而死。
```

**象征**：
- 星空 = 无限 = 真理
- 海洋 = 遗忘 = 权力的胜利
- 气泡 = 知识的短暂

**萨尔贡的感应**：
```
（回到现在）

萨尔贡突然从梦中惊醒。

他满身冷汗。

他梦见了一个希腊人，站在海边，向虚空展示一个正方形。

"看，"那人说，"对角线与边长，没有共同度量。"

然后海浪吞没了他。

萨尔贡知道，那不是梦。

那是...记忆？不，是某种...回响？

那些在他之前的异常者。那些看见真理的人。

他们的命运，都是死亡。
```

### 场景4：代价（结尾，500字）

**时间**：回到现在，第二天
**地点**：河边
**人物**：萨尔贡（独自）

**场景目标**：
1. 萨尔贡理解"算法代价"
2. 他的决心（不放弃）
3. 强烈的钩子
4. 为第7集铺垫

**关键段落**：
```
萨尔贡坐在河边，想着希帕索斯。

"看见不该看见的东西的人，"他低语，"都会死。"

希帕索斯看见了无理数，死了。
母亲看见了...什么？她也死了。

现在，轮到他了。

萨尔贡看着自己的手指。

他可以选择忘记。可以选择假装一切正常。可以像其他人一样，生活在无知的安全中。

但他已经看见了。

61个凹点。瞳孔里的光。母亲的教导。

他不能假装看不见。

"那么，"他对自己说，"我就做希帕索斯没有做完的事。"

"我要让真理，活下去。"

身后，监视者仍在。

但这一次，萨尔贡没有回头。

他已经准备好了。
```

**钩子**：
> 明天，他要开始寻找答案。
> 从61个点开始。

**主题**：
- 知识需要代价
- 但值得追求
- 传承与责任
