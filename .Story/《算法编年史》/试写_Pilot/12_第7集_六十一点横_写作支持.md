## 场景结构（4场景，3500字）

### 场景1：研究（开头，800字）

**时间**：清晨
**地点**：萨尔贡的秘密藏身处（河边洞穴）
**人物**：萨尔贡（独自）

**场景目标**：
1. 萨尔贡拿出母亲留下的泥板
2. 对比两个61点图案（他的和母亲的）
3. 开始寻找规律
4. 建立主动探索的氛围

**关键段落**：
```
萨尔贡坐在洞穴中，面前摆着两块泥板。

一块是他自己无意识敲击的。一块是母亲留下的。

两块泥板上的61个凹点，排列方式几乎相同。

"这不是随机的，"他低语，"这是...某种语言？"

他开始计数。

横向：11个点
纵向：6行
但交点重复计算...

他试着连线。
点与点之间，形成了几何图形。
三角形。正方形。圆的一部分。

"这是...地图？"他困惑地说。

但指向哪里？
```

**研究方法**：
- 计数
- 连线
- 寻找几何图形
- 猜测含义

**钩子**：
> 他突然意识到，这些点的排列，和夜空中的某些星星的位置相似。
> "难道...这是星图？"

### 场景2：无意识（发展，1000字）

**时间**：当天下午
**地点**：作坊内
**人物**：萨尔贡（试图控制无意识敲击）

**场景目标**：
1. 萨尔贡尝试触发无意识状态
2. 部分成功：能感受到但不能完全控制
3. 发现敲击的节奏
4. 二进制的暗示

**关键段落**：
```
萨尔贡坐在工作台前，盯着一块新的泥板。

他试着放松。试着让自己进入那种...空白的状态。

但越是努力，越是清醒。

"不要控制，"他想起母亲的话，"要倾听。"

他闭上眼睛，深呼吸。

然后，他听到了。

不是声音，是...节奏。

嗒、嗒、嗒。

不是随意的，是...有规律的。

短、长、短、短、长...

他睁开眼睛，发现手指已经在泥板上移动。

61次敲击。同样的图案。

但这次，他记住了节奏。

短、长、短、短、长...

像是某种...代码？
```

**二进制暗示**：
- 短 = 0
- 长 = 1
- 61个点 = 61位二进制信息
- 但萨尔贡还不能理解

**关键发现**：
> 他数了敲击的次数。
> 61次，但节奏不同。
> 有些是单次敲击（短），有些是双次（长）。
> 
> "短、短、长、短、长、长..."
> 他试着记录下来。
> 
> 像是一种...只有两种状态的语言。

### 场景3：河边老人（转折，1200字）

**时间**：傍晚
**地点**：河边
**人物**：萨尔贡，河边老人（CLUE-OLDMAN-01回收）

**场景目标**：
1. 河边老人再次出现
2. 他观察萨尔贡的研究
3. 给予暗示性的指导
4. 回收CLUE-OLDMAN-01，埋下CLUE-OLDMAN-TRUTH-01

**关键段落**：
```
萨尔贡在河边画着61个点的图案。

"你在找什么？"

萨尔贡猛地抬头。河边老人站在芦苇丛中，像上次一样。

"我在找..."萨尔贡犹豫，"我在找答案。"

老人走过来，看着地上的图案。

"61个点，"老人说，"61扇门。"

"什么？"

"每扇门，通向一个真相。你已经看见了几扇？"

萨尔贡困惑："我...我不知道。"

老人微笑："你母亲看到了30扇。我...我看到了71扇。"

萨尔贡震惊："你认识我母亲？"

老人：（点头）"我是她的...导师。就像你母亲是你的导师一样。"

他指着图案："这不是地图，孩子。这是...源代码。"

"源代码？"

"宇宙的源代码。61个参数，定义了一切。"

老人转身离去。

"继续研究，"他说，"但不要告诉任何人。特别是...所罗门之门。"
```

**关键揭示**：
- 61个点 = 61扇门 = 61个真相
- 母亲看到了30扇
- 老人看到了71扇（暗示他是第71柱）
- 这是"源代码"

**回收的线索**：
- CLUE-OLDMAN-01：河边老人首次出现（第1集埋下）

**埋下的新线索**：
- CLUE-OLDMAN-TRUTH-01：老人的身份
- CLUE-SOURCE-CODE-01：源代码的概念
- CLUE-30-DOORS：母亲看到的30扇门

## 数学概念

### 61的特殊性
- 60是巴比伦基数（60进制）
- 61 = 60 + 1（超越完满）
- 61是第18个素数
- 61 = 5² + 6²（两个平方数之和）

### 二进制暗示
- 短 = 0
- 长 = 1
- 61个点 = 61位二进制
- 萨尔贡还不能理解，但读者可以暗示
