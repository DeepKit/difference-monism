## 场景结构（4场景，3500字）

### 场景1：救援（开头，800字）

**时间**：深夜（紧接第8集）
**地点**：神庙前的街道
**人物**：萨尔贡（被捕），河边老人，士兵

**场景目标**：
1. 萨尔贡被押往神庙
2. 河边老人出现阻止
3. 展示老人的力量/智慧
4. 救出萨尔贡

**关键段落**：
```
士兵们押着萨尔贡，走向神庙。

突然，一个身影从阴影中走出。

"等一等，"河边老人说。

士兵队长：（警惕）"你是谁？让开！"
老人：（平静）"我是第71柱。"

士兵们震惊。第71柱？七十二柱之一？

老人：（走近）"这个孩子，我要带走。"
队长：（强硬）"不可能。他是异常者，必须接受审判。"
老人：（微笑）"你们确定要和我动手吗？"

他举起手，在空中画了一个图案。

61个点，连成一个复杂的几何图形。

士兵们看着那个图案，开始...困惑。他们的眼神变得茫然。

"你们没有看到任何人，"老人说，"你们抓到了异常者，但他逃脱了。你们追不上。回去报告吧。"

士兵们机械地点头，然后...转身离去。

萨尔贡震惊地看着这一切。

"这是...什么？"他低语。

老人：（转身）"源代码的应用。以后我会教你。"
```

**力量展示**：
- 老人是第71柱（所罗门之门的高层）
- 使用"源代码"的能力
- 不是暴力，而是...影响/控制

**钩子**：
> "来吧，"老人说，"是时候让你知道真相了。"

### 场景2：真相（发展，1000字）

**时间**：紧接着
**地点**：老人的秘密居所（河边小屋）
**人物**：萨尔贡，河边老人

**场景目标**：
1. 老人解释所罗门之门
2. 解释七十二柱
3. 解释第71柱的含义
4. 揭示萨尔贡是第73柱

**关键对话**：
```
老人："所罗门之门，不是神庙，不是组织。它是一种...算法。"
萨尔贡："算法？"
老人："一种控制信息的算法。它选择谁能看到真相，谁不能。"

他点燃一盏灯。

老人："七十二柱，是算法的执行者。第1柱到第72柱，都是被选中的人。"
萨尔贡："被选中？"
老人："看到源代码的人。但能控制自己，不成为威胁。"

老人看着萨尔贡："你...你是不同的。你是第73柱。"
萨尔贡：（震惊）"第73柱？但你说只有72..."
老人：（微笑）"72是完满。73是...错误。溢出。不可控。"

他停顿："你是异常中的异常。你是算法的漏洞。"
萨尔贡："那我...我该怎么办？"
老人：（严肃）"学习。成长。然后...选择。"
```

**关键揭示**：
- 所罗门之门是"算法"（控制系统）
- 72柱是执行者
- 萨尔贡是第73柱（"错误"）
- 第73柱是威胁，也是希望

**回收的线索**：
- CLUE-OLDMAN-01：河边老人（本集揭示身份）
- CLUE-OLDMAN-TRUTH-01：老人的真实身份

### 场景3：教导（转折，1200字）

**时间**：第二天
**地点**：河边（日出）
**人物**：萨尔贡，河边老人

**场景目标**：
1. 老人开始教导萨尔贡
2. 教导如何控制无意识敲击
3. 教导如何看见"源代码"
4. 建立导师-学徒关系

**关键段落**：
```
日出时分，老人和萨尔贡站在河边。

"你的无意识敲击，"老人说，"不是疾病。是能力。源代码在通过你表达。"

萨尔贡："但我控制不住..."
老人："因为你试图控制。不要控制。要...引导。"

老人示范。

他的手指在空中移动，没有触碰任何东西。但萨尔贡看到，空气中留下了光的轨迹。

61个点，连接成线，形成图案。

"看见了吗？"老人问。
萨尔贡：（震惊）"我...我看见光的轨迹..."
老人：（满意）"很好。你开始看见了。"

他指导萨尔贡："闭上眼睛。感受你的手指。不要想，不要控制。只是...感受。"

萨尔贡闭上眼睛。

起初，什么都没有。
然后，他感觉到了。

节奏。短、长、短、短...

二进制的节奏。

"不要控制它，"老人说，"跟随它。"

萨尔贡的手指开始移动，在空中画出图案。

不是无意识的，是有意识的。

第一次，他感觉到了...控制。

不是强制的控制，是和谐的共舞。

"这就是第73柱的能力，"老人说，"不是服从源代码，是...与它共舞。"
```

**能力展示**：
- 老人能看见/操作"光的轨迹"（源代码的可视化）
- 萨尔贡开始学习控制
- 不是抑制，而是"共舞"

**埋下线索**：
- CLUE-CONTROL-01：控制能力的开始
- CLUE-DANCE-01：与源代码共舞

### 场景4：传承（结尾，500字）

**时间**：一周后
**地点**：河边
**人物**：萨尔贡，河边老人

**场景目标**：
1. 展示一周的学习成果
2. 老人给予任务
3. 强烈的钩子
4. 为第10集铺垫

**关键段落**：
```
一周后，萨尔贡已经能基本控制自己的能力。

他不再无意识敲击。他能在需要时，看见源代码的轨迹。

"你学得很快，"老人说，"比你母亲还快。"

萨尔贡："我母亲...她真的是异常者？"
老人：（点头）"是的。第73柱的前任。她逃脱了所罗门之门20年，直到..."
他停顿："直到她选择了死亡，保护你。"

萨尔贡沉默。

老人：（严肃）"现在，我要给你一个任务。"
萨尔贡："什么任务？"
老人："所罗门之门有72柱。第1到第72。你知道第72柱是谁吗？"
萨尔贡：（摇头）
老人：（压低声音）"第72柱，就是所罗门之门本身。它是...算法的人格化。"

萨尔贡震惊。

老人："你的任务，是找到第72柱。理解它。然后...决定。"
"决定什么？"
"决定你是否要成为它的一部分，还是...摧毁它。"

萨尔贡看着老人。

"我是第73柱，"他低语，"错误。溢出。不可控。"

他握紧拳头。

"那么，让我成为最大的错误。"
```

**钩子**：
> "找到第72柱，"老人说，"找到所罗门。"

**主题**：
- 传承与责任
- 选择的自由
- 从受害者到行动者

## 能力系统

### 源代码的能力
1. **看见**：看到源代码的轨迹（光的线条）
2. **引导**：控制无意识敲击，与源代码共舞
3. **影响**：影响他人的感知（老人对士兵做的）

### 第73柱的特殊性
- 不是服从源代码，是与之共舞
- 不可预测，不可控制
- 既是威胁，也是希望

---

## 写作要点

### 能力的展示
- 不要太科幻，保持神秘感
- 用感官描写（光的轨迹）
- 强调"共舞"而非"控制"

### 教导的过程
- 不要一蹴而就
- 展示学习和失败
- 强调内心的转变

### 第72柱的悬念
- 只说它是"所罗门之门本身"
- 不说清楚是什么
- 为后续铺垫
