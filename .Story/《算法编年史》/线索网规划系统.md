## 一、5条主线定义（核心骨架）

### 主线1：信使传承线（The Messenger Line）
**核心**：10位信使的觉醒、传承与转世  
**功能**：提供人物载体，让读者有情感锚点  
**出现频率**：每5-10集推进一次  
**占比**：约15%

**10位信使**：
1. 萨尔贡（第1-500集）- 美索不达米亚
2. 花拉子米（第501-1000集）- 阿拉伯
3. 斐波那契（第1001-1500集）- 意大利
4. 哥白尼/伽利略（第1501-2000集）- 天文学
5. 笛卡尔/费马（第2001-2500集）- 解析几何
6. 牛顿/莱布尼茨（第2501-3000集）- 微积分
7. 巴贝奇/阿达（第3001-3500集）- 机械计算
8. 图灵（第3501-4000集）- 计算机理论
9. AI研究员（第4001-4500集）- 人工智能
10. 读者化身（第4501-5000集）- 元小说觉醒

---

### 主线2：源代码线（The Source Code Line）
**核心**："六十一点横"的本质与演变  
**功能**：全书核心悬念，终极真相的载体  
**出现频率**：每10-20集出现一次  
**占比**：约10%

**演变轨迹**：
- 第1纪：泥板上的二进制刻痕
- 第2纪：圆的分割（π的逼近）
- 第3纪：机械齿轮的排列
- 第4纪：打孔卡与电路
- 第5纪：数字代码与AI

**终章 reveal**：第5000集揭示为算法的自我意识

---

### 主线3：编译器对抗线（The Compiler Conflict Line）
**核心**：控制与自由的永恒对抗  
**功能**：提供叙事张力，冲突引擎  
**出现频率**：每15-30集出现一次  
**占比**：约10%

**各纪元的"编译器"**：
- 第1纪：所罗门之门（祭司集团）
- 第2纪：帝国与教会（权力对知识的控制）
- 第3纪：行会与学术界（知识的垄断）
- 第4纪：国家与资本（技术控制）
- 第5纪：平台与算法（数字控制）

**对抗手段演变**：暴力 → 制度 → 技术 → 算法

---

### 主线4：算法进化线（The Algorithm Evolution Line）
**核心**：算法从简单到复杂的文明进步  
**功能**：展示算法的可能性，正面叙事  
**出现频率**：每3-5集出现一次  
**占比**：约20%

**演进路径**：
```
确定性几何（第1纪）
    ↓
无限逼近（第2纪）
    ↓
机械自动化（第3纪）
    ↓
形式化逻辑（第4纪）
    ↓
涌现智能（第5纪）
```

**每个Epoch的突破**：
- 第1纪：欧几里得几何、毕达哥拉斯定理
- 第2纪：穷竭法、代数诞生
- 第3纪：解析几何、微积分
- 第4纪：布尔代数、图灵机
- 第5纪：神经网络、深度学习、AI觉醒

---

### 主线5：算法代价线（The Algorithm Cost Line）⭐ **新增核心主线**
**核心**：算法对人类社会的负面影响与异化  
**功能**：平衡算法进化的光明面，展示算法的边界与危险  
**出现频率**：每3-5集出现一次，与进化线交替或并行  
**占比**：约20%

**与进化线的关系**：**镜像对照**
```
算法进化（光明）          算法代价（阴影）
────────────────────────────────────────
几何学诞生              知识垄断与暴力
代数传播                算法用于压迫征税
机械计算                时间规训与劳动异化
统计学问世              优生学与大屠杀
AI诊断疾病              算法偏见与歧视
```

**各纪元的代价类型**：
- **第1纪**：知识暴力（希帕索斯因无理数被处死）、劳动异化（金字塔奴隶）
- **第2纪**：经济压迫（算法精确征税导致饥荒）、知识垄断（历法控制农时）
- **第3纪**：时间异化（机械钟规训人类）、系统性风险（金融投机泡沫）
- **第4纪**：科学暴力（优生学）、毁灭性力量（核导弹计算）
- **第5纪**：算法歧视（AI偏见）、劳动替代（结构性失业）、思维控制（信息茧房）

**终章整合**：算法选择提问（包容代价），而非给出答案（忽视代价）

### 5条主线的编年体分布

**不是每集都推进所有主线**，而是：

| 集数类型 | 占比 | 主线数量 | 功能 |
|---------|------|---------|------|
| **主线推进集** | 44% | 1-2条推进 | 推动核心叙事 |

**关键设计**：主线像河流，副线像支流，插播像湖泊——可以独立存在，也可以交汇。

## 二、每集内容规划公式

### 2.1 四维锚定公式

**每集必须包含**：

```
【集内容】= 【时间锚】+ 【空间锚】+ 【人物锚】+ 【线索网】

【时间锚】= 具体年份 + 历史背景 + 季节/时辰
【空间锚】= 具体地点 + 空间特征 + 感官细节
【人物锚】= POV角色 + 状态变化 + 目标/障碍
【线索网】= 1主线 + 2支线 + 1线索点 + 3-5素材
```

### 2.2 线索组合表（每集选择指南）

**主线索选择（必选1条）**：

| 集数范围 | 主线索A | 主线索B | 建议权重 |
|---------|---------|---------|----------|
| 1-1000 | 信使觉醒 | 源代码初现 | A:70%, B:30% |
| 1001-2000 | 传承接续 | 知识传播 | A:50%, B:50% |
| 2001-3000 | 技术突破 | 编译器对抗 | A:60%, B:40% |
| 3001-4000 | 形式化危机 | 人性坚持 | A:40%, B:60% |
| 4001-5000 | AI觉醒 | 读者觉醒 | A→B渐变 |

**副线索选择（必选2条，不重复）**：

从12条副线索中选2条，要求：
- 与主线索有**张力**或**互补**
- 与Epoch重点维度**对应**
- 能植入**至少2个素材**

**组合示例**：
```
第2500集（第3纪，牛顿时代）：
- 主线索：算法进化（微积分诞生）
- 副线索1：战争冲突（英荷战争背景）

- 素材：牛顿-莱布尼茨争议 + 棱镜实验 + 炼金术
```

**暗线索选择（必选1条，每5集1次）**：

不是每集都需要，但每5集必须有1条暗线索出现：
- 欺骗操控（信息不对称）
- 偏见歧视（系统性偏差）

- 未知混沌（留白与神秘）

## 五、写作执行工具

### 5.1 每集规划检查表

**写作前必须确认**：

```
□ 时间锚：具体年份？历史背景？
□ 空间锚：具体地点？感官细节？
□ 人物锚：POV是谁？状态如何？目标？
□ 主线索：哪条？推进还是伏笔？
□ 副线索A：哪条？与主线关系？
□ 副线索B：哪条？与主线关系？
□ 暗线索：本集是否出现？
□ 核心素材：哪个？如何推动情节？
□ 辅助素材：2个？提供什么背景？
□ 隐喻素材：1个？用于什么描写？
□ 本集钩子：悬念是什么？
□ 长程钩子：埋下什么伏笔？
□ 收线索：回收什么前文？
```

### 5.2 线索冲突快速生成器

**输入**：2条线索 + 1素材
**输出**：情节冲突

示例：
```
输入：
- 线索A：算法进化（追求最优解）
- 线索B：艺术审美（追求美感）
- 素材：WAR-006火牛阵（混乱战术）

输出情节：
将军面对敌人，算法建议最优阵型（秩序），
但艺术家建议火牛阵（混乱美感），
最终选择混乱，因为敌人无法预测——
最优不是计算出来的，是创造出来的。
```

### 5.3 素材调用流程

```
1. 确定本集Epoch → 查看该Epoch重点维度
2. 选择主线索 → 从对应维度选核心素材
3. 选择副线索 → 从素材库查关联素材
4. 检查暗线索 → 是否需要系统脆弱性揭示？
5. 融合素材 → 用主素材推动情节，辅助素材丰富背景
6. 植入钩子 → 素材中的悬念点 → 本集钩子
7. 标记线索 → 更新线索追踪表
```

## 七、执行建议

### 7.1 立即建立的工具

1. **线索追踪表**：Excel/Notion表格，记录每条线索状态
2. **素材-线索映射图**：可视化哪些素材支撑哪些线索

4. **线索回收日历**：标记何时回收哪条线索的伏笔

### 7.2  Pilot集测试

**建议先用3-5集测试**：
- 第1集：测试线索启动
- 第38集：测试线索交织
- 第99集：测试线索高潮
