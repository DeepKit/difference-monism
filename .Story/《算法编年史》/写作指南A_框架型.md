### 1.2 契约撰写12步法

**步骤1-2：选坐标、定类型**
- 根据当前集数确定phase（六阶）和mode（五态）
- 选择scene_type（6种类型之一）
- 系统自动加载对应模板、陷阱图、测试策略

**步骤3-4：定时空、定人物**
- 填写spacetime（与历史事实核对）
- 确定POV角色和在场角色
- 检查角色状态一致性（从前一集继承）

**步骤5-7：定落点、定出口、定信息门**
- must_land：必须是可校验的命题
  - ✅ 好例子："萨尔贡发现数据里藏着不属于他的变量名"
  - ❌ 坏例子："萨尔贡有所领悟"（无法量化）
- exit_to：方向+情绪+钩子
- information_gate：明确 reveal/withhold

**步骤8-9：定红线、定弹性**
- taboos：绝对不能违反的规则
- must_not_reveal：悬念保护清单
- deviation_budget：human_confirmed必须为true

**步骤10：写直觉**
- intuition_notes：纯自然语言，系统不解析
- 用于AI生成时的氛围参考

**步骤11-12：声明副作用**
- character_state_diffs：角色状态变化
- foreshadow_ops：伏笔操作（埋/推进/收）

## 二、六阶节奏控制（ASTO动力学）

### 2.1 六阶叙事特征

| 六阶 | 叙事特征 | 典型场景 | 写作要点 |
|------|----------|----------|----------|
| **a 混沌** | 信息碎片、氛围铺垫、多义性 | 开场、梦境、回忆 | 意识流、感官细节、不解释 |
| **b 秩序** | 规则建立、关系明确、逻辑清晰 | 对话推进、计划制定 | 信息交换、人物关系确认 |
| **c 流变** | 渐变积累、张力蓄压、悬念推进 | 线索串联、疑点堆叠 | 节奏加速、细节累积 |
| **d 脉冲** | 突变、高潮、不可逆转折 | 真相揭露、对抗爆发 | 压缩时间、强烈冲突 |
| **e 崩解** | 旧秩序瓦解、失控、情感冲击 | 信念崩塌、关系破裂 | 失控感、情感释放 |
| **f 归元** | 沉淀、收束、新平衡建立 | 独处反思、余韵留白 | 慢节奏、留白、新起点暗示 |

### 2.2 纪元内的六阶分布

**标准分布（每纪元1000集）**：

```
混沌期：1-50集（5%）
  └─ 新信使觉醒，世界观建立
  
秩序期：51-200集（15%）
  └─ 算法知识系统化，人物关系确立
  
流变期：201-600集（40%）
  └─ 算法应用、文明演进、矛盾积累
  
脉冲期：601-800集（20%）
  └─ 重大转折（所罗门崩溃、图灵之死）
  
崩解期：801-900集（10%）
  └─ 旧纪元秩序瓦解，向新纪元过渡
  
归元期：901-1000集（10%）
  └─ 封印完成，向下一纪元传递
```

**警示**：AI典型产出会均匀分配六阶篇幅，必须人工调整！

### 2.3 七序权重分配（场景内部结构）

**标准场景（3500字）的七序权重**：

```
觉醒序（开场）：10-15%
  └─ 角色进入情境，感官触发
  
感知序（观察）：15-20%
  └─ 环境描写，信息接收，察觉异常
  
解析序（思考）：25-35%
  └─ 内心推理，分析局势，建立假设
  
干预序（行动）：15-25%
  └─ 采取行动，对话交锋，做出反应
  
设计序（决策）：5-10%
  └─ 形成计划，做出决策，重新定义目标
  
回溯序（反思）：10-15%
  └─ 审视后果，自我反思，情绪沉淀
  
消解序（收束）：5-10%
  └─ 场景收束，情绪落位，暗示延续
```

**不同scene_type的权重调整**：

| 场景类型 | 重点七序 | 权重调整 |
|----------|----------|----------|
| dialogue_driven | 干预序 | +10% |
| introspection | 解析序+回溯序 | +15% |
| action_sequence | 干预序 | +20% |
| revelation | 觉醒序+消解序 | +10% |
| atmosphere | 感知序 | +25% |

## 三、单元写作规范（每集3500字）

### 3.1 3+3+3+1时间结构

```
总时长：10分钟（约3500字）

前3分钟（约1000字）：钩子（危机/发现/悬念）
├─ 功能：立即抓住读者
├─ 技巧：从动作或对话开始，避免冗长描写
└─ 示例："萨尔贡三十岁，伊什美三十三岁。他们已经被囚禁了两年。"

中3分钟（约1200字）：发展（人物关系/算法知识/世界展开）
├─ 功能：建立情境，铺垫高潮
├─ 技巧：交替进行人物互动和知识植入
└─ 要点：知识必须通过人物困境呈现，而非说教

后3分钟（约1000字）：高潮（决策/对抗/顿悟）
├─ 功能：情感或智力的顶点
├─ 技巧：压缩时间，增加紧迫感
└─ 示例：萨尔贡触碰所罗门额头，说"我要成为错误"

最后1分钟（约300字）：心法+预告（知识封装+下集钩子）
├─ 功能：给读者收获感和期待感
├─ 技巧：心法用比喻，钩子用悬念
└─ 结构：【今日算法心法】+【下一集预告】
```

## 六、单双集结构模式

### 6.1 单数集结构（叙事-钩子-引导追更）

**核心目标**：推动主线剧情，制造悬念，引导用户追更和分享

| 阶段 | 功能 | 内容要点 |
|------|------|----------|
| **开篇（5%）** | 承接上集 | 简短回顾上集结尾的悬念或冲突 |
| **发展（60%）** | 主线推进 | 主角面对挑战，展示算法/数学思维解决问题 |
| **高潮（20%）** | 核心揭示 | 关键发现、突破或转折 |
| **钩子（10%）** | 悬念制造 | 抛出新的谜团、危机或惊人发现 |
| **引导（5%）** | 行动召唤 | "下一集将揭示..."、"分享给朋友一起解谜" |

**特点**：
- 每集解决一个小问题，但引出更大问题
- 结尾必须有"未完成感"
- 适合在关键节点设置付费点（如重大揭示前）

### 6.2 双数集结构（支线展开-世界观-人物-回收线索）

**核心目标**：丰富世界观，深化人物，回收/铺设线索

| 阶段 | 功能 | 内容要点 |
|------|------|----------|
| **开篇（10%）** | 支线引入 | 切换到副线人物或新场景 |
| **展开（50%）** | 多线叙事 | 展示不同视角，揭示世界规则 |
| **交织（25%）** | 线索连接 | 支线与主线产生关联或碰撞 |
| **揭示（10%）** | 信息释放 | 回收旧线索或铺设新线索 |
| **过渡（5%）** | 回归主线 | 自然切回主线或预示下集交汇 |

**特点**：
- 节奏相对舒缓，注重氛围和细节
- 让读者"喘口气"，同时获得新信息
- 适合世界观揭秘和人物关系深化

### 6.3 两种结构的配合节奏

```
第1集（单）：河边的孩子发现泥板 → 钩子：神秘符号的含义？
第2集（双）：切换到祭司视角，揭示符号背景 → 回收：符号与神殿有关
第3集（单）：主角解读符号，遭遇危机 → 钩子：被监视了！
第4集（双）：监视者的身份和动机 → 回收：神殿的阴谋
...
```

```
  │   ╱   ＼___     ___
  │  ╱         ＼___╱   ＼___
  │_╱                        ＼
  │
  └────────────────────────────→ 时间
    引入   展开    交织    沉淀
```

#### 6.4.2 时间节奏控制

| 技巧 | 单数集 | 双数集 |
|------|--------|--------|
| **时间压缩** | 多用（几小时→几百字） | 少用（实时或延展） |
| **场景切换** | 快切（3-5个场景） | 慢切（1-2个场景深入） |
| **对话节奏** | 短句、快节奏 | 长句、可停顿 |
| **内心独白** | 简短（危机中无暇思考） | 可长（有时间反思） |

#### 6.4.3 视角切换策略

**单数集视角**：
- 70%主角POV
- 20%对手/反派POV（制造紧张）
- 10%全知（必要信息补充）

**双数集视角**：
- 40%主角POV（沉淀、反思）
- 40%配角POV（展开支线）
- 20%反派POV（揭示阴谋）

#### 6.4.4 钩子设计技巧

**单数集钩子类型**：
1. **危机型**："刀已经架在脖子上"
2. **发现型**："他看到了不该看的东西"
3. **选择型**："两条路，选错了就是死"
4. **信息型**："原来真相是..."（戛然而止）

**双数集钩子类型**：
1. **情感型**："她第一次说出了真心话"
2. **预示型**："那个符号，他好像在哪里见过"
3. **关联型**："原来这两件事是相连的"
4. **悬念型**："那个人，究竟是谁？"

#### 6.4.5 闭环与开放的平衡

**单数集**：开放为主（70%），小闭环为辅（30%）
- ✅ 好：解决了眼前危机，但引出更大阴谋
- ❌ 坏：什么都没解决，又堆了一堆问题

**双数集**：闭环为主（60%），开放为辅（40%）
- ✅ 好：完成了一个支线任务，但留下情感悬念
- ❌ 坏：完全闭环，读者没有期待了

#### 6.4.6 单双集配合的黄金比例

**标准模式（推荐）**：
```
单-双-单-双-单-双（1:1交替）
适合：主线推进期、人物成长期
```

**加速模式**：
```
单-单-双-单-单-双（2:1）
适合：高潮前夕、危机密集期
```

**减速模式**：
```
单-双-双-单-双-双（1:2）
适合：世界观展开期、情感沉淀期
```
