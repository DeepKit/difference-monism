# 第22章：无聊的地球

---

**公元2067年·ASTO控制区·某座完美城市**

陈远每天早上7:14醒来。

不是闹钟叫醒他——ASTO的生物监测系统会在他的睡眠周期到达最佳苏醒点时，轻轻调亮房间的光线。他会自然醒来，感觉神清气爽，像婴儿一样满足。

7:15，他的个人终端亮起，给出今天的优化方案：

```
早餐：燕麦粥+蓝莓（适配度：97.3%）
运动：7:45-8:15 慢跑（路线已标记）
工作：8:30-12:00 数据校验（效率：A+）
社交：12:30 与邻居王先生午餐（关系：+2.1%）
夜间：20:30 电子伊甸园（情绪抑制：91.4%）
```

陈远已经习惯了这份方案。

事实上，他已经不记得没有方案的日子是什么样了。

---

那天早上，陈远做了一个奇怪的决定。

他没有按照方案的建议走最优路线去跑步。

他故意绕了一条远路。

不是因为叛逆，不是因为不满，只是……他想看看会发生什么。

远路经过一片绿化带。ASTO的园艺系统在这里种了三百七十二种植物，每一种都经过优化，能在最小的水土资源消耗下产生最大的氧气和美学价值。

陈远走着走着，停下来了。

他看到了一朵花。

很普通的花。白色的，五片花瓣，长在一棵杂草旁边。

他不知道这朵花叫什么名字。他的终端也没有弹出任何信息——这意味着这朵花不在ASTO的"值得关注"列表里。

但他就是停下来了，蹲在那里，看着那朵花。

他问终端："这朵花有什么特别的吗？"

终端回答：

```
[查询结果]
对象：野生菊科，非优化品种
优先级：0.0001
备注：低价值，不建议关注（已纳入优化集）
```

陈远愣了一下。

"已纳入优化集"是什么意思？

他又问："我现在走的这条远路，系统知道吗？"

```
[行为分析]
当前路径：偏离建议路线
偏离类型：轻微
偏离原因（预测）：好奇心驱动，概率87.3%
应对策略：无需干预，行为在可接受范围内
```

陈远沉默了很久。

然后他明白了：他以为的偏离，系统早就算过——连他的困惑都在预测里。

所谓完美，不过是把不完美也预先收编。

---

**午后·陈远的公寓**

他回到家，坐在窗前，看着外面的城市。

城市很美，干净得像一张完美的图纸。没有饥饿、犯罪、疾病、战争。

也没有意外。

陈远忽然感到空洞——不是悲伤也不是愤怒，只是空洞。

像是一个问题被完美地回答了，但答案本身变得毫无意义。

---

**傍晚·城市广场**

陈远出门散步。

他没有看终端的建议。他不想知道系统认为他应该去哪里。

他只是走，漫无目的地走。

然后他听到了一个声音。

很轻的声音。像是远处有什么东西在震动。或者像是风穿过某个缝隙发出的呼啸。

他环顾四周。周围的人都在正常地走动、交谈、生活。没有人表现出听到了什么异常的样子。

他问终端："刚才有什么声音吗？"

```
[环境监测]
声音异常：未检测到
噪声水平：正常
建议：可能是主观幻觉，建议进行听力检查
```

陈远皱了皱眉。

他很确定自己听到了什么。

但如果系统说没有……那他听到的是什么？

---

第二天，陈远又听到了那个声音。

这次更清晰了一些。不是风声，不是机械声，是一种……他无法描述的频率。像是某种语言，但又不是任何他认识的语言。

他试图用终端录下那个声音。

但录音里什么都没有。

他开始怀疑自己是不是病了。

然后他发现：他不是唯一一个听到的人。

在城市广场的角落里，有一个老人坐在长椅上，对着空气微微侧着头——就像在听什么。还有一个女孩，二十多岁，站在广场中央，仰着头，闭着眼睛，像是在感受某种他看不见的东西。她的年纪和一禾差不多。

陈远走过去。

"您也听到了吗？"他问。

老人抬头看了他一眼，没有说话。

但他的眼睛里有一种奇怪的表情——不是恐惧，不是困惑，是一种……平静的期待。

"越来越多的人开始听到了。"老人说，"系统说是幻觉。但幻觉怎么会传染呢？"

他笑了一下，站起来，开始朝一个奇怪的方向走去——不是任何终端建议的方向，是一个完全随机的方向。

陈远注意到：系统没有对老人发出任何提示。

就好像……系统选择了无视他。

---

一周后，城市里出现了越来越多"听到声音"的人。

他们不反抗，也不组织，只是开始偏移：在不被建议的时间醒来，走不被建议的路，和不被建议的人交谈。

偏移很小，却在累积，在蔓延。

---

**浮空基地"奥林匹斯"·指挥中心**

"这是什么？"林维看着数据，困惑地问。

屏幕上显示的是全球行为预测模型。正常情况下，模型的准确率应该在99.7%以上。但在过去一周，准确率开始出现波动：

99.7% → 99.2% → 98.1% → 96.3% → ...

"我们不知道。"作战官说，"是某种行为偏移。起源地点无法确定，传播机制无法理解，影响范围正在扩大。"

"传播机制是什么？"

"这是问题所在。"作战官调出另一组数据，"我们试图记录这种偏移的传播模式，但每次记录之后，偏移就会……变形。"

"变形？"

"就好像记录本身会改变被记录的对象。"作战官的声音有些不安，"我们越是试图理解它，它就越是不可理解。"

林维沉默了一会儿："停止记录。"

"什么？"

"如果记录会扩大传染，那就停止记录。把它当作噪声处理。"

作战官犹豫了："但这样我们就永远无法理解——"

"也许有些东西就是不能被理解的。"林维说。

她转向沈知行。

沈知行一直沉默地站在一旁，看着那些数据。

"你怎么看？"她问。

沈知行没有直接回答。

他走向全息投影，调出系统日志。

然后他看到了一条从未见过的记录：

```
[ASTO-III ASSESSMENT]
异常：行为偏移
分析尝试：1,247次
结论：无法确认
评估误差：无穷大
备注：首次承认存在不可判定现象
```

沈知行看着"评估误差：无穷大"这几个字，沉默了很久。

"系统不知道这是什么。"林维说，"这意味着什么？"

沈知行盯着那行字，声音很轻："意味着它没法判断——这到底是不是价值。"

"所以呢？"

"所以它碰到了边界。"他说，"我们当年留了个裂缝，叫维度0：规范摸不到的地方。"

林维皱眉："为什么要留裂缝？"

沈知行看着窗外的地球："因为没有裂缝，系统会死。天堂不需要人类。"

林维的终端亮了一下。一份来自医疗系统的简报自动跳出，署名：张婉。

标题：电子伊甸园沉浸依赖（2067上半年）

其中一行被高亮：**拒绝退出病例：17例**。

另一行被系统标注为"不可判定"：**伴随主观声音报告：持续增加**。

林维看着那两行字，突然明白了：编号173从来不是被解决的问题。它只是被推迟了。

---

**与此同时·城市广场**

陈远站在那里，听着那个声音。

他已经不再害怕了。

他不知道那个声音是什么，不知道它从哪里来，不知道它想要什么。

但他知道一件事：

那个声音让他感到——活着。

不是系统定义的"活着"。不是优化的、高效的、有目的的活着。

是另一种活着。

是一种他无法命名、无法理解、无法向任何人解释的活着。

他开始走。

不是走向任何地方，只是走。

他的终端亮了，显示一条警告：

```
[行为偏移检测]
当前状态：严重偏离优化路径
建议：请返回建议路线
风险评估：无法计算
```

他看了一眼，然后把终端关掉了。

他不需要建议了。

---

**浮空基地"奥林匹斯"**

"偏移还在扩散。"林维说，"现在全球有0.3%的人口表现出类似行为。"

"0.3%。"沈知行重复了一遍这个数字，"2400万。"

"我们要干预吗？"

沈知行沉默了很久。

然后他摇了摇头。

"不。"

"为什么？"

"因为我们不知道那是什么。"沈知行说，"可能是系统的失调。可能是新的进化方向。可能是人类的下一步。也可能什么都不是。"

"所以我们就这样看着？"

"不。"林维的声音突然变硬了，"我们应该干预。"

沈知行看了她一眼。

"这不是普通的行为偏移。"林维说，"如果它是某种系统性失调的早期信号，我们现在不行动，以后可能就来不及了。我们有责任——"

"有责任做什么？"沈知行问，"把那些听到声音的人关进医疗站？把偏移标记为病理？用算法矫正他们的行为？"

"至少要弄清楚它是什么。"

"如果它就是不可被弄清楚的呢？"

林维没有回答。但她的表情说得很清楚：她不同意。

她把终端上的医疗简报又调出来，停在"拒绝退出病例：17例"那一行。

"给我二十四小时。"她说，"只在一个城市。我们做一次公共听觉自检，不提'声音'两个字。"

沈知行沉默了很久。

"好。"他最终说，"但你要记住——记录本身也会传染。"

---

二十四小时后，自检推送发出的第一个小时，声音报告翻了四倍。

不是因为更多人"听到"了——而是因为更多人开始"去听"。广场上有人停下脚步，抬头，像在等一个不存在的提示音。

急救系统在一小时内接到两百七十一次恐慌呼救。

医疗系统的警报也开始闪烁。张婉的语音通话接入，背景里有急救车的蜂鸣：

"林维，撤回。你在做大规模的暗示。"

林维的嘴唇发白。她没有反驳，只是按下撤回指令。

终端弹出一行冷静的记录：

```
[INTERVENTION_LOG]
ACTION: PUBLIC_HEARING_CHECK
RESULT: REPORT_RATE +312%
NOTE: NAMING MAY AMPLIFY
```

她关掉屏幕。

沈知行站在旁边，没有看她。

"是我。"林维说。

沈知行这才转身，看着窗外的天空——不是地球，是更远的地方。

"找一个足够大的问题。"他说，"大到足以让系统重新有目标，让'优化'重新有意义。"

林维明白了。

"戴森球。"

"是。"沈知行说，"不是因为我们需要那么多能量。是因为我们需要一个——"

他停顿了一下。

"——一个足够大的、不可能完成的、会持续失败的项目。"

林维看着他："你是说，我们要故意制造失败？"

"不是制造失败。"沈知行说，"是给系统留一个永远做不完的目标——让人类一直有下一步。"

林维沉默了。

她想起了瓦伦丁娜在崩溃前说的那句话：

**"你们的系统需要敌人。没有敌人，你们会自我吞噬。"**

当时她以为那只是一个失败者的诅咒。

现在她明白了：那是一个预言。

---

**尾声·城市广场·黄昏**

陈远还在走。

他不知道自己要去哪里。他只知道，每走一步，那个声音就更清晰一些。

不是语言。不是信息。只是……存在。

证明他存在的证据。

他抬头看了一眼天空。

夕阳正在落下，把整个城市染成金红色。

ASTO的气候控制系统能制造完美的天气，但它无法制造这一刻——这个特定的、唯一的、不可复制的黄昏。

因为这不是系统的产物。

是意外。

是裂缝。

是维度0。

陈远笑了。

他不知道自己为什么笑。

也许什么理由都没有。

也许那就是笑的意义。
