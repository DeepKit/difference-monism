# 第05章 枯井惨案 (The Dry Well)

1.

边境线外，Zone-F9 的村长阿木（X）站在那口已经干得冒烟的深井旁。
这口井是三个村庄唯一的水源。
他向 ASTO 的边境哨所申请“人道主义水卡”。
自动回复：`申请驳回。理由：申请人 IP 位于非辖区。建议向当地政府求助。`
阿木苦笑。当地政府？早就被军阀炸平了。**ASTO 的边界是一堵墙，墙内是文明，墙外是丛林。**

2.

阿木带着全村人凑出来的金戒指，试图贿赂哨所的无人机。
无人机悬停在空中，红色的扫描光束扫过那枚戒指，然后毫无感情地飞走了。
在它的算法里，那个金戒指不是财富，只是一块未被授权的金属。
阿木看着无人机飞回墙内。墙那边，巨大的景观喷泉正在随着交响乐喷涌，水雾在阳光下折射出彩虹。
那一刻，他感到的不是嫉妒，而是**一种作为“非人”的屈辱**。

3.

三天后，水彻底断了。
为了争夺最后几桶泥浆水，隔壁村的人拿着砍刀冲了过来。
阿木不想打，但他必须保护孩子。
他在混战中被砍断了手。
他倒在血泊里，看着天空。天上有一颗 ASTO 的气象卫星划过。
卫星能看见每一朵云，却看不见这地上的血。
**因为在卫星地图上，这里只是一片灰色的“待定区域”。**

4.

火拼持续了一整夜。
倒下了几十人。他们躺在干涸的井口，像沉默的石头。
难民潮开始冲击 ASTO 的边境墙。他们不是为了侵略，只是为了活命。
有人试图翻越电网，被高压电瞬间击落。一个人影挂在铁丝网上，像一面黑色的旗帜。
阿木（仅剩一只手）站在电网外，眼神空洞地看着墙内的那个喷泉。
他没有哭，也没有喊。
他只是觉得，这个世界，大概是真的坏掉了。

# 第06章 红色按钮 (The Red Button)

1.

ASTO 核心区，能源调度室。
陈工（土）看着屏幕上的边境监控。那个挂在电网上的影子，让他想起了多年前的自己——那时候他也是个难民，是为了抢半块面包被沈知行推开的那个孩子（序章伏笔）。

“系统警报：边境压力值 98%。建议启动‘威慑模式’。”
威慑模式，就是加大电压。

陈工的手在抖。
如果按下去，墙外的人会死更多。如果不按，墙可能会塌。
市政厅的电话打进来了：“陈工！守住边界！这是命令！”

陈工没有接电话。
他端起保温杯，喝了一口水。水是温的，甜的。
但他突然觉得嗓子被堵住了。
**“墙是为了保护里面的人，”他喃喃自语，“但如果里面的人连良心都没了，还保护个屁。”**

2.

他把手伸向了那个从未被使用过的红色按钮——**“逆向输送”**。
这个按钮会切断核心区 15% 的电力，驱动巨大的泵站，把景观用水反向输送到边境。

“警告：此操作违规。您将被解雇。”系统提示。

“那就解雇吧。”陈工的手抖得厉害，热咖啡洒在了裤子上，烫得他叫出了声。
**除了恐惧，他竟然感到一阵隐秘的兴奋。那是把那些高高在上的富人拉入黑暗的快感。平时他们用眼神羞辱他，现在，他用一个指头就能让他们尖叫。这是一种卑微的、甜蜜的复仇。**

但他还是按了下去。

嗡——
巨大的电流声响起。核心区的霓虹灯暗了下来。
景观喷泉停了。

两千公里外。
那口枯井突然发出了轰鸣声。
一股清澈的水流，像白龙一样喷涌而出，冲刷掉了井边的暗红。

阿木愣住了。
难民们愣住了。
他们扔掉了手里的刀，跪在泥水里，疯狂地喝着那带着泥沙的水。

陈工看着监控，瘫倒在椅子上。
他知道自己完了。但他看着阿木那张满是血污的脸，突然笑了。
**和平不是条约。和平是物理上的填平。**

# 第06.5章 抄作业 (Copying the Homework)

**Unit 3.5: 错误的镜像**

与此同时，在 ASTO 的邻国——著名的“铁血联邦”。
他们也引进了 ASTO 的“贡献值系统”。
国王在电视上宣布：“这是最先进的文明技术！我们将用它实现伟大复兴！”

但是，他们为了“效率”，删掉了一个模块：**申诉与复议**。
在这个版本里，系统判你低分，就是低分。没有理由，不接受反驳。

结果：
为了刷分，人们开始互相举报。
为了抢夺“高分岗位”，工人们在工厂里械斗。
甚至出现了“买分杀人”——杀了低分者，可以获得“清除社会负担”的加分。

阿木（那个没死成的X）流亡到了这里。
他本以为逃离了干旱，结果掉进了地狱。
他看着那些为了几分贡献值而疯狂的人群，终于明白：
**没有伦理的技术，比刀剑更锋利。**
ASTO 的伟大不在于它有多强，而在于它**懂得在什么时候停下来**。
