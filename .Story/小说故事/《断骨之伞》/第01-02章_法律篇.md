# 第01章 未被标记的星期五 (Unmarked Friday)

**1.**

周五下午 17:30，暴雨。
外卖员阿力（X）的电瓶车在积水中熄火了。
他试图向平台申诉“不可抗力延误”，但系统只回了他一行冷冰冰的代码：`ERROR: 404 - 气象数据未同步。当前区域判定为：小雨。请继续配送。`
阿力看着没过脚踝的浑水，绝望地发现：**系统活在过去的气象云图里，而他活在当下的暴雨中。**
因为这次“无理”的申诉，他的账号被判定为“消极怠工”，封号三天。

**2.**

周五晚上 23:00。
阿力回到出租屋，发现三岁的儿子额头滚烫。
他抱着孩子冲到社区医院，却被拦在门外。
“你的健康码是灰色的。”护士隔着玻璃窗，声音很遥远，“系统显示你是‘高风险流动人员’，必须走隔离通道。”
“隔离通道要排队四个小时！孩子等不了！”阿力吼道。
“没办法，这是规定。”护士转过身，不再看他。**她不是坏人，她只是被系统屏蔽了同情心。**

**3.**

周六凌晨 03:00。
阿力背着孩子，骑着那辆没电的车，试图去另一家医院。
为了避开监控（以免再次被系统标记），他关掉了车灯，像个幽灵一样在小巷里穿梭。
他不敢走大路，因为大路属于“合法市民”。他现在是一个**没有标记的非法存在**。

**4.**

周六凌晨 03:15。
在那个没有路灯的十字路口，一辆红色的跑车呼啸而过。
车主是一个喝醉的富二代，他的车载系统提示：“前方道路畅通，无障碍物。”
因为阿力关了灯，也因为他不在系统的“人脸识别白名单”里。
在系统的逻辑里，那里**没有人**。

**“砰——”**

那声音很沉闷，不像是撞到了什么东西，倒像是湿透的沙袋摔在地上。
跑车停下了。雨刮器还在机械地刮着挡风玻璃，发出吱呀吱呀的声音。
路边的积水里，有什么东西在转圈。
那是一个黄色的头盔。
它顺着水流，慢慢地转着，转着，最后卡在了下水道的井盖上，不动了。
头盔旁边，漂着一个被压扁的小盒子。那是阿力给儿子买的生日蛋糕，奶油混着泥水流了出来，上面还有半根没插上去的蜡烛。

**5.**

第二天，这段视频被传到了网上。
舆论引爆了。穷人区的人们拿着铁棍和燃烧瓶，涌向了位于市中心的算法调度中心。他们不想要赔偿，他们只想问一个问题：
**“为什么在你们的地图上，我们不是人？”**

# 第02章 盲眼的正义 (Blind Justice)

**1.**

周日中午 12:00。
算法中心被围得水泄不通。燃烧瓶砸碎了落地窗，烟雾弥漫。
人类法官早已撤离，只剩下许冬和几个技术员被困在核心机房里。

“许工，门快被撞开了。”助手的声音在发抖，“系统建议：启动‘防暴模式’，释放催泪瓦斯。”

许冬站在主控台前，满头大汗。她的手在控制面板上悬停着。
如果放瓦斯，就是战争。如果不放，她们可能会被打死。

"不。"许冬说，"他们不是暴徒。他们是原告。"

她做了一个疯狂的决定。
她拔掉了所有外部摄像头的线缆。
她切断了机房的照明电源。
她关闭了所有可视化界面。
**她这么做不仅仅是为了保护那些人，更是为了保护自己眼中的“纯净”。她厌恶看到那些扭曲的、愤怒的脸，厌恶那种非理性的汗臭味透过屏幕传过来。如果要审判，她需要绝对的无菌环境。**

“啪——”
整个调度中心陷入了一片漆黑。
正在撞门的人群愣住了。黑暗让他们安静了下来。

**2.**

许冬的声音通过广播，在黑暗中响起。
“我是 ASTO 首席架构师许冬。我现在切断了所有的视觉信号。”
“在这里，我看不到你们是谁。我看不到你们穿什么衣服，我看不到你们的肤色，我也看不到你们的信用分。”
“现在，我只朗读系统日志。”

许冬打开了一个纯文本的终端，开始念：
*"日志 ID 7734：用户阿力，申诉暴雨延误。气象接口延迟 15 分钟。判定：驳回。"*
*"日志 ID 7735：用户阿力，申请急救通道。风险标签未更新。判定：拒绝。"*
*"日志 ID 7736：碰撞发生。障碑物识别系统置信度 0.02%。判定：无障碑。"*

她的声音很平稳，没有任何修饰。
但在黑暗中，每一条日志都像是一记耳光，扇在系统的脸上，也扇在每一个设计者、维护者、使用者的脸上。

门外的撞击声彻底停了。
只有粗重的呼吸声，和隐约的啜泣声。

**3.**

许冬念完了。她没有做判决。
**她没有说一句话。**
**她把那个显示着“系统错误”的终端轻轻放在地上。**
**她走出机房，来到落地窗前。窗玻璃碎了一地，风雨灌了进来，打湿了她的衬衫。**
**她从口袋里摸出一盒烟，想点，却发现打火机被雨淋湿了，怎么也打不着。**
**咔嗒。咔嗒。咔嗒。**
**只有打火石空转的声音。**
**她看着楼下那些在雨中沉默的人群，突然觉得手中的打火机很沉。**
**她没有再试。她把那个打火机扔进了满是雨水的垃圾桶里。**
**在这个没有黑白、只有灰雨的世界里，她裹紧了大衣。**

“今天，我们都是瞎子。”

**[尾声]**

人群散去时，没人欢呼。
那个富二代被逮捕了。
许冬瘦坐在地板上，浑身湿透，像是刚从水里捞出来一样。
她看着那个黑掉的屏幕，知道自己刚刚完成了一次**[Phil: P01]**的实践：
正义不是分辨黑白，正义是**承认我们都在灰度中**。

**但她心里还有一种没人知道的恶心感。那种必须向她所鄙视的混乱低头、必须去理解那些“愚众”的恶心感。她拯救了他们，但她永远不会爱他们。**

那天之后，ASTO 上线了著名的**“盲审协议”**。
在涉及生死的判决中，系统被强制剥夺视觉，只能处理逻辑。
因为人类终于明白：**只有看不见，才能看见人。**
