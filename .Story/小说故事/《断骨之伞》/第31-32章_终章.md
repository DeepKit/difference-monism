# 第31章 新生 (The New Born)

1.

产房外。
沙尘暴（Unit 2 的余波）还在刮。
窗户被沙砾打得噼啪作响。
产妇小林（X）躺在待产床上。她是高龄产妇，有心脏病史。
系统评估：`风险等级：高。建议剖腹产。`
但医院断电了（Unit 3 的余波，虽然恢复了，但不稳）。
备用发电机正在给新生儿保温箱供电，手术室的灯忽明忽暗。

2.

小林的丈夫在走廊里求医生：“能不能转院？”
医生摇头：“外面风沙太大，直升机飞不了。”
“那怎么办？”
“只能顺产。”
丈夫看着医生，觉得他很冷漠。其实医生也很绝望。他刚做完三台手术，手都在抖。
**在极端环境下，医疗不是奇迹，是赌博。**

3.

小林开始大出血。
她感到生命在流逝。
她看着天花板上的灯。灯闪了一下，灭了。
黑暗中，她听到了隔壁床传来的婴儿啼哭声（主角的孩子）。
她想：“那是我的孩子吗？”
护士告诉她：“不是。您的孩子……还在肚子里。”
小林闭上了眼睛。
她觉得自己的身体变成了一个空壳，一个为了孕育新生命而被榨干的容器。
在系统的死亡记录里，她只是一个数据：`产妇死亡。死因：循环系统衰竭。`

4.

灯亮了。
小林死了。
她的孩子也没保住。
生命永远停在了那一刻。
丈夫跪在地上，发出了野兽般的嚎叫。
他冲进产房，想看最后一眼妻子，但被保安拦住了。
“别进去！”保安喊道，“正在消毒！”
丈夫疯了。他开始砸门，砸墙，砸一切能砸的东西。
“那是我的妻子！不是污染物！”

隔壁床。
林医生（水）抱着主角（沈知行的孙子/或许是许冬的孩子，留白）刚出生的孩子。
孩子还在哭。
那哭声和隔壁的砸门声混在一起。
新生与死亡，仅一墙之隔。

# 第32章 第一次呼吸后的窒息 (Suffocation after the First Breath)

1.

沈知行（金）站在观察室的玻璃窗前。
他看着那个刚出生的孩子。
孩子浑身通红，皱巴巴的。
护士正准备给孩子吸氧。
突然，监护仪发出了尖锐的警报。
`血氧下降。心率骤停。`

2.

孩子吸入了第一口空气。
但这空气里，或许有一粒来自Unit 15的霉菌，或许有一粒来自Unit 2的沙尘。
孩子的肺部发生了剧烈的痉挛。
生理性窒息。
他在抽搐。
他的脸变成了紫色。

系统的时间显示：`2056-05-20 03:33:33`。
秒针突然停了。
不是没电，不是故障。
就是停了。
停在了 `33` 秒。
仿佛时间本身也在这一刻窒息了。

沈知行看着那停跳的秒针。
他没有喊医生。
他只是死死盯着那个孩子。
他在心里默念：“呼吸。呼吸。呼吸。”

3 秒后。
秒针跳到了 `36`。
孩子“哇”的一声，哭了出来。
那哭声很响亮，穿透了玻璃，穿透了风沙，穿透了隔壁的死亡。

3.

许冬走了过来。
他看了一眼隔壁（小林的尸体已经被运走了）。
“X 去哪了？”他问。
沈知行没有回答。
他指着那个正在大哭的孩子。
“他就是 X 的继承人。”

“什么意思？”
“每一代文明，都是踩着上一代的尸体出生的。”沈知行说，“这个孩子吸入的第一口空气里，有 X 的呼出的废气。”
“我们无法切除痛苦，无法切除死亡。”
“我们只能带着它们，继续呼吸。”

窗外，雨又开始下了。
这次不是酸雨，是普通的春雨。
雨水冲刷着窗户上的沙尘。
树（Unit 2）在雨中生长。
灯（Unit 14）在雨中亮着。
电网（Unit 3）在雨中生锈。

沈知行深吸了一口气。
空气里有消毒水的味道，有血腥味，也有泥土的味道。
这不好闻。
但这很真实。

“走吧。”沈知行说，“路还长。”

他撑起那把断了骨的伞，走进了雨里。
许冬跟在后面。
他们没有回头。
因为身后的产房里，那个孩子正在用尽全力，发出在这个不完美世界上的，第一声呐喊。

**（全书完）**
