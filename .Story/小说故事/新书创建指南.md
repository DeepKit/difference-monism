# 新书创建指南：《点赞刑法》与爆款逻辑解析

> **核心目标**：借鉴市场爆款逻辑，打造一部“易发表、易创收”的短篇科幻/反乌托邦小说。
> **对标作品**：宁航一《十四分之一》（又名《必须犯规的游戏》）
> **项目代号**：《点赞刑法》

---

## 第一部分：爆款逻辑拆解——以《十四分之一》为例

### 1. 作品画像
*   **书名**：《十四分之一》（必须犯规的游戏）
*   **作者**：宁航一
*   **结构**：暴风雪山庄模式（主线） + 14个独立悬疑短篇（支线）。
*   **核心钩子**：14个悬疑作家被关在一起，每人讲一个故事，讲得不好的会死。谁是隐藏的主办者？
*   **市场反响**：畅销书，有声书播放量超1800万，影视/剧本杀改编热门。

### 2. 成功要素（优点）
*   **极致的“模块化”阅读体验**：
    *   **碎片化友好**：虽然是长篇，但核心内容是14个独立短篇。读者可以随时拿起放下，像看连续剧一样。这非常符合现代读者的阅读习惯（知乎盐选、短视频解说）。
    *   **类型杂糅**：14个故事涵盖了恐怖、推理、奇幻、怪谈等多种类型，总有一款适合你。
*   **强悬念钩子**：
    *   “谁是主办者？”这个终极悬念贯穿始终，迫使读者为了知道结局而忍受中间的平庸部分。
    *   “不犯规就会死”的规则设定，制造了持续的生存压力。
*   **低门槛**：
    *   不搞复杂的诡计，主打脑洞和反转。故事通俗易懂，不挑读者。

### 3. 避坑指南（缺点）
*   **主线拉胯，人物扁平**：14个人物像工具人，缺乏深度。主线结局被普遍吐槽“烂尾”、“神棍”（催眠/精神分裂解释一切）。
*   **警示**：在《点赞刑法》中，我们要**避免“为了反转而反转”**。结局的震撼应来自于**逻辑的必然**，而非机械降神。

## 第二部分：新书《点赞刑法》策划案

### 1. 核心商业定位
*   **目标平台**：知乎盐选 / 每天读点故事 / 公众号 / 短视频脚本
*   **篇幅**：3-5万字（短篇连载，3-5章）
*   **标签**：反乌托邦、赛博朋克、高概念、爽文、人性实验
*   **一句话梗概**：在“信用分决定生存”的世界，一个被系统抹杀的“透明人”，反而获得了不受监控的绝对自由，并以此通过“点赞”和“差评”来审判特权阶级。

### 2. 核心设定：AR信用社会
*   **视觉奇观**：每个人额头上悬浮着一个发光的数字（AR隐形眼镜强制植入）。
    *   **金名（90-100分）**：特权阶级。走路自动亮红毯特效，噪音自动屏蔽，犯罪有“初犯豁免权”。
    *   **白名（60-89分）**：普通市民。小心翼翼，甚至不敢在街上大声说话，生怕被路人随手点个“踩”。
    *   **灰名（<60分）**：**透明人**。
        *   **物理屏蔽**：在别人的视野里，你是一团马赛克；你的声音会被AR降噪算法过滤成静音。
        *   **社会性死亡**：买不了东西，坐不了车，没人能看见你，没人能听见你。你活着，但你不存在。

### 3. 故事大纲（三幕式）

#### 第一幕：坠落（黄金开篇）
*   **主角**：李默，职业“刷分师”。专门帮富二代、流量明星刷好感度，精通系统的每一个G点。
*   **激励事件**：李默为了救一个被霸凌的灰名老头，得罪了金名富少。富少发动粉丝对他进行“差评轰炸”。
*   **转折**：一夜之间，李默从85分跌破60分。他眼睁睁看着自己的手在别人眼中变成马赛克，他大喊大叫却没人回头。他变成了透明人。

#### 第二幕：地下城与审判（爽点密集）
*   **发现Bug**：李默绝望中发现，虽然系统屏蔽了他，但也意味着**监控摄像头识别不了他**！他成了系统中唯一的幽灵。
*   **地下城**：他遇到了其他“透明人”——被差评的医生、被网暴的老师。他们建立了一个不需要分数的地下互助网。
*   **反击**：李默利用“透明”属性，潜入金名富少的豪宅，收集他们的罪证（偷税、吸毒、虐待）。
*   **点赞刑法**：李默并不直接曝光，而是利用刷分师的技术，黑入富少的账户，给他们疯狂**刷分**——刷到溢出，刷到系统判定数据异常，触发“反作弊机制”，直接封号。
*   **名场面**：富少在直播中，额头的金字突然变成鲜红的“ERROR”，然后瞬间被全网屏蔽，当场发疯。

#### 第三幕：崩塌（升华主题）
*   **高潮**：系统管理者发现李默这个Bug，试图招安他，许诺给他“永久金名”。
*   **抉择**：李默发现系统的底层逻辑是“制造焦虑以维持统治”。只要有分数，就永远有奴隶。
*   **结局**：李默拒绝招安，利用最高权限发动了“归零计划”——删除了所有人的分数数据。
*   **尾声**：AR眼镜失效。人们走上街头，第一次看清彼此真实的脸，而不是额头上的数字。恐慌之后，是久违的自由。

## 第三部分：前三章细纲与试读

### 第一章：把你的五星给我
*   **场景**：高端餐厅。李默正在帮一个暴发户“刷”用餐礼仪分。暴发户不小心打碎盘子，李默迅速操作，把周围顾客的“厌恶值”数据转移到自己身上。
*   **冲突**：暴发户保住了分数，李默掉了0.5分。李默收钱，冷漠离去。
*   **铺垫**：回家路上，李默看到一个灰名老头在乞讨，路人视而不见（因为被系统屏蔽了）。李默犹豫了一下，没敢施舍（和灰名互动会掉分）。
*   **结尾钩子**：李默接到一个大单，要去帮顶流明星“洗白”虐猫丑闻。

### 第二章：差评海啸
*   **场景**：明星豪宅。李默目睹明星虐待助理，忍无可忍出手制止。
*   **高潮**：明星大怒，在社交平台挂出李默的ID：“这个人偷我东西。”
*   **视觉奇观**：李默走在街上，眼前的世界突然变红。无数个“-1”像弹幕一样向他飞来。路人对他指指点点，随手点击“鄙视”。
*   **结局**：李默看着镜子里的自己，额头上的数字从65狂掉到59。瞬间，世界安静了。周围的人像NPC一样无视他穿过。

### 第三章：幽灵的特权
*   **场景**：李默饥寒交迫，试图去便利店偷面包。监控探头扫过他，红灯没亮——系统把他当成了噪点。
*   **转折**：李默意识到自己获得了“隐身挂”。他大摇大摆地走进明星的豪宅，当着明星的面（明星看不见他）吃光了桌上的大餐，还用明星的手机发了一条自首微博。
*   **爽点**：明星看着手机自己动起来，吓得尿裤子。全网沸腾，明星分数暴跌。

2.  **投稿策略**：
    *   **首选**：知乎盐选制作人投稿（需投递前三章+大纲）。
    *   **备选**：公众号“每天读点故事”。
3.  **ASTO联动**：在故事中植入彩蛋——这个信用分系统就是**Titan公司**开发的早期版本（Beta版）。李默发现的地下城入口，墙上画着一个Origin的标志。

## **1. 方向定位对比**

| 维度   | 《未尽之算》           | 《属集师/ASTO哲学悬疑》          |
| ---- | ---------------- | ----------------------- |
| 核心主题 | 规则 vs 人性、算法冷酷    | 存在变迁 vs 自由意志、属集哲学       |
| 氛围   | 冷峻职场、Office Noir | 智性悬疑 + 存在主义惊悚           |
| 哲学感  | 隐性，透过案件展示        | 显性，直接用属集/五态/七序展开        |
| 叙事结构 | 单集案件 + 尾巴 Hook   | 1→5→6→7→1 循环嵌套，108集宏大循环 |
| 主角定位 | 系统纠错者，数学极强       | 属集整理师，掌握存在异常，哲学感更强      |
| 市场潜力 | 智性职场悬疑，连载感好      | 蓝海哲学硬核爽文，结构精密，概念即剧情     |

✅ **结论**：
《未尽之算》更适合短篇/连载职场悬疑类网文，读者容易入门、快速上手；
ASTO方向则适合长期系列化、哲学+智性爽文粉丝群体，但门槛高。

## **2. 可以从《未尽之算》借鉴/升级的点**

1. **莫比斯环结构**

   * 《未尽之算》的“回到起点”概念可以和ASTO的**1→5→6→7→1循环**结合：

     * 每一季处理五态中的一个态，每个案件都是循环的一部分，既保持连载悬念，又体现哲学闭环。

2. **余数思维 / 系统漏洞**

   * 可以直接映射到ASTO的**六阶+七序**操作：

     * 六阶的“混沌→秩序→流变→脉冲→崩解→归元” → 每个案件的时间节奏
     * 七序的“感知→解析→干预→设计→物化→回溯→消解” → 主角解决异常的标准步骤
   * 保持职场悬疑的“可操作性”，又强化哲学感。

3. **冷峻职场风 + 系统UI描写**

   * ASTO小说里“属集流形可视化” + 系统界面描述 = 完美融合《未尽之算》的Office Noir感
   * 保持低饱和度、客观叙事，强化智性爽感。

4. **角色定位升级**

   * 《未尽之算》主角数学极强 → ASTO方向可升级为**属集整理师/哲学实务者**
   * 保留“纠错者”逻辑，但扩展到哲学层面的“风险层守护者”。

## **3. 升级建议：形成“哲学职场悬疑”混合风格**

### **标题建议**：

* 《未尽之算》 → 《属集之算》或《流形之算》
* 保留“算法/计算”意象，同时映射ASTO的哲学属性。

### **结构设计**

* 每季围绕一态（I-V）展开，每22集一个循环
* 每集案件既是“职场悬疑”，又映射一个哲学概念
* 案件结尾保留尾巴，暗示更大循环

### **叙事手法**

* 保留冷峻职场风格
* 系统UI+属集可视化
* 案件解决过程体现**逻辑漏洞/余数思维/哲学悖论**
* 最终呈现“莫比斯环”式闭环：主角既在修补系统，又在守护自由意志

### **角色设计**

* **女主**：历知遥 → 职场哲学家/属集师
* **男一**：付思远 → 技术理性伙伴，智性支撑
* **男二**：秦寓 → 阴影/挑战者，推动哲学悬疑冲突

---

💡 **总结**：
你完全可以把《未尽之算》的**冷峻职场悬疑结构**与ASTO哲学概念融合，形成一个**哲学职场悬疑爽文**的新类型。

* 单集悬念 → 职场案件
* 季度循环 → 五态展开
* 七序操作 → 案件解决逻辑
* 风险层守护 → 终极哲学深度

这样既保留市场易接受度，又强化独创性和哲学厚度。
