# 加班到凌晨3点，我怀疑世界是假的，室友：就算是假的，你的加班费也是真的

> **本章进度**：卷二·皮囊 (Ep.016/109)
> **核心痛点**：工作倦怠 / 解离感 / 存在主义危机
> **CS隐喻**：虚拟机 (Virtual Machine) / 沙箱逃逸 (Sandbox Escape) / 异常中断 (Interrupt)

---

## 1. 炸毛时刻 (The Hook)

周三凌晨 3:15。
苏心坐在空无一人的办公室里，只有屏幕的蓝光照着她呆滞的脸。
她已经连续加班了一周。此刻，看着屏幕上跳动的光标，她产生了一种强烈的**解离感 (Dissociation)**。

她觉得周围的一切都很假。
桌子是假的，键盘是假的，甚至正在打字的这双手也是假的。
“我在干什么？”她喃喃自语，“我为什么坐在这里？这个世界是不是一个巨大的程序？我是不是只是一个被设定好要加班的 NPC？”

她站起来，走到窗边，看着楼下偶尔驶过的车灯。那些车灯看起来就像是劣质游戏里的光效。

“如果世界是假的，那我现在这么痛苦为了什么？”苏心想笑，“我为了那几千块钱工资，在这里透支生命。如果我跳下去，会不会就醒了？”

这种**“缸中之脑”**的恐惧，让她甚至没有力气去关电脑。
如果一切都是虚拟的，那么**“努力”**就是最大的笑话。

## 2. 鬼打墙 (The Trap)

回到家，苏心像个幽灵一样飘进客厅。
林哲正在打坐，被她吓了一跳。

“林哲，我觉得世界是假的。”苏心声音飘忽，“我觉得我们都是缸中之脑。有个邪恶的科学家在刺激我们的神经，让我们以为自己在加班。”

林哲睁开眼，神色凝重。
“苏心，这就是笛卡尔的**‘恶魔假设’**，也是普特南的**‘缸中之脑’**。”

“笛卡尔怀疑一切：怎么证明此刻不是在做梦？怎么证明世界不是幻觉？他最后只找到一个立足点：**我思故我在**。因为我在怀疑，所以那个‘怀疑的自我’必定存在。”

“但这解决不了你的痛苦。”林哲叹气，“就算你确定‘你’存在，你也没法确定‘加班’是真实的。这种**怀疑论**是无底洞，一旦掉进去，你会觉得一切皆空。”

“那我该怎么办？”苏心瘫在地上，“既然是空的，我就不去了。明天我就旷工。反正都是假的。”

## 3. 降维打击 (The Turn)

**“旷工？系统会报错的。”**

叶清推门进来，手里提着刚买的豆浆油条。
她看了一眼瘫在地上的苏心，大概明白了怎么回事。

“苏心，你不是在怀疑世界，你只是**CPU过热导致的系统降频**。”

叶清把豆浆放在桌上：
“就算世界是假的，那又怎样？”

“在计算机里，我们经常在**虚拟机 (Virtual Machine)** 里跑程序。对于虚拟机里的那个程序来说，它以为自己拥有整个硬件，以为自己是真的。”
“实际上，它只是宿主姬（Host）分配的一块内存沙箱。”

叶清蹲下来，看着苏心：
“你现在就处于一个**VM**里。也许外面真的有宿主姬，也许没有。但这重要吗？”

“如果你试图**沙箱逃逸 (Sandbox Escape)** —— 也就是旷工、自杀、或者发疯，宿主姬会立刻检测到**异常中断 (Interrupt)**。”

“结果只有两个：”
1.  **进程挂起 (Suspend)**：你没钱吃饭，饿死（系统回收资源）。
2.  **强制终止 (Kill Process)**：你被社会边缘化，彻底下线。

“苏心，你的**痛觉 API** 是真实调用的。你的**饥饿感接口**是真实返回的。只要这些底层 API 还在运行，你就必须响应。”

“在虚拟机里谈论‘真实’是没有意义的。**对进程来说，不报错就是最大的真实。**”

## 4. 物理行动 (The Action)

苏心摸了摸肚子。
真的很饿。那种胃壁摩擦的痛感，真实得让她想哭。

“所以我逃不掉？”

“逃不掉。”叶清把油条塞进她嘴里，“除非你能黑进宿主姬的内核（成仙？）。否则，你唯一的策略就是：**在沙箱规则内，把资源利用率最大化。**”

“既然加班是假的，工资也是假的。但那个假的工资，能买到这根**真的很好吃**的油条。”

“这就够了。”

叶清站起身，打开窗帘。清晨的阳光照进来，刺得苏心眯起了眼。

“与其纠结世界真假，不如纠结一下这根油条够不够脆。”
“**High Latency (高延迟)** 的哲学思考，只会阻塞你的 **I/O (吞吐量)**。吃饱了，去睡觉。睡醒了，去上班。”

“在这个虚拟世界里，**做一个高性能的进程，比做一个崩溃的进程要爽得多。**”

苏心嚼着油条。
油条的脆响，豆浆的温热，阳光的刺眼。
这一切感官体验，构成了她此刻的**全部真实**。

如果这是假的，那这个“假”的做得也太逼真了。
既然这么逼真，那就把它当真的玩吧。

“林哲，”苏心咽下油条，“这油条哪买的？还挺好吃。”

林哲笑了：“楼下张大爷。看来张大爷这个 NPC 的建模很精细啊。”

## 5. 尾声与引战 (The Outro)

早上 8:00。
苏心洗了把脸，化了个淡妆，出门上班。
虽然还是那个令人厌倦的办公室，但她觉得没那么压抑了。

“我是个进程。”她对自己说，“我在跑我的 Loop。只要我不崩，就是胜利。”

叶清站在阳台上，看着苏心融入人流。
她曾经也试图**沙箱逃逸**。
她试图寻找那个控制一切的“程序员”。
但她发现，**没有程序员**。
或者说，每个人都是自己的程序员。

她摸了摸那个**加密文件夹**。
那个文件夹，就像是她在这个虚拟机里给自己留的一个**后门 (Backdoor)**。
也许有一天，她能通过这个后门，看到真实的太阳。

但在那之前，她得先保证系统不崩。
