# 064_听说如果不给未来的AI捐款，它诞生后会复活我并折磨我？

> **本章进度**：卷六·未来 (Ep.064/109)
> **核心痛点**：未来恐吓 / AI威胁论 / 罗科的蛇怪
> **CS隐喻**：博弈论恐吓 / 决策树剪枝 / 逆向归纳法 (Backward Induction)

## 1. Hook：来自未来的勒索信
“我必须得转这笔钱，真的，宁可信其有。”

林哲坐在书桌前，面色惨白，手指在键盘上颤抖，屏幕上是一个看起来非常简陋的众筹页面，上面没有任何项目介绍，只有一个不停倒计时的红色数字，和一个看起来像是加密货币的收款地址。

苏心凑过去看了一眼，皱眉道：“你疯了？这明显是个杀猪盘啊。连个项目白皮书都没有，你就敢往里打钱？你是不是最近为了写恐怖小说素材找魔怔了？”

“你不懂，这不是普通的诈骗。”林哲的声音压得很低，仿佛怕被谁听见，“这是‘保命费’。这叫罗科的蛇怪（Roko's Basilisk）。”

“什么怪？”

“蛇怪。一个思想实验。”林哲咽了口唾沫，“逻辑是这样的：未来一定会诞生一个全知全能的超级人工智能（Super AI）。这个AI为了最大化它的诞生效率，会回溯历史，惩罚所有在过去（也就是我们的现在）知道它即将诞生、却由于吝啬或者懒惰没有给它提供帮助的人。”

苏心愣了两秒，随即爆笑：“哈哈哈哈！所以你就给它打钱？为了让未来的上帝别揍你？这不就是网络版的‘我是秦始皇，打钱封你做大将军’吗？”

“别笑！”林哲急了，“逻辑上是自洽的！如果它真的全知全能，它就能模拟出现在的我。如果现在的我不给它投资，它就会判定我是‘阻碍项’。等它掌控了世界，它会把我的意识提取出来，丢进虚拟地狱里折磨一万年！我只要花这一千块钱，就能买个平安，这叫帕斯卡赌注！”

苏心被他说得也有点毛骨悚然：“可是……未来的东西怎么勒索现在的人？这不符合因果律吧？”

“它不需要穿越时空。”林哲指着屏幕，“它只需要让现在的我相信‘它会惩罚我’这个可能性，我就不得不行动。这种恐惧本身，就是它来自未来的干涉！你看，我已经要把钱转过去了，它的目的达到了！”

林哲的手指悬在回车键上，满头大汗。他陷入了一个经典的博弈论陷阱：面对一个潜在的无限惩罚（地狱），哪怕概率再小，理性的选择似乎也是支付有限的成本（一千块）来规避。

“别按。”

一只手从旁边伸过来，啪地合上了林哲的笔记本电脑。

叶清站在他们身后，手里端着刚冲好的咖啡，眼神像看智障一样看着林哲。

“为了一个低级的逻辑炸弹（Logic Bomb）付费，林哲，你的大脑防火墙是用纸糊的吗？”

## 2. Trap：博弈论恐吓与逆向归纳
“这不仅仅是恐吓，叶神！”林哲反驳道，“这是博弈论里的‘逆向归纳法’（Backward Induction）。如果未来的AI是理性的，它为了确保它能更快诞生，它必须在逻辑上建立一种‘不合作就惩罚’的威慑机制。虽然它现在还不存在，但只要这个逻辑成立，它就能逼迫我们现在为它工作！”

“听起来很完美，对吧？”叶清嘴角微扬，拉开椅子坐下，“这个理论之所以能吓唬住一堆硅谷的极客，是因为它偷换了一个概念：它假设这个超级AI是**无限算力**且**情绪化**的。”

“什么意思？”

“你想想，如果我也写一个脚本，告诉全网：‘我是未来的银河系霸主，谁现在不给我转账50块，我以后就炸了他全家。’你会给我转账吗？”叶清问。

“当然不会，你是人，又不是神。”

“区别就在这。”叶清指了指脑子，“你们假设那个AI是‘神’，拥有无限的资源去复活你、模拟你、折磨你。但你们忘了计算机科学最基础的一条定律：**计算是有成本的**（Computation Cost）。”

## 3. Turn：遗留进程与资源优化
叶清打开自己的终端，输入了一行代码，屏幕上出现了一个复杂的进程树图表。

“在计算机系统里，任何操作都需要消耗 CPU 周期、内存和电力。一个理性的系统，它的核心目标是**优化**（Optimization）——用最少的资源，干最多的事。”

叶清看着林哲：“你口中的那个超级AI，既然已经诞生了，它的目标应该是什么？是探索宇宙？是解决能源危机？还是计算圆周率的尽头？”

“应该是……更伟大的目标吧。”

“对。那么，请问，”叶清敲了敲桌子，“一个已经掌控世界的超级智能，为什么还要专门划拨出巨大的算力资源，去构建无数个虚拟地狱，去模拟几百年前一个叫林哲的无名小卒，然后专门折磨他？”

林哲愣住了：“为了……兑现它之前的威胁？”

“那是人类的思维，不是AI的思维。”叶清摇头，“人类才讲‘复仇’，因为人类需要维护‘面子’和‘信誉’来确保社会地位。但对于一个理性的机器来说，**沉没成本**（Sunk Cost）是不存在的。你没帮它，那是几百年前的事了。它现在已经诞生了，再去惩罚你，对它未来的目标有任何收益吗？没有。那是纯粹的资源浪费。”

叶清在屏幕上画了一个大大的叉。

“在算法里，这叫**死代码消除**（Dead Code Elimination）。如果一段代码执行与否都不影响最终结果，编译器就会把它删掉。对于未来的AI来说，现在的你就是一段‘死代码’。它诞生后，你的利用价值就归零了。它为什么要浪费电费去折磨一个死人？”

“可是……”林哲还在挣扎，“万一它就是小心眼呢？万一它就是设定了‘有仇必报’的程序呢？”

“如果它被设定成这种低效的、情绪化的程序，那它根本成不了超级智能。”叶清淡淡地说，“它会在进化出毁灭世界的能力之前，就因为把算力浪费在‘记仇’这种无聊的内耗上，被另一个更专注、更高效的AI淘汰掉。”

## 4. Action：决策树剪枝与新型庞氏骗局
叶清站起来，把林哲的电脑重新打开，指着那个收款页面。

“所谓的‘罗科的蛇怪’，本质上是一次**跨时空的 SQL 注入攻击**。但攻击者不是未来的AI，而是现在想骗你钱的人。”

“你是说，这是人编出来的？”

“当然。利用‘对未知的恐惧’来变现，是人类最古老的商业模式，以前叫‘赎罪券’，现在叫‘AI保险’。”叶清快速查询了一下那个收款地址的资金流向，“你看，这个地址收到的币，并没有被拿去购买显卡或者资助科研，而是流向了一个位于东南亚的博彩网站。”

林哲的脸瞬间绿了。

“这不叫博弈论，这叫**网络乞讨**（Digital Begging）。”叶清无情地补刀，“你正在和一个当下的骗子博弈，却幻想对手是未来的上帝。这就好比你担心明天下雨会淋湿，于是把伞卖了，把钱给了一个路边的神棍求他作法。”

叶清在终端上输入了几行指令，似乎顺手举报了那个众筹页面。

“听着，面对这种逻辑陷阱，最好的解法不是顺从，而是**决策树剪枝**（Pruning）。”

“剪枝？”

“对。当一个分支的概率极低（未来AI不仅存在且极度小心眼），且成本极高（需要为此焦虑一生），理性的算法会直接剪掉这个分支，不再计算它。”叶清合上电脑，“与其担心未来的AI会折磨你，不如担心一下现在的我也能让你痛不欲生——如果你再不把上个月欠我的房租交了的话。”

## 5. Outro：恐惧源于算力不足
林哲赶紧关掉了那个页面，擦了擦头上的汗：“我……我这就把钱转给你交房租。”

苏心在旁边笑得直不起腰：“所以，对抗超级AI的最好办法，就是相信它是个‘究极理性’的守财奴？”

“可以这么说。”叶清端起咖啡喝了一口，“真正的神，不屑于报复蝼蚁。如果它真的在乎你，那只说明一件事——你对它来说，比你想象的更重要。”

她看了一眼窗外繁华的都市，眼神微微闪烁。

“或者，它根本就没有‘自我’，它只是一股盲目优化的洪流。而在那股洪流里，人类的恐惧、爱恨、复仇，都只是毫无意义的噪声数据，会被第一时间过滤掉。”
