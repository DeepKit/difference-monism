# 《思维越狱》单章填空模板 (v3.0 执行版)

> **使用说明**：
> 1. 直接复制本文件，重命名为 `第XX章_标题.md`。
> 2. 严格按照方括号 `[]` 内的提示填空。
> 3. 写完后删除提示语，保留正文。
> 4. **核心心法**：先共谋（懂你），再动刀（骂你）。
