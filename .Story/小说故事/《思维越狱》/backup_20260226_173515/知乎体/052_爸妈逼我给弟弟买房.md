# 爸妈逼我给弟弟买房，“只有你能救他”，室友：拔管！你的器官不属于公共资源库

> **本章进度**：卷五·抉择 (Ep.052/109)
> **核心痛点**：扶弟魔 / 原生家庭吸血 / 道德绑架
> **CS隐喻**：私有变量 (Private Variable) / 封装 (Encapsulation) / 访问控制 (Access Control)

---

## 1. 炸毛时刻 (The Hook)

周六晚上 10:00。
苏心挂断了家里的电话，气得把手机砸在沙发上。
屏幕裂了一条缝。

“他们疯了！”苏心尖叫，“我弟弟结婚要买房，首付差 20 万。我妈让我把我的存款全拿出来！还说如果不拿，弟弟就要分手，这个家就散了！”

“我说那是我辛辛苦苦攒的嫁妆钱。我妈竟然说：‘你弟弟是你亲弟弟！他的事就是你的事！你怎么这么自私？只有你能救他！’”

苏心抱着膝盖大哭：“为什么？从小到大，好东西都给弟弟，脏活累活都归我。现在长大了，还要吸我的血！我是不是欠他们的？我是不是生下来就是为了给弟弟当燃料的？”

她陷入了**“血缘诅咒”**：
因为有血缘关系，所以你的资源（钱、器官、未来）默认是家族的**公共财产**。如果你想私有化，你就是背叛者。

## 2. 鬼打墙 (The Trap)

林哲叹了口气，给苏心倒了杯水。

“苏心，这在伦理学上，叫——**小提琴家思想实验 (The Violinist)**。”

“朱迪斯·贾维斯·汤姆逊提出：假设你一觉醒来，发现自己被绑架了，背上插着管子，连着一位著名的小提琴家。小提琴家得了绝症，只有你的肾脏能过滤他的毒素。如果拔管，他就会死。医生说：‘只需要连 9 个月，他就能痊愈。如果你现在拔管，就是杀了他。’”

林哲指着苏心：“你弟弟就是那个小提琴家。你妈就是那个医生。他们用‘他会死（分手/家散）’来绑架你。虽然他需要你的帮助，但问题是：**你有义务用你的身体/金钱去维持他的生命吗？**”

“在传统伦理中，血缘被赋予了神圣性。拒绝帮助亲人，会被视为‘不义’。所以你会内疚，你会觉得真的是自己‘杀’了弟弟的幸福。”

苏心哭着说：“那我能拔管吗？拔了管，我会不会被唾沫星子淹死？”

## 3. 降维打击 (The Turn)

**“拔。因为你的器官是 Private 的。”**

叶清的声音从书房传来。她正在写一个 Java 类。

“什么 Private？”苏心问。

叶清走出来，看了一眼苏心那个裂了屏的手机。

“在面向对象编程 (OOP) 里，最重要的概念之一就是 **Encapsulation (封装)**。”

“一个类（Class）的成员变量，默认应该是 **Private (私有)** 的。这意味着：只有这个类自己的方法（Method）可以访问和修改它。外部类（哪怕是子类、父类）都无权直接读取。”

叶清指着苏心：
“你是 `Class SuXin`。你的存款、你的肾脏、你的时间，都是你的 `private` 变量。”
“你妈、你弟，是外部类 `Class Family`。”

“他们试图直接访问你的 `wallet` 变量：`SuXin.wallet.withdraw(200000)`。”
“在代码里，这行指令会直接报错：**Access Denied (访问被拒绝)**。”

“为什么？”
叶清轻声道：“因为**封装性**是系统安全的基石。如果所有变量都是 **Public (公有)** 的，任何外部对象都能随意修改你的状态，那你的系统早就乱套了。你的钱会被取光，你的时间会被占满，你会变成一个**没有任何边界的僵尸对象**。”

“你妈觉得你的钱是‘公共资源库’，是因为她还在运行 **过程式编程 (Procedural Programming)** 的旧代码（没有封装概念，全局变量满天飞）。”

“但你是一个现代化的 **对象 (Object)**。你有权定义自己的访问控制列表 (ACL)。”

## 4. 物理行动 (The Action)

苏心听懂了。
原来拒绝不是自私，是**系统安全设置**。

“那我该怎么做？直接拒绝？”

“对。”叶清点头，“不仅要拒绝，还要**封闭接口**。”

“如果外部对象想要访问你的私有变量，正规途径是调用你的 **Public Getter/Setter** 方法。”
“比如：`request_help()`。”
“而在你的 `request_help()` 方法里，你可以写上逻辑判断：”

```java
public void request_help(int amount) {
    if (amount > 5000) {
        throw new Exception("Go away."); // 拒绝
    } else {
        // 借点小钱可以，因为不影响系统稳定性
        wallet.withdraw(amount);
    }
}
```

叶清拿过苏心的手机：
“现在，给你妈发一条消息（调用接口返回值）：”
“**‘妈，我有自己的生活规划。这 20 万我有用，不能给弟弟。我可以借给他 5000 块救急，这是我的底线。’**”

“如果她骂你自私？”
“那是她的系统在报错。**Error Log (错误日志)** 是打印在她的控制台上的，不是打印在你的控制台上的。你只需要 **Ignore (忽略)**。”

苏心深吸一口气。
她看着手机屏幕。
那种“欠他们”的感觉消失了。取而代之的是一种**界限感**。
我是我。你们是你们。
我们有血缘接口，但没有**内存共享**。

她发了消息。
然后，把手机设成了静音。

## 5. 尾声与引战 (The Outro)

周日。
家里果然炸锅了。电话轰炸，亲戚轮番上阵。
但苏心就像一个配置了高级防火墙的服务器，默默地把所有恶意流量都挡在了外面。

“林哲，”苏心说，“我第一次觉得，被人骂‘白眼狼’也挺爽的。至少我的狼窝（钱）还在。”

叶清正在写代码。
“**Good fences make good neighbors.**” 她淡淡地说，“好的封装，才能造就好的系统。”

她看了一眼那个**缺角的算盘**。
算盘的每一档都是独立的。
如果珠子能随便乱窜，那算出来的数，就谁也不敢信了。
