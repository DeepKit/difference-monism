# 094_角色如何吃人

> **本章进度**：卷八·人群 (Ep.094/109)
> **核心痛点**：权力腐蚀 / 角色异化
> **CS隐喻**：权限滥用 (Privilege Escalation) / Sudo / 审计日志 (Audit Log)

## 1. 制服
“我不认识他了，真的。”

苏心坐在咖啡馆里，神情恍惚。

“老张以前多好一个人啊。我和他同批进公司，他还会帮我带早餐，说话轻声细语的。上周他升了小组长，我本来还挺为他高兴的。”

“结果才过了一个星期，他就像换了个人。昨天开会，他居然指着我的鼻子骂：‘苏心，你这种低级错误也好意思交上来？不想干就滚蛋！’那种眼神，充满了傲慢和鄙夷，就像在看一只蚂蚁。”

苏心眼圈红了：“难道权力真的有毒吗？还是他以前的善良都是装的？”

林哲叹了口气：“可能是‘新官上任三把火’吧，想立威。”

“不，不是立威。”叶清冷冷地说，“是被**吞噬**了。”

## 2. 笼子
“听说过**斯坦福监狱实验**吗？”

叶清打开平板，展示了一组黑白照片。

“1971年，津巴多教授找了24个心理健康的大学生，随机分成两组：一组扮演狱警，一组扮演囚犯。实验在斯坦福大学的地下室进行。”

“结果不到两天，‘狱警’们就开始虐待‘囚犯’。他们剥夺囚犯的睡眠，强迫他们做羞辱性的动作，甚至有人以此为乐。而‘囚犯’们则变得唯唯诺诺，甚至出现了精神崩溃。”

“最可怕的是，这些‘狱警’在实验前都是普通的好学生。但只要穿上制服，拿起警棍，他们就迅速变成了暴君。”

叶清看着苏心：“这说明：**性格并不决定行为，环境和角色才决定行为**。当系统赋予一个人绝对的权力且不受监管时，人性中的恶会被瞬间激活。”

## 3. Root 权限
“在系统管理里，这叫**权限滥用**（Privilege Abuse）。”

叶清解释道：“老张以前是普通用户（User），他的权限受限，只能读写自己的文件。现在他被提升为管理员（Admin），拥有了 `sudo` 权限，可以随意修改、删除你们的文件，甚至可以终止你们的进程（开除）。”

“对于一个没有受过专业训练的管理员来说，这种‘生杀予夺’的快感是极具成瘾性的。他会忍不住去执行那些高危指令，仅仅是为了验证自己的权力。”

“更关键的是，如果系统没有**审计日志**（Audit Log）——也就是没人监管他的行为，没人能撤销他的操作——那么这种滥用几乎是必然的。”

“不是老张变坏了，是那个‘小组长’的角色（Role）把他吃掉了。现在的他，只是权力的一个**载体**。”

## 4. 一顿火锅
“那该怎么办？辞职？”苏心问。

“辞职是最后的手段。在此之前，你可以尝试唤醒他的**自我**。”

叶清建议道：“现在的他已经完全沉浸在 `Admin` 角色里了。你要做的，是强制他切换回 `User` 视角。”

“下次他再骂你，你可以冷静地看着他说：‘老张，抛开工作不谈，我觉得你最近状态有点紧绷。咱们周末去吃个火锅放松一下？’”

“这句话的作用是：提醒他，除了‘组长’，他还是‘老张’，还是你的‘朋友’。这叫**上下文切换**（Context Switch）。”

“如果这招没用，”叶清耸耸肩，“那就说明他的自我已经被彻底覆盖了。这时候，你也别把他当朋友了，把他当成一个**出故障的系统组件**。保护好自己，收集证据（Log），然后向上级举报（Report Bug）。”

## 5. 脱下制服
几天后，苏心试着用了叶清的方法。老张愣了一下，眼里的戾气消散了一瞬，露出了一丝疲惫。

“唉，最近压力确实大……”

虽然老张没有完全变回以前那个老好人，但他对苏心的态度确实缓和了不少。

“看来，制服虽然能吃人，但只要时不时脱下来透透气，还是能抢救一下的。”苏心感叹。

“是的。”叶清点头，“所以，永远不要迷信权威，也不要迷信好人。要迷信**制度**。”

“只有把权力关进笼子（沙箱 Sandbox）里，人才不会变成野兽。”
