# 只有一张ICU床位，给年轻的流氓还是年老的教授？我手在抖，室友：别抖，交给权重算法

> **本章进度**：卷五·抉择 (Ep.049/109)
> **核心痛点**：道德困境 / 资源分配 / 职业伦理
> **CS隐喻**：加权轮询 (Weighted Round Robin) / 优先级队列 (Priority Queue) / QoS (服务质量)

---

## 1. 炸毛时刻 (The Hook)

周五凌晨 2:00。医院值班室。
苏心握着电话，手心全是汗。
“苏医生，ICU 刚空出一张床位。急诊送来两个病人，都如果不进 ICU 可能会死。您决定给谁？”

**病人 A**：75 岁，退休教授，脑出血。为人谦和，桃李满天下，但有基础病，预后不佳。
**病人 B**：22 岁，街头混混，斗殴被捅伤。有案底，但身体底子好，救活了能活很久。

苏心看着两份病历，脑子里像是有两个小人在打架。
“救教授吧！他对社会贡献大！”
“救年轻人吧！生命是平等的，他还那么年轻！”

护士在电话那头催：“苏医生？快点！两个人都等不起！”

苏心崩溃了：“我怎么能决定谁该死谁该活？我是医生，不是上帝！为什么要把这种选择题抛给我？”

她陷入了**“上帝视角的瘫痪”**：
作为一个凡人，被迫行使“神”的裁决权。这种道德重负让她根本无法做出决策，只想逃跑。

## 2. 鬼打墙 (The Trap)

回到家（第二天补休），苏心还在发抖。
“林哲，我最后选了教授。但我一直在想，那个年轻人是不是因为我而死了？我是不是杀了人？”

林哲叹了口气，给苏心倒了杯热牛奶。

“苏心，这在伦理学上，叫——**电车难题 (Trolley Problem)**。”

“一辆失控的电车冲过来，轨道上有 5 个人。你可以拉动拉杆，让电车变道，撞死另 1 个人。你拉不拉？”

林哲分析道：“这里有两种道德直觉的冲突。”
“**功利主义 (Utilitarianism)**：追求最大幸福，杀 1 救 5 是对的。那你应该救年轻人（因为他余生更长）。”
“**义务论 (Deontology)**：生命是无价的，不能被计算。那你没法选。”

“无论你选谁，你都会感到内疚。因为你在这个选择中，扮演了**主宰者**的角色。这种角色本身就是一种诅咒。”

苏心哭着说：“那我以后还要面对无数次这样的选择吗？我会疯的！”

## 3. 降维打击 (The Turn)

**“别疯。你不需要当上帝。你只需要当个调度器。”**

叶清的声音从书房传来。她正在给服务器配置负载均衡策略。

“什么调度器？”苏心问。

叶清走出来，看了一眼那两份病历（复印件）。

“苏心，在计算机系统里，资源（CPU、内存、带宽）永远是稀缺的。进程（任务）永远是过载的。”
“如果操作系统每次都要纠结‘这个进程该不该杀’，那系统早就崩了。”

叶清打开电脑，展示了一段代码：
```python
def schedule(tasks):
    # 加权轮询算法
    scores = []
    for t in tasks:
        score = (t.survival_rate * 0.6) + (t.social_value * 0.4)
        scores.append(score)
    return max(scores)
```

“我们不谈道德。我们只谈**算法 (Algorithm)**。”

叶清指着屏幕：
“你现在的痛苦，是因为你试图用**感性**去处理一个**资源分配问题**。”
“你需要引入**优先级队列 (Priority Queue)**。”

“设定好权重参数 `Weights`：”
*   `w1` = 存活率 (Survival Rate)
*   `w2` = 剩余寿命 (Life Expectancy)
*   `w3` = 社会贡献 (Social Contribution)

“然后，把病人扔进算法里跑一遍。谁的分数高，资源就给谁。”

“**把道德压力卸载给算法。**”叶清冷冷地说，“你只是一个**执行者 (Executor)**，不是**决策者 (Decider)**。决策的是那套经过伦理委员会审核的**规则**。”

“如果规则说‘年轻人优先’，你就救混混。如果规则说‘存活率优先’，你就救教授。”

“**不要让自己成为那个拉拉杆的人。让规则去拉。**”

## 4. 物理行动 (The Action)

苏心愣住了。
“可是……规则是冷的。人是热的啊。”

“正因为人是热的，所以人会犯错，会偏心，会崩溃。”叶清说，“只有冷的规则，才能保证**最大的公平**。”

“在网络工程里，这叫 **QoS (Quality of Service)**。”
“当带宽不够时，路由器会根据 QoS 策略，自动丢弃低优先级的包（比如 P2P 下载），优先保证高优先级的包（比如 VoIP 语音）。”

“路由器会为此感到内疚吗？不会。因为它只是在执行 `Policy`。”

叶清拿过苏心的笔记本，帮她写了一个简单的打分表：
1.  **预后评分**（0-10）
2.  **年龄评分**（0-10）
3.  **紧急程度**（0-10）

“下次遇到这种情况，别用脑子想。直接填表，算分。”
“**分数高的进 ICU。分数低的转普通病房。**”

“如果家属闹事，你就把这个表拍在桌子上：‘这是医院的 **SLA (服务等级协议)**。我也没办法。’”

苏心看着那个表格。
虽然看起来很冷血，但她突然觉得……那种要把她压垮的“上帝感”消失了。
她变回了一个医生。一个专业的、执行医疗规范的医生。

“好。”苏心深吸一口气，“下次我就填表。”

## 5. 尾声与引战 (The Outro)

周六。
苏心回医院上班。又遇到了抢救室爆满的情况。
这次她没有哭，也没有抖。她冷静地拿出评分表，快速勾选，然后指挥护士：“3 床进 ICU，5 床联系转院。”

同事看着她，有点惊讶：“苏医生，你今天好果断。”

苏心笑了笑：“按规则办事，没心理负担。”

林哲听说了这事，感叹：“虽然解决了苏心的痛苦，但把生命量化成数字……总觉得有点赛博朋克的悲凉。”

叶清正在写代码。
“**量化是公平的前提。**”她淡淡地说，“不量化，就会变成‘谁有钱谁进’或者‘谁关系硬谁进’。那才是真正的悲剧。”

她看了一眼那个**缺角的算盘**。
算盘也是量化工具。
在资源匮乏的世界里，只有**算计**，才能让更多人活下来。
