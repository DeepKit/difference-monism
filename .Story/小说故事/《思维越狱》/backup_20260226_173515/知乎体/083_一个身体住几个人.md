# 083_一个身体住几个人

> **本章进度**：卷七·自我 (Ep.083/109)
> **核心痛点**：逃避责任 / 人格切换幻想
> **CS隐喻**：User Profile 切换 / 账号绑定 / 权限继承

## 1. Hook：替身打工人
“我决定了，我要开发我的第二人格。”

苏心站在镜子前，一脸严肃地对自己说道。

林哲正躺在沙发上看电影《致命ID》，被她这突如其中二的发言整不会了：“你又要搞什么幺蛾子？”

“你看电影里那些多重人格患者（DID），”苏心指着屏幕，“通常都有一个软弱的主人格，和一个超级能干、冷酷无情、甚至会格斗的副人格。当主人格受欺负时，副人格就会‘唰’地一下出来接管身体，大杀四方。”

苏心眼里闪着星星：“我也想要一个这样的副人格！我给他起名叫‘苏·钮轱辘·强’。以后每天早上闹钟一响，我就切换成‘苏强’去上班，去跟老板吵架，去写那些该死的PPT。等下班打卡的一瞬间，再切回‘苏心’，负责吃火锅、刷剧、花钱。”

“这样我就只负责享受人生，痛苦都让‘苏强’去扛。多完美的计划！”

林哲翻了个白眼：“你当这是玩网游呢？还带双开练级的？”

## 2. Trap：多重人格与账号切换
“理论上，这在操作系统里是可行的。”

叶清的声音传来。她坐在电脑前，正在通过命令行切换用户。

`su - root`

“在 Linux 系统里，一个物理机器可以同时存在多个**用户配置**（User Profile）。`User_A` 和 `User_B` 可以拥有完全不同的桌面背景、文件权限、甚至操作习惯。”

“多重人格障碍（DID）在某种程度上，就是大脑为了应对极端创伤，强制创建了一个具有高权限的 `Root` 账户，或者一个具有隔离环境的 `Guest` 账户，用来处理那些主人格无法处理的‘非法操作’（比如被虐待）。”

苏心兴奋了：“看！叶神都支持我！快告诉我怎么创建新账号？我要求不高，智商 180，精通编程和金融就行。”

## 3. Turn：数据库是共享的
“你想得美。”叶清冷冷地泼了盆冷水。

“首先，操作系统虽然支持多用户，但**底层数据库**（Database）和**硬件资源**（Hardware）是共享的。”

叶清指了指苏心的身体。

“就算你切成了‘苏强’去上班，他熬夜写的 PPT 消耗的是**你**的脑细胞；他跟老板吵架飙升的皮质醇（压力激素）伤害的是**你**的血管；他坐一天办公室导致的腰椎间盘突出，最后疼的也是**你**。”

“你以为你切号了就能逃避痛苦？不，**账单是寄到物理地址的**。无论哪个账号消费，最后扣的都是这具肉体的血条。”

## 4. Action：没有外挂，只有练级
“其次，”叶清看着苏心，“你以为切个号就能变强？”

“在系统里，新建一个用户，默认是**空配置**（Empty Profile）。你现在的脑子里，并没有预装‘智商 180’和‘精通金融’的技能包。”

“如果你现在强行分裂出一个‘苏强’，他大概率只是一个**脾气更暴躁的傻瓜**。因为你的知识库（Knowledge Base）里只有‘如何摸鱼’和‘哪家奶茶好喝’。你并没有喂给大脑足够的专业数据。”

“你想切换到 `User_Rich`（富人账号），但你的系统里根本没有这个用户。你只有 `User_Worker`（打工苏心）和 `User_Lazy`（懒惰苏心）。”

叶清关掉终端窗口。

“与其幻想开外挂切号，不如老老实实给当前账号**练级**。技能是学出来的，不是分裂出来的。你想让‘苏强’帮你写代码？那你得先自己学会写代码，然后把这个技能写入你的大脑皮层。”

“等你练满了级，你不需要切号，你本人就是钮轱辘·苏心。”

## 5. Outro：我也想有人替我
苏心泄气地瘫在沙发上：“我就知道没这么好的事。看来还是得我自己去上班。”

“不过，”苏心突然转头看叶清，“叶清，你这么厉害，是不是因为你有很多个人格？比如一个负责写代码，一个负责修电脑，一个负责怼人？”

叶清愣了一下，眼神中闪过一丝不易察觉的落寞。

“也许吧。有时候我也希望，能有一个人格替我去面对那些……无法面对的东西。”

她下意识地摸了摸自己的太阳穴，那里曾经回荡着那个声音。

“但遗憾的是，在这个残酷的物理世界里，我们都是**单线程、单用户**的生物。除了硬抗，别无他法。”

叶清站起身：“好了，‘苏强’同学，你的 PPT 写完了吗？没写完的话，今晚熬夜的还得是‘苏心’。”

苏心惨叫一声，认命地打开了电脑。

**【下集预告】**
苏心午睡醒来，一时分不清自己是在做梦还是在现实。她突然想到了庄周梦蝶。
万一我现在就是在做梦呢？万一醒来后我是一只蝴蝶，或者是一个插着管子的电池呢？
叶清告诉她：别纠结真假。谁的分辨率高，谁就是现实。
