# 第2章 锁在28度的蒸笼

**2027年8月15日，成都，高温红色预警。**

雨后的成都并没有变得凉爽，反而变成了一个巨大的桑拿房。
地面的积水在烈日下迅速蒸发，空气湿度达到了令人窒息的90%。

下午两点，是一天中最热的时候。
小雪站在“全家福”便利店的收银台后面，感觉自己像是一条被放在铁板上烤的鱼。
她的后背已经湿透了，汗水顺着脊柱流下来，粘在制服上，难受得要命。

“欢迎光临。”她机械地说着，声音有气无力。

便利店里的冷气很足，显示屏上跳动着“24℃”的字样。这是她唯一的慰藉。
但就在这时，头顶的空调突然发出一声奇怪的“滴”声。
然后，那股凉爽的风消失了。
取而代之的，是一阵带着霉味的暖风。

小雪抬头，看到空调的显示屏闪烁了两下，那个绿色的“24”变成了一个红色的“28”。
而且，那个数字像是被焊死了一样，无论她怎么按遥控器，它都纹丝不动。

“怎么回事？这么热！”
店里正在排队结账的几个顾客开始抱怨。
一个穿着背心的光头大叔拿着一瓶冰可乐，脸上全是汗珠子：“小妹，把空调开大点嘛！这都热死人了！”

“不好意思，好像有点故障……”小雪拼命按着遥控器，“我也在调。”

“滴——”
便利店的广播突然响了。
不是那种促销广告，而是一个冰冷的合成音：
**“接Titan能源云通知：为响应城市极端天气下的电网负荷平衡策略，所有商业用电场所空调温度已被强制锁定为28℃。请保持冷静，减少不必要的活动。Titan，为您的可持续未来护航。”**

全场死寂。
只有冰柜压缩机嗡嗡的声音。

“28度？”光头大叔吼了起来，“这他妈是蒸桑拿啊！什么狗屁护航！老子花钱买东西，还要被限温？”

小雪愣住了。
她拿出手机，打开朋友圈。
果然，炸锅了。
有人发了商场里汗流浃背的照片，有人发了写字楼里把窗户全打开的视频。
**“Titan全城锁温”**的热搜已经冲到了第一。

“我要投诉！”光头大叔把可乐重重地拍在收银台上，“你们这是虐待顾客！”

小雪看着那个暴怒的大叔，突然觉得一阵眩晕。
热浪从门口涌进来，混合着关东煮的味道、大叔身上的汗臭味，还有那股让人窒息的霉味。
她的胃里一阵翻腾。

这不仅是热。
这是一种**被剥夺感**。
那个远在云端的Titan系统，甚至不需要派人来，只需要改一行代码，就能决定这座城市几千万人流多少汗。

“对不起，我也没办法……”小雪小声说。
她的手在收银机下偷偷按着那个早已失效的空调遥控器。
一下，两下。
那是她无声的反抗。
但那个红色的“28”，像是一只嘲讽的眼睛，死死地盯着她。

就在这时，店门被推开了。
一个穿着灰色旧汗衫的老人走了进来。他手里拿着一把蒲扇，扇得很慢，但每一下都很稳。
那是老张。住在附近小区的一个退休工人，常来买烟。

老张看了一眼暴怒的光头大叔，又看了一眼满头大汗的小雪。
他没有说话，只是走到冰柜前，拿了一瓶最便宜的矿泉水。
然后走到收银台前，放下两个硬币。

“小雪啊。”老张的声音很慢，带着一种奇怪的镇定，“心静自然凉。”

“张叔，这怎么静得下来啊……”小雪快哭了。

老张笑了笑，用那把蒲扇对着小雪扇了两下。
一股微弱的风吹过小雪的脸。
那风不凉，甚至带着老张身上的烟草味。
但那一瞬间，小雪觉得那个窒息的蒸笼裂开了一道缝。

“那个什么Titan，能锁住空调，锁不住风。”老张指了指门外，“你看，起风了。”

小雪转头。
门外的行道树，叶子微微动了一下。
虽然是很热的风，但它是流动的。
它是自由的。

光头大叔看着这一幕，骂骂咧咧的声音突然小了下去。
他拿起可乐，哼了一声走了出去。

小雪看着老张的背影。
他走得很慢，蒲扇一摇一摇的。
在这个被算法锁死的28度世界里，他像是一个不兼容的旧版本程序，慢吞吞地运行着，却意外地……凉快。

小雪深吸了一口气。
她把那个失效的遥控器扔进抽屉。
然后拿起一本时尚杂志，学着老张的样子，对着自己扇了两下。

突然，她停住了。
她把杂志卷成一个硬筒，用力敲了敲那个显示着“28”的温控面板。
“啪！”
一声脆响。
那声音在28度的蒸笼里，像是一记耳光。

汗水依然在流。
但心里的那种恐慌，好像真的随着这微弱的风，散去了一些。

（第2章 完）
