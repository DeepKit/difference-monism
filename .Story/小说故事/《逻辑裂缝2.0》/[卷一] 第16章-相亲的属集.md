# 第16章 相亲的属集

**时间：2024年9月21日（周六）下午**  
**地点：成都市锦江区某茶餐厅**

---

林小踏进茶餐厅时，化了一个"良家女子"妆。

不是老K要求的那种妆——那种妆容是为了让受害者产生"这个女人很性感但又不轻浮"的错觉。今天的妆容是为了让她妈妈满意：淡雅、知性、温柔,带着一点脆弱感。

**属性切换：【欺诈师】→【相亲对象】**

她母亲已经坐在靠窗的位置,对面坐着一个穿白色衬衫的男人。三十出头,戴着细框眼镜,手腕上是劳力士的蚝式恒动。

"小小,来来来,这是张工。"母亲的声音里带着讨好,"在华为做架构师。"

张工站起来,礼貌地伸出手:"你好,我是张文轩。"

林小握手。他的手心是干的,力度适中,持续时间恰好2.3秒——她下意识地计时了。这是她在老K那里学会的习惯：握手时长代表着对方的社交训练程度。

"你好。"

他们坐下。

张文轩很会聊天。他先是礼貌地问了林小的工作,当听到"在一家小公司做数据分析"时,他点点头,没有追问细节——这是高情商的表现,不让对方难堪。

然后他开始展示自己。

"我现在主要负责5G核心网的架构优化。前段时间刚完成一个项目,把时延降低了15%。"他的语气是轻描淡写的,但眼睛里有一丝期待被追问的光芒。

林小配合地"哇"了一声。

他继续:"不过这行压力大,经常加班。但我还是会坚持健身——每周至少三次。"他撩起袖子,露出紧实的小臂,"保持身体健康很重要。"

**属性展示：【高收入】【技术能力】【自律】【身材管理】**

林小在心里默默给他的"展示列表"打勾。

她太熟悉这套了。

在老K的话术培训里,有一整套"优质对象人设建立"的流程:
1. 展示经济实力(但不能太炫耀)
2. 展示能力(但要谦虚)
3. 展示自律(证明长期价值)
4. 制造稀缺感(暗示追求者多)

张文轩正在完美地执行这套流程。

"我平时也喜欢看书。"他从包里拿出一本《人类简史》,"最近在读这本,挺有意思的。"

**属性补充：【有文化】【有深度】**

林小看着那本书。书脊是崭新的,显然没怎么翻过。但封面有故意制造的折痕——这是一个细节,为了显得"这本书我真的在读"。

她曾经在老K那里学过同样的技巧:在跟高知受害者聊天时,桌上要放一本哲学书,但不能太新,要显得"我经常读"。

"你呢?平时有什么爱好?"张文轩问。

林小正要回答,他的手机震动了。

他看了一眼屏幕,皱眉:"不好意思,公司的事。"

他接起电话:"喂?嗯……这个方案我昨天不是说了吗……行吧,我晚上回去改。"

挂掉电话后,他苦笑:"没办法,技术人员就是这样,随时待命。"

**属性强化：【责任感】【被需要】**

林小突然感到一阵恶心。

不是因为他装,而是因为她太清楚他为什么要装。

这是一场交易。

他展示他的"属性集":【高收入】【技术能力】【自律】【有文化】【责任感】。期待她用她的"属性集"交换:【年轻】【温柔】【体贴】【生育能力】。

然后两个人的"属性集"会合并成一个新的"家庭对象",继续在社会的算法里运行。

没有人是真实的。

每个人都在展示一个经过优化的版本,就像老K教她的:"你要成为他们心目中完美女友的样子,而不是你自己。"

"你怎么了?"张文轩注意到她的沉默,"是不是太热了?"

他体贴地给她倒水。

**方法调用：caring.considerateAction()**

林小接过水杯,看着杯子里的水面微微晃动。

她想起《ASTO核心逻辑》里的一个概念:

```python
class Person:
    def __init__(self):
        self.attributes = {}
        self.真实自我 = None  # 永远为 None,因为不存在
    
    def switch_role(self, target_audience):
        """根据受众切换展示的属性集"""
        return self.attributes.filter(适合[target_audience])
```

她现在正在运行的是:
```python
林小.switch_role("相亲对象")
```

而张文轩正在运行:
```python
张文轩.switch_role("优质男性")
```

但没有人在运行:
```python
真实的自己.show()
```

因为那个函数会返回 `None`。

"小小,你在想什么呢?"母亲碰了碰她的手。

林小回过神:"没什么,就是想到工作的事。"

张文轩立刻接话:"工作重要,但也要注意休息。其实我觉得,女孩子嘛,不用那么拼。找个靠谱的人,有个稳定的家庭,比什么都重要。"

**隐藏属性暴露：【传统性别观念】**

林小的母亲眼睛一亮,连连点头。

林小看着张文轩真诚的眼神,突然想笑。

她想问他:你知道吗,我白天的工作是骗老人的钱。我现在坐在这里,用的是跟骗他们一样的技巧在跟你聊天。我们都在表演,但你以为你的表演是真实的。

但她只是笑了笑:"嗯,你说得对。"

相亲继续进行。

张文轩继续展示他的"优质属性",她母亲继续满意地点头,而林小继续扮演那个"温柔知性的相亲对象"。

一个小时后,相亲结束。

走出茶餐厅时,成都的秋天已经有了凉意。

张文轩礼貌地说:"今天聊得很开心,我们加个微信吧,以后多联系。"

林小扫了他的二维码。

然后他开着他的奥迪离开了。

母亲挽着林小的胳膊,兴奋地说:"这个小伙子不错!有上进心,有责任感,关键是人家条件好还不嫌弃咱们……"

林小没说话。

她看着成都的夜幕降临,路灯一盏一盏亮起来。

她想起陈易凡在那个雨夜说的话:

"每个人都是一段代码,但没有人知道自己的源代码是什么。"

她现在知道了。

她的源代码是 `None`。

她是一个空的"属性集"容器,可以根据需要填充任何属性:【欺诈师】【孝顺女儿】【温柔女友】。但当所有属性都被清空时,剩下的是什么?

什么都没有。

她打开手机,看到张文轩已经发来了微信:

"今天很高兴认识你。我觉得我们挺合适的,希望以后能多了解彼此。😊"

她没有回复。

她打开了另一个聊天窗口——那是老K的窗口。

老K发来了今天的"业绩目标"和三个新的"目标客户资料"。

**任务队列：**
- [ ] 回复张文轩(扮演:温柔女友)
- [ ] 联系目标客户A(扮演:投资顾问)
- [ ] 给妈妈打电话(扮演:孝顺女儿)
- [ ] ~~做真实的自己~~(错误:对象不存在)

林小把手机收起来。

秋风吹过锦江,水面上倒映着成都的灯火。

她站在桥上,突然感到一种深入骨髓的孤独。

不是因为没有人陪伴,而是因为——

**没有人知道真正的她是谁。**

**包括她自己。**
