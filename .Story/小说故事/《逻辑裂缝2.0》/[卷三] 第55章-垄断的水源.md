# 第55章 垄断的水源

**2027年10月15日，大凉山，阿土列尔村。**

秋天的凉山，最缺的就是水。
村里的老式水渠已经干涸了。为了解决灌溉问题，陈易凡之前远程指导小雪，用ASTO逻辑设计了一套“智能分水系统”。
原理很简单：通过土壤湿度传感器检测每块地的需水量，然后自动控制水闸的开关，实现“按需分配，杜绝浪费”。

这是一个完美的逻辑。
但小雪今天发现，完美的逻辑正在制造灾难。

她站在梯田边，看着眼前的景象。
属于村长家那几亩花椒地，水流哗啦啦地淌着，土壤湿润得甚至有些泥泞。
而旁边几户贫困户的玉米地，却干裂得像老人的手背，玉米叶子枯黄卷曲。

“怎么回事？”小雪问负责看守水闸的村民阿牛，“系统坏了？”

阿牛支支吾吾：“没坏。系统显示的，花椒地‘优先级最高’。”

“优先级？”小雪皱眉，“我设置的参数里，粮食作物才是P0级啊！”

“那个……”阿牛看了看四周，压低声音，“村长前两天找了个懂电脑的亲戚，说是来‘优化系统’。然后……然后就这样了。”

小雪冲进控制室（其实就是一个破木屋）。
她打开那台旧笔记本电脑，查看后台日志。
果然，代码被改了。
在核心分配逻辑里，多了一行判断：
`if (owner == "Village_Leader") { priority = MAX_VALUE; }`
`// 备注：花椒是经济作物，能给村里创收，所以优先供水。`

小雪气得手抖。
这行代码是谁写的？
不是黑客，不是Titan，而是村长的那个读技校的侄子。他用的正是ASTO开源的模块。

她拿出手机，拨通了陈易凡的电话。
信号依然不好，声音断断续续。

“亦凡，”小雪的声音在风中颤抖，“你的ASTO被黑了。”
“被谁？Titan？”
“不。被**贪婪**。”

小雪看着窗外那片绿油油的花椒地，和那片枯黄的玉米地。
“你告诉我，技术是中立的。你说刀可以杀人，也可以切菜。
但我现在看到的，是拿着刀的人，正在用你的‘切菜指南’，更高效地切断别人的喉咙。”

陈易凡在电话那头沉默了。

“你知道吗？”小雪继续说，“以前村长想多占水，还要跟人吵架，还要顾忌面子。
现在不用了。
他只需要指着屏幕说：‘这是系统算的，最科学，最优化。’
没人敢反驳系统。
你给他们造了一个神，而这个神，现在成了恶霸的打手。”

“把代码改回来。”陈易凡说，“我可以远程……”

“改回来有什么用？”小雪打断他，“明天他侄子又会改回去。只要控制权还在村长手里，算法就永远姓‘村’。
亦凡，我们错了。
我们以为给了他们工具，就是给了他们公平。
其实我们只是给了强者更强的武器。”

电话那头传来一阵嘈杂的电流声。
小雪挂断了电话。
她走出木屋，走向那片干枯的玉米地。
她没有去改代码。
她拿起一把锄头，开始挖土。
她要把那个自动水闸旁边挖开一个缺口，让水流出来。

这是一个极其原始的动作。
这是一个违反ASTO逻辑的动作（破坏系统完整性）。
但这是此刻，唯一正义的动作。

水流了出来，混着泥沙，流进了玉米地。
几个村民看到了，也跑过来帮忙。
他们不敢动电脑，但他们敢动锄头。

小雪看着浑浊的水流。
她明白了。
对抗技术暴政的，从来不是更好的技术。
而是**锄头**。
是那些如果不反抗、就活不下去的人手里的锄头。

（第55章 完）
