# 第67章 阿姨们的算法

**2028年5月21日，成都，339电视塔下。**

广场舞依旧。
领舞的还是张阿姨。
队伍里多了几个新面孔，也少了好几个熟悉的老人（老张已经不再来跳了，他忙着反诈）。

今天，张阿姨的动作有点不一样。
以前，她总是追求“标准”，谁跳错了她就要说。
但今天，她跳得很随意。
音乐是《荷塘月色》，但她跳出了迪斯科的感觉。

“张姐，你这是啥子跳法哦？”一个阿姨问。
“这叫‘随心所欲法’。”张阿姨笑着说，“小雪走的时候跟我说，跳舞不是为了给别人看的，是为了让自己开心的。只要开心，咋个跳都是对的。”

她不知道什么是ASTO，也不知道什么是“不确定性”。
但她用自己的身体，诠释了Origin最核心的哲学：
**Life is not a bug to be fixed. It's a feature to be experienced.**
（生活不是一个需要修复的Bug，而是一个需要体验的特性。）

而在广场的角落里，几个年轻人正在用手机直播。
他们是Titan的员工。
他们正在测试Titan新推出的一款名为“Flow”（流）的社交APP。
这款APP的核心算法，正是基于陈易凡开源的“诚实的不确定性”协议。
它不再推荐“最完美”的内容，而是推荐“最真实”的内容。
比如张阿姨这支乱糟糟、但充满生命力的舞蹈。

直播间里，弹幕刷屏了。
“好治愈啊！”
“这才是生活！”
“我也想去跳！”

张阿姨不知道自己火了。
她只是在跳舞。
在339电视塔的阴影下，在算法的注视下，她跳出了一首属于自己的、无法被定义的诗。

（第67章 完）
