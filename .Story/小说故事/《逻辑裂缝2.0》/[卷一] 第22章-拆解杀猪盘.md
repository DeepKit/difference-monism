# 第22章 拆解杀猪盘

**时间：2024年9月29日（周日）晚22:00**  
**地点：小雪的出租屋**

---

小雪的出租屋只有15平米，但收拾得很干净。

此刻，这间小屋里弥漫着一股浓烈的劣质香水味。那是方姐身上的味道。

方姐是小雪的邻居，也是她在便利店的同事。三十五岁，离异，渴望爱情。

"你看，他今天又给我发了99朵玫瑰的红包。" 方姐捧着手机，脸上泛着少女般的红晕，"他说等他在新加坡的项目做完，就回来接我。"

手机屏幕上，那个叫"Kevin"的男人的头像是一个在游艇上喝香槟的背影。

**对象：Kevin**  
**属性：【高富帅】【深情】【忙碌】【懂投资】**

小雪看着那些聊天记录。

如果是以前，她会由衷地为方姐高兴。但自从接触了ASTO的逻辑训练后，她看世界的方式变了。

她看到的不是情话，是**脚本**。

"方姐，能让我看看你们这周的聊天记录吗？" 小雪问。

"干嘛？羡慕啊？" 方姐虽然这么说，但还是得意地把手机递了过去。

小雪打开了她在便利店收银机背面偷偷画的那个"ASTO逻辑环"。

她开始比对。

**输入数据：Kevin的回复时间**
- 早上 08:00 - 08:30：早安问候（固定）
- 中午 12:00 - 12:30：午餐分享（图片精美，但从不露脸）
- 晚上 22:00 - 23:00：深情交流 + 诱导投资

**异常检测：**
- 9月25日（周三）：他说在开会，回复延迟。
- 9月27日（周五）：他说在开会，回复延迟。
- 延迟的时间差都是**45分钟**。

"方姐，你看这里。" 小雪指着屏幕，"周三和周五，他说他在开会。但是你看他发给你的这张午餐图。"

那是两张看起来完全不同的法餐图。

"怎么了？"

"光线不对。" 小雪调出了手机自带的修图软件，拉高了对比度，"周三这张图，窗外的影像是下午两点的阳光。周五这张，虽然是晚上发的，但盘子边缘的反光显示窗外是同一个角度的阳光。"

"而且，" 小雪深吸了一口气，"这两张图的元数据（Exif）都被抹去了。如果是随手拍的，不会这么干净。"

方姐的脸色变了变："你懂什么？人家是大老板，用的手机肯定跟咱们不一样。"

"还有这个。" 小雪点开了Kevin的一条语音。

*"宝贝，我也很想你，但是现在这个币种走势很好，你先把那五万块转进去，我们为了未来一起努力。"*

小雪打开了另一个APP——那是陈易凡给她的"逻辑过滤器"。

她把这段语音输入进去。

**分析结果：声纹匹配度 99.9% -> 合成语音库 V3.0**

"方姐，这是AI合成的声音。他在念稿子。"

"你胡说！" 方姐一把抢过手机，"你就是嫉妒我！你自己找不到男朋友，就看不得我好！"

"方姐，真的是假的。这是一个典型的'杀猪盘'脚本。" 小雪急了，"你看他的话术，先建立情感依赖，然后展示财力，最后切入投资。每一步都卡在心理学的弱点上。"

"我不听！我不听！" 方姐捂住耳朵，"Kevin是真的！他还给我看过他的身份证！"

"身份证也可以P的……"

"够了！"

方姐猛地站起来，椅子被撞翻在地。

"李小雪，我把你当姐妹，你为什么要毁我的梦？" 方姐的眼睛红了，眼泪把那层厚厚的粉底冲出两道沟壑，"你知道我这几年过得有多苦吗？好不容易有个人关心我，嘘寒问暖，是不是骗子有那么重要吗？"

"至少他让我觉得自己是个女人，不是个只会扫条形码的机器！"

小雪愣住了。

她看着歇斯底里的方姐，突然意识到自己犯了一个逻辑错误。

**逻辑错误：【真理】 != 【幸福】**

在ASTO的逻辑里，真是真，假是假。但在方姐的生活里，"假"是她活下去的唯一动力。

Kevin是不是真的不重要，Kevin提供的**情绪价值**是真的。

但是……

小雪想起了老张。那个在巷口哭得撕心裂肺的老张。

如果不拆穿，方姐的下场就是老张。

"方姐。" 小雪的声音冷了下来，"你可以不信我。但你那五万块钱，是你给豆豆存的手术费。"

提到儿子豆豆，方姐的身体僵住了。

"如果这钱没了，豆豆的手术怎么办？"

方姐的眼神开始涣散。她慢慢地坐回地上，手机滑落在脚边。

屏幕上，那个Kevin又发来了一条消息：
*"宝贝，怎么不回话了？是不是睡了？爱你晚安。"*

多么温柔。多么完美。

"哇——"

方姐突然嚎啕大哭。

那种哭声不是伤心，是绝望。是梦醒之后的虚无。

小雪走过去，抱住方姐。

方姐在她怀里拼命挣扎，拳头雨点般落在小雪背上："都怪你！都怪你！为什么要告诉我！让我傻一辈子不好吗！"

小雪忍着痛，一声不吭。

她看着窗外漆黑的夜空。

她救了方姐的钱。

但她杀了方姐的梦。

**逻辑是冰冷的刀。它能切除肿瘤，但也会切断血管。**

小雪突然明白了陈易凡为什么总是皱着眉头。因为拿着这把刀的人，注定要看到人性最血淋淋的一面。

"对不起，方姐。" 小雪在心里轻轻地说，"但我们必须醒着。"

因为睡着的人，会被吃掉。
