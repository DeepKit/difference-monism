# 前言：关于《作品》的三重世界

你即将阅读的，不仅仅是一部关于电影的小说。
这是一个关于**创造、控制与代价**的三重奏。

在《作品》的宇宙中，三条线索交织并行：

1.  **🎥 现实线**：一个失败的导演沈既白，为了寻找人生的意义，决定重启他尘封多年的剧本《ASTO》。他面临着资本的压迫、伦理的拷问和家庭的撕裂。这是**“因”**。
2.  **🎞️ 电影线**：沈既白正在拍摄的那部科幻电影。在一个被系统（ASTO）完美控制的未来世界，主角阿空试图寻找系统外的可能性。这是**“果”**，也是思想的显影。
3.  **🌐 Web3线**：隐藏在代码与区块链深处的真相。真正的 ASTO 并非为了控制，而是为了自由。这是**“源”**。

本书呈现的，是**【电影虚拟线】**的完整故事，同时穿插着现实世界的投影。
当你翻开下一页，你将进入那个既是虚构、又无比真实的 ASTO 世界。

请记住总导演的一句话：
**“作品一旦完成，就不再属于作者。”**

现在，它属于你了。
