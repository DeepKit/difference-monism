# 【共识】第二章：寻找共识 (Finding Consensus)

## 2.1 父子琴音共振

寒假前两天，县城老家。空院里飘着刚雨霁过后的浓郁静谧。院子里的煤核树光滑发亮，只剩下几枚没被摘净的红柿子，像挂在枝头的红灯笼。

沈既白穿着一身深蓝色的中山服，戴着无框眼镜，一派典型的学院派打扮。

父子合力把沉重的石磨撬开一条缝。

“看行吗？”沈既白抹了一把额头上的汗，“看了我发给你的‘大槐’了吗？那个‘猴子推石头’的比拟，你觉得怎么样？”

沈知行停下了手里的动作，推了推眼镜，眼神有些迷惘：“爸，说实在话吗？”

“当然。”

“我觉得有点……范畴错误吧。”沈知行斟酌着用词，“海德格尔说过，只有‘此在’（Dasein）才能真正地‘去存在’。猴子推石头，那是本能，是物理运动。你不能把物理世界的结果因果，和人类历史的行动（Action）混为一谈。这在哲学上是站不住脚的。”

沈既白沉默了，没有反驳。他指着那块刚被撬开的石磨：“你看这块石磨。”

他用力推了一下，石磨低沉地发出沉闷的吱呀声，转了半圈，压碎了几粒谷子，吓跑了一群正在“听”的蚂蚁。

“刚推这一下，是物理运动吧？”

“是。”

“草被压下去了，蚂蚁逃跑了，水的流向变了。对那群蚂蚁来说，它们的世界是不是被‘改变’了？”沈既白问。

沈知行愣神：“那是物理后果，不是历史意义。意义是人赋予的。”

“但在蚂蚁眼里，刚刚发生的那个‘事件’，和一颗烟花没有区别。”沈既白蹲下来，看着那群惊慌失措的蚂蚁，“这就是 ASTO 设定的第一层：**先在力**——先于意义、先于叙事的力。世界在有人赋予意义之前就已经在运转：石磨在那儿，重力在那儿。猴子推一下，风吹一下，或者我推一下，本质都是施加一个力，让某些属性的变迁速度变快或变慢。”

沈知行愣了一下。他没想到父亲会用这种方式解释。

“如果不认识这一点，你的哲学就是空的——”沈既白站起来，拍了拍手上的灰，“你以为人是上帝，都能空手创造历史。其实人只是在加或减那些本来就要发生的事。”

沈知行有些不悦：“好，就算物理层面都一样。但伦理不一样啊！猴子推石头不需要负责，人需要承担责任。”

“对！”沈既白提高了声音，眼神变得锐利，“这正是我要说的。猴子推石磨，那只是顺手——它不多想。但你——沈知行，你想去大城市，你想离开县城——你是反思着来的。你要让水往不该流的地方流。”

沈知行抬头，看着父亲。

“你为了‘更好的未来’（优化属性），不只改了你自己，也改了我们这个家的‘场域’。你会离开，我和你妈会变成空巢老人。这也是代价。”沈既白的声音平缓下来，字字诛心，“你敢承担这个代价吗？”

屋子里一片寂静。只有远处传来的鸦鸣。

沈知行握着父亲的手指微微发颤。他一直以为自己是在寻求理想，从未想过，这对别人的生活意味着什么。

## 2.2 虚拟线：三阶工具幻象

2056年，ASTO 数据标注中心。

这是一个巨大的白色空间，数千个工位整齐排列，像是一个巨大的蜂窝。每个工位上都坐着一个戴着神经连接头盔的操作员。

影-阿空（编号04）坐在第17排第4个位置。他的眼前是不断流动的全息数据流：人脸、行人轨迹、信用评分、情绪指数……

他的工作很简单：系统会初筛出“潜在风险者”，而他需要做最后的确认。

左眼眨一下是“安全”，右眼眨一下是“高风险”。

他每天要眨眼两万次。他觉得自己不是人，只是系统的一个义肢。这样想着，反而踏实。

**第一阶：自然扰力**

啪。

一张新的照片弹出来。一个中年男人，神情疲惫，行人轨迹显示他经常在非授权区域活动。

系统提示：疑似非法集会组织者。

阿空没有多想。右眼眨了一下。

高风险。

他继续处理下一个。工作就是这样，不需要想太多。

**第二阶：意识扰力**

一周后。

阿空坐在餐厅里，吃着标准化的营养餐。突然，屏幕墙上播放了一条新闻：

*【正义中心通报】编号731号公民经改造后成功回归社会。他正在学习如何正确地上网。*

画面上，一个男人坐在白色的房间内，对着镜子练发音。他的发音很标准，嘴角的弧度精确到小数点后两位。

但他的眼眶是空的。

镜头再近一点：男人面对的“镜子”其实是一块带刻度的玻璃，左下角有一行小字：*录制中*。

画外音像导演一样冷静：“编号731，第三遍。句尾停顿0.3秒，尾音收住。嘴角上扬两度。别看镜头，去看你自己。”

男人照做，嘴角拉出一个“合格”的弧度：“我理解了。我支持社区情绪稳定。我会正确上网。”

“重来。”画外音说，“你不需要感动，你需要可预测。”

屏幕角落弹出评分：情绪一致性 92→94。

男人的眼眶依旧空荡。


阿空认出了那张脸。

是上周他标注的那个中年男人。系统标注为“高风险”，是因为系统检测到他经常在非授权区域活动。但那个区域，是他已故妻子的墓地。

系统没有这个参数。

在系统里，没有参数就等于不存在。

阿空也没有问。

那天晚上，阿空躺在床上，反复告诉自己：“我只是工具。责任在系统，不在我。”

但那个空荡荡的眼眶，总在他闭上眼的时候浮现。

**第三阶：反思性扰力**

啪。

又一张新的照片弹了出来。

阿空正准备习以为常地眨眼，动作却突然停住了。

照片上的人，有着一头乱糟糟的卷发，嘴角挂着玩世不恭的笑，手里拿着一把破旧的吉他。

那是**孔鸣**。他失散了三年的弟弟？！

系统红色的警告框在发热：
**【目标：孔鸣（ID: 9527-X）】**
**【风险评估：极高。】**
**【原因：传播非标音频（摇滚乐），扰乱社区情绪稳定。】**
**【建议操作：高风险（送入矫正中心）。】**

倒计时开始：3……2……1……

那个陌生男人的空眼眶突然和弟弟的笑脸重叠在一起。

如果选【高风险】，弟弟就会变成那样。

如果选【低风险】，系统会立即检测到他的违逆，他和弟弟会被一起“清洗”。

1……

阿空的手指在颤抖。

他突然明白了一件大事：“我只是工具”这句话本身，就是共谋。他把责任外包给了系统，但在场的是他，不是系统。

在倒计时归零的前一秒，阿空做出了选择。

他没有眨眼。他伸手拔掉了神经连接线。

警报声骤然响起，打破了白色空间的死寂。

## 2.3 现实线：组局的代价

大年初三，北京一家不知名的咖啡馆里。

沈既白面前坐着两个人。

左边是**韩深**，塌鼻子驼背，眼神阴郁。曾经是业内最有前途的编辑，一纸封杀后，只能靠玄幻网文为生。

右边是**苏念**，容貌精致，却掩盖不住疲惫。独立制片人，刚从一场不愉快的相亲里逃出来。

“我没钱。”苏念抱着头，丧气地陷在座位里，“如果是来找我改剧本的，免谈。我现在写亲情文的小说稿，不用动脑子。”

“我没时间。”韩深看了看手表，“我老婆给了我最后通牒，今年不出门就回老家考公。老沈，你的项目……风险太大，回报周期太长。不适合现在的我。”

沈既白从包里拿出了两份打印好的剧本，推到他们面前。

“韩深，”沈既白看着那个颓废的男人，“我知道你在玄幻里埋了什么。你写‘战神’，写的其实是你对这个世界的愤怒——但那是‘自在态’的愤怒，只能发泄。我这儿有个剧本，能把你的愤怒‘编码’成真正的能量。我不改你，我翻译你。”

“得了。”韩深按灭烟头，“老沈，你能不能说人话？什么自在态、编码不编码的，听得我头疼。简单说就是——别把故事装裱起来，起来干活。对吗？”

沈既白愣了一下，然后笑了：“对。就是这个意思。”

韩深的手指动了一下，眼神依然阴郁，但没有说话。

沈既白转向苏念：“苏念，你不想被家庭‘封版’，对吗？像剪辑封版那样，一锤定音，谁也改不了。”

“你想证明你不是父亲的遗产女儿，而是苏念。这部电影，说的就是‘解封’——一个人如何打破系统给定的最优路径。”

苏念的手指在桌子下悄悄地划着不存在的东西。

荒芜。长时间的荒芜。

韩深拿起剧本，翻开了第一页。苏念起身倒了杯茶，又放下。

“大概多少？”苏念问。

“不多。但我卖了老家那块宅基地。”沈既白说。

“三万。”苏念咳了一声，嘴角勾起了一个久违的笑意，“算我一个。反正亲情文也写腻了。”

“我只给你半年。”苏念从包里拿出一支笔，“半年后如果没起色，我就回老家结婚。”

沈既白看着他们，心里的那块石头终于落了地。

还有希望。

虽然是在废墟之上。

沈既白举起咖啡杯，向两人示意了一下。

“合作愉快。”

韩深和苏念也举起了杯子。

就在杯子碰撞在一起的那一刻，沈既白的手机又震动了一下。

还是那个陌生号码。

“小心方磊。”沈既白的手抖了一下，从荷包里掏出几颗薄荷糖。

他抬起头，正好迎上方磊走过来的目光。方磊端着咖啡，身后跟着两个助理，气场过人。

“老沈，真巧啊。”方磊笑着走了过来。

沈既白把手机反扣在桌面上。
