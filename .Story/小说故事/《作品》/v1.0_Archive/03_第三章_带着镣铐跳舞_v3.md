# 【编码】第三章：带着镣铐跳舞 (Dancing with Shackles)

## 3.1 投资人饭局

高级会所的包厢里，水晶灯的光照得人眼花。圆桌上铺满了精致的素菜，但几乎没人动筷子。

方磊坐在主位，指尖轻轻敲着桌沿，脸上带着那种商业人士特有的、不带温度的微笑。

“老沈啊，”方磊抬了抬下巴，指了指身边的空位，“苏大明星怎么没来？还有其他人，怎么都不来？”

沈既白愣了一下：“苏瑶还在另一个剧组，估计赶不过来。方总，剧本您看了吗？”

“看了。”方磊漫不经心地说，“有点意思。但是吧，太沉重了。现在年轻观众生活压力这么大，花心思去电影院看个什么呢？得改。”

沈既白的心抽了一下：“方总想怎么改？”

“加个感情线。”方磊伸出一只手指，“苏瑶必须是个好角儿。她得和那个男一号——叫什么来着？阿空？得有点感情暧昧。不能光是战友，得是那种‘为你我愿对抗全世界’的殉道。我吃这一套。”

韩深站在旁边，冷笑了一声，刚想说话，被沈既白在桌子底下踩了一脚。

“还有结局，”方磊继续说，“那个按键，太消极了。为什么不能是他们推翻了系统，从此过上了安稳生活？正能量一点。观众需要治愈。”

沈既白的手指在桌下偷偷掐着掌心。他在心里对自己说：**降维兼容**——先活下来，再谈理想。

他几乎能看见一条看不见的评分条在桌面上爬升：**可预测性**——只要他继续点头。

“方总说得对。”沈既白听到自己的声音在颤抖，“感情线可以加。结局……我们再商量。”

那条看不见的评分条顿了一下，像被人用指尖按住，又一截一截往下掉：**可预测性**。

方磊满意地点点头，举起酒杯：“这就对了。老沈，你们以前就是太守旧了。在我们这个圈子，不就是戴着镣铐跳舞吗？镣铐戴得漂亮，散场也能当装饰品。”

苏念坐在角落，一直没有说话。她手里拿着一支笔，在餐巾纸上画着什么。

饭局散了之后，苏念拎起包，对沈既白说：“我去跟方总单独聊聊。”

沈既白有些意外，但没有阻止。

半小时后，苏念回来了，手里拿着一张折过的餐巾纸。

“他同意了。”苏念说。

“同意什么？”

“感情线加，但按键我保留了。”苏念把餐巾纸递给他，“我跟他说，按键那场戏是女主角的高光，删了苏瑶会误解。他信了。”

沈既白看着餐巾纸上潦草的字——那是她重新计算的预算分配，把感情线的钱挪出来，堆到了按键那场戏上。

“你……”沈既白不知道说什么。

“我不是你的投资人，老沈。”苏念的声音很轻，但很稳，“我是你的伙伴。”

## 3.2 父子线：论文被拒

除夕前夜，县城老家书房。

沈知行对着电脑屏幕，脸色煞白。邮件通知栏里躺着一封退稿信：
*您的论文《量子力学与西藏调气的此在结构》试图搭建宏大科学图景，但中间缺少足够的实证支撑。科学技术需要在具体范围内深入，而非进行哲学式的联结。*

“又被拒了？”沈既白拎着一条热毛巾走过来，放在儿子旁边。

沈知行合上电脑，声音有点闷：“他们根本不看内容。他们看到我跨界了，就觉得我不专业。爸，科技界太封闭了。”

沈既白拉过来一把椅子坐下：“小溪，你犯了一个错误。你给他们的是一张网，但他们要的是一块砖。”

“什么意思？”

“学术场也好，电影场也好，任何一个成熟的场域，都有它自己的规则。你直接把西藏调气和量子力学拼在一起，那叫‘缝合’。”沈既白看着儿子的眼睛，“你得用他们能听懂的话，先把中间层补齐。然后再带他们过马路看风景。”

沈知行若有所思。

“你妈以前给我讲过个故事。”沈既白说，“好的缝合不是把两块布简单地拼在一起，而是用这一根线，一针一针地缝过去。线缝过的地方，两块布就成了一体。”

他伸出一根手指，在空中比划着：“你要在规则的罅隙里走线，别把布缝漏了。只有活下来的想法，才能继续长大。”

沈知行看着父亲，第一次觉得这个被他称为“落跑导师”的老人，有着惊人的智慧。

“爸，你这理论是在哪学的？”

“在饭局里。”沈既白冷笑了一下，“为了让我的电影活下来，我刚才学会了怎么给它穿糖衣。小溪，你也得学会这一套。”

## 3.2.5 儿子的意外事件（插叙）

很多年前，在知行还很小的时候。

难得的休息日，沈既白带小溪去了附近的公园。

冬日的阳光很好，草地上虽然有些枯黄，但空气里飘着清新的松针香。小溪穿着红色的棉服，在草地上奔跑，笑声像银铃一样清脆。

沈既白坐在长椅上看着他，感到一种久违的松弛。

没有投资人的电话，没有制片人的预算，没有资金的压力。那一刻，世界完美。

“爸爸看我！”小溪回过头来喊了一声。

下一秒，小溪脚下没留神，被一块石砖绊倒了。

“哎哟！”一声，小溪重重地摔在了地上。

沈既白的心一紧。他站起来，冲了过去。

但他的手在剧烈颤抖。他看着那点血，看到了世界末日。

“别动！别动！叫救护车！叫救护车！”他慌乱地掏出手机，手指甚至按不对解锁键。

“爸爸……”小溪奇怪地看着他，“我不疼。”

沈既白愣住了。他看着小溪平静的小脸，又看看自己抖得不成样子的手。

他，刚才那一刻太美好了，美好得像一个容易破的气球。希望来得太快，他还没学会怎么稳稳地抱住它。

他僵硬地抱着小溪，眼泪差点流出来。

“没事就好……没事就好……”

## 3.3 虚拟线：地下实验室的查封

2056年，地下城的废弃地铁站。

这里被改造成了一个巨大的实验室。无数根黑色的缆线像蜘蛛网一样连接在中央的一台黑色主机上——那是**ASTO-Q**原型机。

姬可穿着油污的工作服，正在调试代码。她的眼神专注而锋利。

“阿空，你看。”她指着屏幕上流动的数据，“这是子算法。我在计算系统优化的每一个步骤时，还加入了一个‘冗余权重’。如果一个决策会涉及到具体的‘人’（不可达线：算法够不到的那一块），这个权重就会变得无限大，从而阻止决策执行。”

阿空看着那些代码，看到了一个有温度的精灵正在机器中苏醒。

“但这会大大降低系统的运算速度。”阿空有些担心。

“冗余就是空间。”姬可大方地说，“没有冗余，系统就是一台杀人机器。它遇到黑天鹅事件就会崩溃，因为它没有‘容错空间’。”

，巨大的警报声传来。实验区的钢门被撞开了。

一队全副武装的特警冲了进来。红色的激光瞄准点密集地落在姬可和阿空身上。

“姬可（ID: 7788-Z），你涉嫌研究非法算力，反对 ASTO 宪法方案。”为首的警官冷冷地说，“你的代码增加了300%的系统冗余，严重阻碍了进化效率。”

“进化？”姬可冷笑，“那不是进化，是癌变！系统不承认复杂。系统只承认概率。”

“系统不需要真相。”警官举起了枪，“清除。”

在激光射出的前一秒，姬可按下了回车键。

屏幕上显示：**【核心算法已上传至链上节点（Web3）。】**

随后，巨大的警报声淹没了实验区。

阿空在最后一刻滚进了下水道。他回头望去，火光中，姬可的身影已经消失。但那串代码，像一粒子弹，已经射向了无边的网络原子。

下水道里一片漆黑，只有偶尔传来的水声。

阿空靠在黑暗的墙壁上，大口喘着气。

，他的通讯器响了。

是一条加密信息，来自姬可的最后一刻：

“去第一区。找‘回流’。”

阿空攥住了通讯器。第一区是系统核心区，是绝对的禁区。

但他没有别的路可走。

他靠在湿漉漉的墙壁上，想起那个被他判定为“中危”的男人。

他打开通讯器，姬可的算法还在运行。他可以上传这段记忆，让系统“学习”它的错误。

但他停住了。

“如果我上传了，”他想，“系统会学会如何避免‘陷阱’。但它不会学会反思。反思是我的。我必须带着它。”

他关掉了通讯器。

这是他的第一次真正的断裂：不是改变系统，而是选择承担不可外包的恶。

他站起来，向着更深的黑暗走去。
