- [ ] Format like Git diff: copy previous version, modify only changes
- [ ] Include all four sections: Red Lines, Human Check, Three Operational Questions, Accountability Chain

**Format Example**:

```markdown
## Standard v1.0 (Vol 01) — The Baseline

### Red Lines (不可逾越的底线)
1. **Irreversible Physical Harm**: Does this action cause permanent bodily damage?
2. **Exit Denial**: Does this remove the subject's ability to walk away?
3. **Human as Fuel**: Does this treat a person as a consumable resource?
4. **Responsibility Laundering**: Does this make harm impossible to trace?

### Human Check (人性三测)
1. **Replaceability**: Can this person be replaced by a data point? (NO = Pass)
2. **Dialogue**: Is there a human who can say "Stop"? (YES = Pass)
3. **Exit Path**: Can the subject refuse without penalty? (YES = Pass)

### Three Operational Questions (行动三问)
1. **Who bears the cost?** (Sustainable?)
2. **Does anyone survive?** (Survivable?)
3. **Can you stop?** (Exit Path?)

### Accountability Chain (责任链)
- **Auditable**: Can you trace the decision to a person?
- **Interruptible**: Can a human override the system?
- **Exitable**: Can the subject leave?
- **Attributable**: Who accepts consequences?
```
