## ⚡ Quick Start Guide

### If you are writing a Chapter (Ep):
1.  Check **[TIMELINE.md](TIMELINE.md)** for the date.
2.  Open **[写作指南-Western-Style.md](写作指南-Western-Style.md)** for the "Clinical Noir" checklist.
3.  Use **[小案结构.md](小案结构.md)** to ensure you hit the Docket/Evidence/Sophie beats.
4.  End with a **Sophie's Board** update.

### If you are outlining a new Volume:
1.  Open **[42卷去重复地图.md](42卷去重复地图.md)** to get the Volume Signature (Case Type, Outcome, etc.).
2.  Open **[STANDARD_EVOLUTION_MAP.md](STANDARD_EVOLUTION_MAP.md)** to get the "vX.X" update text for the ending.
3.  Use **[反杀弹药库-Western.md](反杀弹药库-Western.md)** to pick a Malicious Compliance mechanic.

### If you are designing Ream Content:
1.  Read **[叙事结构与变现策略.md](叙事结构与变现策略.md)**.
2.  Remember: All story content (Ep1-6) eventually goes free on Royal Road. Ream subscribers get (a) advance chapters (15-25 ahead) and (b) exclusive depth materials: **Adjudication Stack** + **Unredacted Files** + **Reasoning Trace** + **Counterfactuals**. You don't sell the *plot*. You sell the **understanding**.

## 🎯 Key Terminology (v4.0)

| Term | Definition |
| :--- | :--- |
| **104_C** | The universal "Safety Valve" code used to freeze/override the system. Later weaponized. |
| **Red Lines** | The absolute prohibitions (Irreversible Harm, No Exit) defined in The Standard. |
| **Human Check** | The requirement for a human to accept liability in a decision chain. |
| **Adjudication Stack** | The "Ethical Source Code" — Ream-exclusive deep-dive material (v1.0...v4.11). Full Standard text + Reasoning Trace + Unredacted Files + Counterfactuals. Never appears on Royal Road. |
| **Clinical Noir** | The narrative style: Detached facts, brutal math, hidden emotion. |
| **Sophie's Board** | The investigative dashboard at the end of chapters (Red String, Status). |
| **NewCare** | The antagonist corporation (Insurance/Compliance). |
| **The Standard Act** | The final legislative result of Mia's war (Vol 42). |
