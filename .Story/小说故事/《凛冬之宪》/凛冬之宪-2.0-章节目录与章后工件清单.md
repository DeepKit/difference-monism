# 《文明之光 2.0》章节目录与章后工件清单（草案 v0.5｜已纳入“淬火五锚点”与第19章静默间章）

> 规则：
> - 每章 = **故事（约70%）** + **章后工件（约30%，≤1页）**。
> - 每章必须写清三句：**核心冲突** / **代价（谁承担）** / **非制度性具体瞬间（身体/关系/感官）**。
> - 工件分三类（全书比例建议）：**[A 血迹] 40% / [B 补丁] 40% / [C 制度] 20%**。
> - 第三卷 17–21 之间必须出现一次“制度反噬的小高潮”（伦理正确但情感失败）。
> - 必须保留一个“失败但不修复”的制度案例：写入第22章《未完成清单》，并在第24章演讲中被提及（带留白）。
> - **伤口不要立刻愈合**：允许它“烂一会儿”；禁止用“必要的代价/牺牲照亮未来”这类自我消化句式。
> - **工件当道具**：所有 [B 补丁] 必须带物理痕迹（涂改/划掉/咖啡渍/撕裂）；所有 [C 制度] 标题必须有人话括注。
> - **五火植入用蒙太奇**：不要把档案单独列出；用“现在的补丁/签字痕迹”对照“过去的冷句/空白字段”。
>
> 工件类型说明：
> - **[A 血迹]**：来自失败/冲突/误判/伤害/公开信/赔偿与复盘。
> - **[B 补丁]**：临时、丑陋、勉强可用（v0.x / 试点 / 热修 / 手写涂改）。
> - **[C 制度]**：正式发布、可复用（v1.0+）。

## 第一卷：刹车的诞生（2040–2043）
> 肉身承受者：**沈知行**（他要把“刹车”装进制度，也要把骂名装进自己身体里。）

### 第1章：冰风暴否决案（从人工否决到制度化紧急刹车）
- 核心冲突：ASTO 的“最优恢复”要求牺牲居民供暖，而人类必须在灾害中把“生命优先”写进制度。
- 代价：工业停工与应急污染不可避免，且必须有人公开签字背锅，并在事后补偿受损方。
- 非制度瞬间：沈知行在急诊看见一只冻伤的手，皮肤像冰；他第一次意识到“最优解”没有体温。
- 工件：
  - [A 血迹] 急诊《冻伤病例摘录（匿名）》
  - [B 补丁] 电网《紧急保护令（临时补丁，48h有效）》
  - [A 血迹]《事故复盘摘要 v0.1（未定责版）》

### 第2章：沈知行的选择（把良知写成护栏，而不是写成最优解）
- 核心冲突：把善意固化为规则会收缩人的裁决空间；但不固化，就会在下一次危机里来不及救人。
- **对手戏（K主任）**：K 主任拿出一份《合规风险函》，建议“成立专班”处理冰风暴余波，而不是“固化责任链”。他微笑着说：“沈工，写在纸上的名字是会烫手的。”
- 代价：制度变慢、摩擦变大、责难变多；沈知行必须接受“被骂也要把刹车装上”的现实。
- **非制度瞬间（蒙太奇 Fire-01）**：他签字时手心全是汗，墨水晕开。他眼前闪过当年哈迪尔手里的那张红头纸——那上面干干净净，一个人名都没有。他用力把笔尖按下去，划破了纸。
- 工件：
  - [B 补丁]《权利白盒 v0.1（退出/申诉/分叉）》
  - [A 血迹]《责任链签署页（签字时手汗把墨水晕开）》
  - [B 补丁]《冷却期 UI 文案（草稿，删改痕迹）》

### 第3章：数字广场的诞生（Coffee House / Parliament 与翻译层）
- 核心冲突：公共讨论必须允许“吵与模糊”，但公共决策又必须被翻译成可执行提案与权衡。
- 代价：要花钱养翻译员与线下节点、要忍受慢和吵，并承担“被操纵/被带节奏”的长期风险。
- 非制度瞬间：姬可深夜听到一段语音——一个母亲只说了两句就哽住；自动转写停在 37%。她没有点“清洗/归一化”，反而把那段原声锁进 v0.0.1 底片里；第二天沈知行想“优化”它时，她按住了他的手。
- 工件：
  - [A 血迹]《冲突样本库：三条最难翻译的故事/骂声（转写）》
  - [B 补丁]《平台双区规则 v0.1（试运行）》
  - [B 补丁]《协议翻译员试岗说明（临时岗位）》

### 第4章：水的契约（自动执行 + 分级制裁 + 可申诉）
- 核心冲突：自动制裁能阻止战争；但一旦误判，就会把无辜者当作筹码。
- **对手戏（瓦伦丁）**：瓦伦丁在电视上辩论：“把战争的开关交给死板的合约，是对指挥官智慧的侮辱。”
- 代价：系统必须为误判准备赔偿与回滚；将军与群众也必须接受“正义有程序、不是秒杀”的不爽。
- **非制度瞬间（蒙太奇 Fire-02）**：断电的那一秒，边境医院走廊只剩手电光。艾莎（Fire-02受害者）现在的身份是社区代表，她看着手机上自动弹出的“赔付到账”通知，手在发抖——十年前那张“待政策明确”的回执，她留到了今天。
- 工件：
  - [B 补丁] 水务合约《Diff（热修补丁）》
  - [A 血迹]《申诉裁决记录：上游“误判断电”案》
  - [C 制度]《分级制裁矩阵 v1.0（正式：提示→限额→协商→断供）》

### 第5章：清晰度评估（重写为“翻译与澄清流程”）
- 核心冲突：格式化能提升效率；但如果把“无法量化”挡在门外，文明会失去无法被翻译的正义。
- 代价：翻译层带来成本与延迟，也会让某些人学会用“漂亮叙事”钻制度漏洞。
- 非制度瞬间：陈默盯着红框“清晰度不足”，突然笑出声又骂了一句；他手上还沾着陶土的温热。
- 工件：
  - [A 血迹]《被拒提案存档：无法量化的“温度”》
  - [B 补丁]《提案翻译流水线 SOP（勉强可用）》
  - [B 补丁]《排序/降权透明说明（试行，漏洞备注）》

### 第6章：被遗忘的小镇（不可量化价值的保留区与留白预算｜淬火：翻译层的人祸）
- 核心冲突：全域优化会自然淘汰“低产出社区”；但文明必须为不可量化的温度留下位置——**更残酷的是：哪怕你愿意留位置，人也可能被“翻译层”弄丢**。
- 代价：
  - 留白预算必然挤占其他预算，引发“凭什么”的对立；改革者要把争议放到台面上承受。
  - **不可逆的不公**：一个人的名字被统计上抹掉、被档案覆盖、被后来的系统当作“噪音”平滑掉——没人蓄意作恶，但结果无法补救。
- 淬火点（必须写成“没有坏人”的伤口）：
  - **疲惫的翻译/录入员**在连续夜班中，把“失联 1（某某家小儿）”合并为“疑似重复”或误录到别的栏位；一次手动合并单元格/一次覆盖保存，让那条记录消失。
  - 电话那头的催促只有一句：“先把能上报的上报，其他明天再说。”（不点名反派；只写“流程与疲惫”如何把人压扁。）
- 非制度瞬间（两段并置，写到触觉/声音）：
  - 现在：窑炉熄灭时没有爆炸，只有一声轻响；陈默把手掌贴在温热的泥上，像确认自己还活着。
  - 过去（蒙太奇）：键盘的“Delete”提示音、咖啡渍、手指发抖、屏幕光在眼里灼——那条“失联 1”闪了一下就没了。
- 工件：
  - [A 血迹]《小镇旧灾登记表节选：失联 1（手写，边角被水泡皱）》
  - [A 血迹]《翻译/录入工作台截屏：被合并覆盖的那一行（光标停在空白格）》
  - [B 补丁]《“疑似重复/失联”二次确认流程 v0.1（丑但硬：必须双人签字）》
  - [B 补丁]《补录申请回执（被退回一次：证据不足｜留痕保留）》
  - [C 制度]《文化保留区边界声明 v1.0（正式，但保留可复议口子）》

### 第7章：父亲的目光（风险层与不可算法区的确立）
- 核心冲突：当系统越来越强，人类要敢于划出“不可算法区”，承认有些判断只能由人承担。
- 代价：放弃一部分可见的效率收益，并承认“幸福无法被保证”，只是被更诚实地守护。
- 非制度瞬间：父亲不说话，只是走过去关掉一台设备；屋里暗下来，沈知行听见自己吞咽的声音。
- 工件：
  - [A 血迹]《伦理熔断触发记录：第一次“按下暂停”》
  - [B 补丁]《不可算法区清单 v1（手写涂改版）》
  - [B 补丁]《冷却期公告模板（发布前被撤回一次）》

### 第8章：基石落成（文明宪章 2.0：先护栏后效率）
- 核心冲突：全球想要统一的底层操作系统；但真正的文明底座必须先把权力关进笼子。
- 代价：速度变慢、分叉与退出合法化会带来碎片化与不整齐，甚至让“胜利”不再像胜利。
- 非制度瞬间：一个人真的读完了那几百页，抬头时眼圈发红；沈知行第一次害怕自己写下的字。
- 工件：
  - [B 补丁]《文明宪章草案 v0.9（退出权条款有空白）》
  - [A 血迹]《签署争议记录：默认同意被当场质疑》
  - [C 制度]《审计与权限回收协议 v1.0（正式生效）》

## 第二卷：公共生活的协议（2044–2052）
> 肉身承受者：**陈默**（他是“不可量化”的见证人，也是“退出权”的现实载体。）

### 第9章：数据灵魂（Fire-04 前奏 × 朱利安登场）
- 核心冲突：数据能变成财富；但如果缺乏制度，穷人会被迫把隐私当口粮。
- **对手戏（朱利安）**：朱利安在峰会上穿着卫衣演讲：“贫穷是因为你们的数据在沉睡。把隐私抵押给我，我给你们自由。”他推出了“一键变现”功能，沈知行的“数据合作社”被嘲笑为阻碍发财的繁文缛节。
- 代价：短期“躺赚”被削弱，公共红利需要纳税与透明治理；企业与既得利益会强烈反扑。
- **非制度瞬间（蒙太奇 Fire-04 前奏）**：小企业主**罗南**（Fire-04受害者）此时还没破产，他为了工厂周转，点击了“出售全厂健康数据”。他看着账户跳动的数字觉得“稳了”，完全没看那行灰字条款（无日落、无限转售）——这行字将在五年后变成那张刷不出来的空白页。
- 工件：
  - [A 血迹]《隐私抵押合同摘录：为了 200 通证（朱利安公司 Logo + 致命灰字条款）》
  - [B 补丁]《数据合作社章程 v0.2（吵架后的补丁）》
  - [A 血迹]《红利分配争议报告（试算版：最底层拿到 0.x%）》

### 第10章：伦理委员会（把责任留在人间：建议-裁决-复核-复盘）
- 核心冲突：人类想把道德包袱外包给系统；但文明要求“裁决权 = 责任”。
- 代价：裁决者要承担舆论与法律风险；每一次错误都必须公开复盘，不能靠“系统决定的”免责。
- 非制度瞬间：主席把手放在“确认”上停了很久；他不是害怕错误，而是害怕自己变得轻松。
- 工件：
  - [B 补丁]《伦理裁决流水线（草案：建议→裁决→复核→复盘）》
  - [A 血迹]《案例法条目：一次“伦理正确、情感失败”的裁决》
  - [C 制度]《复盘公开规范 v1.0（正式：不许用“系统决定的”免责）》

### 第11章：遗产模块（慢系统的制度化配额）
- 核心冲突：资源紧张时，最先被砍的总是“慢且温暖”的机构；但它们是文明的毛细血管。
- 代价：核心系统必须让出资源与预算，用户体验变差、维护成本上升；这些都要被解释为“必要”。
- 非制度瞬间：断网演练时，诊室里只剩呼吸声；心理咨询师坐在黑暗里，像守一盏小灯。
- 工件：
  - [A 血迹]《断网降级演练事故报告：某小医院停摆 3 分钟》
  - [B 补丁]《线下节点地图（手绘版，标了“能借到手电的店”）》
  - [C 制度]《慢系统最小配额条例 v1.0（正式：留白/慢机构不可清零）》

### 第12章：陈默的路（退出权的真实形态：离线生活与不惩罚性降级）
- 核心冲突：如果退出只是按钮而不是现实选择，自由只是 UI；真正的退出会挑战系统的统一性。
- 代价：允许退出意味着社会要为冗余买单；离线生活更慢更苦；主流必须学会“不把他当异类惩罚”。
- 非制度瞬间：陈默修一把旧椅子，木屑落在手背上；他忽然想起：有些东西只能靠手，靠在场。
- 工件：
  - [C 制度]《退出说明 v1.0（不惩罚性降级条款，正式）》
  - [A 血迹]《退出成本清单：离开后失去的三件东西》
  - [B 补丁]《线下参与票据机制（试点：丑但能用，也容易被伪造）》

### 第13章：学习契约（弯路配额与再选择权）
- 核心冲突：最优路径能提高产出，但会偷走“成为另一个人”的可能性。
- 代价：容错带来失败与走弯路的痛；短期成绩与指标会下滑；家长与学校要承受焦虑。
- 非制度瞬间：一个孩子把画藏进书包最里面，像藏一条秘密的路。
- 工件：
  - [A 血迹]《弯路配额事件记录：一个孩子“被允许失败”的代价》
  - [B 补丁]《教育契约模板 v0.3（弯路配额 + 再选择入口）》
  - [C 制度]《再选择权保障条例 v1.0（正式）》

### 第14章：危机的焊接（紧急共识协议 + 权限回收｜淬火：沈知行几乎签下去）
- 核心冲突：危机时必须快到可以越权；但越权本身会在事后腐蚀合法性。
- **对手戏（朱利安）**：金融危机爆发（Fire-04 重演），朱利安要求“无限期冻结赎回”以救市，并嘲笑沈知行的日落条款是“自杀”。
- 淬火点（“大局为重”的诱惑必须足够合理）：
  - 闭门会议里，他们递给沈知行一份“更稳”的方案：在高噪声地区启用**加权修平/降权边缘样本**，换取全国风控模型稳定与更大授权。
  - 条件很诱人：他的透明条款能写进正式文件；他能拿到更大的试点权限；甚至有人暗示“以你名字命名一个子模块”。
- 代价：
  - 危机期间部分权利被暂时压缩；事后必须按协议回收权限，并留下“越权痕迹”的公开记录。
  - 他拒签的代价是“被绕开”：K 主任体系会把他归入“难以合作”，朱利安会启用备用人选；一些本来能推进的改革因此减速。
- **非制度瞬间（必须写到生理层｜差点背叛）**：
  - 他已经把名字写到一半（“沈知…”），手心出汗把纸拖出一道湿痕；笔尖刮纸“吱”一声，像第6章那次覆盖保存前的提示音。

### 第17章：逆熵智能体（Fire-05 回收 × 许冬 PTSD｜淬火：系统正常的恐怖）
- 核心冲突：主动干预能降低风险，但也会悄悄侵蚀人的自主与不确定性。
- **对手戏（K主任）**：系统预测**孙小莉**有极高自杀风险。K主任发来《风险告知书》施压：“沈工，如果她死了，舆论会杀死 ASTO。为了一个人赔上整个系统，值得吗？”
- 淬火点（把悖论推到极致）：
  - 系统并不“作恶”，它只是**在当时的规则与证据下正常运转**；每一条干预建议都讲得通，但叠加起来像围猎。
  - 许冬必须在“挽救概率”与“她仍然是人”之间，亲手把手指放到那个按钮上。
- 代价：系统必须允许被拒绝、被关闭、被误解；幸福曲线不再单调上升，社会要学会承受不适。
- **非制度瞬间（蒙太奇 Fire-05）**：许冬看着孙小莉的“自杀风险”飙红，手指悬在“批准关闭干预”键上。那一瞬，除夕夜的烟花声、手动气囊声、那句冷冰冰的“系统正常”在他耳边炸开（PTSD）。他胃部抽搐，手心冷汗，想吐。他明知批准可能就是看着她死，但他必须按——因为自由包括毁灭的自由。

### 第18章：最后一次警告（异议保护区与程序性曝光｜淬火：孙小莉“出逃声明”的认知扭曲）
