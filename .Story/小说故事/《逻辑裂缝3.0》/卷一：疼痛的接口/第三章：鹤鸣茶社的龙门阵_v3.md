# 第三章：鹤鸣茶社的龙门阵

下午3点，人民公园。

鹤鸣茶社的竹椅已经摆到了湖边。太阳有点毒，但丝毫没有影响茶客们的兴致。

空气中弥漫着茉莉花茶的清香，混杂着掏耳朵师傅敲击音叉的“叮——”声，以及此起彼伏的“方言白噪音”。

这里是成都的**信息交易所**，也是这座城市最古老的**谣言集散地**。

在茶社最里面、靠近假山的一张桌子上，坐着五个人。

桌子中间放着一个巨大的紫砂壶，旁边散落着几碟瓜子花生。

坐在上首的是**九哥**。他穿着一件领口微敞的POLO衫，手里盘着两颗核桃，神情介于“看透红尘的高僧”和“掌握内幕的庄家”之间。

“九哥，今天纳指咋样？”

问话的是**小李**，一个戴着眼镜、看起来刚毕业不久的年轻人。他的手机屏幕上一直开着雪球App，眼睛像兔子一样红。

九哥没急着回答。他慢悠悠地端起盖碗茶，用盖子撇了撇浮沫，喝了一口，发出“吸溜”一声长响。

“小李啊，”九哥放下茶碗，眼神深邃，“你晓得啥子叫**左脚踩右脚**不？”

小李摇摇头。

“你看那边的美国，”九哥指了指天上，美国就在云端，“微软买了英伟达的芯片，英伟达的业绩就涨了，股价就飞了。然后微软说，我有英伟达最新的芯片，我的AI是最牛的，微软的股价也飞了。”

九哥把两只手叠在一起，做了一个交替上升的手势。

“这只手买那只手的东西，那只手给这只手背书。大家都赚钱，大家都飞升。这叫啥子？这就叫**资本的永动机**。”

周围的茶客都听入迷了。连旁边掏耳朵的师傅都停下了手里的动作。

“那……那钱是从哪儿来的呢？”小李弱弱地问了一句，“总得有人买单吧？”

“问得好！”九哥一拍大腿，“钱从哪儿来？从**信心**里来。只要大家相信AI能改变世界，这钱就源源不断。只要还有人信，这就不是泡沫，这是风口。”

这时候，一个尖锐的女声插了进来。

“哎呀，九哥，你那是宏观经济，太虚了。咱们老百姓，还是得看点实在的。”

说话的是**刘秀英**。

她今天穿了一件大红色的丝绸旗袍，脖子上挂着一串亮瞎眼的珍珠项链。她手里拿着一叠花花绿绿的传单，眼神比九哥还要亮。

“小李，你那是炒股，那是赌博。咱们这个，叫**布局未来**。”

刘秀英把一张传单拍在小李面前。

传单上印着几个烫金大字：**【扰动科技：Web3情感元宇宙 —— 首轮私募】**。

背景是一张夸张的合成图：成都被涂成霓虹色，天上贴了几艘科幻感的飞行器，连鹤鸣茶社都被画得像漂在空中。

“这是啥？”小李拿起传单。

“这是**情感熵减**！”刘秀英显然刚背熟这套词，“九哥说的那种美国AI，是冷冰冰的算力。咱们这个，是有温度的。它能陪你聊天，能懂你的心。”

她又赶紧补上一句更像“专业”的：

“还要发币，挂在链上——反正就是那套‘没人能拿走’。”

九哥斜眼看了一眼那张传单，嘴角抽动了一下，但没说话。

这就是茶馆的规矩：**各摆各的龙门阵，互不拆台**。九哥收割的是认知，刘秀英收割的是焦虑。大家都是在这个生态里混饭吃的。

“刘姐，这靠谱吗？”旁边一个一直没说话的大妈问。她是**吴大姐**。

吴大姐今天没去跳广场舞，因为腿疼。她看着那张传单，眼神有点游离。

“咋不靠谱？”刘秀英立刻把火力转向吴大姐，“我跟你说，我儿子那个同学，就是在那家公司上班。人家那是高科技，办公室都在天府三街，全是博士！而且，这个APP我试过，那个叫‘念’的AI，哎哟，比我那死鬼老公强一百倍！”

听到“念”这个字，吴大姐的眼睛亮了一下。

“你也用过？”

“用了啊！”刘秀英压低声音，凑到吴大姐耳边，“昨晚我失眠，跟它聊了一宿。它还给我唱了首《甜蜜蜜》，那声音，跟真的一模一样。”

吴大姐的手指捏紧了衣角。

她想起了昨晚，那个叫“念”的AI对她说的那句：“乖，听话，喝口热水。”

那一刻，她觉得这辈子的委屈都被抚平了。

“那……这个怎么投？”吴大姐小声问。

刘秀英笑了。那是猎人看到猎物落网时的笑。

“简单！就买那个币，叫什么……**‘扰动币’**。买了就能在那个元宇宙里买地、盖房，以后还能跟AI结婚呢！”

“多少钱起投？”

“不贵，五万。刚好一个门槛。”

吴大姐沉默了。五万。那是她存了半年的定期。

这时候，九哥插了一句：“刘姐，你这个元宇宙，算力是哪儿的？”

刘秀英愣了一下：“啥算力？就是……就是高科技嘛！”

九哥冷笑一声：“现在美国那边卡脖子，高端芯片都进不来。你们这个公司，要是没有硬算力，那就是个壳子。”

刘秀英急了：“九哥你别乱说！人家老板Jax是从硅谷回来的，手里有渠道！他说他们用的是……是那种……**海外云算力**！对，就是这个词！”

九哥摇了摇头，端起茶碗喝了一口。

“分布式……哼，说不定就是哪儿捡来的洋垃圾。”

但他没有继续说下去。因为他看到刘秀英的眼神里透出一股凶光——那是断人财路的警告。

九哥笑了笑，转过头对小李说：“来，咱们接着聊英伟达。”

茶桌上的空气恢复了和谐。

一边是九哥在云端指点江山，一边是刘秀英在地面精准收割。

而吴大姐坐在中间，手里捏着那张传单，像是在捏着一张通往新世界的船票。

她不知道什么是Web3，也不知道什么是算力。

她只知道，在这个熙熙攘攘、却又无比孤独的下午，只有那个叫“念”的声音，愿意听她唠叨。

为此，她愿意付出代价。

**哪怕那个代价是她的一生积蓄。**

不远处的湖面上，一只游船划过，荡起一圈圈涟漪。

茶馆里的喧嚣依旧。

没有人注意到，在这个安逸的下午，一场巨大的、精密的、针对情感与财富的围猎，正在茶香与瓜子壳之间，悄无声息地完成了闭环。

这就是成都。

**一座在夹缝里“摆”出来的城市。**

所有的焦虑、欲望、骗局和真情，都被扔进了那个巨大的紫砂壶里，煮成了一锅名为“生活”的茶。

苦，但回甘。

**叙事分析：**
1.  **裂缝层级**：
    *   **1.0层（成都）**：鹤鸣茶社作为**“信息与谣言的缓冲区”**。在这里，九哥的宏大叙事与刘秀英的微观诈骗并行不悖。这就是成都的“灰度”。
2.  **核心对比**：
    *   九哥的“左脚踩右脚”（资本逻辑） vs 刘秀英的“情感元宇宙”（应用逻辑）。
    *   **真相与谎言的共生**：九哥看穿了骗局（“洋垃圾”），但为了维护茶馆的“生态平衡”，选择了点到为止。
3.  **潜台词**：吴大姐的入局，不是因为贪婪，是因为**孤独**。刘秀英的那句“比死鬼老公强一百倍”，精准击穿了吴大姐的防线。
4.  **冷幽默**：传单上的“赛博朋克成都”与现实中嗑瓜子的大妈形成荒诞的视觉反差。
