# 第15章 诊所里的泄漏 (Leak in the Clinic)

**时间：202X年8月22日（周四）上午10:00**  
**地点：原点工作室（电脑维修部）**

---

魏远江的工作室，在白天会变身为一个“电脑维修部”。
门口挂着一块手写的牌子：**“专业维修电脑、手机、打印机。不修健康码弹窗。”**

这是他唯一的合法收入来源。
虽然最近生意清淡（大家都出不来），但今天上午生意不错。
几个社区的大妈正在排队修手机。她们是趁着买菜的时间溜出来的。

“小陈啊，我这个手机咋个越来越卡哦？”李嬢嬢（昨天刚拉了老郑入局的那位）把手机递给魏远江，“是不是病毒哦？还是行程码存太多了？”

魏远江接过手机。
屏幕油腻腻的，贴膜都翘边了。
他熟练地连上数据线，打开调试工具。

“嬢嬢，你这后台开了二十几个APP，能不卡吗？”魏远江一边说，一边帮她清理进程。

突然，他的手指停住了。

在那个“短剧出海APP”的后台日志里，他看到了一行异常的数据流。
`Upload: /data/contacts -> 192.168.1.1`
`Upload: /data/sms -> 192.168.1.1`
`Upload: /data/location -> 192.168.1.1`

这个APP，不仅在偷电，还在偷数据。
它在疯狂地上传李嬢嬢的通讯录、短信和实时位置。

而且，上传的终点，还是那个内网地址。
这意味着，这些数据并没有发到国外，而是发到了……**本地的某个服务器**。

魏远江的心沉了一下。
他想起昨晚那个叫“Forest”的人发来的“罪证网”。
那张网的中心，是资金流向。
而现在，他看到了另一张网：**信息流向**。

这不仅仅是一个庞氏骗局。
这还是一个**精准信息采集器**。
他们收集这些老人的数据，是为了什么？
卖给保健品公司？卖给诈骗团伙？还是……卖给催收公司？

“小陈，好了没得？”李嬢嬢催促道，“我还要去收菜呢。”

魏远江拔掉数据线。
他看着李嬢嬢那张毫无防备的脸。
他想告诉她真相。
告诉她，她的隐私已经被那个“张总”看光了。告诉她，她的亲戚朋友可能马上就会接到诈骗电话。

但他张了张嘴，却什么也没说。
因为他知道，李嬢嬢不会信。
在她眼里，那个APP是生金蛋的鸡。谁敢动她的鸡，她就跟谁急。

“好了。”魏远江把手机递回去，“嬢嬢，以后少下点乱七八糟的东西。”

“哎呀，晓得晓得。”李嬢嬢接过手机，喜滋滋地走了。

魏远江看着她的背影。
他转过身，回到自己的电脑前。

他打开了昨晚Forest发的那个网页。
**《一场精心策划的掠夺》**。

现在，他要在上面加一个新的图层：**《被窃取的隐私》**。

他开始编写一个新的脚本。
这个脚本的功能很简单：**反向追踪**。
既然那个APP在上传数据，那他就可以顺着这条数据流，找到那个**物理服务器**的位置。

只要找到了服务器，就能找到老K的老巢。

屏幕上，一个绿色的光点开始在地图上移动。
它穿过天府大道，穿过绕城高速，最后……
停在了地图上的一片空白区域。
**云服务器。节点动态跳变。**
上一秒在新加坡，下一秒在爱尔兰。
他根本定不了位。

魏远江看着那个不断跳动的绿点。
他感到的不是兴奋，而是**面对庞然大物的无力感**。

“原来，恶魔住在云端。”
他自言自语。

就在这时，小陈（外卖员）推门进来，手里提着两份盒饭。
“师父，吃饭了！今天有红烧肉！”

魏远江关上屏幕。
“好，吃饭。”

他接过盒饭，大口吃了起来。
红烧肉很香。但他吃了三口，发现自己**尝不出味道**。

“小陈，”魏远江突然问，“这肉咸吗？”
“咸啊！师傅你口味重？”小陈奇怪地问。

魏远江心里咯噔一下。
味觉丧失。
这是……那个症状？

他放下筷子，看着窗外的空调外机。
一只麻雀正在上面筑巢。

他不知道自己是不是中招了。
如果是，那这个工作室就完了，连带这一栋楼都要被封控。
ASTO的源代码还在本地服务器上。

“没事。”魏远江重新拿起筷子，“可能是太累了。”

他决定，今晚就开始把代码往分布式节点上备份。
不管是不是病毒，系统都必须有容灾备份。

他不知道那是哪里。
但那是**离他很远**的地方。
