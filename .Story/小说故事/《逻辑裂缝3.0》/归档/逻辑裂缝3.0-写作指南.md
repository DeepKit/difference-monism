## 1. 灵魂级戒律 (The Soul)

### ⚠️ 最高指令：不要证明，只展示
*   **Don't**：证明"漏洞是对的"。
*   **Do**：一次次展示"没有漏洞，人会死"。
*   **Don't**：让小说成为ASTO的说明书。
*   **Do**：让ASTO成为人物在绝境中随手抓起的一块砖头。

### 🚫 三大禁令
1.  **禁止魏远江说教**：智慧体现在"不做什么"（不黑进银行、不评判美丑），而不是"说了什么"。
2.  **禁止林晓总结**：不需要独白说"我代表受压迫者"。只需要写出她的痛（救护车上的电话、吞私钥时的哽咽）。
3.  **禁止强行升华**：结局不要乌托邦。骗子还在换地方，系统还在升级。**只要老郑的花园还剩一半，就是胜利。**

### ✅ 三大触感法则
1.  **思想动作化**：
    *   "反完美主义" → **老郑留下那棵歪脖子树**。
    *   "系统无情" → **催收短信每30秒跳一条，精准如心跳**。
    *   "人性光辉" → **行长按指纹时，手抖留下的模糊红印**。
2.  **细节歧义化**：小陈的路是捷径还是陷阱？马桶花盆是艺术还是垃圾？**不给标准答案**。
3.  **失败者尊严**：写出被定义为"废品"的人，如何**像人一样站立**的瞬间。

### ⚡ 核心原则修正
*   **结局作为暂存区**：已修改，未提交，未推送。不是拒绝结局，而是**诚实地保持开放**。
*   **章节标题双声道**：奇数章用**身体性标题**（暴雨中的螺丝刀），偶数章用**代码标题**（Error 503），形成系统与肉身的对撞。
*   **文明的分形结构**：**从膝盖到星尘，四层同构**。
    *   **宇宙层**：地球在虚空中带着偏差运行了45亿年。宇宙没给地球发"宜居许可证"，地球只是偶然地、带着错误地存活。
    *   **国家层**：中国在世界的技术封锁、规则禁令、话语权围剿中，在裂缝里自主创新。被定义为"发展中国家"却要重新定义"发展"本身。中国是世界上最大的"老郑"。
    *   **城市层**：成都在严格管控的缝隙中保留了麻将声。红线里的麻将，静默期的葱。不写对抗，写"带着黄码活下去"。
    *   **个人层**：老郑在被冻结账户旁种草药。小陈在碎屏的雨夜里奔跑。林晓在吞私钥的哽咽中完成反杀。
    *   **贯穿线索**："带着错误运行"不是无奈的妥协，而是**存在本身的方式**。
*   **时空胶囊**：**2020-2022 成都**。大喇叭喊核酸，背景音是麻将声。健康码变色，生活继续运转。被系统判定的风险，被肉身承受并转化。

## 2. 叙事线索与人物重构

### 2.1 线索一：技术哲学与纯爱（大脑与心）
*   **魏远江**：
    *   **定位**：ASTO的**传道者**而非发明者（源自《思维越狱》）。
    *   **动作**：在茶馆用筷子蘸水画图，而不是在全息屏上演讲。
    *   **感情**：与林晓的**纯爱救赎**。不是"一见钟情"，而是**"认出"（Recognition）**。林晓通过代码想象了一个完美的他，却发现真实的他更破碎、更真实。
    *   **关键**：日内瓦演讲是**失败**或**被误解**的，唯一的亮点是他拿出的**"电路板标本"**（长在废弃电子垃圾上的野草）。
*   **林晓 (Lin Xiao)**：
    *   **定位**：**孝心驱动的分裂者**（删除"伪精英"）。能力越强，作恶越高效的现代性张力。
    *   **背景**：2017年起家庭变故（母瘫痪妹照料，父癌需50万）。为救命钱加入老K的Web3/DAO公司。
    *   **铺垫**：从卷一开始铺垫她**咀嚼代码/数字**的习惯，为最后**吞私钥**做生理准备。
    *   **关系**：自卑于魏远江的"干净"，但渴望被救赎。

### 2.2 线索二：全景骗局与灰色生存（脊柱）
*   **骗局矩阵**：
    *   **Web3/DAO**（林晓）：用"去中心化"包装的集权掠夺。
    *   **短剧/黑茶**（李嬢嬢）：理直气壮的"国家战略"推广。
    *   **套路贷**（阿强）：从暴力催收（泼油漆）到软暴力（AI外呼、爆通讯录）。
*   **反派众生相**：
    *   **老K**：冷血资本家。结局不是被抓，而是**金蝉脱壳**，摇身一变成为"招商引资重点客户"。罪恶升级。
    *   **阿强**：底层催收员。反水不是因为方姐下跪，而是因为方姐**站起来走了**——这种"权力的真空"让他崩溃。

### 2.3 线索三：万千园与市井烟火（血肉）
*   **核心场景**：**"万千园之城"**（千园PK季）。
    *   **机制**：还政于民，资金直达社区。ASTO负责**"管钱不管人"**（防刷票、审计）。
    *   **资金来源**：**暧昧性**。可能是P2P爆雷的罚没款，或者是企业洗白的钱。**钱不干净，但地是真的，花是真的。**
    *   **图腾**：**马桶花盆**。审美之争的核心，对抗标准化景观的武器。
*   **人物**：
    *   **老郑**：在被骗光积蓄后，通过种草药找回尊严。
    *   **小陈**：外卖骑手，在"骑士驿站"花园获得喘息。
    *   **方姐**：银行卡冻结事件主角。单亲妈妈，孩子重病。

### 2.4 线索四：系统的缝隙与文明的韧性
*   **赵警官（民警）**：
    *   **定位**：**系统的翻译者**与**黄码世界的陪伴者**。
    *   **高光时刻**：**马路牙子办公**。因健康码变黄无法进门，在雨中搬椅子坐路边为老郑办业务。背后的新闻播报国家被封锁。
    *   **隐喻**：如果系统不发绿码，我们就带着黄码活下去。
*   **小吴（网格员）**：
    *   **定位**：**微小权力的拥有者**与**静默期的保供者**。
    *   **高光时刻**：**全城静默下的一根葱**。本该铲除的花园，在封控期成为全楼的葱源。她选择"看不见"，并组织分发。
    *   **隐喻**：在寸草不生的时候，我们在墙角种出一点点春天。

## 3. 全书分卷大纲 (Logic Rift 3.0)

### 卷一：疼痛的接口 (Pain Interface)
*   **核心任务**：建立痛感，铺垫背景（2020-2022 成都），植入ASTO氛围。
*   **关键章节**：
    *   第1章：**暴雨中的螺丝刀与绿码**（小陈出场。暴雨淋湿屏幕扫不出码，进不去小区，比超时更绝望）。
    *   第2章：**Error E6: Comm Fail**（老郑出场。**赵警官首次出场，疲惫地解释冻结，背景音是大喇叭喊核酸**）。
    *   第3章：**林晓的账单**（林晓出场，家庭重担与疫情封控的双重挤压）。
    *   第4-10章：**骗局的铺开**（黑茶、短剧、Web3初现）。
    *   第11-15章：**ASTO的氛围**（民间对"容错"的误读与实践，魏远江在茶馆的微小扰动）。

### 卷二：逻辑的灰度 (Gray Logic)
*   **核心任务**：骗局深入，情感萌芽，ASTO显影。
    *   第16-20章：**相亲与见面会**（林晓的自卑，魏远江的"认出"）。
    *   第21-25章：**老郑的崩盘**（算力币爆雷，老郑绝望，马桶花盆初现）。
    *   第26章：**README.md (uncommitted)**（小雪的生存清单，ASTO灵魂确立）。
    *   第27-30章：**套路贷的獠牙**（阿强的视角，软暴力催收。**赵警官再次出场，关掉屏幕，教老郑"物理断网"**）。

### 卷三：系统的反噬 (System Backlash)
*   **核心任务**：矛盾总爆发，人性高光。
    *   第31-35章：**万千园的兴起**（千园PK，大妈拉赞助。**小吴登场，全城静默期间，保护方姐的葱，成为全楼的生命线**）。
    *   第36-40章：**Timeout Exception**（救护车上的十个电话，方姐银行卡冻结。**赵警官在雨中马路牙子办公，背景新闻播报国家遭遇技术封锁**）。
    *   第41-45章：**行长的指纹**（人性战胜规则）。
    *   第46-50章：**林晓的觉醒**（发现DAO的后门。**看着新闻里的国家突围，感叹"原来国家也是个老郑"**）。

### 卷四：带着错误运行 (Running with Errors)
*   **核心任务**：不完美的结局，韧性的生存。
    *   第51-60章：**苏黎世的反杀**（林晓吞私钥，狼狈但决绝）。
    *   第61-65章：**日内瓦的误读**（演讲失败，但电路板标本留下了痕迹）。
    *   第66-70章：**阿强的反水**（面对沉默的崩溃）。
    *   第71-73章：**// TODO: Fix in next release**（结局：老K变身。**赵警官与小吴在宣传栏前对视，确认"系统里大家都挺幸福"，然后继续修补系统**）。
