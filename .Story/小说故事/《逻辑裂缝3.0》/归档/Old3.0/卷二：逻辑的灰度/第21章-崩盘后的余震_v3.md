# 第21章 崩盘后的余震 (Aftershocks)

**时间：202X年8月26日（周一）上午09:00**  
**地点：原点巷子 -> 废品回收站**

---

老郑开始捡废品了。
不是为了环保，是为了活下去。

那7000块没了。他的存折里只剩下50块。
下个月的房租、水电、豆豆的伙食费，都成了悬在头顶的刀。

他每天早上五点起床，背着那个最大的编织袋，去附近的垃圾桶翻找。
纸板、塑料瓶、易拉罐。
他学会了怎么把纸箱踩扁，怎么把矿泉水瓶里的水倒干净。

“老郑，你也来啦？”
在回收站门口，他遇到了李嬢嬢。

李嬢嬢看起来老了十岁。
头发白了一半，以前常穿的那件红旗袍不见了，换成了一件灰扑扑的旧外套。
她手里提着一袋子快递纸箱。

“李妹儿……”老郑想打招呼，但嗓子有点哑。

李嬢嬢看到老郑，眼神躲闪了一下。
“张哥，我……我对不起你。”

“不说那些了。”老郑摆摆手，“都过去了。”

他看着李嬢嬢手里的纸箱。
“这些……能卖多少钱？”

“三毛五一斤。”李嬢嬢叹了口气，“这一袋子，也就两块钱。”

两块钱。
以前李嬢嬢在茶馆里，一壶茶就是三十块。
那时候她谈论的是“美元分红”、“国家战略”。
现在，她的世界缩水成了“三毛五一斤”的废纸。

“张哥，你恨我吗？”李嬢嬢问。

老郑沉默了一下。
他看着李嬢嬢那双粗糙的手，上面全是灰尘和划痕。
他想起了那个在派出所门口哭得瘫在地上的女人。

“不恨。”老郑说，“咱们都是……韭菜。”

他从兜里掏出一瓶水，递给李嬢嬢。
“喝口水吧。天热。”

李嬢嬢接过水，眼泪又流了下来。
“张哥，我儿子……昨天又来要钱了。他说我不给钱，他就去跳楼。我没办法啊……”

老郑心里一紧。
原来，每个人的背后，都有一个无底洞。
李嬢嬢拉人头，不是为了自己享福，是为了填那个洞。

“别怕。”老郑拍了拍她的肩膀，“只要人还在，就有办法。”

两人并排坐在回收站的台阶上。
脚下是成堆的垃圾。
头顶是灰蒙蒙的天空。

“李妹儿，那个马桶花盆，你看到了吗？”老郑问。

“看到了。”李嬢嬢擦了擦眼泪，“就在巷子口那个废墟里。种的啥子？”

“薄荷。”老郑笑了，“等长出来了，我给你摘点。泡水喝，清火。”

李嬢嬢也笑了。
“那个马桶……挺好看的。”

在那一刻，两个被系统抛弃的老人，在垃圾堆旁达成了一种奇怪的和解。
他们没有钱，没有尊严，没有未来。
但他们有彼此。
还有那个种在马桶里的希望。
