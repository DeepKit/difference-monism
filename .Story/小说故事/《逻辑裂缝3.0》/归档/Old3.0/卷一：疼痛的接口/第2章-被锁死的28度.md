# 第2章 被锁死的28度

**时间：202X年8月14日（周三）下午18:30**  
**地点：老郑的廉租房（成都市二环路内某老旧小区）**

---

老郑回到家的时候，浑身都已经湿透了。
但他感觉不到冷。因为他的心比身上更冷。

半小时前，他在派出所。

因为经过了一个风险区，他的健康码变黄了，连派出所的大门都没进去。
那位赵警官——人挺好，但也没办法——搬了把椅子，坐在大门口淋着雨给他查了半天。

最后告诉他一个消息：
“老郑，你的卡被‘反诈中心’冻结了。属于二级关联账户。系统自动锁的。”

“那……那咋个解嘛？”老郑当时在雨里问。

“难。”赵警官关掉了那个闪着红字的屏幕，像是想帮他省点电，又像是想帮他省点绝望，“系统判定的，解封要走流程，起码半年。而且……你这笔钱，可能已经被划走了。”

老郑不知道自己是怎么走回来的。
他的手里攥着那张湿漉漉的报警回执，就像攥着一张废纸。

推开家门。
“爷爷回来啦！”
小孙子豆豆从里屋跑出来，手里拿着一个变形金刚玩具。

老郑强挤出一个笑脸。他不能让豆豆知道，爷爷的钱没了，爷爷被系统判定为“风险”了。

“哎，慢点跑。”老郑摸了摸孙子的头，“今天作业做完没有？”

“做完了。但是爷爷，好热啊。”豆豆指了指墙上的空调，“它不转了。”

老郑抬头。

那是一台老式的格力空调，外壳已经泛黄。此刻，导风板紧紧闭着，像是睡着了。

屋里闷热得像个蒸笼。虽然外面在下暴雨，但这这种老房子的保温层太差，白天的热气全憋在里面出不去。湿度表显示：**32℃，85%**。

“咋个回事嘛？”

老郑拿起遥控器，按了一下“开”。

“滴”。

空调响了一声，导风板缓缓打开。冷气刚刚吹出来，还没碰到豆豆的脸，就听见“滴滴滴”三声急促的报警音。

导风板又关上了。

显示屏上跳出一个红色的代码：**E6**。

老郑不懂什么是E6。他只知道，如果不修好，今晚豆豆肯定睡不着觉，明天上学就会打瞌睡。

他拿出手机，熟练地打开了一个名为“全能维修”的APP。这是一个在公交车上遇到的大学生教他下的，说是现在修东西都用这个，价格透明，师傅正规。

老郑眯着眼睛，在屏幕上戳戳点点。

**【故障报修：空调不制冷】**
**【预约时间：立即】**

提交。

屏幕上转了一个圈。

**【系统提示：当前区域订单量过大，排队人数：128人。预计上门时间：明天上午10:00。】**

“明天上午？那今晚咋个办？”

老郑急了。他又点了一次“加急”。

**【加急服务：需支付“高峰调度费”50元。是否支付？】**

50元。
老郑的手抖了一下。
如果是以前，他舍不得。但今天，他刚在派出所听说那几十万都没了，这50块钱反而显得像个笑话。

“付！”
老郑咬牙点了“支付”。
输入密码。
指纹确认。

屏幕转了两圈。

**【支付失败：您的银行卡已被冻结。错误代码：FROZEN_ERR_09】**

老郑愣住了。
他死死盯着那个红色的单词。

在派出所，那是赵警官嘴里的一句解释。
在家里，这变成了豆豆头上的汗珠，变成了这台死活不转的空调。

原来“冻结”不是说钱动不了。
是说你的生活，从此**寸步难行**。

他想换微信支付。
**【余额不足】**。微信里的钱昨天刚提现到卡里，准备凑给豆豆交择校费。

全都被锁在那个该死的系统里了。

“爷爷，我热……”豆豆开始哭闹，小脸通红。

老郑放下手机，手无力地垂下来。
手机滑落在凉席上，发出沉闷的响声。

他从床底下翻出一把蒲扇。

“不哭不哭，爷爷给你扇。”

这就是系统的逻辑。当需求（暴雨天电器故障高发）远远大于供给（师傅数量有限）时，价格杠杆失效了。系统直接进入了**“熔断模式”**。

老郑不甘心。他想起了以前小区门口贴满的小广告。那时候，只要打个电话，楼下的王师傅提着工具箱就来了，给根烟抽，二十块钱就能搞定。

但现在，楼下的墙壁刷得雪白。那些小广告早就在“文明城市创建”中被铲干净了。

王师傅呢？
听说王师傅因为没有“特种作业操作证”，被APP平台封号了，回老家种地去了。

现在，老郑手里只有这个APP。
这个界面精美、流程规范、但他妈的就是**叫不来人**的APP。

“爷爷，我热……”豆豆开始哭闹。

老郑放下手机，从床底下翻出一把蒲扇。

“不哭不哭，爷爷给你扇。”

他坐在床边，一下一下地摇着蒲扇。
风很小，但这风是**真实**的。

他看着那个显示着“E6”的空调。
E6是什么意思？
后来魏远江告诉他，E6是**“通讯故障”**。是室内机和室外机“失联”了。

多讽刺啊。
在这个万物互联的时代，机器和机器失联了，人和人也失联了。
老郑想找个修空调的人，中间隔着APP、隔着算法、隔着排队系统、隔着加急费。
最后发现，谁也找不着。

他只能靠这把破蒲扇。

这就是**被锁死的28度**。
系统承诺给你恒温的幸福，但当系统崩溃时，你连打开窗户呼吸的权利都没有（因为窗户被防盗网封死了）。

老郑摇着扇子，看着窗外的暴雨。
雨水打在防盗网上，碎成一片片白雾。

他突然想起那个大学生跟他说的话：
“大爷，以后我们要搞那个什么……元宇宙。在里面，你想住大别墅就住大别墅，想吹空调就吹空调，永远不坏。”

老郑当时笑了笑，没说话。

现在他想：
**那个元宇宙里，有蒲扇吗？**
如果没有，那万一停电了，是不是连风都没有了？

豆豆终于睡着了。
老郑的手腕酸得不行。
他停下来，给自己倒了一杯凉白开。

喝水的时候，他看到手机屏幕亮了一下。
是一条短信。

**【反诈中心提醒：您的账户存在异常交易，已限制非柜面交易。请携带相关证件前往开户行核实。】**

没有退款短信。
因为根本没支付成功。

老郑看着那条短信，只觉得浑身发冷。
这种冷，比空调管用多了。

他放下杯子，重新拿起蒲扇。
一下，两下。

在这座拥有两千万人口、千亿级算力、无数智能终端的超级城市里，
这把蒲扇，成了老郑维护孙子睡眠的**最后一手**。
