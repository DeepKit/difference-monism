# 第8章 便利店里的阈值切割 (Threshold Cutting)

**时间：202X年8月18日（周日）凌晨01:30**  
**地点：中和场，7-11便利店**

---

凌晨一点半的便利店，是城市的“深夜食堂”。
这里有刚下夜班的程序员，有准备出摊的早点摊主，还有像魏远江这样昼夜颠倒的“修仙者”。

魏远江推门进来的时候，收银台后面的小雪正在犯困。
她头一点一点的，像只啄米的小鸡。脸上的口罩滑下来了一半，露出了稍显苍白的嘴唇。

“欢迎光临。”
她被门铃声惊醒，条件反射地喊了一句，声音有点哑。赶紧把口罩拉上去，遮住大半张脸。

魏远江走到柜台前，指了指后面的烟架：“一包红塔山。软的。”

小雪揉了揉眼睛，先拿起桌上的测温枪，对着魏远江的手腕“滴”了一下。
“36度5。正常。”
然后转身去拿烟。
她的动作很慢。因为她的手腕上缠着厚厚的肌内效贴布（膏药）。
这是长期扫码留下的职业病——腱鞘炎。

“一共11块。”小雪把烟放在柜台上，拿起扫码枪。

“滴。”
支付成功。

魏远江拿起烟，刚要走，停住了。
他的目光越过小雪的肩膀，落在了收银机的背面。

那里贴着一块巴掌大的硬纸板。
纸板很粗糙，是从废弃纸箱上剪下来的，边缘还带着毛刺。
上面用黑色记号笔写着几行字，字迹歪歪扭扭，但很用力。

**【小雪的生存算法 v1.0】**

1. **允许报错**
   *（如果今天扫码慢了，或者找错钱了，不要觉得自己是个废物。机器才不犯错，人是会犯错的。）*

2. **不要优化掉“没用”的东西**
   *（比如方姐的玫瑰花梦，比如小李的创业冲动，比如下班路上那一分钟的晚霞。那些东西没用，但是那些东西让我们活着。）*

3. **相信随机性**
   *（就像相信雨。有时候淋湿不是倒霉，是天空想摸摸你。）*

4. **善良 > 正确**
   *（如果逻辑告诉我应该推开那个求助的老人，那就把逻辑扔了。因为心会痛，逻辑不会。）*

魏远江盯着那块纸板，看了很久。
久到小雪开始觉得不自在。

“那个……”小雪脸红了，试图用身体挡住那块纸板，“那个是我乱写的。”

魏远江抬起头，看着小雪。
他的黑眼圈很重，眼神里总是带着一种看透一切的疲惫。但此刻，那疲惫里裂开了一道缝，透出一点光来。

“写得很好。”魏远江说。

“啊？”小雪愣住了。
平时来店里的客人，要么催她快点，要么嫌她动作慢。从来没有人夸过她写的字。

“我是做系统的。”魏远江指了指那块纸板，“我们总是想把系统做得完美，把所有错误都杀掉。但你这里写着‘允许报错’。”

他笑了笑，拆开烟盒，抽出一支烟，放在鼻尖闻了闻。
“这比我写的代码高级。”

小雪更不好意思了：“我……我不懂什么代码。我就是觉得，如果不这么想，我就快变成这个扫码枪了。”

她举起手里那把黑色的扫码枪。
“滴、滴、滴。每天几千次。有时候我做梦都在听这个声音。”

魏远江点点头。
他太懂那种了。他虽然不扫码，但他每天敲击键盘的声音，也是另一种“滴滴滴”。

“你叫什么名字？”魏远江问。

“小雪。”

“我叫魏远江。”
他从口袋里摸出一支马克笔——那是他随身带着用来画草图的。
他拿起笔，悬在那块纸板上，想写点什么。
但最后，他什么也没写。

他把笔收了回去。
“如果我有团队，”魏远江看着小雪，眼神很认真，“我会把你这几条规则挂在墙上。这比我见过的所有管理学都管用。”

他拿着烟走了。
自动门“叮咚”一声关上。

小雪看着那块纸板。
在歪歪扭扭的“小雪的生存算法”下面，虽然没有多出任何字，但她觉得那个叫魏远江的人，像是看懂了她的秘密。

她不知道的是，就在转身出门的那一刻，魏远江走出便利店，发现自己忘了买打火机。
他也没回去。
他不记得自己笑了，但脸上的肌肉是酸的。

这一夜，ASTO的灵魂，在一个24小时便利店的收银机背后，被一个女孩重新定义了。
不是在实验室，不是在硅谷，而是在这充满了关东煮味和烟味的市井里。
