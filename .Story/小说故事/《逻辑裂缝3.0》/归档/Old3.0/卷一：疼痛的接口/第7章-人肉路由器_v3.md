# 第7章 人肉路由器 (Sneakernet)

**时间：202X年8月17日（周六）上午11:30**  
**地点：天府三街写字楼 -> 华西医院**

---

小陈的膝盖还没好利索，但他不能停。
昨天那单差评扣的15块钱，加上手机屏幕的维修费，让他这个月的“生活线”又逼近了一步。

他接到了一个“跑腿单”。
**备注：文件急送。从天府三街XX大厦送到华西医院。不可接触配送。**
**打赏：20元。**

这是一个肥单。
小陈想都没想就抢了。

到了写字楼楼下，保安拦住了他：“外卖不准进客梯，去走货梯。对了，把行程卡打开看一眼。”
小陈熟练地打开行程卡，绿色的箭头在碎裂的屏幕下显得有点扭曲。

“进去嘛。口罩戴好。”保安挥挥手。

货梯门口排着长队，全是送餐的骑手。大家保持着一米的距离，沉默得像一群等待被运送的货物。

小陈看了看时间，咬牙给下单人打了个电话。
“你好，我是骑手。保安不让上楼，能不能麻烦您下来拿一下？”

电话那头是一个女人的声音，很疲惫，也很冷漠：“我走不开。你想办法上来。不然投诉你。”
嘟。挂了。

小陈握着手机，愣了两秒。
他想骂人。但他不敢。
因为他知道，对方动动手指给个差评，他这一天又白干了。

他只能去挤货梯。
电梯里全是各种饭菜的味道，混着汗味。
旁边一个骑手问他：“哥们，腿咋了？”
“没事，磕了一下。”小陈把重心移到右腿上。

到了36层。
前台没人。
他按照备注找到了那间办公室。
那是一个巨大的、没有隔断的办公区。几百个工位密密麻麻地排着，每个人都对着电脑，像养鸡场里的鸡。

“谁是……林小姐？”小陈小声问。

没人理他。
大家都在忙着敲键盘，或者盯着屏幕上的K线图。

终于，一个角落里的女人抬起头。
她穿着职业装，但头发有点乱。桌上堆满了文件和吃剩的外卖盒。
是林晓。

“这里。”她招了招手。

小陈走过去。
“您的跑腿单。”

林晓抬起头。
她的目光落在小陈身上，停顿了一秒。
她看到了他还在滴水的裤腿，还有那个即使隔着裤子也能看出来的、肿胀的膝盖轮廓。
那种“带着伤还在跑”的姿态，让她忽然想起了一个人。
那是三年前的她自己。
那时候，她刚做完阑尾炎手术第三天，为了不被扣全勤奖，捂着肚子去赶地铁。在早高峰的拥挤中，她也是这样，用一条腿支撑着身体，咬着牙不让自己倒下。

那一瞬间的重叠，让她冷漠的面具裂开了一道缝。

“送到华西医院神经内科32床。交给那个叫林雨的女孩。告诉她，这是……保险单。”

小陈接过文件袋。很沉。
他看了一眼林晓。
这个女人看起来很光鲜，但他闻到了一股味道。
不是香水味。
是**熬夜的味道**。那是长时间不洗澡、靠咖啡续命、焦虑发酵出来的酸味。
这味道他太熟悉了。他在网吧包夜的大神身上闻到过，在他自己身上也闻到过。

“好的。”小陈说。

他转身要走。
，林晓叫住了他。
“等一下。”

她从抽屉里拿出一瓶红牛，递给他。
“外面热，喝吧。”

小陈愣住了。
刚才在电话里还要投诉他的女人，现在居然给他水喝？

他接过红牛。冰凉的罐体贴在手心，那种冷意顺着血管传到心脏，让他打了个激灵。
“谢……谢谢。”

---

去医院的路上，小陈一直把那个文件袋抱在怀里。
他不知道里面装的是什么。
也许是几百万的合同，也许是救命的药。

但自己像一根光纤。
或者说，像一个**路由器**。

在这个城市里，有些东西是互联网传不过去的。
比如这瓶红牛的温度。
比如那个文件袋的重量。
比如那种“我也很难，但我也想对你好一点”的微妙情绪。

这些东西，只能靠人来传。
靠他这种断了一条腿、骑着破电动车、在烈日下流汗的人来传。

他是**Sneakernet**（人肉网络）。
他是这座赛博城市里，最后一条物理宽带。

到了医院。
他把文件袋交给了那个叫林雨的女孩。
女孩在哭。
“姐……她还在加班吗？”

小陈点了点头：“她在忙。她让你放心。”

女孩擦了擦眼泪，从包里拿出一颗糖。
“哥，吃糖。辛苦了。”

小陈捏着那颗糖，走出医院大门。
外面阳光刺眼。
他剥开糖纸，把糖塞进嘴里。
甜的。

膝盖没那么疼了。
他骑上车，给魏远江发了一条微信：
“师父，我今天不想骂人了。我觉得我挺有用的。”

魏远江回了一个字：
“嗯。”
