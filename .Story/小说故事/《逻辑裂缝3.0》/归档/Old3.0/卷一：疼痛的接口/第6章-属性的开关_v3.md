# 第6章 属性的开关 (User Profile Switch)

**时间：202X年8月16日（周五）晚上20:00 -> 21:30**  
**地点：老K科技公司 -> 华西医院**

---

晚上八点，公司的中央空调准时停机。
这是老K定下的规矩：“加班可以，但电费得省。真正的奋斗者，心里有火，不需要空调。”

林晓坐在闷热的办公室里，额头上渗出一层细密的汗珠。
她的妆有点花了，粉底卡在鼻翼两侧，像一道裂纹。

**当前属性：【技术总监】**
*   **任务**：审核“DAO治理代币”的空投合约。
*   **目标**：确保没有明显的漏洞，同时确保留有一个极其隐蔽的“超级管理员权限”（Backdoor）。
*   **状态**：执行中。

屏幕上是几千行Solidity代码。
林晓的手指在键盘上飞快地敲击，像是在弹奏一首无声的钢琴曲。

`function emergencyWithdraw() public onlyOwner { ... }`

她盯着那个 `onlyOwner` 修饰符。
这就是那个“后门”。
在白皮书里，这个函数被解释为“应对黑客攻击的紧急机制”。
但实际上，这就是老K跑路时的“收割镰刀”。

林晓感到一阵恶心。
她从抽屉里拿出一块巧克力，撕开包装纸，塞进嘴里。
狠狠地嚼。

**咀嚼动作检测：**
*   **咬合力**：25kg
*   **频率**：3Hz
*   **目的**：缓解焦虑，抑制呕吐感。

这是她最近养成的习惯。每当写下一行违心的代码，她就要吃一块硬东西。巧克力、坚果、甚至冰块。她需要用牙齿的痛感，来压住胃里的翻腾。

“林总，代码还没审完吗？”
老K推门进来，满身酒气。他刚从一个“高端商务局”回来，脸上带着那种成功人士特有的油腻红光。

“快了。”林晓没抬头，“有个溢出风险要修一下。”

“修什么修！”老K走过来，拍了拍她的椅背，“只要能跑通就行。那些韭菜又看不懂代码。他们只看白皮书上的‘年化收益300%’。”

他凑近林晓，压低声音：“晓晓，这次发币之后，你的提成够你在天府新区付个首付了。好好干，别想那些没用的。”

林晓的手指僵了一下。
首付。
不，她不要首付。她要的是那50万的手术费。

“知道了。”她咽下嘴里最后一点带苦味的巧克力，“K总放心。”

老K满意地走了。
林晓看着屏幕。
光标在那个 `emergencyWithdraw` 函数上闪烁。

她深吸一口气，在函数的注释行里，加了一行看似乱码的字符：
`// 0x48656c70206d65`

这是十六进制。
翻译过来是：**Help me.**

这行代码不会被执行，甚至不会被大多数人注意到。
它就像是一个溺水者在深海里吐出的一个小气泡。
无声，无用，但真实存在。

提交。
推送。
关机。

---

**21:30。华西医院，神经内科住院部。**

林晓走进病房前，先去了一趟洗手间。
她对着镜子，熟练地补妆。
擦掉鼻翼的卡粉，涂上显气色的口红，理了理头发。然后，她拿出包里的一瓶酒精喷雾，从头到脚喷了一遍——虽然医院门口已经消杀过了，但她怕把外面的“风险”带给免疫力低下的妈妈。

“茄子。”

那个微笑很完美。温柔，自信，充满希望。

**属性切换：【技术总监】 -> 【孝顺女儿】**

她推开病房的门。

妈妈躺在床上，半边身子不能动，嘴角歪斜，口水顺着下巴流下来。
妹妹正在给妈妈擦身子，累得满头大汗。口罩挂在下巴上，勒出一道深痕。

“姐，你回来啦。”妹妹看到她，眼睛亮了一下，“吃饭了吗？”

“吃过了。”林晓撒谎，“公司聚餐，吃的海鲜。”

她走到床边，握住妈妈那只还能动的手。
“妈，今天怎么样？”

妈妈看着她，嘴里发出含糊不清的“啊啊”声。
但眼神是亮的。那是看到“全家希望”时的眼神。

“姐，医生今天又催费了。”妹妹小声说，“爸那边的靶向药也快吃完了。”

林晓脸上的微笑没有崩。
就像是一个训练有素的AI，无论输入什么负面数据，输出永远是正向反馈。

“没事，钱的事你别管。”林晓从包里拿出一个信封，递给妹妹，“这是两万块。先交上。”

这是她透支了三张信用卡套出来的。

“姐，你太厉害了！”妹妹接过钱，崇拜地看着她，“大公司就是不一样。”

林晓看着妹妹那双清澈的眼睛。
她想起了老K那双贪婪的眼。
想起了茶馆里李嬢嬢那双狂热的眼。

这个世界，是不是只有骗子和傻子？
而她，夹在中间，是个**清醒的帮凶**。

“妈，我要去趟苏黎世出差。”林晓一边给妈妈按摩僵硬的手臂，一边轻声说，“公司派我去……学技术。回来就能升职加薪了。”

妈妈虽然听不懂“苏黎世”，但听懂了“升职加薪”。她激动地拍了拍床单，另一只歪斜的手试图竖起大拇指。

林晓看着那只扭曲的大拇指。
她的心像被那把生锈的螺丝刀搅了一下。

她低下头，把脸埋在妈妈的掌心里。
那只手很粗糙，有老人味，有消毒水味。
但那是**真的**。

比区块链上的Hash值，比账户里的数字，比老K的承诺，都要真一万倍。

“妈，等我回来。”
她在心里说。
如果我能回来的话。
