# 第1章 Error 503: Knee Not Found

**时间：202X年8月14日（周三）下午17:30**  
**地点：成都，天府三街 -> 中和场**
**环境：暴雨，全城常态化防控中**

---

暴雨是在下午五点半准时落下的。

就像是有人在云端输入了一行 `weather.set(RAIN, HEAVY)`，连过渡的阴天都没有，天空直接黑成了一块死屏。紧接着，水像鞭子一样抽了下来。

在这个特殊的年份，暴雨和**“弹窗”**一样，总是来得猝不及防。

小陈的电动车停在天府三街的十字路口。口罩被雨水打湿，死死地贴在嘴上，让他呼吸困难。

雨水顺着头盔的缝隙流进脖子里，又冷又黏。但他没空擦。他的眼睛死死盯着固定在车把上的手机屏幕。

屏幕上，外卖平台的接单界面正在疯狂闪烁。

**【系统派单】**
*   **取餐**：喜茶（环球中心店）
*   **送达**：中和新村3号院
*   **距离**：4.2km
*   **限时**：35分钟
*   **预计收入**：6.5元

**倒计时：29s... 28s...**

小陈的手指悬在“接受”按钮上，停住了。

他的左膝盖正在隐隐作痛。那种痛不像撞击，像是有把生锈的螺丝刀插在骨缝里，每转动一下轮子，螺丝刀就跟着转一圈。

这是“职业病”。也是系统留在他身上的第一个**Bug**。

“接不接？”

如果不接，今天的活跃度就会掉，明天的派单权重大降。
如果接了，中和场那边的路全是积水，一旦摔了，这一周都白干。

**系统提示：【暴雨天气，配送费已上调1.2元】**

1.2元。
这刚好是一个馒头的价格。

小陈咬了咬牙，点了接受。

“这一单跑完就收工。”他对自己说。这已经是他今天对自己说的第八次了。

---

电动车冲进雨幕。

天府大道的路面很好，柏油黑得发亮。两侧的写字楼像巨大的发光墓碑，矗立在雨雾中。

小陈骑得很快。他的大脑里有一张比高德地图更精确的地图：
哪里有积水坑，哪里红绿灯读秒快，哪里的保安不拦外卖车。

这是**“肉身算法”**。

但今天，算法失效了。

刚拐进中和场的小巷子，一辆黑色的奥迪车变道。

没有转向灯。

小陈猛捏刹车。

轮胎在湿滑的路面上发出一声刺耳的尖叫。车身侧滑，像一块失控的肥皂。

“砰！”

并没有撞上奥迪。小陈连人带车摔进了路边的水坑里。

奥迪车停都没停，径直开走了。溅起的泥水泼了小陈一身。

小陈趴在水里，第一反应不是疼，而是去看手机。

屏幕碎了。
但还在亮。

**【订单即将超时，剩余时间：8分钟】**

还好，还好。

他试图站起来。

左腿传来一阵剧痛。那把生锈的螺丝刀，这次像是直接扎穿了膝盖骨。

“嘶——”

他倒吸一口凉气，重新跌坐回水里。

雨还在下。
巷子里的污水漫过了他的脚踝，漂着几个白色的塑料饭盒和烂菜叶。

这就是成都的折叠处。
五分钟前，他还在那个被称为“国际城南”的CBD，看着那些衣着光鲜的白领喝咖啡。
五分钟后，他像一只断腿的狗，趴在城中村的臭水沟里。

手机震动了一下。

**【系统监测到异常停留。是否遇到困难？】**
**A. 车辆故障**
**B. 身体不适**
**C. 交通管制**
**D. 其他**

小陈看着那几个选项。

如果选B，这单会自动取消，但他会被扣分，甚至封号三天（因为系统会判定他“不具备配送能力”）。
如果选A，需要上传修车照片。他车没坏。

**没有一个选项叫“我摔倒了，但我还能爬”。**

系统不需要英雄。系统只需要**零件**。
一个坏了的零件，唯一的命运就是被替换。

小陈擦了一把脸上的泥水。

他没有选。

他咬着牙，用右腿支撑着身体，硬生生地站了起来。

扶起车。
把洒了一半的喜茶扶正（还好封口严实，没洒太多）。

“还有7分钟。”

他重新跨上车，拧动油门。

左腿已经麻木了。也好，麻木了就不疼了。

他在雨中狂奔。
像一个带着致命旧伤，却依然在全速运行的程序。

---

**18:05。中和新村3号院门口。**

巨大的蓝色帐篷堵在门口，像一道不可逾越的关卡。
大喇叭里循环播放着：“请出示健康码、行程卡，扫场所码进入。”

小陈停下车，膝盖钻心地疼。他掏出手机，想要扫墙上的二维码。

但是，屏幕碎了。
那道裂纹刚好横切过摄像头的位置，而且屏幕上全是泥水。

他对准二维码。
屏幕一片模糊。
再试。
还是模糊。

“搞快点嘛！后面堵起了！”后面的外卖员在按喇叭。

小陈急了，用袖子使劲擦屏幕，越擦越花。
“大哥，”小陈冲着帐篷里的保安喊，保安穿着防护服，看不清脸，“我手机摔坏了，扫不起，能不能通融一下？我就送进去放在架子上。”

“扫不起？”保安的声音闷在面罩里，毫无感情，“扫不起就在门口等着。喊业主自己出来拿。”

“她这会儿不接电话啊！”

“那不得行。没得绿码，谁晓得你是不是从那个区过来的。”保安指了指不远处的封控区，“昨天刚拉走两车，你负责嗦？”

小陈愣住了。
他看着那个近在咫尺的小区大门。
只是送一杯奶茶进去。
平时只需要十秒钟。
现在，因为这一块碎掉的玻璃，因为这个看不见的系统规则，这十秒钟变成了天堑。

**【订单即将超时】**

他绝望地看着手机。
雨还在下。保安像个白色的雕塑一样立在那里，寸步不让。

最后，还是业主终于接了电话，不耐烦地让他把奶茶放在门口的湿凳子上。

“您的外卖……放门口了。”

三秒钟后，小陈的手机震动了一下。

**【差评提醒：餐品破损。扣款：15元。】**

这一单，他赚了7.7元（含暴雨补贴）。
扣掉15元。
倒贴7.3元。

还得加上修手机屏幕的钱，大概200元。
还有那条不知道断没断的腿。

小陈站在楼道里，看着那扇紧闭的防盗门。

楼道里的声控灯灭了。
黑暗中，他笑了一下。

那笑声干涩，像两块生锈的铁片在摩擦。

他没有哭。
他想起老家的一句俗话：眼泪流进眼睛里，就看不见路了。

他一瘸一拐地走下楼。

雨还在下。
整个城市都在雨中运行，精准，冷漠，且完美。

只有他的膝盖，像一个无法被修复的**Error 503**，在隐隐作痛。

而那颗在45亿年前就被宇宙抛入虚空的蓝色行星，也在带着这个膝盖、带着这些破碎的屏幕，继续转。宇宙没给它发过"宜居许可证"，它只是带着偏差，带着错误，一转再转。

小陈不知道的是，他膝盖里的那颗螺丝钉，和地壳板块运动、和太阳黑子周期、和国家被封锁时的芯片突围，遵循的是同一个法则：

**带着错误运行。**
