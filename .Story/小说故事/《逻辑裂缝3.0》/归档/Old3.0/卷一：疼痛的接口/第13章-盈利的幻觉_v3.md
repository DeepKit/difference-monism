# 第13章 盈利的幻觉 (Illusion of Profit)

**时间：202X年8月20日（周二）下午17:30**  
**地点：原点巷子 -> 菜市场**

---

老郑今天走路的姿势有点不一样。
腰板挺直了，步子轻快了。虽然戴着蓝色的医用口罩，但眼角的笑意藏不住。

他刚刚从李嬢嬢那里拿到了第一笔“分红”。
**800元。**

李嬢嬢说：“这是三天的收益！三天就有40%！比你在家隔离强多了！”

他走进菜市场。
门口的大喇叭喊着：“进入市场请扫码测温，严禁购买野生动物。”
老郑熟练地伸出手腕测温，然后在那个贴满了绿色贴纸的手机上点了几下。

“张大爷，绿码还是这么绿啊！”保安开玩笑。
“那是，身体好着呢！”老郑中气十足。

“张大爷，今天买点啥？”卖肉的王屠户问。
平时老郑都是绕着肉摊走的，或者只买那种处理的碎肉。

但今天，老郑停下了脚步。
他指着那块最好的五花肉：“给我来二斤。要肥瘦相间的。”

王屠户愣了一下，手里的刀都停了：“张大爷，今天不过节啊？”

“孙子想吃红烧肉。”老郑笑得脸上的皱纹都舒展开了，“给他做顿好的。”

王屠户手起刀落，切下一块漂亮的肉。
“好嘞！再送你两根葱！”

老郑提着肉，又去买了条鱼，还买了豆豆最爱吃的草莓。
那草莓真贵，30一斤。
要是以前，老郑看都不敢看。
但今天，他买了半斤。

“反正明天还有分红。”他想。

回到家，老郑系上围裙，开始忙活。
红烧肉在锅里炖着，咕嘟咕嘟冒着香气。
那种香气，是富足的味道，是希望的味道。

豆豆放学回来了，闻到香味，书包都没放下就冲进厨房。
“哇！红烧肉！爷爷万岁！”

老郑夹了一块肉，吹了吹，塞进孙子嘴里。
“烫不烫？”
“不烫！好吃！”豆豆吃得满嘴流油。

老郑看着孙子狼吞虎咽的样子，眼眶有点湿。
这才是日子啊。
以前那种抠抠搜搜、为了几毛钱菜钱跟人讨价还价的日子，终于要过去了。

他拿出手机，打开那个“短剧APP”。
看着上面跳动的数字。
**今日预估收益：+260元。**

这个数字比红烧肉还香。

就在这时，手机响了。
是李嬢嬢发来的语音。
“张哥，公司那边说，因为咱们这批种子用户表现好，特批了一个‘超级加倍’活动。只要再追加五千，以后每天的分红翻倍！名额只有十个，你要不要？”

老郑的手抖了一下。
五千。
他没有五千。
他的存折里只剩下那点“棺材本”了。

但是……翻倍啊。
如果每天能分500，一个月就是一万五。
那不仅能给豆豆报奥数班，还能给家里换个新空调。那个E6代码，他再也不想看见了。

老郑看了一眼正在啃骨头的豆豆。
他想起1998年下岗那天，老伴拉着他的手说：“没事，只要咱们一家人在一起，吃咸菜也是香的。”
现在老伴走了，但存折还在。那是他和老伴三十年的积蓄，也是这个家的底气。

“拼了。”
他想。为了豆豆，为了这个家。

“李妹儿，给我留个名额。我明天一早去银行取钱。”

放下手机，老郑觉得心跳得很快。
那是兴奋，也是恐惧。
像是在赌桌上梭哈的前一秒。

窗外，天色暗了下来。
乌云又开始聚集。
老郑没注意。他只顾着给豆豆夹肉。
睡前，他把手机放在枕边。屏幕还亮着，显示那个跳动的数字。他看了一眼，觉得像是一个**心跳监测仪**。

“多吃点，长身体。”
他说。

而在几公里外的一栋写字楼里。
“张总”看着后台的数据，嘴角勾起一丝冷笑。
“鱼咬钩了。准备收网。”
