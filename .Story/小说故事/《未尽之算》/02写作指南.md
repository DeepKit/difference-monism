# 《未尽之算》写作指南 v7.0

> 本文件定义"怎么写"——平台策略、单集模型、爽点工程、排版规范、检查清单。
> 世界观与角色谱系见 `架构总纲.md`；42卷逐卷概要见 `执行大纲.md`。

---

## 一、核心定位

给"清醒的系统困兽"看的认知爽文。

**平台**：All in 知乎盐选。
**核心受众**：
1. 体制内的清醒者 (60%)：公务员/国企中层，爽点是"恶性合规"反杀
2. 大厂的算法囚徒 (30%)：互联网/金融人，爽点是"古代算法比现代还精明"
3. 智性爽文爱好者 (10%)：高学历人群，社交传播主力

---

## 二、标题规范

**格式**：`《未尽之算》` + 知乎体观点句/悬念句

- 必须以`《未尽之算》`开头
- 能制造"点进来看"的冲动，可挂靠热点但必须与本集内容直接相关
- 不是章节序号，是独立的知乎问题标题，要能单独成立
- 书名号《》引电影/书名，单尖括号〈〉引篇名/歌名

---

## 三、内容形态

- **卷 (Volume)** = 一个知乎问题 + 一个盐选专栏
- **集 (Episode)** = 回答的长篇连载（单集3000字+）
- **每卷** = 4个小案 × 弹性6集 = 24集

---

## 四、单集模型（ASTO引擎）

### 4.1 六阶节奏（每卷24集）

| 阶段 | 集数 | 功能 |
| :--- | :--- | :--- |
| 混沌 (Chaos) | 1-4 | 抛出死局 + B线切入 + C线知识 |
| 试探 (Probe) | 5-8 | 第一次反杀尝试（必须失败） |
| 深渊 (Abyss) | 9-12 | 代价升级 + 伦理困境 |
| 反杀 (Strike) | 13-16 | 恶性合规反杀（高潮） |
| 余震 (Aftershock) | 17-20 | 系统反扑 + 补丁升级 |
| 清算 (Settlement) | 21-24 | 审计板公开 + 交接 + 下卷钩子 |

### 4.2 单集结构

```
Hook（前300字）→ 死局/悬念
正文 → 三线编织（生活线 + 学问线 + 历史线）
付费卡点 → 卡在主角做决定的那一刻
尾钩 → 下一集悬念
```

### 4.3 付费点设计

- 每集在2/3处设付费墙
- 必须卡在"主角做决定"的那一刻
- 付费后立刻给爽点，不能让读者觉得被骗

---

## 五、爽点工程

### 5.1 恶性合规（核心机制）

不是反抗，是严格按字面意思执行规则，直到系统自己崩溃。

**结局轮换**：惨胜50% / 假胜（系统吸收胜利）30% / 彻底失败20%

### 5.2 核心三角

| 角 | 内容 | 读者体验 |
| :--- | :--- | :--- |
| 生存 | 主角能否活下来？ | 爽 |
| 伦理 | 救一人还是救一群人？ | 虐 |
| 元叙事 | 倒计时 + 审计板 | 沉浸 |

### 5.3 金句规则

- 每集至少一句可传播的扎心短句
- 标粗处理，适合截图传播
- 风格：冷幽默 + 社会洞察，不鸡汤

---

## 六、三线编织规则

### 6.1 生活线（温度）

- 必须有：吃饭、睡觉、发呆、身体反应等感性锚点
- 如果一集全是逻辑推演，强行插入一个感性锚点
- 碎片类型：火锅、体检报告、手机型号、称呼变化、租金

### 6.2 学问线（锋利）

- 行业知识必须转化为"武器"，不能变成科普
- 读者看完要觉得"学到了一招"

### 6.3 历史线（厚度）

- B线历史镜像：每卷对应一个中国历史朝代/事件
- 古今对照必须形成"原来一千年前就这样了"的震撼
- 朝代去重：相邻卷不重复同一朝代

---

## 七、人物写作策略

### 7.1 X的交接四件套

每代X离场时必须完成：
1. **遗物**：下一代X无意中捡到，触发部分记忆觉醒
2. **伤疤**：上一代X的代价在下一代身上留下痕迹
3. **口头禅**：继承一句话，但含义随语境变化
4. **未完成**：上一代没做完的事，成为下一代的起点

### 7.2 灰度规则

- 没有纯善纯恶的角色
- 反派必须有"他也没办法"的时刻
- 主角必须有"他也做了脏事"的时刻
- 系统不是邪恶的，它只是在执行逻辑

### 7.3 见证者规则

- 苏怀/阿K 是读者的眼睛，不是主角
- 他们记录但不干预，偶尔提供情报
- 交棒点（卷25→26）必须有情感仪式

## 八、产品层

### 8.1 审计板（每卷必备）

- **PUBLIC版**：状态信息，读者可见
- **UNREDACTED版**：证据链 + 裁决栈 + 反事实后果 + 系统补丁

### 8.2 倒计时

- 每卷标注"距离系统停机还有 XXX 集"
- 从卷01的1008递减到卷42的0

### 8.3 评论区引战话题

- 每集预设1-2个可引发讨论的伦理问题
- 目的：提升互动率，不是真的引战

---

## 九、排版规范

- 段落间空一行
- 对话不加引号，用破折号或直接写
- 金句加粗
- 知识点用 `>` 引用块
- 每集末尾加分割线 `---`

---

## 十、防疲劳策略

### 10.1 去重复地图

相邻两卷不得在以下维度重复2项以上：
- X的职业领域
- 历史朝代
- 反派类型
- 代价类型
- 结局色调

### 10.2 呼吸卷

每5卷插入1卷"呼吸卷"：节奏放缓，侧重生活线，让读者喘气。

### 10.3 长篇连载生存法则

- 每卷可独立阅读，新读者从任意卷入坑
- 长线伏笔是彩蛋不是承诺——看到了惊喜，没看到不影响当前故事
- 作者给自己留"随时可以在某个节点优雅收束"的余地
