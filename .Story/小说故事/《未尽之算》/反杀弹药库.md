# 《未尽之算》反杀弹药库 (恶性合规诡计集)

> **使用说明**：
> 本文档是全书42卷的“爽点核武库”。
> 每卷创作前，从中挑选一个最适配的核心诡计，作为高潮反杀手段。
> **核心逻辑**：不掀桌子，利用规则本身杀人。

---

## Type A: 字面意义流 (Literal Interpretation)
*利用语言的歧义，严格执行字面意思，违背立法本意。*

1.  **【无墨之字】**（已用/卷01）：规则要求签字，未规定墨水。用血/水签字，字迹消失，无法追责。
2.  **【一毫米的误差】**：规则要求“城墙高三丈”。严格执行三丈，不考虑地基沉降。城墙合规倒塌，压死贪官。
3.  **【全员重伤】**：规则“重伤者可免役”。全村互相把对方打成“法定重伤标准”（如断一指），集体免役。
4.  **【一饭之恩】**：规则“赈灾粥需插筷不倒”。煮成固态饭团插筷子，虽然极少米量也能不倒，以此嘲讽形式主义。
5.  **【宵禁的漏洞】**：规则“宵禁期间不得在街上行走”。所有人改成“爬行”或“跳跃”，因为规则只写了“行走”。
6.  **【沉默的供词】**：规则“犯人必须画押”。犯人把供词撕碎吃下去，然后在肚皮上画押。
7.  **【精准的迟到】**：规则“迟到一刻钟斩首”。全军精准迟到14分59秒，导致战机延误，但无人可杀。
8.  **【合规的裸奔】**：规则禁止穿“奇装异服”。于是所有人不穿衣服（无装无服），无法定罪。
9.  **【白马非马】**：规则“禁止马匹入城”。骑驴、骑牛、骑猪入城，瘫痪交通。
10. **【无限的午餐】**：规则“公务接待需四菜一汤”。把“一汤”做成澡盆那么大，四菜是四粒花生米。

## Type B: 程序正义流 (Malicious Bureaucracy)
*利用繁琐的流程和审批链条，瘫痪系统效率。*

11. **【蒸骨三千年】**（已用/卷01备选）：申请最高级别尸检，耗时三年，拖死限期。
12. **【无限递归】**：为“申请表”本身申请一个“申请许可”，再为“申请许可”申请“许可申请”。
13. **【万民上书】**：规则“凡百姓申诉，必立案存档”。发动全城十万百姓，每人申诉“丢了一根针”，把衙门档案库撑爆。
14. **【死循环审批】**：A部门审批需B部门盖章，B部门盖章需A部门审批。两边同时发起，系统死锁。
15. **【合规的拥堵】**：规则“过关需查验路引”。商队把路引撕成碎片，让守卫一片片拼，完全合规配合检查。
16. **【逐字朗读】**：规则“宣读圣旨需在大庭广众”。找个口吃的人念，念了三天三夜，拖延处斩时间。
17. **【精准报销】**：把一笔巨款拆分成一万笔“一文钱”的小额开支，让账房算盘打到起火。
18. **【全员证人】**：审案需传唤证人。把全城人都列为“目击证人”，法庭永远开不了庭。
19. **【翻译的艺术】**：规则“番邦文书需翻译”。把文书翻译成一种失传的方言，再申请找翻译专家，耗时十年。
20. **【格式陷阱】**：指出上级公文的格式错误（如避讳字），要求退回重写，以此驳回死刑令。

## Type C: 系统冲突流 (Rule Conflict)
*利用两个不同系统的规则矛盾，让系统自我攻击。*

21. **【忠孝两难】**：利用“丁忧”规则（父母死需守孝三年）对抗“征召”规则。全军突然“父母双亡”。
22. **【货币战争】**：利用“官方汇率”与“黑市汇率”的价差。合规兑换，把国库洗空。（索罗斯战法）
23. **【保密vs公开】**：规则A“灾情需上报”，规则B“机密不得外泄”。把灾情定为“绝密”，导致无法赈灾，最后因“无人知情”而免责。
24. **【神权vs皇权】**：利用“天降祥瑞”或“日食”规则，迫使皇帝下罪己诏，停止暴政。
25. **【避讳杀人】**：给仇人起名叫皇帝的名字（或谐音），导致他在任何文书上都无法被提及，成为“不存在的人”。
26. **【税收悖论】**：利用“鼓励开荒免税”和“人头税”的矛盾。把所有人户籍迁到荒地，变成“无税之国”。
27. **【左右互搏】**：用左手签发的“通行证”去通过右手设立的“封锁线”。
28. **【时间的缝隙】**：利用历法变更（如改元），在两个年号的交接空档期作案。
29. **【身份叠加】**：我是官（有豁免权），我也是商（有经营权）。利用双重身份在系统间套利。
30. **【法律的死角】**：在两国边境线上作案，左脚在宋，右脚在辽。两国都无管辖权。

## Type D: 资源耗尽流 (Resource Exhaustion)
*合规地消耗系统资源，直到系统崩溃。*

31. **【八百里送菜单】**（卷02预设）：利用“加急文书”无限制，发送海量垃圾信息（菜单、问候），跑死所有驿马。
32. **【免费的午餐】**：利用“赈灾粮领取无限制”。全城人排队领粮，领完倒掉或喂猪，直到粮仓通过“合规渠道”被掏空。
33. **【全员各种】**：利用“举人免税”。全村挂靠在一个举人名下，导致该县税收归零。
34. **【纸张战争】**：要求所有公文“一式一百份”存档，耗尽衙门的纸张预算。
35. **【蜡烛战术】**：利用“夜间审案需点灯”。通宵审案，耗尽灯油，导致无法办公。
36. **【铁钱危机】**：利用铁钱价值低于铸造成本。收集铁钱熔化卖铁，合法套利，导致货币体系崩溃。
37. **【监狱满员】**：全城轻微犯罪（如随地吐痰），主动投案，把监狱塞满，导致重刑犯无处关押。
38. **【医保穿透】**：利用“名贵药材报销”。全员开昂贵补品，把医保基金通过合规手段买空。
39. **【弓箭回收】**：草船借箭变体。诱敌射箭，回收利用，耗尽敌方军火。
40. **【合规的拥挤】**：所有人到钱庄“存一文钱”，再“取一文钱”，循环操作，挤兑导致钱庄瘫痪。

## Type E: 极端合规流 (Extreme Compliance)
*把规则执行到荒谬的极致，导致结果反转。*

41. **【绝对安全】**：为了“防火安全”，把所有房子拆了，只剩空地。确实不起火了，但人冻死了。
42. **【零误差】**：为了“账目精准”，花费一万两银子去寻找丢失的一文钱。
43. **【全员禁言】**：为了“防止谣言”，禁止所有人说话。导致军令无法传达，军队溃败。
44. **【极致环保】**：为了“保护耕地”，禁止在耕地上盖房、走路、拉屎。最后人全饿死。
45. **【绝对公平】**：为了“公平分配”，把一个饼切成原子级大小，谁也吃不到。
46. **【永不犯错】**：为了“不犯错”，什么都不做。彻底的躺平，导致系统停摆。
47. **【信息透明】**：把所有机密全部公开，导致敌人看光底牌，但也导致敌人被海量信息淹没。
48. **【极致礼貌】**：两军交战，先互通书信礼仪三天，错过战机。
49. **【按闹分配】**：系统规定“有困难解决困难”。全员装困难，比谁更惨。
50. **【最后一人】**：系统规定“最后一人关灯”。所有人都不走，灯永远亮着，耗尽能源。
