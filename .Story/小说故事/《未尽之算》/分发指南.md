# 《未尽之算》分发与运营指南 v1.0

## 1. 平台矩阵策略 (7:2:1 黄金法则)

### 主战场：知乎盐选 (70% 精力 - 核心变现)
*   **定位**：高净值用户收割机，认知变现主阵地。
*   **逻辑**：用户画像与“清醒的系统困兽”完美重合。
*   **产品形态**：“问题 - 答案”式连载。
    *   **公式**：每卷 (24集, ~5万字) = 一个知乎问题 + 一个盐选专栏。
    *   **问题示例**：
        *   "为什么所有'完美制度'最终都会逼死人？"
        *   "历史上那些'背锅侠'后来都怎么样了？"
        *   "古代打工人如何对抗KPI？"
*   **专栏结构**：
    *   **免费部分 (前50%)**：历史X/张三的困境（情绪共鸣）。
    *   **付费部分 (后50%)**：三人组讨论 + 透镜指南（认知升级）。
*   **营销打法**：**降维打击**。
    *   开设专栏《〈未尽之算〉的1008张蓝图》。
    *   直接公布“四部曲演进图”：肉身重负 → 符号异化 → 数据幽灵 → 未尽余数。
    *   展示“创造世界的源代码”，吸引创始级铁粉。

### 放量渠道：番茄小说 (20% 精力 - 流量与IP)
*   **定位**：泛用户流量池，短剧IP孵化器。
*   **逻辑**：利用算法推荐获取海量曝光，直通字节系短剧生态。
*   **适配改造**：
    *   **切片化**：将四部曲拆分为独立短剧单元。
    *   **去哲学化**：弱化阿K的吐槽，强化X的视觉化“发疯”时刻。
    *   **进度可视化**：标题增加全局进度（如：`第114集 | 总计 1008集`），将长篇压力转化为史诗工程的陪伴感。

### 口碑阵地：豆瓣阅读 (10% 精力 - 品牌背书)
*   **定位**：影视版权敲门砖，文学性认证。
*   **逻辑**：吸引80/90后高学历群体，博取正午阳光等影视方关注。
*   **差异化标签**：“生活悬疑”、“类型文学创新者”。
*   **内容侧重**：
    *   强调“考据细节”（宋代验尸格目、明代驿站算法）。
    *   营造“冷峻现实主义”质感。

## 2. 内容策略校准：情绪工程

### 五点爆破法 (单卷/24集节奏)
| 爆点位置 | 情绪类型 | 具体设计 | 运营目的 |
| :--- | :--- | :--- | :--- |
| **第1集开头** | **愤怒共鸣** | 主角X被系统无理压榨的具体场景（如：验尸还要扣钱）。 | 免费，建立代入感。 |
| **第6集结尾** | **压抑峰值** | X被逼至绝境，系统露出獠牙（如：不签字就杀头）。 | 付费卡点前奏。 |
| **第12集中段** | **智力快感** | X发现系统漏洞（未尽之算），读者获得“我也能看穿”的爽感。 | **付费转化关键点**。 |
| **第18集高潮** | **道德撕裂** | 两难抉择：救一个人 vs 保全更多人。 | 评论区站队，引爆流量。 |
| **第24集结尾** | **代偿满足** | X付出代价但完成反杀，系统输理，人心赢规。 | 续订下一卷钩子。 |

### “发疯”的升级定义：恶性合规 (Malicious Compliance)
*   **旧理解**：暴力掀桌、情绪失控。
*   **新策略**：**平静的疯狂**——严格执行系统规则，直到系统因逻辑闭环而崩解。
*   **示例**：
    *   *原设计*：X拒绝签字，被流放。
    *   *升级设计*：X签字，但用毛笔蘸血签，字迹三天后消失。系统无法惩罚，因为“规则只规定要签字，没规定用什么墨”。

## 3. 战略升维：元叙事营销

### 核心资产：1008集全景地图
*   **策略**：不藏着掖着，将“宏大结构”作为第一卖点。
*   **话术**：“这不是一本小说，这是一场跨越1008集、贯穿四重系统牢笼的生存实验。地图在此，你敢跟到底吗？”
*   **结构性钩子**：
    *   用“系统升级”替代单纯的“打怪升级”。
    *   让读者期待“下一个阶段的牢笼是什么样”，而非仅仅“下一个案子怎么破”。

### 哲学内核：冰山策略
*   **水面之上 (90%)**：扎心的职场故事、历史悬疑、社会潜规则。
*   **水面之下 (10%)**：严密的ASTO逻辑。
    *   **彩蛋设计**：每卷末附带“系统原理”解读（如：本卷漏洞对应ASTO物化态IV的映射裂缝），满足极客读者的考据欲。

## 4. 开局执行方案 (前5集 Demo)

*   **核心钩子**：**“算法的必然” vs “人心的偶然”**。
*   **开篇场景 (第1集)**：
    *   **任务**：七日验明108具无名尸，否则扣光俸禄。
    *   **系统建议**：随机验10具，其余报瘟疫，效率最大化。
    *   **X的抉择**：看着第一具尸体手中的半块糖，骂了一句“去你妈的效率”，然后开始**从第一具一具具验下去**。
    *   **意义**：用“无望的恪守本职”对抗“高效的系统逻辑”。
