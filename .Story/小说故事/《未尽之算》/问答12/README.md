# 《未尽之算》知乎养号问答12篇 — 运营指南

## 人设定位
**"用古代故事照亮现代困境的历史洞察者"**
- 主打"历史+现代"跨时空对照，独特视角差异化
- 辅以"恶性合规"方法论，展示思维深度
- 语言风格：口语化、短段落、有金句、不学术

## 策略
- **前8篇（干货篇）**：纯知识输出，不提小说，建立"这人说的有道理"的信任
- **后4篇（钩子篇）**：在干货基础上自然引出《未尽之算》，不硬广

## 发布节奏建议
- 每1-2天发一篇，约3周发完
- 先发01-04（最易传播的话题），观察数据
- 05-08根据前四篇反馈调整发布顺序
- 09-12在粉丝量到一定基数后再发

## 注意事项
1. **新号初期不要在回答里放外链**，先养权重
2. **前8篇发布时标题不带《未尽之算》**，纯粹以"知乎用户"身份回答

5. **关注对应话题并先回答几个简短问题**，避免新号一上来就发长文被限流
