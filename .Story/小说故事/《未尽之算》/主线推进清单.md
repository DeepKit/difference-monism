# 《未尽之算》主线推进清单 (Master Threads)

> 本文档用于追踪全书 1008 集并行的几条核心叙事线。每一集除了讲案子，必须至少推动以下一条线。

---

## 1. X代际线 (The Operator Lineage)
*   **推进载体**：**职位交接**。
*   **节奏**：每 5-6 卷一次换班（按八代操作者）。
*   **状态**：
    *   **X-01 (卷01-05)**：陈默 → 医疗事故死亡（系统补丁v1.01触发的“合规死亡”），留加密硬盘。
    *   **X-02 (卷06-10)**：罗生 → 主动沉默/自我放逐，留标注线索的硬盘。
    *   **X-03 (卷11-16)**：单一凡 → 跳楼自杀，留 Key #1（餐巾纸密码）。
    *   **X-04 (卷17-21)**：迟归 → 被驱逐出境，留 Key #2（备课笔记）。
    *   **X-05 (卷22-27)**：冉路 → 被系统吸收，留 Key #3（王博士产出）。
    *   **X-06 (卷28-33)**：凌蓝珏 → 执照吊销/行业除名，留 Key #4（老林产出）。
    *   **X-07 (卷34-38)**：白夜 → 替阿K挡死，归拢全部碎片交阿K。
    *   **X-08 (卷39-42)**：苏小米 → 主动离开，完成最终解密。

## 2. 见证者/档案线 (The Witness Chain)
*   **推进载体**：**《未尽之算》档案 / 地下广播**。
*   **节奏**：全程贯穿，苏雨桐（卷01-27） → 阿K（卷28-42）。
*   **功能**：
    *   每一卷末尾的 **[审计板]** 由见证者签署。
    *   每一代 X 的 **[交接记录]** 由见证者发布。
    *   卷27末苏雨桐自我隔离，交接笔记与5件物品给阿K。
    *   负责把散落的“小胜”串成“大历史”。

## 3. MacGuffin 碎片线 (The Key Fragments)
*   **推进载体**：**锁定音频文件 + 4份密钥碎片**。
*   **节奏**：全书解密。
    *   **Start (卷10)**：获得锁定文件（陈默遗物）。
    *   **Key #1 (卷15)**：金先生线索（餐巾纸助记词）。
    *   **Key #2 (卷21)**：吴老师线索（备课笔记密码）。
    *   **Key #3 (卷27)**：苏雨桐/王博士线索（算法纸条）。
    *   **Key #4 (卷33)**：老林线索（警号牌刻痕）。
    *   **End (卷42)**：集齐碎片，解锁文件（爷爷陈守一的遗言/自由）。

## 4. 系统熵增线 (The Entropy Clock)
*   **推进载体**：**倒计时 / 系统日志 [SYSTEM_LOG]**。
*   **节奏**：每集头部可见。
*   **阶段**：
    *   **卷01-10**：系统稳态，清除个别坏点。
    *   **卷11-20**：系统开始主动“吞噬”资源以维持低熵（金融吸血）。
    *   **卷21-30**：系统算力过载，开始依赖“算法独裁”。
    *   **卷31-42**：系统熵增不可逆，走向热寂/崩溃/重启。

## 5. 历史镜像 B线 (The History Mirror)
*   **推进载体**：**苏雨桐的史料 / 历史平行故事**。
*   **节奏**：每集 B 轨。
*   **功能**：证明“太阳底下无新事”。现代的算法暴力，在古代都有对应的人力暴力版本（如：健康码 vs 编户齐民；算法外卖 vs 站赤急递）。

## 6. 反派进化线 (The Antagonist Evolution)
*   **推进载体**：**对手层级升级**。
*   **节奏**：四部四阶。
    *   **第一部**：执行者（中层官僚/KPI机器）。杀人是为了保饭碗。
    *   **第二部**：定义者（立法者/资本家）。杀人是用“契约”和“定义权”。
    *   **第三部**：算法（无脸模型）。杀人是“优化参数”。
    *   **第四部**：镜像/系统本身（叶女士/招安者）。杀人是“把你变成系统的一部分”。

### 7.5 社会氛围三阶段

8个X不是同一批人，不存在"主角团"的集体成长弧线。时间线的情绪载体是**中国社会的变迁**：

*   **高速增长期（2006-2012 / 肉身部）**：
    *   经济高速增长，互联网崛起，外卖/快递行业爆发。
    *   社会心态：机会遍地，"只要努力就能改变命运"。
    *   系统状态：稳态运行，清除个别坏点。
    *   X的处境：个体被碾碎（陈默的"合规死亡"、罗生的黑名单），但社会整体向上，没人在意底层的代价。
*   **移动互联网全面渗透期（2012-2018 / 符号部 + 数据部前半）**：
    *   金融创新/P2P/共享经济，移动支付改变一切。
    *   社会心态：焦虑开始蔓延，"中产陷阱""消费升级"。
    *   系统状态：开始主动"吞噬"资源以维持低熵（金融吸血），算力过载。
    *   X的处境：符号暴力取代肉身暴力（单一凡被"定义"杀死、迟归被"合法"驱逐），系统学会了用规则杀人。
*   **监管收紧与算法治理期（2018-2025 / 数据部后半 + 余数部）**：
    *   数据安全法、个人信息保护法、平台经济整顿、算法推荐管理规定。
    *   社会心态："躺平""内卷""润"，集体性的疲惫与清醒。
    *   系统状态：熵增不可逆，走向热寂/崩溃/重启。
    *   X的处境：系统已经无处不在，反抗方式被逐一封堵（冉路被吸收、凌蓝珏被社死、白夜用命换时间），直到苏小米选择"不玩了"。
