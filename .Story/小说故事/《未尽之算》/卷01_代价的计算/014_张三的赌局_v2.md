# 第14集：张三的赌局

> **《未尽之算》·第14集·张三的赌局**

死牢里没有光，只有气味。

发酵的排泄物味，陈年血迹的铁锈味，还有死老鼠腐烂的甜腥味。但此刻，这些味道统统被另一种霸道的气息盖过了。

猪肉大葱。

刚出笼的，带着面粉发酵香气和滚烫肉汁的，猪肉大葱包子。

张三蜷缩在墙角的烂稻草里，喉结像生锈的轴承一样剧烈上下滑动。三天没喝水，他的喉咙早已干裂起皮，每一次吞咽都像是在吞一把碎玻璃。

但他还是想吞。

周主簿蹲在他面前，手里捏着那个白胖暄软的包子，轻轻掰开一半。

“滋——”

热气冒出来，带着那股能把人魂魄勾走的肉香。张三甚至能看清肉馅里那一丁点翠绿的葱花，和还在滋滋冒泡的油脂。

“想吃吗？”周主簿的声音很温柔，像是在哄一个孩子。

张三拼命点头。他的手已经被打断了，只能像条蛆一样在地上蠕动，试图把脸凑过去。

“我也想给你吃。”周主簿叹了口气，把包子凑到张三鼻尖下晃了晃，又缩回去，“但规矩你也懂。一手交钱，一手交货。”

“我……没钱……”张三的声音哑得像破风箱。

“你有。”周主簿笑了，那笑容在昏暗的油灯下显得格外狰狞，“化人场那天，你从那丫头身上撕下来的红布条。那上面有血手印。把它给我，这一整盒包子都是你的。”

提到那块红布，张三的瞳孔缩了一下。

那是铁证。是那个权贵杀人的指纹。

“我……没有……”

“别装了。”周主簿失去了耐心，把半个包子扔在地上，用脚尖碾碎，“我不急。你那瞎眼老娘大概挺急的。听说她还在巷口等你回去送米呢。你说，要是皇城司的人去了……”

张三浑身一抖。

“别动她！”

“那就拿出来！”周主簿厉声喝道。

这是一个死局。交出去，证据销毁，真凶逍遥，他成帮凶。不交，他饿死，老娘也活不成。

生存本能在尖叫：**给他！活着比什么都重要！**

张三颤颤巍巍地把手伸进怀里，摸到了那块硬邦邦的布条。那上面沾着女孩的血，已经干结发黑。

周主簿眼睛亮了，贪婪地伸出手。

就在指尖即将触碰的瞬间。

张三的手突然缩了回去。

他把布条塞进了嘴里。

“你干什么？！”周主簿大惊。

张三没有回答。他抢过那个被碾碎的半个沾泥的包子，连同那块粗糙的、带着血腥味的布条，一起塞进喉咙。

**嚼碎它。**

布条像砂纸一样摩擦着口腔，卡在喉咙口，令人作呕。但他死命地往下咽。

“吐出来！给我吐出来！”周主簿扑上来掐他的脖子，指甲深深嵌入他的肉里。

窒息感袭来。张三翻着白眼，脸涨成猪肝色，但他的腮帮子依然在疯狂咀嚼。

**把冤屈咽下去。把良心咽下去。把证据变成烂在肚子里的屎。**

“咕咚。”

一声沉闷的吞咽声。

那团混合着肉香、泥土和血证的硬块，硬生生地滑进了食道。

张三被掐得直翻白眼，但他笑了。满嘴是血沫地笑了。

“没了。”

“证据……在我肚子里。”

“想要？剖开拿啊！”

现代。安全屋。

陈默盯着屏幕上的倒计时：`00:29:59`。

老赵给出的最后通牒。半小时后，物理回收程序启动，这里将变成毒气室。

阿K手里拿着那块从古墓里挖出来的黑色骨头。那是张三的胸骨。哪怕过了一千年，哪怕已经被烧成了灰，骨质里依然检测出了特殊的纤维成分。

他把证据吞了。他把证据变成了自己骨头的一部分。

“他在看着我们。”陈默说。

“默哥，发吧。”方屿指着那个“发送给全城”的红色按钮，“把死亡名单发出去。只要曝光那份名单，舆论就会爆炸，系统会被迫停机。”

“不行。”陈默声音冰冷。

“为什么？！”苏怀急了，“这是唯一的核武器！”

“因为那是脏弹。”陈默拔掉了网线，“名单一发，社保系统崩盘，所有依赖系统维持生命的重症病人都得死。为了救林小草一个人，拉上全城几千个病人陪葬？那我们和老赵有什么区别？”

“那怎么办？等死吗？”

陈默没有说话。他打开了一个代码编辑器。

屏幕上一片漆黑，只有光标在闪烁。

“张三吞下了证据，因为他要把证据烂在肚子里，不让坏人销毁。”

“今天，我们要吞下这份名单，因为我们不能让它伤害好人。”

陈默的手指悬在键盘上。

“我们要做的，不是摧毁系统。而是让系统**噎死**。”

“噎死？”阿K愣住了。

“系统最怕什么？不是黑客，不是病毒，是**逻辑悖论**。”

陈默开始敲击代码。不是复杂的攻击脚本，而是一个简单的签名函数。

`System.Sign(User_ID);`

但在 User_ID 的位置，他没有输入名字。

他拖入了一张图片。

那是林小草画的画：一个穿着束缚衣的疯子，正在试图拥抱自己。

在这张图片的底层元数据（Metadata）里，陈默写入了一行递归逻辑：

```javascript
function AssessValue(Human h) {
    // 薛定谔的价值判断
    return !AssessValue(h); 
}
```

“这是什么？”阿K凑过来，倒吸一口凉气，“如果他是正资产，则返回负；如果他是负，则返回正？这是一个死循环！”

“对。这就是我要喂给系统的‘布条’。”

陈默按下了回车。

“系统必须对每一个签名进行价值评估。这是写在它底层宪法里的绝对指令。它无法跳过，无法忽略。”

“当它读到这个签名时，它会陷入无限的自我纠结。”

“真即是假。假即是真。”

“它会调动所有算力去解这个死结，就像一条吞掉自己尾巴的蛇。”

屏幕上的进度条开始疯狂闪烁。

并没有什么惊天动地的爆炸声。

只有风扇的声音。

嗡——嗡——

那是安全屋机柜里的散热风扇。与此同时，几公里外的社保局大楼里，数万台服务器的风扇也正在同一秒开始狂转。

一场无声的高烧，开始了。
