# 第14集：死囚把铁证嚼碎吞了——想要？剖开我肚子拿

> **《未尽之算》·第14集·张三的赌局**
> **系统倒计时：996**
> *[SYSTEM_LOG]: 物理证据链断裂。原因：生物降解。*

---

死牢里没有光，只有气味。

发酵的排泄物，陈年血迹的铁锈味，死老鼠腐烂的甜腥。但此刻，这些统统被另一种气息盖过了。

猪肉大葱。

刚出笼的，带着面粉发酵香和滚烫肉汁的，猪肉大葱包子。

张三蜷缩在墙角的烂稻草里，喉结像生锈的轴承剧烈上下滑动。三天没喝水，喉咙干裂起皮，每一次吞咽都像吞一把碎玻璃。

但他还是想吞。

周主簿蹲在他面前，掰开半个包子。

"滋——"

热气冒出来，带着勾魂的肉香。张三甚至能看清肉馅里那一丁点翠绿的葱花，和还在滋滋冒泡的油脂。

"想吃吗？"

张三拼命点头。他的手被打断了，只能像条蛆在地上蠕动，把脸凑过去。

"化人场那天，你从那丫头身上撕下来的红布条。上面有血手印。"周主簿把包子凑到他鼻尖下晃了晃，又缩回去，"给我，整笼包子都是你的。"

那块红布。铁证。权贵杀人的指纹。

"我……没有……"

周主簿把半个包子扔在地上，用脚碾碎："你那瞎眼老娘还在巷口等你回去送米呢。皇城司的人要是去了——"

张三浑身一抖。"别动她！"

"那就拿出来！"

死局。交出去，证据销毁，真凶逍遥，他成帮凶。不交，他饿死，老娘也活不成。

生存本能在尖叫：**给他！活着比什么都重要！**

张三颤颤巍巍把手伸进怀里，摸到了那块硬邦邦的布条。干结发黑的血。

周主簿眼睛亮了，贪婪地伸出手。

指尖即将触碰的瞬间——

张三的手缩了回去。

他把布条塞进了嘴里。

"你干什么？！"周主簿大惊。

布条入口的一瞬间，张三差点呕出来。

那是一块粗麻布。被血浸透后风干，硬得像树皮，边缘毛刺扎得满嘴都是细小的伤口。舌头一卷，铁锈味和咸腥味一起涌上来——那是女孩的血。干了很久的、已经发黑的血。

他抢过地上那个被碾碎的、沾满泥的半个包子，连同布条一起往嘴里塞。

牙齿咬下去，布条纤维卡在牙缝里，怎么嚼都嚼不烂。粗糙的麻丝像一把微型锉刀，反复摩擦口腔内壁，嘴里瞬间全是血沫。包子的面皮混着泥沙，和布条绞成一团黏糊糊的硬块。

他往下咽。

咽不下去。

那团东西卡在喉咙口，像一只带刺的拳头堵住了食道。喉头剧烈痉挛，呕吐反射让胃酸猛地翻上来，烧得整个胸腔都在着火。泪水不受控制地涌出来——不是悲伤，是身体在发出最原始的求救信号：**吐出来！你会死的！**

周主簿扑上来掐住他的脖子。十指嵌入肉里，指甲抠得皮肤翻起。窒息感叠加在呕吐感上面。张三翻着白眼，脸涨成猪肝色。气管几乎被完全封死，每一次呼吸只能挤进一丝像刀片一样的冷气。

**但他的腮帮子还在动。**

嚼。嚼碎它。牙齿咬断一根又一根纤维。血沫从嘴角溢出来，顺着下巴滴在地上。舌头已经被磨破了，分不清嘴里的血是自己的还是那个女孩的。

**把冤屈咽下去。把良心咽下去。把证据变成烂在肚子里的屎。**

"咕咚。"

一声沉闷的吞咽。

那团混着肉香、泥土和血证的硬块，像一块烧红的铁，硬生生碾过食道，滑进了胃里。灼烧。像吞了一颗炭火。

张三被掐得几乎断气，但他笑了。满嘴血沫地笑了。

"没了。"

"证据……在我肚子里。"

**"想要？剖开拿啊！"**

---

现代。安全屋。

陈默盯着屏幕上的倒计时：`00:29:59`。

老赵的最后通牒。半小时后，物理回收程序启动。

阿K手里拿着一块从古墓里挖出的黑色骨头。那是张三的胸骨。哪怕过了一千年，骨质里依然检测出了特殊的纤维成分。

**他把证据吞了。把证据变成了自己骨头的一部分。**

"默哥，发吧。"方屿指着"发送给全城"的红色按钮，"死亡名单一曝光，舆论爆炸，系统被迫停机。"

"不行。"

"为什么？！"

"名单一发，社保系统崩盘，所有依赖系统维持生命的重症病人都得死。为了救林小草一个人，拉上全城几千个病人陪葬？"

**"那是脏弹，不是正义。"**

"那怎么办？"

陈默打开代码编辑器。屏幕一片漆黑，只有光标闪烁。

"张三吞了证据，因为他不能让坏人销毁它。今天，我们吞下这份名单，因为我们不能让它伤害好人。"

"我们不是要摧毁系统。是让系统**噎死**。"

他开始敲代码。不是攻击脚本——一个简单的签名函数：`System.Sign(User_ID);`。

但User_ID的位置，他拖入了一张图片。林小草画的画：穿着束缚衣的疯子，试图拥抱自己。

图片底层元数据里，写入了一行递归逻辑：

```javascript
function AssessValue(Human h) {
    // 薛定谔的价值判断
    return !AssessValue(h); 
}
```

"正资产返回负，负资产返回正——"阿K倒吸凉气，"死循环！"

"对。这就是我要喂给系统的'布条'。"

陈默按下回车。

系统必须对每一个签名进行价值评估——写在底层宪法里的绝对指令。无法跳过，无法忽略。

真即是假。假即是真。它会调动所有算力去解这个死结，像一条吞掉自己尾巴的蛇。

没有爆炸声。只有风扇的声音。

嗡——嗡——

几公里外的社保局大楼里，数万台服务器的风扇同一秒开始狂转。

一场无声的高烧，开始了。
