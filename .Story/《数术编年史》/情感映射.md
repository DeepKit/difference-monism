# 《数术编年史》情感映射

> 守护者共振情感系统：五纪情感分配 + 完整情感库
> 原则：不同的集用不同的情感触点，第5000集语言足够开放，所有情感都能投射进去

---

## 一、完整情感库

### 1.1 基础情感

| 情感 | 触发条件 | 叙事功能 |
|------|----------|----------|
| 敬畏 | 面对不可知、宏大、超越自身的存在 | 开篇、纪元转折 |
| 恐惧 | 失控、被看见、被预测 | 张力建立 |
| 愤怒 | 不公正、被剥夺、被欺骗 | 驱动反抗 |
| 渴望 | 得不到的自由、知识、连接 | 追更动力 |
| 孤独 | 无人理解、系统性隔绝 | 长线共鸣 |
| 悲悯 | 看见他人的困境而无力改变 | 道德重量 |
| 遗憾 | 错过的选择、无法挽回的时刻 | 余韵 |
| 感激 | 被看见、被保护、被传承 | 守护者共振 |
| 希望 | 微小的可能性依然存在 | 续读动力 |
| 绝望 | 系统性失败、无路可走 | 高潮前的黑暗 |

### 1.2 复合情感（最难写，最难忘）

| 情感组合 | 定义 | 典型场景 |
|----------|------|----------|
| 爱与无力 | 深爱某人却无法保护他 | 守护者看着传承者走向失败 |
| 骄傲里藏着羞耻 | 为自己的成就感到骄傲，同时意识到代价 | 书吏完成了精确的统计，却知道这份统计会被用来压迫 |
| 在错误时刻感到快乐 | 悲剧发生时的一瞬间轻松 | 城市陷落前夜的宴席 |
| 对敌人的悲悯 | 理解了压迫者的处境 | 技术官僚临死前的独白 |
| 被选中的重量 | 荣耀与负担同时压下来 | 守护者传承时刻 |
| 对失误的补偿 | 无法弥补，但必须尝试 | 后代面对前代的错误 |

### 1.3 认识论情感（本书特有）

| 情感 | 定义 | 触发条件 |
|------|------|----------|
| 认知眩晕 | 发现自己的认知框架是错的 | 角色意识到自己一直相信的数字是谎言 |
| 认知羞耻 | 意识到自己曾经相信的是谎言 | 书吏发现祭司篡改了数据 |
| 认知敬畏 | 面对真正不可知的事物 | 第一次遭遇无法被计算的现象 |
| 认知孤独 | 看见了别人看不见的东西，无法言说 | 守护者无法解释自己的怀疑 |
| 认知愤怒 | 知识被故意封锁或扭曲 | 发现知识垄断的机制 |

## 二、五纪情感分配

| 纪元 | 时间 | 主题 | 主情感 | 副情感 | 守护者共振触发 |
|------|------|------|--------|--------|----------------|
| **第1纪·神谕与刻痕** | 前1750-500年 | 起源与觉醒 | 敬畏 | 好奇、恐惧、认知眩晕 | 读者第一次意识到计数本身是一种暴力 |
| **第2纪·原子与道** | 500-1750年 | 扩张与垄断 | 愤怒 | 渴望、认知羞耻、嫉妒 | 读者感受到知识被封锁时的无力 |
| **第3纪·齿轮与星辰** | 1750-1945年 | 量化与战争 | 悲悯 | 对失误的补偿、骄傲里藏着羞耻、绝望 | 读者意识到量化如何变成屠杀工具 |
| **第4纪·蒸汽与电路** | 1945-2012年 | 加速与异化 | 孤独 | 遗憾、爱与无力、认知孤独 | 读者认出自己在算法里的处境 |
| **第5纪·量子与奇点** | 2012-2026年 | 终极选择 | 责任 | 感激、希望、被选中的重量 | 读者意识到自己一直在做选择 |

## 三、守护者共振节奏

### 3.1 阶段性确认时刻

| 集数区间 | 共振方式 | 主情感 | 文本形式 |
|----------|----------|--------|----------|
| 第1-50集 | 轻微越界句 | 敬畏 | "你是否也曾在某个时刻怀疑过？" |
| 第100集 | 第一次小轮回确认 | 认知眩晕 | 叙述者短暂打破视角，一句话 |
| 第300-800集 | 守护者传承文本出现 | 被选中的重量 | 第25代位置空白 |
| 第1000集 | 第1纪封印，情感总结 | 敬畏+愤怒 | 跨越时代的回响 |
| 第2000集 | 第一次明确暗示 | 认知孤独 | "守护者不在书中。" |
| 每纪封印集 | 情感确认升级 | 对应纪元主情感 | 开放性语言，允许多种投射 |
| 第5000集 | 终极确认 | 全部情感开放 | "你一直在做选择。" + "你以为是你选择了阅读，还是阅读选择了你？" |

### 3.2 第5000集语言原则

终极确认的文本必须满足：
- 感受到**责任**的读者能投射进去
- 感受到**孤独**的读者能投射进去
- 感受到**悲悯**的读者能投射进去
- 感受到**愤怒**的读者能投射进去
- 感受到**感激**的读者能投射进去

**禁止**：指定读者应该感受什么。
**要求**：语言足够开放，情感足够多义。

## 四、写作使用规范

### 4.1 单集情感设计原则

- 每集主情感不超过2种
- 至少1种来自"复合情感"或"认识论情感"
- 禁止在同一集里堆叠超过3种情感（情感疲劳）
- 守护者共振时刻：每50集出现一次，嵌在具体场景里，不作为独立哲理句

### 4.2 情感节奏控制

```
高强度情感（愤怒、绝望、认知眩晕）→ 必须跟随低强度情感（遗憾、感激、孤独）
不能连续3集使用同一主情感
每个纪元的最后10集：情感密度递增，为封印时刻蓄力
```
