# 《数术编年史》人物内心档案

> 框架之外的人：私人欲望、秘密、恐惧、底线
> 原则：这里的内容与数术命框架无关。这是让框架活起来的燃料。
> 禁止：神秘主义内容、科幻内容。所有设定必须在历史现实主义范围内。

---

## 配角系统设计

### 结构：1主 + 2复用 + 4新

每个case（纪元/主角）的人物配置：

| 位置 | 数量 | 来源 |
|------|------|------|
| 主角 | 1 | 本case专属 |
| 复用配角 | 2 | 从复用池选，跨case出现，角色可换但位置不变 |
| 新建配角 | 4 | 从新建池选，本case专属 |

### 10种配角类型

**复用池**（每case选2，跨纪元有叙事逻辑）

| 类型 | 功能 |
|------|------|
| 传承者 | 接收或传递使命/秘密/知识 |
| 见证者 | 旁观并记录，叙事视角 |
| 镜像者 | 相似选择，不同命运，人性循环暗线载体 |
| 导师/学徒 | 知识传递的两端，守护者传承微观体现 |

**新建池**（每case选4，本case专属）

| 类型 | 功能 |
|------|------|
| 情感锚 | 主角最在乎的人，情感代价来源 |
| 对立者 | 价值观相反但动机可理解，道德撕裂 |
| 催化剂 | 触发主角关键转变后退场 |
| 背叛者 | 被信任后背叛，信任崩塌 |
| 制度之手 | 体制/权力的具体化身 |
| 牺牲者 | 因主角选择而付出代价，道德重量 |

### 禁止事项
- 禁止神秘主义内容（预言、神迹、超自然力量）
- 禁止科幻内容（技术奇点、未来科技）
- 所有角色行为动机必须在历史现实主义范围内

---



每个角色包含四项：
1. **私人欲望**：与数术命框架完全无关的那个东西
2. **从不说出口的那句话**：可能永远不出现在正文里，但出现在每一个场景里
3. **最害怕的具体画面**：不是抽象的"失败"，是一个具体场景
4. **会为之背叛原则的那一件事**：他的底线在哪里断裂

## 主要角色

### 萨尔贡（第1纪·尼普尔书吏）

- **私人欲望**：想拥有一块属于自己的土地，种麦子，不再为任何人计数
- **从不说出口的那句话**："我其实不在乎数字是否正确，我只是不想被打。"
- **最害怕的具体画面**：看到自己的儿子坐在泥板前，用同样的姿势抄写同样的数字
- **会为之背叛原则的那一件事**：伊什美的性命被用来威胁他

---

### 伊什美（第1纪·河边老人的学生）

- **私人欲望**：想成为一个让人记住名字的人，不是因为他做了什么，而是因为他是谁
- **从不说出口的那句话**："我跟着老人，不是因为我相信他说的，而是因为他是第一个问我想要什么的人。"
- **最害怕的具体画面**：被带走时回头，看到萨尔贡站在原地没有动
- **会为之背叛原则的那一件事**：如果老人要求他，他会做任何事

---

### 河边老人（第1代守护者·CHAR-023原型）

- **私人欲望**：想在死之前看到一个不需要他的人——真正学会了，不再需要他
- **从不说出口的那句话**："我教的不是数字，我教的是怀疑。但我自己也不确定怀疑是否够用。"
- **最害怕的具体画面**：学生用他教的方法，为权力服务，而且做得很好
- **会为之背叛原则的那一件事**：如果他的知识会直接导致一个孩子死亡，他会选择沉默
