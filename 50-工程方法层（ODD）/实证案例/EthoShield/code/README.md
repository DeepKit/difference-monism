## 核心指标说明

| 指标 | 名称 | 理想值 | 当前效果 |
|------|------|--------|---------|
| Demographic Parity | 人口统计差异 | <= 0.1 | 0.15 → 0.02 |
| Disparate Impact | 差别影响比率 | >= 0.8 | 0.42 → 0.88 |
