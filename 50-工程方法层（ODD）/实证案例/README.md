# ODD 实证案例目录说明

> **定位**：本目录收纳 ODD 的案例实验、代码样本、任务记录与概念验证材料。
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）
>
> **当前状态**：本目录属于案例实验与概念验证工作区，不属于 ODD 首轮公开主包。

## 边界

- 本目录中的样例代码、测试脚本、任务记录和计划稿默认按研发件理解
- `EthoShield/`、`Runtime/` 与大量 `*.py` 样例更适合作为概念验证或教学辅助，而非正式理论包
- 若对外正式公开 ODD，应优先发送主论文、白皮书、发布说明和根入口
