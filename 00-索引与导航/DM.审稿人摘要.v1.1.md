# 差异一元论 · 审稿人摘要（v1.1）

> **用途**：供编辑、审稿人或外部评议者在 `1-3` 分钟内识别 `DM` 的最小主张、失败线与证据入口。
>
> **定位**：`送审/征评辅助页`，不是成熟理论宣言。

---

## 可宣称 / 不可宣称

`可宣称`：

- `DM` 已形成统一主链、失败条件接口与首批案例档案，可被定位为研究纲领型送审/征评包
- 意识、制度、文明三域都已有第一层公开引用起点，但默认只支持 `限定性主论证`
- 制度域与文明域的第一层案例已补入 `四项硬字段`，公开比较句只有在回指 `P99.27` 正式记录时才允许下判断；当前已形成 `八组` 正式记录

`不可宣称`：

- `DM` 已经完成同行评审、长期外部验证或意识问题的终局闭合
- `DM` 已经用整包材料证明某种制度/文明路径总体最优
- `AI 回评` 不可被当作外部背书；它只应被视为 `内部预审工具 / 外部征评辅助`

---

## 三个主张

1. `本体地基`
`DM` 把世界写成 `差异 -> 结构 -> 边界 -> 继承 -> 层级涌现` 的主链；`非任意约束性` 被明确写成最低形而上学承诺，而不是纯方法方便语。

2. `意识接口`
`DM` 在意识问题上主张最小内部性、自指、回卷继承与主体级整合构成“可感之位”的最低门槛；并把“完整第三人称翻译不可得”写成可受击败的负面边界，而不是终局闭合。

3. `制度/文明接口`
`DM` 把 `续存` 定位为 bearer condition，而非最高价值；其制度与文明增量主要在于：把 `纠错能力 / 纠错载体 / 退出兑现度 / 深度伤害` 压成最小可复核字段，并要求跨案例比较回指 `P99.27` 正式记录。当前第一层制度/文明案例已随案落下这些字段，比较中间层已有 `八组` 正式记录。

---

## 三条失败线

1. `地基失败线`
若出现比 `差异 + 非任意约束性` 更薄、却同样覆盖主链的起点语言，`P00` 必须缩限或改写。

2. `意识失败线`
若纯外部模型能长期无剩余解释稳定第一人称区分轮廓，并吃掉 `内部区分 / 记忆沉积 / 后续选择绑定`，`P04` 的最小内部性需要重写。

3. `制度/文明失败线`
若高纠错结构在跨案例、跨时段中稳定构成竞争劣势，或永久例外结构在关键指标上系统优于可复归结构，`P09` 的核心判断必须缩限或改写。

## 投稿策略

- `不要整包直投`
整套 `DM` 更适合作背景包、征评包与理论定位包。

- `优先单焦点论文`
更适合先拆成：`P04 最小内部性 / 第三人称翻译负面边界`，或 `P09 纠错能力核心性 / 危机复归程序` 这类单焦点稿件。

- `本摘要的作用`
帮助审稿人先抓住 `3 个主张 + 3 条失败线 + 3 个证据入口`，而不是一上来被整包体量压住。
