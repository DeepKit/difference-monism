## 一、三分钟定位

### 1.1 DM 是什么

> **DM（差异一元论）是一套以“差异-结构-边界-继承-层级涌现”为统一语法，贯通本体、心灵、认识、规范与制度分析的哲学研究纲领。**

### 1.2 DM 不是什么

- 不是强还原论
- 不是心物二元论
- 不是“万能解释器”
- 不是把“续存”偷换成最高价值
- 不是已经完成同行评审与案例核验的成熟终稿

### 1.3 DM 的自我定位

> **一个以体系化写法呈现的研究纲领。**

这意味着：

- 它已形成可连续阅读的主链
- 它已具备可争论、可受损、可缩限的核心主张
- 它仍保留若干开放接口，不冒充闭合体系

最短判断：

> **DM 已形成可进入正式送审与外部评议的理论包，但并不冒充闭合终稿。**

推荐搭配：

- DM.审稿人摘要.v1.1.md

### 1.4 可宣称 / 不可宣称

`可宣称`：

- `DM` 已形成统一主链、失败条件接口与首批案例档案，可作为送审/征评包外发
- 三域已有第一层公开引用起点，但默认只支持 `限定性主论证`
- 制度域与文明域的第一层案例已补入 `纠错能力 / 纠错载体 / 退出兑现度 / 深度伤害` 四项硬字段；公开比较句只有在回指 `P99.27` 正式记录时才允许下判断，当前已形成 `八组` 正式记录

`不可宣称`：

- `DM` 已完成同行评审、长期外部验证与整包终局定稿
- `DM` 已完成意识问题的第三人称终局闭合，或已给出制度/文明总体胜负裁决
- `AI 回评` 不可被当作外部背书；它只应被视为 `内部预审工具 / 外部征评辅助`

## 二、v1.1 的关键升级

### 2.1 P06：认识论收紧

本版新增了三项关键内容：

1. `认知偏差三重约束`
能量约束、持存筛选约束、模型-世界差距约束

2. `生产性缺陷 / 毁灭性缺陷`
把创造性启发与结构性僵化区分开来

3. `与当代认识论对话`
显式补入与构造经验主义、enactivism、可靠主义的对话

最小可检查摘要：

- `能量约束` 解释为什么有限主体会系统性依赖压缩、启发式与低成本认知路径
- `持存筛选约束` 解释为什么认知并不总是最精确，但长期自毁型模式会被结构性淘汰
- `模型-世界差距约束` 解释为什么知识不可能无损复制世界，而只能持续管理误差

边界声明：

> **P06 直接解释的是单体认知何以必然受限；科学共同体、同行评议与重复实验则属于建立在这种受限性之上的社会认识论纠错结构，而不是被 P06 单章单独闭合。**

核心定位：

> **认识不是镜像复制，而是有限主体在扰动中对稳定约束作出的边界化描述。**

### 2.2 P10：传统对话收紧

本版新增并收紧了：

- 与结构实在论、过程哲学、现象学、缘起论的差异定位
- 与 illusionism 的真实分歧点
- 与 enactivism 的最低共识与最小扩展判据

最小可检查摘要：

- `与 illusionism 的分歧` 不在于是否自然化，而在于解释重点放在“为何会有这类内省判断”还是“什么结构产生了内部性”
- `与 enactivism 的分歧` 不在于是否承认耦合，而在于 DM 只在满足 `负熵维持 + 反馈修正 + 边界自维` 时，才把非生物系统纳入最小 enactive 候选

核心定位：

> **DM 的新增性不在于发明全新口号，而在于把多条传统资源重新接线为跨层主链。**

### 2.3 P02 / P11：不可还原性跃迁与开放边界分工

本版已把：

- `三个不可还原性跃迁` 的完整论证稳定放入 `P02`
- `P11` 收紧为开放边界、研究纲领地位与接口管理

分工纪律：

> **P02 负责正式主张，P11 负责边界管理；P11 不再重复承担主链承重论证。**

### 2.4 P11：物理接口降格

物理接口在本版中只保留为：

- 非主链承重的概念接口
- 可继续检验的候选方向
- 不反向支撑 P00-P11 主链合法性的开放出口

核心定位：

> **物理映射是预留出口，不是当下地基。**

纪律补句：

> **即便这条物理接口最终失败，被关闭的也只是 RM→DM 的一条扩展路径，而不是 P00-P10 主链的概念合法性；后者立足于最小操作化起点、主链一致性与概念经济性。**

## 四、这轮最值得外部评审压的五个问题

1. `同一语法`
它现在是否已进入最低构成纪律，而不只是描述统一

2. `最小内部性`
它是否仍只是 GNW / IIT / 功能主义的换名版本，还是已给出保守但明确的边界裁决

3. `P06 的认识论收紧`
三重约束是否足够解释认识边界，而又未滑入过强的适应主义

4. `P10 的传统对话`
它是否真的在和真实对手对话，尤其是 illusionism 与 enactivism

5. `制度 / 文明分析`
DM 的增量是否真的会改变诊断分辨率，而不仅是给既有判断换一套语言

1. `差异起点的最薄合法性`
2. `偏序非对称性的最终形式化`
3. `感质闭合的最后一步`
4. `价值锚点与制度案例的更强桥接`
5. `反向约束的量化阈值`
6. `物理接口的非承重式验证`

最短纪律：

> **承认开放边界不是缺陷；把开放边界伪装成已闭合，才是送审风险。**
