## 一、三分钟定位

### 1.1 DM 是什么

> **DM（差异一元论）是一套尝试用“差异-结构-边界-继承序-层级涌现”同一语法，贯通本体、心灵、认识与规范的研究纲领。**

### 1.2 DM 不是什么

- 不是强还原论
- 不是心物二元论
- 不是“万能解释器”
- 不是把“续存”偷换成最高价值

### 1.3 当前最小新增性

1. `时间性重写`
把时间箭头退回到“痕迹继承进入后续成立条件”的偏序结构

2. `意识定位`
把意识写成“最小内部性 + 四层半收窄”，不伪装第三人称闭合

3. `规范与制度生成链`
把规范、责任、权利、尊严与危机治理放回主体间结构、记忆载体、角色载体与程序载体的生成链条中处理

### 1.4 到 v0.9.8 前已完成的收紧

1. `弱本体一元论`
降格为背景公设，不再冒充最强新增点

2. `同一语法`
已补“弱强度”说明与跨域桥接规则，避免被读成空泛修辞

3. `四层半模型`
已补“硬收窄 / 软收窄”区分，并明确单一案例不能压住整套模型

### 1.5 当前停线的务实标准

本包当前追求的，
不是把一切问题一次性闭合，
而是先达到两条最低线：

1. `解释世界`
它能比主要竞争框架更稳定地解释自己最看重的关键现象组

2. `指导实践`
它能在责任、制度、危机与文明判断上给出可复查、可争论、可操作的最小模板

### 1.6 v0.9.8 本轮新增的四个硬收紧

1. `同一语法 → 从排除线继续压成构成约束`
已不只说“复杂整合 / 协调稳定 / 秩序外形”不能直接上升，而进一步写成：缺少某些结构位置时，某些高层现象不是“暂未出现”，而是“原则上不能稳定成立”

2. `制度域 → 用 ASTO 的五态 / 六阶 / 七序 加第二层判据`
已明确制度健康不只看纠错载体列表，还看：有没有 `定向态`、危机能否从 `脉冲` 回到 `归元`、以及 `回溯 / 消解` 是否仍被制度保留

3. `主张 B / C 的必要条件句继续保留`
继续把最小内部性与规范力量写成缺位即不得上升的构成句，而不只保留软性的表述边界

4. `实践裁决的差异对象更明确`
继续坚持 D/E 不是先看制度外形，而是先看 `纠错载体是否进入定向态`、`危机是否可从脉冲回到归元`、以及 `回溯 / 消解` 是否被封死，防止实践判断退回外形自由主义

### 1.7 v0.9.8 的最短阅读提醒

若你只看这次最关键的新增，请优先看四处：

## 二、这次请重点评什么

请优先评以下四件事：

1. `主链是否清楚`
2. `最小新增性是否真实`
3. `失败线是否真实`
4. `案例与案例组是否足以压住它声称支持的内容`

### 2.1 同一语法的最低桥接纪律

DM 当前不把“同一语法”理解成各域内容完全同一，
而只要求五类结构角色可以稳定对齐：

1. `差异单位`
2. `边界判据`
3. `继承方式`
4. `约束传递`
5. `层级生成`

最短定义：

> **同一语法不是共享词汇，而是共享“缺什么结构位置，就长不出什么高层现象”的构成纪律。**

最小跨域例子：

| 结构角色 | 心灵域 | 规范/制度域 |
|----------|--------|-------------|
| 差异单位 | 内部状态、记忆痕迹、感受区分 | 行动类型、角色位置、承诺状态、程序节点 |
| 边界判据 | 哪些状态属于同一主体的内部处理链 | 哪些成员、行动与裁决属于同一共同体或同一程序 |
| 继承方式 | 先前状态如何进入后续感受与选择 | 先例、承诺、档案、裁决如何进入后续允许/禁止/追责条件 |
| 约束传递 | 一个内部状态如何改写后续可行路径 | 一次破约、一次裁决、一次例外如何改写后续制度路径 |
| 层级生成 | 分散处理何时形成主体级整体 | 零散互动何时形成稳定规范、组织或制度 |

若某个跨域说法连这五格都填不出来，
DM 认为它只能算启发性类比，
还不算真正桥接。

但 DM 当前不再把这五格只当事后填表工具。

更直接的最低生成 / 构成约束是：

1. `意识域`
若缺少 `稳定边界 + 内部可区分 + 继承回卷 + 主体级整合`，
则最小内部性不可能稳定成立；
复杂整合、功能拟态或输入输出可模拟，
都不得直接上升为主体级内部性

2. `规范域`
若 `公共标记 + 角色位置 + 程序载体` 长期缺位，
则规范力量不可能跨成员替换持续成立；
协调稳定、模仿、惯例与短期均衡，
都不得直接上升为规范力量

3. `制度域`
若 `边界改写权 + 错误识别权 + 退出解释权` 长期并到同一节点，
并同时封死 `回溯 + 消解`，
则健康纠错不可能长期成立；
秩序稳定或高执行力，
都不得直接上升为制度健康

因此 DM 当前至少排除三种偷换：

```text
复杂整合 = 内部性
协调均衡 = 规范力量
秩序稳定 = 制度健康
```

更短的总括句是：

> **DM 的同一语法若要成立，就不能只会事后填五格；它还必须说明：当哪些结构位置缺位时，某些高层现象不是“暂未出现”，而是“原则上不能稳定成立”。**

## 三、本次只交付五条核心主张

### 主张 A：弱本体一元论

> 不为不同层级另设断裂的本体起点，但也不承诺高层可被低层无剩余还原。

当前地位：

- `背景公设 / 最小附加假设｜结构描述`

失败线：

- 若高层属性的约束边界始终无法映射到低层结构属性，而 DM 又只能诉诸彼此断裂的额外起点，则其弱一元论需缩限为更弱的层级并存论

### 主张 B：意识 = 最小内部性 + 四层半收窄

> 意识不是第三人称闭合对象，而是最小内部进入的结构前提；最小内部性不是粒子级微体验，而是某类边界化、自相关并可继承的组织达到阈值后才出现的结构属性。当前最低候选阈值至少要求：稳定边界、内部状态可区分、这些状态进入自身后续成立条件、并在主体级形成整合。具体感质之所以是“这种而非那种”，需经硬收窄、谱系沉积与软收窄的多层收窄。

补充区分：

> **DM 当前不把“高整合”“全局可利用”或“输入输出上可模拟”直接等同于最小内部性。它坚持的额外条件是：该组织要把自己的前态差异，持续卷回自己的后续成立条件，并形成主体级整合。**

本体地位收紧：

> **DM 当前把最小内部性写成“阈值随附的高层组织事实”：它不另加第二种神秘质料，但也不把第一人称内部进入降格成任意的第三人称重命名。第三人称结构条件给出的是构成门槛；第一人称则是同一结构在达到该门槛后不可互换的内部进入方式。**

再收紧一句：

> **DM 当前在这里真正想解释的，不是“凭空生成了一块心灵质料”，而是“何时出现了一个不能被第三人称进入方式无剩余替代的内部进入位”。**

再压成 ASTO 语言：

> **第一人称不是额外内容，而是同一组织在扰动链连续性中的内部递归切片位；第三人称则总经由行为、报告、信号、文本与仪器读数等介质形成外部切片。第三人称可以对应、压缩、预测甚至高精度模拟第一人称轮廓，但仍不能无剩余替代第一人称，因为它改变不了自己所处的切片位置。**

边界案例的当前保守裁决：

```text
火焰：不是强候选
蚁群整体：更像强协调系统，而不是单一主体强候选
当前大语言模型：不应仅凭自我报告与流畅互动直接赋予完整道德地位
```

但 DM 也约束自己：

> **若未来非生物系统在持续时窗内同时满足四项阈值条件，DM 不得因其非生物身份而把它排除出候选区。**

当前地位：

- `解释性模型 + 开放边界｜结构描述`

失败线：

- 若纯外部功能模型可长期无剩余解释第一人称区分轮廓，则最小内部性需重写
- 若四层半模型关键层长期失配，则桥接需缩限或放弃

### 主张 C：规范发生可自然化，但规范力量不可被纯收益吃掉

> 规范可从重复互动、记忆与归责中发生学地长出；但规范一旦形成，其约束地位不能被纯收益语言无剩余吃掉。这里的“规范力量”回答的是规则如何以公共位置稳定成立，而不是哪些规则因此自动正当。

最小发生链：

> **DM 当前把 `B -> C` 的最小桥梁写成：内部区分 → 他者预期 → 公共标记 → 角色位置 → 程序载体。**

补充定位：

> **这条链不是唯一历史顺序，而是最低发生语法：不同文化里几步可以重叠、压缩或由同一机制同时承担；但若这五类结构位置长期都没有稳定占位，规范力量就仍会悬空。**

这条链的用途，
不只是解释“规范怎么长出来”，
还要区分：

```text
什么只是协调/模仿/惯例
什么才进入规范力量
```

最低机制压缩图：

```text
反复经历的损伤/承诺差异
→ 在主体内形成稳定显著性
→ 显著性与他者反应逐步配对
→ 这种配对被外化为可反复调用的痕迹
→ 痕迹与角色位置绑定
→ 角色位置被程序顺序稳定化
```

在人类场景里的最低补充机制：

> **DM 当前不要求主体先有成熟的命题式心理理论；更低门槛的桥接是：经历历史的同源性，使某类损伤、羞耻、失信与合作破坏先形成稳定显著性；预判失败留下的伤痕，使主体开始学习“他者会如何回应”；而责备、安抚、奖励、示范、命名与记过等教育表达的一致化，则把这种反应预期逐步外化为公共标记。**

补一个最低前提：

> **DM 当前不要求主体先拥有成熟的命题式心理理论；更低门槛的机制是：内部显著性先与他者反应模式稳定配对，再逐步上升为他者预期。共同注意、反应耦合与情感共振，是这一步的最低候选机制。**

再收紧成三个机制槽位：

> **这三者在 DM 里不是经验例子堆叠，而是三个最低机制槽位：同源性提供可比较的内部显著性底板；伤痕提供对他者反应的前瞻学习；教育外化则把这种学习压成可重复调用的公共命名与见证。**

因此 DM 与单纯社会学习/模仿理论的最小差异是：

> **社会学习主要解释“行为如何被学会”；DM 在这里要解释的是：内部显著性如何被压进主体间空间，并最终沉积成可追责的位置结构。若没有可比较显著性、前瞻性反应学习与公共命名/见证，规范力量就上升不到公共标记、角色位置与程序载体。**

一个最小微观模板：

```text
儿童先把疼痛、羞耻、信任破坏经历为内部区分
→ 再预期他者会责备、拒绝或要求补偿
→ 禁令、道歉、奖励、记过成为公共标记
→ 父母、见证者、违背者、受害者出现角色分化
→ 最后才沉积为家庭、学校或共同体的稳定程序
```

最短说明：

> **DM 在这里不是说“规则自己长出来”，而是说：内部经历必须一步步外化为可预期、可见证、可归责、可程序化的公共结构。**

最低排除句：

> **若一条规则只在即时收益一致时有效，人员一换就散，且没有公共标记、角色位置与程序痕迹，它在 DM 中应被判为协调稳定或惯例，而不是规范力量。**

再补一刀：

> **收益语言最多解释为什么共同体愿意设立誓言、档案、法庭与角色分工；规范效力则解释这些载体一旦被公开占位后，违背为何会持续改写追责位置。前者回答“为何设立”，后者回答“设立之后何以仍有效”。**

公共标记的最小识别标准：

1. `第三方可调用`
2. `跨时间可稳定重复`
3. `与角色位置发生绑定`

因此：

```text
长老口头传统可以是公共标记
区块链账本可以是公共标记
无法被第三方调用的一次性私人默契，不算强意义公共标记
```

当前地位：

- `解释性模型｜结构描述`

失败线：

- 若记忆载体、角色载体与程序载体都能被纯收益描述无剩余替代，则该主张需缩限或重写

### 主张 D：续存是承载前提，不是最高价值

> 文明分析离不开续存，但尊严、权利与悲剧性牺牲限制了“为了续存什么都可以做”。价值锚点不是由结构自动推出，而是 DM 在识别高损伤与去人格化路径后作出的显式规范选择。

当前地位：

- `[—｜价值选择]`（以结构识别为支撑，不冒充纯推出）

失败线：

- 若结构识别长期不能稳定区分“可逆损伤”与“制度化去人格化/承诺结构撕毁”，则该价值锚点的结构接口需重整；但价值锚点本身属于显式选择，不冒充经验必然

最小实践裁决模板：

```text
第一步：先公开说明冲突的是哪几个价值锚点
第二步：再区分可逆损伤、不可逆损伤、去人格化伤害与承诺结构撕毁
第三步：裁决时优先避免不可逆的深度伤害
第四步：任何临时覆盖都必须保留复审、申诉与恢复路径
```

最短说明：

> **DM 当前不给价值冲突的总算法，但要求公共裁决至少满足：冲突可说明、损伤可比较、优先避免不可逆去人格化、并保留恢复通道。**

若双方都涉及不可逆伤害，
DM 当前的最低否决序是：

```text
先排制度化去人格化路径
再排永久封死复审与恢复的路径
再比范围 / 深度 / 持续时间 / 是否仍留部分恢复空间
若仍不可消解，则承认悲剧冲突，并公开说明、留档异议、设置期限与补偿
```

### 主张 E：危机压缩允许，但必须可复归

> 危机中可以临时压缩权力，但退出判定权、复审触发器、举证责任与恢复通道不能由压缩权力单方垄断。

DM 的结构理由：

> **若危机压缩把边界改写权、错误识别权与退出解释权永久集中到同一节点，共同体就会同时失去外部误差源、复审链与未来路径空间。可复归性因此不是装饰，而是防止临时压缩沉积为永久继承序重写的最低条件。**

这不是直接按自由主义制度外形下判。

DM 当前先看的是：

```text
错误可见
记忆归档
复审触发
反制衡
回退恢复
```

也就是纠错载体拓扑。

因此：

```text
形式上自由但纠错载体塌空的制度
不因外形自由而自动健康

形式上高控制但仍保留独立归档、复审触发与局部回退的制度
也不因高控制而自动被打成最低档
```

若借 ASTO 的工具箱再压一层，
DM 当前在制度域更具体的判定对象是：

1. `五态`
先看纠错载体有没有从 `编码态 / 物化态` 上升到 `定向态`

2. `六阶`
再看危机压缩能否从 `脉冲` 回到 `归元`

3. `七序`
最后看制度是否仍保留 `回溯 / 消解`

因此最短压缩句可以写成：

> **制度健康的分界，不在于外形是否开放，而在于纠错载体是否上升到定向态、危机能否从脉冲回到归元、以及回溯与消解是否仍被制度保留。**

当前地位：

- `条件性规范`

失败线：

- 若跨可比案例与足够长时段反复显示：永久例外结构在误差敏感度、外部复审强度、不可逆伤害率与退出兑现度这四项上系统优于可复归型危机压缩结构，则危机纪律需重写

补充说明：

- `外部复审强度` 指独立于压缩权力本身的信息源、审查链或制裁链的强度
- `系统优于` 当前至少按“跨时段、跨可比案例中三项持续占优，且剩余一项不出现高幅劣势”理解
- `跨文明比较纪律`：先识别各文明自身的纠错载体，不得把现代公开复审通道预设为唯一标准；但完全依附于同一压缩节点、不能触发复审与回退的机制，不计为强意义上的外部复审

ASTO 二次判据：

- 若制度长期没有 `定向态`，则不应高估其长期健康性
- 若危机措施只能不断续期，却始终不能从 `脉冲` 回到 `归元`，则“临时性”应被降级理解
- 若制度系统性压制 `回溯 / 消解`，则不应把其高执行力误判为高健康度

纠错载体最小识别清单：

1. `错误可见载体`
谁能把错误、损伤与偏差说出来

2. `记忆归档载体`
谁把这些偏差存成档案、史册、先例或统计

3. `复审触发载体`
谁有能力把问题从执行层重新送回审查层

4. `反制衡载体`
哪些力量能在压缩节点之外形成实质阻力

5. `回退恢复载体`
哪些机制能把临时压缩真实拉回常态边界

补充限制：

> **某个载体即使形式上隶属于统治中心，只要它在实践上能触发修正、回退或复审，仍可计为有效纠错载体；只有当它完全无法启动任何回退时，才应计为“无强意义外部复审”。**

最小实务分流：

1. `可接受的临时压缩`
至少三类载体为强，且 `复审触发` 与 `回退恢复` 不能同时为弱/无

2. `高风险临时压缩`
若干载体仍在，但 `复审触发` 或 `回退恢复` 已明显弱化

3. `实质永久例外`
若 `复审触发` 与 `回退恢复` 同时失效，或边界改写权、错误识别权、退出解释权被长期并到同一节点

比较纪律再收紧一刀：

> **即使某结构在局部效率或表面续存上占优，只要它持续制造更高的不可逆伤害率，或长期拒绝退出兑现，DM 当前也不把它轻率记为“系统优于”。**

## 四、三域重点问题

### 4.1 意识域

请优先判断：

1. `最小内部性` 是否只是换名
2. `四层半收窄` 是否真有解释增量
3. 若把麻醉、裂脑、盲视、no-report、联觉与跨文化类别知觉一起看，DM 是否真的比 `GNW / IIT / illusionism / 粗糙功能主义` 多解释了什么
4. `最小内部性` 是否有清楚的阈值位置，而不是滑向泛心论或黑箱突现

### 4.2 规范域

请优先判断：

1. `规范发生` 与 `规范力量` 的区分是否成立
2. 规范力量是否真的拥有独立于收益计算的结构地位
3. 尊严/权利的结构识别与价值选择之间，边界划得是否够清楚
4. 个体的最小内部性如何外化、沉积并结晶为公共的记忆/角色/程序载体

### 4.3 制度/文明域

请优先判断：

1. `纠错能力` 是否真是文明续存分析中的核心变量
2. `高控制低纠错` 的长期风险排序是否过强
3. `危机压缩必须可复归` 是否只是自由主义常识改写，还是有 DM 自己的结构理由
4. 高控制但局部高纠错的案例，是否会迫使 DM 进一步细分文明比较模板

- 整合相关神经指标变化
