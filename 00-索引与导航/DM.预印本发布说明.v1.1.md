# DM 预印本发布说明（v1.1）

> **适用场景**：把 DM 作为 `预印本 / 征评版 / 占位版` 公开发布时使用。
>
> **推荐口径**：`v1.1 统一送审版 / 征评版 / 预印本`
>
> **不推荐口径**：`最终定稿 / 已完成同行评审 / 已外部充分验证的闭合理论`
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）

## 3. 推荐摘要

中文摘要建议：

> 差异一元论（Difference Monism, DM）是一套以“差异、结构、边界、继承与层级涌现”为统一语法的哲学研究纲领。它试图在不预设实体本体论、心物二元论或终极目的论的前提下，重写存在、时间、规律、因果、意识、认识与规范之间的连续主链。版本 `v1.1` 为统一送审版预印本：已形成可连续阅读的主干、明确的失败条件接口与最小外部征评包，但仍处于公开征评、批评吸收与长期验证阶段，不应被表述为已完成同行评审的终局理论。

英文摘要建议：

> Difference Monism (DM) is a philosophical research program that uses difference, structure, boundary, inheritance, and layered emergence as a unified grammar across ontology, mind, knowledge, and normativity. It aims to reconstruct the continuity between existence, time, lawfulness, causality, consciousness, cognition, and normative order without presupposing substance metaphysics, mind-body dualism, or teleological closure. Version `v1.1` is a unified public preprint: it already provides a readable main chain, explicit failure conditions, and a minimal external review package, but it remains a public review edition rather than a finalized, externally validated theory.

## 7. 平台上传纪律

- 主对象只放一个，不同时并列多个“官方版本”。
- 摘要中明确写出 `预印本 / 征评版 / 统一送审版`。
- 不把内部案例档案、工作流和历史草稿当成主文。
- 若平台支持补充材料，优先附 `理论定位申明` 与 `Glossary`。
- 若后续同步到 Zenodo，本说明中的标题、摘要与关键词可直接复用。
- 若读者在仓库或补充材料中看到 `RM / Resonance Monism / 共振一元论 / 共扰一元论`，应按早期命名阶段材料理解；当前公开主链以 `DM -> ASTO -> ECET -> TAT -> ODD` 为准。
