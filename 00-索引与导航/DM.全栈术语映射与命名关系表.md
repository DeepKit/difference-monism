## 二、历史命名与当前处理

| 历史名称 / 前缀 | 当前处理 | 当前公开用法 |
|----------------|----------|--------------|
| `RM` | 早期探索阶段的总标签 | 保留作理论溯源，不作为当前默认总品牌 |
| `Resonance Monism` | `RM` 的英文早期命名 | 仅在历史稿、早期 SCI/探索稿中保留 |
| `共振一元论` | `RM` 的中文早期命名之一 | 保留作历史痕迹，不作为当前公开主名 |
| `共扰一元论` | `RM` 的中文早期命名之一 | 保留作历史痕迹，不作为当前公开主名 |
| `MFO / 互扰流显` | 更早期的探索性技术表述 | 不作为当前公开正典口径 |
| `RM.TAT.*` | 历史文件名前缀保留 | 仅作路径兼容，不表示当前主品牌回切到 `RM` |
| `RM.ODD.*` | 历史文件名前缀保留 | 仅作路径兼容，不表示当前主品牌回切到 `RM` |

最小判断规则：

1. 如果讨论整个当前公开体系，优先使用 `差异一元论体系` 或 `DM -> ASTO -> ECET -> TAT -> ODD`。
2. 如果讨论哲学核心层，使用 `差异一元论（DM）`。
3. 如果看到 `RM / Resonance Monism / 共振一元论 / 共扰一元论`，默认按“早期历史命名阶段材料”处理，除非正文明确声明其仍为当前主对象。

| 层级涌现 | 跨场景桥接 / 前置字段 | 文明层治理判断 | 高影响系统门槛 | 诊断压缩 / 误判控制 | 工程执行闭环 | 首次责任接近 |
| 失败条件 | 例外 / 边界破裂 | 失稳 / 失配 / 失控 | 责任外部化 | 冻结 / 转介 / 复核 | 回滚 / 召回 / 审计 | 责任下滑 / 蔓生 / 引爆 / 沉积 / 割裂 |

这张表的作用不是把五层压成同义反复，而是说明：

- 下层不会替代上层
- 下层是在上层约束下做语法压缩
- 每一层都回答不同问题，但共享同一条“差异 -> 结构 -> 边界 -> 约束 -> 闭合”的连续线索
- `ASTO` 在这里承担的关键职责，是让这些层共享同一组前置字段，而不是各自发明平行状态语言

## 四、当前推荐公开口径

### 4.1 整体体系

推荐：

> **差异一元论体系：`DM -> ASTO -> ECET -> TAT -> ODD`**

不推荐：

- `整个体系当前统一叫 RM`
- `共振一元论仍是默认总品牌`
- `RM、DM、ASTO、ECET、TAT、ODD 只是同一层的多个别名`

### 4.2 分层发布

推荐：

- `DM` 讲哲学地基
- `ASTO` 讲跨场景应用桥接
- `ECET` 讲文明与治理判断
- `TAT` 讲责任架构与责任闭合
- `ODD` 讲工程验证与执行闭环

不推荐：

- 把 `ECET / TAT / ODD` 重新包装成与 `DM` 平行竞争的终极本体论
- 把历史 `RM` 文件名前缀误读成当前公开层级名称

## 五、可直接复用的最短定位句

- `DM`：差异一元论（DM）是一套以差异、结构、边界、继承与层级涌现为统一语法的哲学研究纲领。
- `ASTO`：ASTO 是 DM 向多场景分析、审议、设计与裁决延展时形成的应用桥接框架，不只适用于 AI，并负责给下游层提供统一结构编码与前置字段。
- `ECET`：ECET 是建立在 DM 与 ASTO 之上的文明演化、治理判断与元治理框架。
- `TAT`：TAT 是建立在 DM、ASTO、ECET 之上的责任架构层，讨论高影响系统的责任门槛与责任闭合。
- `ODD`：ODD 是建立在 DM、ASTO、ECET、TAT 之上的工程方法层，把责任闭合落到 AI 产出物治理。
- `COP`：COP 是接触复杂现实后的诊断压缩与风险分流协议层，允许冻结、未知、转介与复核。
- `LMM`：LMM 是现实前线的显影、接引与首次责任接近层，不等于传统营销漏斗。
- `Harness`：Harness 是受控执行环境，负责把上层协议落成可拦截、可留痕、可回滚的运行时。
- `WSM`：WSM 是 `LMM` 内部的前线结构识别子框架，用于把责任流向失衡压成局型标签与弱测量语言。

## 六、文件名与发布纪律

1. 文件名残留旧前缀时，优先在正文前部补一句“历史文件名前缀保留，用于兼容既有引用”。
2. 预印本、Zenodo、公开征评与平台摘要中，优先使用当前层级名，不用历史 `RM` 作为整个体系总名。
3. 若必须提到历史名称，建议固定写法为：

> `RM / Resonance Monism / 共振一元论 / 共扰一元论属于早期命名阶段材料，当前公开主链以 DM -> ASTO -> ECET -> TAT -> ODD 为准。`
