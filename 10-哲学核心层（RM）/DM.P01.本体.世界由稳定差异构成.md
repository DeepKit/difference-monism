# 差异一元论 · 本体
## Difference Monism: Ontic Structures (v5.0)
### DM.P01 · 世界由稳定差异结构构成

> **世界不是由静态物体构成，而是由能够维持自身的差异结构构成。**

---

## 一、P01 的位置

P00已经给出最小地基：

```
差异
→ 结构
→ 边界
→ 持存
→ 时间
→ 规律
→ 因果
→ 涌现
```

P01只做一件事：

> **把“世界是什么”改写为“世界由什么样的结构构成”。**

因此，P01不是文明论，也不是价值论。
它只处理本体层问题。

---

## 二、从物体本体论转向结构本体论

传统物体本体论往往默认：

```
世界先有实体
实体天然有边界
关系只是实体之间的后加连接
```

DM反过来主张：

```
先有差异
差异形成稳定关联
稳定关联形成结构
结构在边界中持存
于是“物”才出现为结构的可识别截面
```

所以：

> **物不是第一性的；物是稳定差异结构的显现形式。**

---

## 三、什么叫“稳定差异结构”

### 3.1 最小定义

> **稳定差异结构 = 在边界条件下，能够反复维持其内部关联方式的差异网络。**

这里的关键词有三个：

1. `差异`
没有差异就无可区分

2. `关联`
没有关联就只有散乱分布

3. `维持`
没有维持就没有存在者，只剩瞬时噪动

### 3.2 为什么“稳定”不等于“不变”

稳定不意味着永不改变。

更准确地说：

```
结构内部可以变化，
但不能在变化中完全失去其可识别方式。
```

因此：

* 冰山融化，结构消失
* 细胞代谢，结构仍续行
* 语言变迁，语法骨架仍可持存

稳定讲的是：

> **变化中的可识别延续。**

---

## 四、世界如何由结构而不是由物组成

### 4.1 “物”只是高压缩称呼

当一个差异结构满足三件事时，
我们就倾向把它当作“一个物”：

1. 有相对清楚的边界
2. 有相对稳定的内部关联
3. 在一定时段内能维持可识别性

因此：

```
物
= 对稳定差异结构的日常压缩命名
```

### 4.2 世界更像结构网络

如果把“物”重新展开，
就会发现世界更接近：

```
结构套结构
网络嵌网络
边界叠边界
```

不是彼此绝对隔离的硬块集合。

所以：

> **世界更像稳定差异结构的层级网络，而不是一袋彼此外在的实体。**

---

## 五、边界、关系与存在者

### 5.1 边界不是外加线条

存在者之所以成其为一，
不是因为有人在外面画线，
而是因为：

```
内部关联强度
高于
与外部的关联强度。
```

这使得一个结构能暂时保持自身。

### 5.2 关系不是次要的

传统观点容易把关系看成次要项：

```
先有物
后有物与物之间的关系
```

DM则更接近：

```
先有差异关系
后有在这些关系中得以稳定的存在者
```

因此关系不是附属，
而是存在者的构成条件。

---

## 六、存在者的最小定义

P01把P00中的结论重新压成一句：

> **存在者 = 具边界并能在扰动中维持其关联方式的差异结构。**

这一定义有三个含义：

1. 存在者不是绝对实体
2. 存在者不是纯主观命名
3. 存在者是本体上真实但历史上暂时的结构稳定项

因此：

```
真实
不等于永恒

暂时
不等于虚假
```

---

## 七、世界的最短改写

如果用一条最短公式表达 P01：

> **世界 = 稳定差异结构及其层级耦合网络。**

这里没有说世界是“纯关系”而无结构，
也没有说世界是“纯物”而无关系。

DM的平衡点是：

```
关系形成结构
结构表现为存在者
存在者继续进入更大结构
```

---

## 八、P01 的边界

P01只完成以下三件事：

1. 世界应从“实体集合”改写为“稳定差异结构网络”
2. 存在者应定义为“具边界、可持存的差异结构”
3. 关系不是次要项，而是存在者成立条件

P01不在此处展开：

* 生命如何出现
* 暂稳态为何优于刚性稳定
* 意识、价值、规范如何形成

这些分别后移到 P02、P03、P04-P09。

---

## 九、结语

P01的核心判断只有一句：

> **世界不是先有物再有关系，而是差异关系先形成结构，结构再显现为我们所说的“物”。**
