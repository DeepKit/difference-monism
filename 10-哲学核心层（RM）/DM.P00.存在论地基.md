## 二、起点：差异

### 2.1 差异不是对象之间的比较

差异不是：

* 两个已成形对象之间的不同
* 主体对对象的识别结果
* 预设 A 与 B 之后的外加关系

差异是：

```
在一切对象化之前，
非同一性自身给出的张力。
```

因此，DM的顺序不是：

```
对象
→ 属性
→ 差异
```

而是：

```
差异
→ 区分
→ 关联
→ 结构
→ 边界
→ 存在者
```

### 2.2 为什么必须从差异开始

如果完全无差异，则：

```
无可区分
无可关联
无可描述
```

所以差异不是某类存在者的属性，
而是存在得以显现的最小前提。

可以压成一句：

> **差异先于实体；实体只是差异结构的后成结果。**

---

## 三、差异为什么能形成结构

### 3.1 差异不是结构的完成态

只有差异，还不足以构成结构。

散乱差异只能给出：

```
可分
但未成形
```

结构则要求：

```
差异之间不是单纯并列，
而是形成可重复的约束关联。
```

### 3.2 结构的最小定义

> **结构 = 差异项之间能够反复维持的约束关联。**

这里的“约束关联”指：

```
一个差异位置的变化，
会系统性影响其他差异位置的成立条件。
```

### 3.3 结构成立的最小条件

我建议用最简三项表达：

1. `可区分`
必须有 A 不是 B

2. `可关联`
A 与 B 之间不能毫无关系，而要有耦合、依赖、限制或排列

3. `可持存`
这种关联不能一闪即逝，而要在扰动下能够重复或续行

因此：

> **差异 + 关联 + 持存 = 结构的最小成立条件。**

### 3.4 为什么会有结构而不只有散乱

并非所有差异组合都能持续。

大量组合会：

```
快速耗散
失去关联
回到背景
```

少数组合则会：

```
在扰动中保持关联
在变化中维持可识别性
```

于是我们看到的“结构”，
正是那些在筛选中得以持续的差异关系。

可以压成一句：

> **结构不是外加形状，而是差异关系在约束筛选下得以持续的结果。**

## 四、边界：结构成其为一

### 4.1 为什么结构必须有边界

如果一个结构没有任何边界，
则：

```
它与背景无可区分
它与他物无可区分
它与自身的延展范围也无可区分
```

这意味着它不能“成其为一个”。

所以边界不是后来的认识论标记，
而是存在者成立的本体条件。

### 4.2 边界的定义

> **边界 = 结构内部关联强度与外部关联强度之间可区分的梯度。**

边界不是绝对墙，
而是：

```
内外关系权重的差异分布。
```

边界因此可以：

```
清晰
模糊
弹性
分层
重叠
```

### 4.3 边界的作用

边界至少完成三件事：

1. `使结构可识别`
它把“这一个”与“非这一个”区分开

2. `使结构可维持`
它限制结构完全溶散到背景中

3. `使结构可互动`
正因有内外区分，结构才可能与其他结构发生耦合

因此：

> **边界不是存在者的附属属性，而是存在者得以成立的条件。**

## 五、持存：存在不是静止，而是续行

### 5.1 存在的最小改写

传统本体论容易把“存在”想成静态在场。

DM改写为：

> **存在 = 结构在约束中维持其可识别性的过程。**

所以“存在者”不是永久不变的核，
而是：

```
在扰动下仍能保持某种可识别延续的结构过程。
```

### 5.2 持存的意义

持存不是永恒不变，
而是：

```
在变化中没有彻底失去自身结构。
```

因此：

* 完全静止不是存在的最高形态
* 完全散乱也不是存在
* 存在是可续行的结构稳定性

### 5.3 存在、边界、持存的关系

可以压成最短链条：

```
差异
→ 约束关联
→ 结构
→ 边界
→ 持存
→ 存在者
```

其中“存在者”不是某个额外添加的实体层，
而是上述链条的结果。

## 六、时间：结构痕迹的单向沉积

### 6.1 时间不是先验容器

DM不把时间看作先行存在的均匀容器。

时间不是：

* 外在钟表的抽象底板
* 所有存在者预先装进去的空框
* 单纯主观感受的投影

### 6.2 时间的定义

> **时间 = 结构痕迹对后续成立条件的单向沉积。**

这里的意思是：

```
一个结构状态一旦发生，
就会改写后续结构的成立条件。

这些改写不会零成本消失，
于是前后关系出现方向性。
```

### 6.3 为什么时间具有不可逆性

不可逆不是神秘属性，
而是因为：

```
已发生的结构沉积
已经进入后续条件网络；
后续路径是在这些沉积上展开的。
```

所以：

> **时间不可逆，不因为宇宙偏爱“向前”，而因为结构沉积会重写后续路径空间。**

### 6.4 时间与历史的区别

在P00里，“时间”只指单向沉积。

它先于：

```
主观记忆
文明史
显式编年
```

后面的“历史”，是时间沉积在生命、社会和文明层的高层形式。

## 七、规律性、定律与公理

### 7.1 世界中的规律性

P00只先讲最深的一层：

> **规律性 = 结构关系在边界条件下反复形成、筛选与沉积后表现出的稳定重复约束。**

也就是说，世界先有的不是公式，
而是某些关系在特定条件下会稳定重复。

### 7.2 科学中的定律

科学定律不是世界本身，
而是：

> **物理定律 = 人类对这些稳定重复约束所作的可检验、高压缩、可计算描述。**

所以定律属于：

```
对规律性的建模表达，
而不是规律性自身。
```

### 7.3 数学中的公理

数学公理也不等于世界原文。

> **数学公理 = 对高稳定形式关系所作的最小操作冻结。**

因此三者必须区分：

1. `规律性`
世界中的稳定约束

2. `定律`
科学对稳定约束的模型压缩

3. `公理`
形式系统内部的最小起点

可以压成一句：

> **世界不是由公式构成的；公式是世界中稳定关系被主体抽象、压缩和组织后的表达。**

## 八、因果：约束的可传递性

### 8.1 因果不是神秘绳子

DM不把因果理解为隐藏在事件背后的超验纽带。

因果更稳的表达是：

> **因果 = 某些结构差异通过可传递的约束链，稳定地改变后续结构成立条件。**

### 8.2 因果为什么可能成立

当一个结构位置的变化，
会系统性地改写另一个结构位置的可行条件，
而这种改写又能反复出现时，
我们就说两者之间具有因果性。

所以因果的核心不是“神秘推出”，而是：

```
约束传递
条件改写
稳定重复
```

### 8.3 因果与规律、时间的关系

三者可以收成一条链：

```
时间
= 结构痕迹的单向沉积

规律
= 沉积中反复保留下来的稳定约束

因果
= 这些约束沿结构链条发生传递的具体方式
```

可以再压一句：

> **因果不是超验力量，而是约束的可传递性。**

## 九、层级与涌现

### 9.1 为什么会出现层级

当下层结构之间形成更稳定、更密集、更可重复的耦合时，
就可能出现一个新的整体结构。

这个整体会表现出：

```
新的边界
新的内部差异分工
新的稳定条件
对下层部分的反向约束
```

这就是层级出现的最小图景。

### 9.2 涌现的最小判据

> **涌现 = 若干下层结构耦合后形成新的整体边界、整体稳定条件与反向约束。**

因此，涌现至少要求四项：

1. `下层结构可耦合`
不是纯并列堆积

2. `出现整体边界`
整体能与背景区分

3. `出现整体条件`
整体的成立不能简化为单个部分的孤立性质

4. `出现反向约束`
整体会反过来限制部分的可行状态

### 9.3 P00 对涌现只做最小到达

P00只给出层级涌现的形式条件，
不在这里完成：

```
生命的具体起源
植物与动物的细分机制
意识的神经实现
文明的制度生成
```

这些内容分别后移到 P02、P04、P05、P08、P09。

## 十一、结语

DM并不把世界理解为一组现成实体，
而把它理解为：

```
差异关系在约束中形成结构，
结构在边界中维持持存，
持存在沉积中展开时间，
时间中的稳定重复表现为规律，
规律的可传递表达为因果，
而更高层结构则在耦合中不断涌现。
```

因此，最短定义可以写成：

> **存在不是静止之物，而是差异结构在约束中的暂时续行。**
