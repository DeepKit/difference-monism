## 一、刚性稳态与暂稳态

### 1.1 稳态的谱系

在持续扰动的背景中，存在两种稳态：

**刚性稳态（Rigid Stability）：**

```
共振结构高度有序
内部变异极低
与外部的共振耦合极少
→ 转换能力 ≈ 0
→ 条件变化时脆性断裂
→ 回归背景
```

例子：
```
晶体：原子位置固定，无法适应外部变化
极权制度：压制所有内部差异，面对新扰动时脆性断裂
单一栽培：消除物种多样性，一种病菌就能导致整体崩溃
```

**暂稳态（Metastable State）：**

```
共振结构动态平衡
保持内部变异储备
与外部保持适度共振耦合
→ 转换能力 > 0
→ 条件变化时转换到新稳态
→ 持续持存
```

例子：
```
生态系统：物种多样性提供转换路径库
弹性制度：保持个体自指共振的同时实现层级耦合
科学范式：可以被新证据修正的理论体系
```

### 1.2 暂稳态的定义

> **暂稳态 = 在当前条件网络下持存 + 条件变化时可转换到新的稳态共振结构**

```
不是永久稳定（在持续扰动的背景中不存在）
不是随时瓦解（那是背景，不是存在）
是在持存与转换之间的动态平衡
```

暂稳态不是妥协，是持续扰动环境下持存的唯一可行形态。

## 二、可转换性：持存质量的度量

### 2.1 什么是可转换性

> **可转换性（Convertibility）：共振结构在条件变化时以低耗散成本转换到新稳态的能力。**

**两个维度：**

```
转换路径长度：
从当前稳态到新稳态必须经过的关系权重变化总量
路径越短，转换成本越低

可达稳态数量：
结构可以转换到的不同稳态的数量
可达态越多，适应范围越广
```

**持存质量的表达：**

$$Q(\text{持存质量}) \propto \frac{1}{L(\gamma_{\text{转换路径}})} \times \text{可达稳态数量}$$

### 2.2 长期持存胜率与转换能力

$$P(\text{长期持存}) \propto \text{转换能力}$$

**筛选逻辑：**

```
易于转换的暂稳态共振结构
↓
条件变化时完成转换
↓
继续持存

刚性稳态共振结构
↓
条件变化时无法转换
↓
脆性断裂，回归背景
```

在漫长的统计时间尺度上，背景中留存下来的共振结构，都是转换能力相对较高的结构。这是统计筛选的必然结果，不是目的论的安排。

### 2.3 回环几何与可转换性的关系

```
回环无方向性
→ 可以与任何方向的新共振结合
→ 可达稳态数量多
→ 可转换性高

回环可嵌套
→ 内部有多个子回环
→ 条件变化时子回环可以重组
→ 转换路径短
→ 可转换性高
```

可转换性是回环几何性质在时间维度上的展开。

自指共振结构从内部逻辑判断，协同路径是更有利于长期持存的策略。这个判断就是价值判断的原型。
