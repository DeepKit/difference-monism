### DM.P04 · 意识的结构前提与硬问题的重写

> **P04 说明“可感之位”，而不在此处完成“所感之相”的生物实现。**

---

## 一、P04 的位置

P00-P03 已经说明：

```
差异

→ 边界
→ 持存
→ 时间
→ 规律
→ 因果
→ 涌现
→ 暂稳态与可转换性
```

P04 继续处理的问题是：

> **在这样的世界里，什么样的结构会出现“内部性”，从而足以承载意识问题。**

因此，P04只处理结构前提：

- 内部性
- 自指
- 内部可区分性

P04不直接完成：

- qualia 的具体质地
- 生物神经实现
- 意义与价值

这些后移到 P05。

---

## 二、硬问题的重写

传统硬问题常被表述为：

```
为什么物理过程会伴随主观体验？
```

这个问法默认了一个压缩前提：

```
物理过程在一边
体验在另一边
两者先天地分离
```

DM不接受这个起点。

更稳的改写是：

> **什么样的结构会形成只能从其内部成立的状态区分。**

这样一来，硬问题就不再是：

```
两种实体如何桥接
```

而变成：

```
一种结构何时具有内部性
何时具有自指
何时具有内部可区分状态
```

---

## 三、内部性

### 3.1 什么叫内部性

> **内部性 = 一个结构对自身状态与外部扰动具有内部区分能力。**

如果一个结构只有输入输出，
却没有内部状态空间的差异组织，
它就不能说具有严格意义上的内部性。

内部性要求至少三点：

1. 有边界
2. 有内部状态
3. 内部状态之间可彼此区分

### 3.2 为什么边界是前提

没有边界，
就没有：

```
内部
外部
内外差异
```

因此意识问题的最深前提不是“大脑”，
而是：

> **存在一个具边界的复杂结构，其内部状态对该结构自身并非全然外在。**

---

## 四、自指

### 4.1 什么叫自指

> **自指 = 结构把自身状态纳入其后续处理条件的能力。**

换句话说：

```
结构不只是处理外部输入，
还会处理“自己当前处于什么状态”。
```

### 4.2 为什么自指重要

若一个结构只能处理外部刺激，
不能把自身状态折回为后续条件，
那它仍是开放反应系统，
但还不足以形成意识问题中的“我正在经历”。

自指使结构不再只是：

```
世界中的一个反应点
```

而开始成为：

```
能把自身纳入世界处理回路的中心。
```

---

## 五、内部可区分性

### 5.1 定义

> **内部可区分性 = 结构内部存在足够丰富且彼此可分的状态空间。**

这意味着：

```
不是只有“有反应/没反应”
而是有多种内部状态差异
并且这些差异会影响后续处理
```

### 5.2 为什么这一步关键

如果内部状态不能彼此区分，
则即便有边界、有自指，
也仍不会形成足够丰富的“可感之位”。

所以意识的结构前提至少是：

1. 有边界
2. 有内部性
3. 有自指
4. 有内部可区分状态空间

可以压成一句：

> **意识问题的结构前提，不是某种神秘材料，而是具边界、自指且内部可区分的高复杂结构。**

---

## 六、第一人称与第三人称

### 6.1 为什么两者不会完全重合

只要一个结构具有内部性，
外部观察者就只能经由：

```
边界
输入输出
外显行为
```

来描述它。

而该结构自身的内部状态，
则是以其内部处理方式成立。

因此：

> **第一人称与第三人称不是两种实体，而是同一结构的两种进入方式。**

### 6.2 为什么这不等于神秘主义

外部描述不能完全替代内部成立，
并不说明有超自然灵魂。

它只说明：

```
结构的内部状态
并不等于
外部对该结构的边界描述。
```

因此 DM 的立场是：

- 不把体验神秘化
- 也不把体验简化成外部描述的同义反复

---

## 七、P04 的边界

P04只在结构层得出：

> **只要世界中出现具边界、自指且内部可区分的复杂结构，就会出现“可感之位”。**

P04不在此处回答：

- 为什么是这种具体感质
- 为什么有愉快/痛苦等效价
- 为什么有意义、价值与行动导向

因为这些还需要：

- 生命史
- DNA 约束
- 神经拓扑
- 生物边界与代谢维持

所以仍保留那句分工：

> **P04 说明“可感之位”，P05 说明“所感之相”。**

---

## 八、P04 的最短结论

`意识问题首先不是两种实体的桥接问题，而是内部性如何出现的问题。`
`内部性要求边界、内部状态与内部可区分性。`
`自指使结构把自身状态纳入后续处理条件。`
`当一个结构具边界、自指且内部可区分时，就出现了承载意识的结构位置。`
`P04只说明意识的结构前提，不在此处关闭 qualia 的生物实现。`

---

## 九、结语

P04的核心立场是：

> **意识不是附着在结构上的神秘副产物，而是某类高复杂结构从其内部成立的方式。**
