# 科学对话层说明

> **定位**：本目录是当前体系面向数学、物理、生命、意识研究的科学接口层、桥接层与对话缓冲层。
>
> **作者 / Author**：Yi Fu（付毅，ODDFounder，fuyi.it@live.cn）
>
> **当前状态**：本目录是公开补充层，不是独立法源层，不直接替代 `DM` 哲学主文，也不冒充已经完成实证闭合的论文层。

## 本层负责什么

1. 把 `DM` 及其下游相关命题压成更可对话的科学接口表达。
2. 区分：
   - 候选映射
   - 结构近似同构
   - 工作性假设
   - 开放边界
3. 标出失败条件、修正条件与后续桥接任务。

## 本层不负责什么

1. 不把哲学直接推出全部科学方程。
2. 不把若干相似性直接写成“科学已被证明”。
3. 不把研究执行、文献补证与投稿包装混成同一层。

## 当前最小公共结构

- `README.md`：本层定位与阅读纪律
- `S01.科学前导.从差异结构到科学接口.md`：当前总论入口
- `S91.桥接节点总表.md`：桥接节点、当前强度与后续补证方向
- `S92.命题强度与失败条件表.md`：命题级别、失败条件与修正层级
- `S93.文献接口与对话对象清单.md`：当前应优先对话的理论、作者与研究方向

## 当前阅读路径

1. 先读 `10-哲学核心层（DM）/README.md`
2. 再读 `S01.科学前导.从差异结构到科学接口.md`
3. 再按需要查：
   - `S91` 看桥接节点
   - `S92` 看命题强度与失败条件
   - `S93` 看文献接口与对话对象

## 使用纪律

1. 若需要哲学主文，回到 `10-哲学核心层（DM）`。
2. 若需要正式论文母本、投稿稿或研究执行件，应回到论文包或研究执行位。
3. 若某条桥接当前只到 `候选映射` 或 `工作性假设`，就不能在对外文本里写成“精确对应”。

## 最短句

> **科学对话层的任务，不是替体系抢占科学结论，而是把相关命题压成更清楚、更可失败、更可比较的科学接口。**
