
而科学层处理的是另一件事：

> **这些本体论判断，如何与现有科学的对象、方法和方程形成接口。**

因此，本文不再使用旧稿中“从哲学严格推导全部科学”的表述。

当前更稳的定位是：

> **DM 提供跨层一致性的本体论框架；科学提供可测量、可计算、可修正的局部模型。**

两者的关系首先是：

> **一致性接口。**

不是：

> **哲学单向推出全部科学细节。**

## 二、为什么要把“严格推导”改成“科学接口”

旧稿最大的张力有三处：

1. 把旧主名 `RM / 共扰一元论` 当成现行名称
2. 把若干科学方程写成了哲学的直接推出物
3. 把意识问题写得过于提前、过于封闭

这些都与现行 DM 主线冲突。

因此，当前科学层必须遵守三条纪律：

1. 现行正式名称只用：
   - `差异一元论`
   - `Difference Monism`
   - `DM`
2. 从本体论到科学之间，优先写：
   - `接口`
   - `映射`
   - `一致性`
   - `近似投影`
   不轻易写成“严格推出”
3. 遇到尚未闭合的桥接，必须保留开放边界

## 三、数学、物理规律与公理，在 DM 中如何理解

### 3.1 数学不是宇宙中的神秘命令

数学更稳地可理解为：

> **对可区分结构、可重复变换与稳定关系的高压缩描述语言。**

它之所以强，
不是因为数学先验地飘在世界之外，
而是因为它抓住了：

```
可区分
可组合
可变换
可保持
```

这些最稳定的结构特征。

### 3.2 公理是什么

公理不是宇宙自己写下的终极法条。

更稳地说：

> **公理是研究者为了稳定展开某类符号结构，而选定的最小起步规则。**

一旦规则选定，
后续许多性质就不是任意捏造，
而是该结构内部的必然后果。

### 3.3 素数与哥德巴赫猜想如何理解

以自然数与整除关系为例：

- `素数` 不是人随口发明的标签
- 它是整数结构在特定可除关系下显出的稳定分层

因此更稳地说：

> **素数是某个符号结构中极难再分解的稳定差异位置。**

哥德巴赫猜想则不是“世界忽然多出一条物理规律”，
而是：

> **在既定整数结构里，一个尚未完成证明的全域分布命题。**

这类数学命题的真假，
不靠经验实验决定，
而靠：

```
定义是否固定
规则是否一致
推演是否成立
```

所以数学的“规律性”与物理的“规律性”并不相同：

- 数学规律更接近结构内必然
- 物理规律更接近经验上稳定重复的约束压缩

### 3.4 数学为什么能描述物理

因为二者都偏向抓住：

> **在变化中仍可保留的稳定关系。**

压缩地说：

> **数学研究可在变换中保持的结构，物理研究可在扰动中重复出现的约束。**

这两者并不相同，
但它们在“寻找稳定性”上深度同构。

### 3.5 数学对象的地位：结构实在，语言主体压缩

DM 对数学对象的立场可以更明确地表达为：

> **数学对象所描述的结构关系是实在的（不依赖主体而存在），但数学语言本身是主体对这些结构关系的压缩表达。**

因此 DM 既不是纯柏拉图主义（数学对象飘在世界之外），
也不是纯约定论（数学全是人的发明）。

更稳地说：

> **数学是被发现的结构关系，用被发明的符号系统表达。**

### 3.6 哥德尔不完备性与渐近线推断的极限

哥德尔第一不完备定理表明：

> **在任何足够强的一致形式系统中，总存在在该系统内部无法证明但为真的命题。**

这对 DM 的认识论（P07 渐近线推断）有直接影响：

渐近线推断在单一形式系统内部，
不是无限逼近全部真理，
而是必然遇到原则性的不可达盲区。

但 DM 可以给出一个结构性回应：

> **主体不被锁死在单一形式系统中。当一个系统的内部盲区被识别，主体可以构建更大的系统来重新逼近。这本身就是可转换性（P03）在认识论层的体现。**

因此：

```
哥德尔极限限制的是单次形式化的覆盖范围
渐近线推断的力量在于跨系统的持续重构
```

可压成一句：

> **渐近线不是在单一坐标系中无限延伸，而是在不断更换坐标系中持续逼近。**

### 3.7 具体概念映射：拓扑不变量与持存

DM 的“持存”概念——变化中的可识别延续——
与拓扑学中的不变量概念具有当前可工作的结构映射关系：

| DM 概念 | 拓扑学对应 |
|---------|-----------|
| 持存 = 跨时间变换维持可识别性 | 拓扑不变量 = 跨连续变换保持不变的性质 |
| 边界 = 内外关系权重的阈差 | 拓扑边界 = 集合与其补集的交界 |
| 可转换性 = 进入新稳态而不断裂 | 同胚 = 保持拓扑性质不变的连续双射 |
| 涌现 = 新的整体边界出现 | 拓扑空间的商空间 = 通过等价关系形成新的整体结构 |

当前更稳的说法不是“已经完成精确同构”，
而是：

> **DM 的本体论可以被工作性地理解为一种动态拓扑视角——研究差异结构在时间变换中保持哪些相对稳定的不变性质。**

这一映射可以被数学进一步检验和修正。

## 四、物理规律如何在 DM 中得到解释

### 4.1 规律不是命令，而是稳定重复

DM 对规律的基本理解已经在 `P00` 固定为：

> **规律 = 结构在重复变换中反复保留下来的稳定约束。**

因此，物理定律更稳地说不是：

> 宇宙外部有人给世界下命令

而是：

> **某些约束在特定尺度、特定条件下稳定重复，以至于可以被压缩为方程。**

### 4.2 为什么同一个世界会有不同层级的方程

因为不同层级抓取的是不同稳定结构：

- 力学方程抓取宏观运动中的稳定约束
- 热力学抓取大数统计中的稳定方向
- 电磁方程抓取带电耦合中的稳定传播关系
- 反应扩散方程抓取局部反应与扩散耦合的稳定形式

因此 DM 与科学的接口不是“一条公式打天下”，
而是：

> **同一世界在不同层级上，出现可被不同语言压缩的稳定约束。**

### 4.3 为什么“时间变慢”不推翻 DM 的时间定义

这里必须区分两层：

1. `本体论时间性`
2. `物理可测的钟表时间`

在 DM 中：

> **时间性 = 结构变换痕迹进入后续成立条件的继承序。**

而物理学中的时间变慢，
说的是：

> **不同物理过程中，时钟的运行速率会随速度、引力势和局部条件而变化。**

因此：

- DM 讨论的是“先后如何可能”
- 相对论讨论的是“具体物理时钟如何计量”

二者不是同一个问题。

更准确地说：

> **本体论时间性先给出先后序与不可逆痕迹；物理时间则是在具体结构中，对这些变化进行计量的可比速率。**

### 4.4 引力与时间的关系如何接上

接近大质量体，
或进入高速运动状态时，
具体物理过程的运行速率会改变，
因此钟表读数会出现差异。

这说明的是：

> **计时过程受结构条件约束。**

而不是：

> **世界最初必须先预设一个绝对钟表，才能有时间性。**

因此 DM 可以把相对论时间理解为：

> **度量时间在不同物理结构中的条件化展开。**

### 4.5 差异起点与物理学的对称性破缺

DM 的核心起点——差异先于实体——
在物理学中有一个当前最强的候选桥接：对称性破缺。

对称性破缺表明：

> **完全对称（完全同一）的状态在物理上是不稳定的，任何微扰都会导致系统自发选择一种不对称（差异化）状态。**

这与 DM 的起点判断深度一致：

```
DM：完全同一不能维持，差异是最小可分性
物理：完全对称不能维持，破缺是稳定态的前提
```

因此 DM 当前可以提出一个工作性假设：

> **差异不是从绝对虚无中凭空产生的，而是对称性破缺的本体论表达——完全同一的状态在结构上不稳定，差异化是稳定存在的前提条件。**

这一断言可以被物理学修正。
如果未来物理学发现存在完全对称且长期稳定的基态，
DM 的起点判断就需要收缩。

### 4.6 DM 的边界形成与普里高津耗散结构

DM 声称：

> **不是所有差异都会形成结构，但能够持续限制彼此可行状态的差异关系，必然已经具有结构性。**

这一判断与普里高津（Prigogine）的耗散结构理论形成当前较强的桥接对应：

普里高津证明：

> **远离平衡态的开放系统，在持续能量流过时，会自发形成有序结构（耗散结构）。**

两者的映射关系：

| DM 概念 | 耗散结构理论对应 |
|---------|----------------|
| 差异关系进入非任意约束 | 远离平衡态时微扰被放大为宏观有序 |
| 边界 = 内外关联强度的阈差 | 耗散结构的空间边界与自组织边界 |
| 暂稳态 = 可转换的持存方式 | 耗散结构本身是远离平衡的暂稳态 |
| 路径依赖与固化 | 耗散结构对初始分岔的敏感依赖 |

因此：

> **普里高津的耗散结构理论，是 DM 在物理层当前最重要的桥接对象之一。它为 DM 的“差异如何自发形成结构”提供了可继续检验的物理接口。**

### 4.7 DM 的不可逆性与热力学熵增

DM 的不可逆性定义是：

> **时间的不可逆性，本质上是历史痕迹进入后续条件网络后的高代价不可清除性。**

热力学的不可逆性定义是：

> **孤立系统的熵不减少。**

两者的关系不是等价，而是层级：

```
DM 层（更一般）：痕迹进入条件网络 → 清除代价高 → 不可逆
热力学层（更具体）：微观状态信息扩散到宏观自由度 → 恢复代价高 → 熵增
```

DM 的不可逆性概念比热力学熵增更一般，
因为它也适用于非热力学系统（如信息系统、生命系统、文明系统）。

但在物理层，DM 的不可逆性应当与熵增一致。

> **如果在物理层发现不可逆性不能被映射为任何形式的信息扩散与恢复代价，DM 的不可逆性定义就需要修正。**

## 五、化学、生命与演化，在 DM 中如何连续起来

### 5.1 化学不是生命，但已出现稳定组合

当结构之间形成较稳定的结合、断裂与重组规律时，
化学层就出现了。

它说明：

> **世界不仅有差异，还有可以重复维持、重复组合的结构关系。**

但化学本身还不是生命，
因为它还未必同时具备：

- 自我维持边界
- 代谢耦合
- 稳定复制
- 谱系继承

### 5.2 生命的最低门槛

生命更稳地可理解为：

> **能够维持边界、持续交换、复制延续并继承差异痕迹的结构。**

这时“历史”不再只是外部观察者的话语，
而进入结构自身：

> **过去的形成过程，已经写进后来者的成立条件。**

### 5.3 植物、动物、人类的连续线

DM 在生命层更稳的连续链是：

```
化学
→ 前生命
→ 植物性生命
→ 动物性生命
→ 高等动物
→ 人类
```

其中：

- 植物首先说明生命是边界、代谢、生长、复制
- 动物说明生命进一步把感觉与移动内化为快速择路能力
- 高等动物说明记忆、学习、预测开始成为核心能力
- 人类则把历史沉积外置为语言、教育、制度与文化

### 5.4 与自创生理论的对比

DM 的生命定义与 Maturana 和 Varela 的自创生理论（autopoiesis）
有高度重叠，但也有明确区别。

**共享部分：**

| 主题 | DM | 自创生理论 |
|------|-----|-----------|
| 核心关注 | 边界维持与自我生产 | 边界维持与自我生产 |
| 生命判据 | 能维持边界并持续交换 | 能自我生产自身组分的网络 |
| 认识论 | 认识是主体结构中的扰动 | 认识是结构耦合中的适应 |

**DM 的新增部分：**

自创生理论主要关注单个生命体当下的边界维持与自我生产，
但较少处理：

1. `谱系继承`
DM 通过 P05 的 DNA 自钥匙概念，
把跨代际的谱系约束纳入生命定义。
自创生理论不直接处理演化与谱系。

2. `感受的短路径压缩`
DM 在 P05 中说明为什么生命会需要主体级短路径感受。
自创生理论承认认知但不深入处理 qualia。

3. `文明层扩展`
DM 把生命的边界维持逻辑扩展到文明层（P09）。
自创生理论主要停留在生物体层。

因此：

> **DM 与自创生理论在生命的边界维持判据上共享核心直觉，但 DM 通过谱系继承、感受短路径和文明扩展，提供了更宽的跨层覆盖。**

## 六、意识问题在科学层如何安放

### 6.1 P04 与 P05 的分工必须保持

现行 DM 已明确区分：

> **P04 说明“可感之位”，P05 说明“所感之相”。**

因此在科学层中，
不能再把意识写成纯本体论一步到位的产物。

### 6.2 当前较稳的写法

关于感受与 qualia，
当前较稳的表达是：

> **qualia 不是纯结构前提的直接产物，而更稳地可理解为生命主体在 DNA 约束的神经-身体整体中形成的短路径暂稳态。**

这里至少包含四个近前提：

1. 生物边界
2. 代谢与持存压力
3. DNA 约束的可感空间
4. 神经-身体拓扑的快速匹配

### 6.3 为什么会需要短路径

因为外部刺激如果总要等到全部分散后果完全展开，
再由主体作出反应，
就太慢。

而在演化筛选中：

> **反应慢的个体更容易死掉。**

所以生命系统更可能留下：

> **把分散后果压缩成主体级整体状态的短路径机制。**

### 6.4 为什么还不能宣称“硬问题已被彻底关闭”

因为 DM 目前较稳地解释了：

- 为什么生命主体会需要感受
- 为什么感受不是任意飘出的
- 为什么感受受 DNA 与神经-身体拓扑约束
- 为什么感觉表现为流变中的暂稳态

但 DM 仍未完全形式化关闭：

> **为什么某一拓扑必然活成为这种而非另一种具体感质。**

因此，这里必须保留开放边界。

### 6.5 与当代意识科学理论的接口表

DM 的意识理论不应孤立发展，
而应明确标出与当代主要意识科学理论的对应关系：

| 理论 | 核心主张 | DM 中的对应概念 | 一致/冲突 |
|------|---------|----------------|----------|
| IIT（整合信息论） | 意识 = 系统的整合信息量 Φ | 内部可区分性 + 自指 + 边界 | 一致：Φ 可被理解为 DM 内部可区分状态空间的量化度量。但 DM 不认为单一标量足以刻画意识的全部结构 |
| GWT（全局工作空间论） | 意识 = 信息进入全局广播 | 主体级短路径 = 把分散后果压缩为全局可用状态 | 当前最强候选接口：GWT 的全局广播可视为 DM 短路径的一种神经实现候选 |
| 预测编码（Predictive Coding） | 感知 = 对预测误差的最小化 | 感觉强度 = 扰动对暂稳态的偏离幅度 × 持存优先级权重 | 一致：预测误差可被理解为 DM 中新扰动对当前暂稳态的偏离度 |
| 高阶理论（HOT） | 意识需要对自身状态的高阶表征 | 自指 = 结构把自身状态纳入后续处理 | 一致：DM 的自指概念与 HOT 的高阶表征在结构上同构 |

DM 与这些理论的关系是：

> **DM 提供跨层的结构框架，这些科学理论提供具体层级的可测量模型。两者之间是框架与模型的关系，而非竞争关系。**

### 6.6 可被神经科学检验的具体预测

DM 的意识理论应当冒险给出可被实验修正的具体预测：

**预测一：短路径速度优先**

DM 声称 qualia 是主体级短路径暂稳态。
这意味着：

> **意识经验的形成速度，应当系统性地快于对同一刺激的完整分析性处理。**

可检验方式：
在同一刺激条件下，比较主观报告的经验形成时间（如情绪反应、疼痛感受）
与完整分析性加工的完成时间（如语义分类、因果推理）。

如果实验稳定证明意识经验的形成并不系统性快于分析性处理，
DM 的短路径模型就需要修正。

**预测二：感觉强度 = 偏离幅度 × 优先级权重**

DM 声称感觉强度取决于：

> **持续扰动对主体当前暂稳态造成的偏离幅度，乘以该偏离在持存结构中的优先级权重。**

这可以被映射为可测量的神经指标：

```
偏离幅度 ≈ 预测误差信号（如 mismatch negativity、prediction error）
优先级权重 ≈ 与持存相关的注意权重（如威胁检测、奖励信号）
感觉强度 ≈ 主观报告的感受强度评分
```

> **DM 预测：预测误差信号 × 持存相关注意权重，应当能预测主观报告的感觉强度。**

如果这一乘积模型被实验稳定否定，
DM 就需要修正 P05 中的感觉强度公式。

**预测三：适应效应的暂稳态解释**

DM 声称感觉会变淡是因为主体形成新的暂稳态。
这意味着：

> **感觉适应不只是感受器疲劳，而是中枢层面的暂稳态重组。**

可检验方式：
在感受器未疲劳的条件下（如通过交叉适应范式），
如果仍然出现感觉强度下降，
则支持 DM 的中枢暂稳态解释。

## 七、科学层最需要守住的诚实边界

当前至少有五个边界不能装作已经完成：

1. `从本体论到具体方程的严格映射`
   现阶段多是接口说明，不是单向演绎闭合

2. `从关系拓扑到可测量量纲的统一换算`
   目前仍缺严格数值桥梁

3. `从经典结构到量子形式化的完整桥接`
   不能把形式相似直接写成量子理论已被推出

4. `从内部性到具体感质的最后闭合`
   当前已明显压缩，但仍未终结

5. `失败条件与预测接口`
   如果没有更清楚的失败标准，理论就容易滑向万能解释器

这些边界不是羞耻，
而是理论成熟所必需的诚实。

### 7.1 具体失败条件清单

如果没有失败条件，
DM 就有滑向万能解释器的风险。

以下是 DM 在科学层当前必须承受的三组失败条件：

**失败条件 A：如果完全对称状态被证明可以长期稳定存在**

DM 的起点判断是“完全同一不能维持”。
这与物理学中“完全对称是不稳定的”一致。

> **如果物理学发现存在完全对称且长期稳定（不发生任何自发破缺）的基态，DM 的起点判断就需要从“差异必然”收缩为“差异在特定条件下必然”。**

**失败条件 B：如果意识经验被证明不需要短路径压缩**

DM 声称 qualia 是短路径暂稳态。

> **如果神经科学稳定证明：意识经验的形成不需要任何形式的信息压缩或快速通道（即完整的分散处理就足以产生意识），DM 的 qualia 模型就需要根本修正。**

**失败条件 C：如果涌现被证明可以完全还原**

DM 声称涌现层具有不可被下层描述完全替代的地位。

> **如果在所有已知涌现案例中，上层性质都被证明可以从下层性质严格推出且无信息损失，DM 的涌现概念就需要从“本体论涌现”收缩为“认识论涌现”（即只是人类计算能力的限制）。**

这三组失败条件不是理论弱点，
而是理论可信度的保证：

> **一个不能被任何事实修正的理论，什么也没有说。**

### 7.2 元框架命题与可修正命题的区分

DM 的命题并非全部同等可被科学修正。
必须区分两类：

**元框架命题（科学实验不能直接推翻）：**

```
差异先于实体
关系先于对象
存在者是暂时的结构稳定项
```

这些是 DM 的概念框架选择，
与实体本体论、主体本体论等框架处于同一层级的竞争关系。
它们的评价标准是一致性、解释范围和概念经济性，
而非单一实验的证伪。

**可修正命题（应当接受科学修正）：**

```
qualia 是短路径暂稳态（可被神经科学修正）
不可逆性来自痕迹进入条件网络的高清除代价（可被物理学修正）
涌现层不可被下层完全替代（可被还原论程序修正）
生命需要谱系继承（可被非 DNA 生命反例修正）
```

因此：

> **DM 的科学对话层必须持续维护这一区分：哪些命题属于框架选择，哪些命题接受科学修正。**

## 八、当前更适合的科学层工作方式

DM 在科学对话层中，
更适合采用以下工作方式：

1. 先给出本体论最小判断
2. 再说明它与哪类科学结构一致
3. 明确哪些地方只是近似投影
4. 明确哪些地方还没有数学闭合
5. 优先寻找可失败、可比较、可收缩的接口

压缩地说：

> **DM 不应急于宣布“我已经推出了全部科学”，而应持续把自己的本体判断压缩成更清楚的科学接口。**

## 十、本文的最短结论

`DM 与科学的关系，首先不是单向推导，而是一致性接口。`
`数学是对稳定结构关系的高压缩语言；渐近线推断在形式系统中有哥德尔极限，但可通过跨系统重构继续逼近。`
`物理规律不是命令，而是结构在特定条件下反复保留下来的稳定约束。`
`DM 的差异起点与物理学的对称性破缺、DM 的边界形成与普里高津耗散结构、DM 的不可逆性与热力学熵增，构成三个优先物理桥接。`
`DM 的生命定义与自创生理论共享边界维持判据，但通过谱系继承与感受短路径提供更宽覆盖。`
`DM 的意识理论应当给出可被神经科学检验的具体预测，包括短路径速度优先和感觉强度的乘积模型。`
`DM 必须区分元框架命题与可修正命题，并持续维护具体的失败条件清单。`

## 十一、结语

本文的核心立场是：

> **差异一元论不是一套已经完成全部科学推导的总公式，而是一套试图把数学、物理、生命与意识放回同一存在论地基上加以重新理解的跨层框架。**

它现在最需要的，
不是更多夸张句，
而是：

> **更清楚的接口、更严格的映射、和更诚实的开放边界。**
